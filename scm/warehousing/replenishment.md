<span data-ttu-id="f1c1c-104">Šioje temoje aprašomos papildymo strategijos, skirtos sandėliams, kuriuose naudojamos modulio Sandėlio valdymas funkcijos.</span><span class="sxs-lookup"><span data-stu-id="f1c1c-104">This topic describes the replenishment strategies that are available for warehouses which use the functionality that is available in Warehouse management.</span></span> <span data-ttu-id="f1c1c-105">Ši informacija netaikoma sandėliavimo sprendimui, kuris prieinamas modulyje Atsargų valdymas.</span><span class="sxs-lookup"><span data-stu-id="f1c1c-105">The information doesn't apply to the warehousing solution that is available in Inventory management.</span></span> <span data-ttu-id="f1c1c-106">Galima naudoti tolesnes papildymo strategijas.</span><span class="sxs-lookup"><span data-stu-id="f1c1c-106">The following replenishment strategies are available:</span></span>

-   <span data-ttu-id="f1c1c-107">**Papildymas pagal bangos poreikį** – naudojantis šia strategija sukuriamas išsiunčiamų užsakymų arba krovinių papildymo darbas, jei bangai kuriant darbą nėra atsargų.</span><span class="sxs-lookup"><span data-stu-id="f1c1c-107">**Wave demand replenishment** – This strategy creates replenishment work for outbound orders or loads if inventory isn't available when the wave creates work.</span></span> <span data-ttu-id="f1c1c-108">Pavyzdžiui, jei apdorojant bangą nėra pardavimo užsakymui reikalingo kiekio, gali būti sukurtas papildymo darbas.</span><span class="sxs-lookup"><span data-stu-id="f1c1c-108">For example, replenishment work can be created if the quantity that is required for a sales order isn't available when a wave is processed.</span></span>
