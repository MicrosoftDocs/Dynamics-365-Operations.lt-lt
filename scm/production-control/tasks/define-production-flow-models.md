## <a name="define-a-production-flow-model"></a><span data-ttu-id="2c722-107">Apibrėžkite gamybos eigos modelį.</span><span class="sxs-lookup"><span data-stu-id="2c722-107">Define a production flow model.</span></span>

5. <span data-ttu-id="2c722-117">Įveskite EPE ciklo dienų skaičių.</span><span class="sxs-lookup"><span data-stu-id="2c722-117">Enter the number of days in the EPE Cycle.</span></span>
    * <span data-ttu-id="2c722-118">Intervalu aprašomas laikotarpis, kai kiekviena gamybos eigos ar darbo elemento parengta dalis bus gaminama bent kartą.</span><span class="sxs-lookup"><span data-stu-id="2c722-118">The interval describes the period when every part produced by a production flow or work cell will be produced at least once.</span></span> <span data-ttu-id="2c722-119">Kuo trumpesnis EPE ciklas, tuo lankstesnis gamybos procesas.</span><span class="sxs-lookup"><span data-stu-id="2c722-119">The shorter the EPE cycle, the more flexible the production process is.</span></span> <span data-ttu-id="2c722-120">Jei EPE ciklas = 0, visą poreikį galima patenkinti per tą pačią dieną.</span><span class="sxs-lookup"><span data-stu-id="2c722-120">If EPE Cycle = 0, all demand can be produced in the same day.</span></span> <span data-ttu-id="2c722-121">EPE ciklas naudojamas planuoti pažangiosios gamybos proceso užduotis.</span><span class="sxs-lookup"><span data-stu-id="2c722-121">The EPE Cycle is used to schedule lean process jobs.</span></span> <span data-ttu-id="2c722-122">Planuojant užduotį su darbo elementu, kurio EPE ciklas = 5, planavimo procesas ieškos darbo langelio pajėgumo, kurio Termino data – EPE ciklas (5 dienos iki termino) siekiant užtikrinti, kad produktą galima pagaminti per šį ciklą.</span><span class="sxs-lookup"><span data-stu-id="2c722-122">When scheduling a job on a work cell with EPE Cycle = 5, the scheduling process will look for capacity on the work cell at Due date - EPE Cycle (5 days ahead of the due date) to ensure that the product can be produced in that cycle.</span></span> <span data-ttu-id="2c722-123">Produkto atsargų gamybos laikas paprastai yra lygus arba didesnis negu susijusio gamybos proceso EPE ciklas.</span><span class="sxs-lookup"><span data-stu-id="2c722-123">The inventory lead time of a product is usually equal to or greater than the EPE Cycle of the related production process.</span></span>

7. <span data-ttu-id="2c722-127">Lauke Planavimo laiko riba įveskite skaičių.</span><span class="sxs-lookup"><span data-stu-id="2c722-127">In the Planning time fence field, enter a number.</span></span>
    * <span data-ttu-id="2c722-128">Planavimo laiko riba apibūdina skaičių dienų, kai pajėgumo rezervavimus susijusiems darbo elementams galima atlikti.</span><span class="sxs-lookup"><span data-stu-id="2c722-128">The planning time fence describes the number of days where capacity reservations can be made for the related work cells.</span></span> <span data-ttu-id="2c722-129">Planavimo laiko riboje įveskite dienų skaičių.</span><span class="sxs-lookup"><span data-stu-id="2c722-129">In the Planning time fence, enter the number of days.</span></span>   <span data-ttu-id="2c722-130">„Kanban“ proceso užduotys, nepatenkančios į šį laikotarpį, nebuvo suplanuotos automatinio planavimo būdu.</span><span class="sxs-lookup"><span data-stu-id="2c722-130">Kanban process jobs that fall outside of this period are not planned with automatic planning.</span></span> <span data-ttu-id="2c722-131">Planavimo laiko ribos paprastai yra du kartus didesnės už vidutinį atsargų gavimo laiką produktams, pagamintiems per produktų eigą arba darbo elementą.</span><span class="sxs-lookup"><span data-stu-id="2c722-131">The planning time fence is typically two times the average inventory lead time of the products produced in a production flow or work cell.</span></span> <span data-ttu-id="2c722-132">EPE ciklas negali trukti daugiau nei pusę planavimo laiko ribos.</span><span class="sxs-lookup"><span data-stu-id="2c722-132">The EPE Cycle should not be more than half the planning time fence.</span></span>     
8. <span data-ttu-id="2c722-133">Lauke Pajėgumo trūkumo reakcija pasirinkite pasirinktį.</span><span class="sxs-lookup"><span data-stu-id="2c722-133">In the Capacity shortage reaction field, select an option.</span></span>
    * <span data-ttu-id="2c722-134">Galimos parinktys: atidėti – atidedama visa planavimo įvykio paklausa kitą pasiekiamą gamybos datą su galimu našumu.</span><span class="sxs-lookup"><span data-stu-id="2c722-134">The options include:   Postpone - Postponing the full demand of the scheduling event on the next available production day, with available throughput.</span></span> <span data-ttu-id="2c722-135">Atšaukti – baigti automatinį planavimo įvykio planavimą ir palikti susijusias užduotis.</span><span class="sxs-lookup"><span data-stu-id="2c722-135">Cancel - End the automatic planning for the scheduling event and leave the related jobs unplanned.</span></span>   <span data-ttu-id="2c722-136">Įtraukti į pageidaujamą datą – planuoti pageidaujamas užduotis užklausų laikotarpiui.</span><span class="sxs-lookup"><span data-stu-id="2c722-136">Add to requested day - Plan the requested jobs for the requested period.</span></span> <span data-ttu-id="2c722-137">Tokiu būdu perkraunamas šios datos langelis ir iš planuotojo reikalaujama peržiūrėti ir rankiniu būdu atlikti veiksmus.</span><span class="sxs-lookup"><span data-stu-id="2c722-137">This overloads the cell for this day and requires the planner to review and to a manual interaction .</span></span>   <span data-ttu-id="2c722-138">Paskirstyti turimiems laikotarpiams – skirtingos planavimo įvykio užduotys paskirstomos visoms pasiekiamoms gamybos datoms pradedant nuo pirmos pasiekiamos datos.</span><span class="sxs-lookup"><span data-stu-id="2c722-138">Distribute to available periods -
