# <a name="end-a-production-order"></a>Baigti gamybos užsakymą

[!include[task guide banner](../../includes/task-guide-banner.md)]

Šioje procedūroje nurodoma, kaip pabaigti gamybos užsakymą. Kuriant šią procedūrą naudojama demonstracinių duomenų įmonė yra USMF. Tai yra paskutinė procedūra iš septynių, kurioje paaiškinamas gamybos užsakymo ciklas.


## <a name="end-a-production-order"></a>Baigti gamybos užsakymą
1. Pasirinkite Gamybos kontrolė > Gamybos užsakymai > Visi gamybos užsakymai.
    * Pasirinkite gamybos užsakymą, kurio būsena yra Paskelbtas baigtu.  
2. Veiksmų srityje spustelėkite Gamybos užsakymas.
3. Spustelėkite Baigti.
    * Šiame puslapyje galite patvirtinti, kad norite baigti gamybos užsakymą.  
4. Spustelėkite skirtuką Bendra.
5. Lauke Data įveskite datą.
6. Lauke Nurašymo metodas pasirinkite „Paskirstymas“.
    * Pasirinkus metodą Paskirstymas, išlaidos už nurašytas medžiagas pridedamos prie pagamintų prekių.  
7. Spustelėkite GERAI.

## <a name="validate-calculation-results"></a>Skaičiavimo rezultatų tikrinimas
1. Veiksmų srityje spustelėkite Valdyti išlaidas.
2. Spustelėkite Peržiūrėti savikainos palyginimą.
    * Pabaigę gamybos užsakymą galite palyginti įvertintą savikainą su realizuota savikaina ir peržiūrėti gamybos nuokrypius.
