# <a name="define-discrete-manufacturing-resource-group"></a>Apibrėžti atskiros gamybos išteklių grupę

[!include[task guide banner](../../includes/task-guide-banner.md)]

Išteklių grupė yra operacijų išteklių rinkinys, kurie paprastai atitinka fizinį darbo elementų išdėstymą, ant gamybos cecho grindų apibrėžtą geltonomis linijomis. Ši procedūra parodo, kaip nustatyti išteklių grupę, skirtą naudoti atskiroje gamyboje. Šią procedūrą galite atlikti naudodami demonstracinių duomenų įmonę USMF arba savo duomenis.

1. Eikite į Išteklių grupės.
2. Spustelėkite Naujas.
3. Lauke Išteklių grupės surinkite reikšmę.
4. Lauke Aprašas įveskite reikšmę.
5. Lauke Teritorija įveskite arba pasirinkite reikšmę.
6. Lauke Gamybos vienetas įveskite arba pasirinkite reikšmę.

## <a name="define-default-operational-parameters"></a>Nustatyti numatytuosius veiklos parametrus
1. Išplėskite skyrių Operacija.
2. Lauke Atliekų procentinė dalis įveskite skaičių.
3. Lauke Nustatymų kategorija įveskite arba pasirinkite reikšmę.
4. Lauke Apdorojimo laiko kategorija įveskite arba pasirinkite reikšmę.
5. Lauke Operacijų planavimo procentas įveskite skaičių.

## <a name="define-operating-hours"></a>Nustatyti veikimo valandas
1. Išplėskite skyrių Kalendoriai.
2. Spustelėkite Pridėti.
3. Lauke Kalendorius įveskite arba pasirinkite reikšmę.

## <a name="add-operations-resources"></a>Pridėti operacijų išteklių
1. Išplėskite skyrių Ištekliai.
2. Spustelėkite Pridėti.
3. Lauke Išteklius įveskite arba pasirinkite reikšmę.
4. Spustelėkite Pridėti.
5. Lauke Išteklius įveskite arba pasirinkite reikšmę.
6. Sąraše raskite ir pasirinkite norimą įrašą.
7. Sąraše spustelėkite saitą pasirinktoje eilutėje.
