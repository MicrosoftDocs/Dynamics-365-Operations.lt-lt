# <a name="fixed-asset-repair-statement-for-lithuania"></a>Ilgalaikio turto remontas Lietuvos biud etui

Šioje temoje paaiškinama, kaip pateikti remonto informacijos ilgalaikio turto ir ilgalaikio turto remonto ataskaitą. Remonto informacija turi būti saugoma visam ilgalaikiui turtui. 

<a name="enter-details-for-a-fixed-asset-repair-statement"></a>Įveskite duomenis ilgalaikio turto remonto pareiškimas
------------------------------------------------

Galite įvesti remonto informacijos ilgalaikio turto ir **ilgalaikio turto remontas** puslapis. Pateikti šią informaciją apie remontą.

|                             |                                                                                                                                       |
|-----------------------------|---------------------------------------------------------------------------------------------------------------------------------------|
| **Field**                   | **Value**                                                                                                                             |
| **Date of repair**          | Data, kai ilgalaikis turtas buvo suremontuotas.                                                                                           |
| **Description**             | Aprašymas remonto.                                                                                                            |
| **Amount spent for repair** | Remonto kaina. **Pastaba:** apskaičiuoti bendrą sumą, spustelėkite į **sumos** mygtuką, ir tada nurodykite skaičiavimo laikotarpis. |
| **Document number**         | Numeris, priskirtas dokumento, sugeneravusio remontui.                                                            |

 

## <a name="generate-the-fixed-asset-repair-report"></a>Ilgalaikio turto remontas ataskaitai generuoti
Ataskaitai generuoti, eikite į **ilgalaikis turtas****&gt; Inquires ir****ataskaitos**&gt;**pranešimai apie**&gt;**ilgalaikio turto remonto**, ir tada nurodykite šias reikšmes.

|                               |                                                                                                                                                                                                                                                                                                                  |
|-------------------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **Field**                     | **Value**                                                                                                                                                                                                                                                                                                        |
| **Book**                      | Pasirinkite knygos, kurių ilgalaikio turto ataskaitai generuoti.                                                                                                                                                                                                                                                    |
| **Nuo dienos, iki šiol**        | Įveskite ataskaitinio laikotarpio pradžios ir pabaigos datas.                                                                                                                                                                                                                                                          |
| **Show report transactions**  | Pasirinkti šį lauką norint įtraukti remonto operacijas ataskaitoje.                                                                                                                                                                                                                                                  |
| **Document number**           | Nurodykite skaičių, kuris yra priskirtas dokumento, sugeneravusio remontui.                                                                                                                                                                                                                           |
| **Print from repair percent** | Įvesti procentais ilgalaikio turto įsigijimo verte, kuris gali būti panaudotas remontas prieš ilgalaikis turtas turi būti pateikta ataskaitoje. Ataskaitoje nurodomas tik tas ilgalaikis turtas, kuriam remonto išlaidos įsigijimo vertės santykis yra didesnis kaip procentas, pagal kurį įvedate šiame lauke. |
