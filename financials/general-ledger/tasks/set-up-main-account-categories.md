# <a name="set-up-main-account-categories"></a>Nustatyti pagrindinės sąskaitos kategorijas

[!include[task guide banner](../../includes/task-guide-banner.md)]

Pagrindinės sąskaitos kategorijos naudojamos finansinių ataskaitų ir „Power BI“ numatytosioms ataskaitoms. Pagrindines sąskaitos kategorijas, sukurtas pagal numatytuosius nustatymus, galima pervardyti, bet jų negalima panaikinti. Ataskaitų ir analizės tikslams galima sukurti ir naudoti papildomas sąskaitos kategorijas. Šioje užduotyje naudojama demonstracinė įmonė USMF.


## <a name="create-a-main-account-category"></a>Pagrindinės sąskaitos kategorijos kūrimas
1. Eikite į dalį Didžioji knyga > Sąskaitų planas > Sąskaitos > Pagrindinės sąskaitos kategorijos.
2. Spustelėkite Naujas.
3. Lauke Pagrindinės sąskaitos kategorija įveskite unikalų pavadinimą.
4. Lauke Aprašas įveskite pagrindinės sąskaitos kategorijos aprašą.
5. Lauke Pagrindinės sąskaitos tipas pasirinkite pagrindinės sąskaitos tipą, kuris bus susietas su kategorija.

## <a name="link-main-accounts-to-account-category"></a>Pagrindinių sąskaitų susiejimas su sąskaitos kategorija
1. Spustelėkite Susieti pagrindines sąskaitas.
2. Sąraše pasirinkite pagrindines sąskaitas, kurios bus priskirtos pagrindinės sąskaitos kategorijai.
    * Pagrindines sąskaitas priskyrus pagrindinės sąskaitos kategorijai ir naudojant tą kategoriją finansinėms ataskaitoms ir analizei, bus sujungti sąskaitų balansai.  
3. Pažymėkite arba išvalykite Susieta parinktis pasirinkti pagrindines sąskaitas.
4. Spustelėkite Gerai.
5. Spustelėkite Taip.
