# <a name="create-a-journal-entry-using-a-template"></a><span data-ttu-id="75d5b-103">Kurti žurnalo įrašą naudojant šabloną</span><span class="sxs-lookup"><span data-stu-id="75d5b-103">Create a journal entry using a template</span></span>

[!include[task guide banner](../../includes/task-guide-banner.md)]

1. <span data-ttu-id="75d5b-106">Didžioji knyga > Žurnalų įrašai > Bendrieji žurnalai.</span><span class="sxs-lookup"><span data-stu-id="75d5b-106">General ledger > Journal entries > General journals.</span></span> <span data-ttu-id="75d5b-107">Spustelėkite Naujas.</span><span class="sxs-lookup"><span data-stu-id="75d5b-107">Click New.</span></span>

3. <span data-ttu-id="75d5b-110">Sąraše raskite ir pasirinkite norimą įrašą.</span><span class="sxs-lookup"><span data-stu-id="75d5b-110">In the list, find and select the desired record.</span></span>
4. <span data-ttu-id="75d5b-111">Sąraše spustelėkite saitą pasirinktoje eilutėje.</span><span class="sxs-lookup"><span data-stu-id="75d5b-111">In the list, click the link in the selected row.</span></span>
5. <span data-ttu-id="75d5b-112">Spustelėkite Eilutės.</span><span class="sxs-lookup"><span data-stu-id="75d5b-112">Click Lines.</span></span>
6. <span data-ttu-id="75d5b-113">Įveskite sąskaitos tipo sąskaitą.</span><span class="sxs-lookup"><span data-stu-id="75d5b-113">Enter an account for the Account type.</span></span>

8. <span data-ttu-id="75d5b-115">Įveskite sumą lauke Debetas.</span><span class="sxs-lookup"><span data-stu-id="75d5b-115">Enter an amount in the Debit field.</span></span>
9. <span data-ttu-id="75d5b-116">Spustelėkite Naujas.</span><span class="sxs-lookup"><span data-stu-id="75d5b-116">Click New.</span></span>
10. <span data-ttu-id="75d5b-117">Įveskite kitą sąskaitos tipo sąskaitą.</span><span class="sxs-lookup"><span data-stu-id="75d5b-117">Enter a different account for the Account type.</span></span>
11. <span data-ttu-id="75d5b-118">Lauke Aprašas įveskite reikšmę.</span><span class="sxs-lookup"><span data-stu-id="75d5b-118">In the Description field, type a value.</span></span>
12. <span data-ttu-id="75d5b-119">Įveskite sumą lauke Debetas.</span><span class="sxs-lookup"><span data-stu-id="75d5b-119">Enter an amount in the Debit field.</span></span>
13. <span data-ttu-id="75d5b-120">Spustelėkite Naujas.</span><span class="sxs-lookup"><span data-stu-id="75d5b-120">Click New.</span></span>
14. <span data-ttu-id="75d5b-121">Lauke Sąskaita nustatykite norimas reikšmes.</span><span class="sxs-lookup"><span data-stu-id="75d5b-121">In the Account field, specify the desired values.</span></span>
15. <span data-ttu-id="75d5b-122">Lauke Aprašas surinkite reikšmę.</span><span class="sxs-lookup"><span data-stu-id="75d5b-122">In the Description field, type a value.</span></span>
16. <span data-ttu-id="75d5b-123">Įveskite sumą lauke Kreditas, kad subalansuotumėte kvitą.</span><span class="sxs-lookup"><span data-stu-id="75d5b-123">Enter an amount in the Credit field to balance the voucher.</span></span>
17. <span data-ttu-id="75d5b-124">Spustelėkite Registruoti.</span><span class="sxs-lookup"><span data-stu-id="75d5b-124">Click Post.</span></span>
18. <span data-ttu-id="75d5b-125">Spustelėkite Funkcijos.</span><span class="sxs-lookup"><span data-stu-id="75d5b-125">Click Functions.</span></span>
19. <span data-ttu-id="75d5b-126">Spustelėkite Įrašyti kvito šabloną.</span><span class="sxs-lookup"><span data-stu-id="75d5b-126">Click Save voucher template.</span></span>
20. <span data-ttu-id="75d5b-127">Ši procedūra daro prielaidą, kad šablono tipas yra Procentas.</span><span class="sxs-lookup"><span data-stu-id="75d5b-127">This procedure assumes a Percent Template type.</span></span> <span data-ttu-id="75d5b-128">Spustelėkite GERAI.</span><span class="sxs-lookup"><span data-stu-id="75d5b-128">Click OK.</span></span>
    * <span data-ttu-id="75d5b-129">• Procentas: kvito sumos konvertuojamos į procentų koeficientus, o tai, pasirinkus kvito šabloną, leidžia taikyti bet kokią sumą.</span><span class="sxs-lookup"><span data-stu-id="75d5b-129">• Percent: The amounts in the voucher are converted into percentage factors, which allows any amount to be applied when the Voucher template is selected.</span></span>  <span data-ttu-id="75d5b-130">• Suma: bus saugomos ir taikomos faktinės sumos.</span><span class="sxs-lookup"><span data-stu-id="75d5b-130">• Amount: The actual amounts will be stored and applied.</span></span>  
21. <span data-ttu-id="75d5b-131">Spustelėkite Pagrindiniai žurnalai.</span><span class="sxs-lookup"><span data-stu-id="75d5b-131">Click General journals.</span></span>
22. <span data-ttu-id="75d5b-132">Spustelėkite Naujas.</span><span class="sxs-lookup"><span data-stu-id="75d5b-132">Click New.</span></span>
23. <span data-ttu-id="75d5b-133">Lauke Pavadinimas spustelėkite išplečiamąjį mygtuką, kad atidarytumėte peržvalgą.</span><span class="sxs-lookup"><span data-stu-id="75d5b-133">In the Name field, click the drop-down button to open the lookup.</span></span>
24. <span data-ttu-id="75d5b-134">Sąraše spustelėkite saitą pasirinktoje eilutėje.</span><span class="sxs-lookup"><span data-stu-id="75d5b-134">In the list, click the link in the selected row.</span></span>
25. <span data-ttu-id="75d5b-135">Spustelėkite Eilutės.</span><span class="sxs-lookup"><span data-stu-id="75d5b-135">Click Lines.</span></span>
26. <span data-ttu-id="75d5b-136">Spustelėkite Funkcijos.</span><span class="sxs-lookup"><span data-stu-id="75d5b-136">Click Functions.</span></span>
27. <span data-ttu-id="75d5b-137">Spustelėkite Pasirinkti kvito šabloną.</span><span class="sxs-lookup"><span data-stu-id="75d5b-137">Click Select voucher template.</span></span>

    * <span data-ttu-id="75d5b-140">Jei yra kitų šablonų, gali reikėti spustelėti Ankstesnis veiksmas ir tada pasirinkti tinkamą šabloną.</span><span class="sxs-lookup"><span data-stu-id="75d5b-140">You may need to click Previous step and then select the correct template if other templates exist.</span></span>

    * <span data-ttu-id="75d5b-142">Sumos laukas rodomas tik jei kvito šablono tipas yra Procentas.</span><span class="sxs-lookup"><span data-stu-id="75d5b-142">The amount field is only displayed if the voucher template is of type Percent.</span></span>  
30. <span data-ttu-id="75d5b-143">Spustelėkite GERAI.</span><span class="sxs-lookup"><span data-stu-id="75d5b-143">Click OK.</span></span>
