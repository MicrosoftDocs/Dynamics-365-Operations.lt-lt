# <a name="create-financial-dimensions-for-retail-channels-and-configure-dimension-values-on-stores"></a><span data-ttu-id="02c63-103"> Kurti mažmeninės prekybos kanalų finansines dimensijas ir konfigūruoti dimensijų reikšmes parduotuvėse</span><span class="sxs-lookup"><span data-stu-id="02c63-103">Create financial dimensions for Retail channels and configure dimension values on stores</span></span>

[!include[task guide banner](../includes/task-guide-banner.md)]

<span data-ttu-id="02c63-104">Ši procedūra padeda kurti mažmeninės prekybos kanalo finansinę dimensiją su dimensijų reikšmėmis ir nurodo mažmeninės prekybos parduotuvių finansinių dimensijų reikšmių konfigūravimo veiksmus.</span><span class="sxs-lookup"><span data-stu-id="02c63-104">This procedure walks through creating a retail channel financial dimension with dimension values and steps to configure financial dimension values on retail stores.</span></span> <span data-ttu-id="02c63-105">Ši tema neapima kitų susijusių veiksmų, pavyzdžiui, dimensijų rinkinių ir sąskaitų struktūrų kūrimo.</span><span class="sxs-lookup"><span data-stu-id="02c63-105">The topic does not include other related steps, such as creating dimension sets and account structures.</span></span> <span data-ttu-id="02c63-106">Šioje procedūroje naudojama demonstracinių duomenų įmonė USRT.</span><span class="sxs-lookup"><span data-stu-id="02c63-106">This procedure uses the USRT company in demo data.</span></span>

1. <span data-ttu-id="02c63-107">Eikite į Didžioji knyga > Sąskaitų planas > Dimensijos > Finansinės dimensijos.</span><span class="sxs-lookup"><span data-stu-id="02c63-107">Go to General ledger > Chart of accounts > Dimensions > Financial dimensions.</span></span>
2. <span data-ttu-id="02c63-108">Spustelėkite Naujas.</span><span class="sxs-lookup"><span data-stu-id="02c63-108">Click New.</span></span>
3. <span data-ttu-id="02c63-109">Lauke Naudoti vertes iš pasirinkite „Mažmeninės prekybos kanalai“.</span><span class="sxs-lookup"><span data-stu-id="02c63-109">In the Use values from field, select 'Retail channels'.</span></span>
4. <span data-ttu-id="02c63-110">Lauke Dimensijos pavadinimas įveskite reikšmę.</span><span class="sxs-lookup"><span data-stu-id="02c63-110">In the Dimension name field, type a value.</span></span>
5. <span data-ttu-id="02c63-111">Spustelėkite Aktyvinti.</span><span class="sxs-lookup"><span data-stu-id="02c63-111">Click Activate.</span></span>
6. <span data-ttu-id="02c63-112">Spustelėkite Uždaryti.</span><span class="sxs-lookup"><span data-stu-id="02c63-112">Click Close.</span></span>
7. <span data-ttu-id="02c63-113">Spustelėkite Aktyvinti.</span><span class="sxs-lookup"><span data-stu-id="02c63-113">Click Activate.</span></span>
8. <span data-ttu-id="02c63-114">Spustelėkite Dimensijos reikšmės.</span><span class="sxs-lookup"><span data-stu-id="02c63-114">Click Dimension values.</span></span>
9. <span data-ttu-id="02c63-115">Uždarykite puslapį.</span><span class="sxs-lookup"><span data-stu-id="02c63-115">Close the page.</span></span>
10. <span data-ttu-id="02c63-116">Spustelėkite Įrašyti.</span><span class="sxs-lookup"><span data-stu-id="02c63-116">Click Save.</span></span>
11. <span data-ttu-id="02c63-117">Uždarykite puslapį.</span><span class="sxs-lookup"><span data-stu-id="02c63-117">Close the page.</span></span>
12. <span data-ttu-id="02c63-118">Eikite į Mažmeninė prekyba ir prekyba > Kanalai > Mažmeninės prekybos parduotuvės > Visos mažmeninės prekybos parduotuvės.</span><span class="sxs-lookup"><span data-stu-id="02c63-118">Go to Retail and commerce > Channels > Retail stores > All retail stores.</span></span>
13. <span data-ttu-id="02c63-119">Sąraše spustelėkite saitą pasirinktoje eilutėje.</span><span class="sxs-lookup"><span data-stu-id="02c63-119">In the list, click the link in the selected row.</span></span>
14. <span data-ttu-id="02c63-120">Perjunkite skyriaus „Finansinės dimensijos“ išplėtimą.</span><span class="sxs-lookup"><span data-stu-id="02c63-120">Toggle the expansion of the Financial dimensions section.</span></span>
15. <span data-ttu-id="02c63-121">Spustelėkite Redaguoti.</span><span class="sxs-lookup"><span data-stu-id="02c63-121">Click Edit.</span></span>
16. <span data-ttu-id="02c63-122">Lauke Mažmeninės prekybos kanalas spustelėkite išplečiamąjį mygtuką, kad atidarytumėte peržvalgą.</span><span class="sxs-lookup"><span data-stu-id="02c63-122">In the Retailchannel field, click the drop-down button to open the lookup.</span></span>
17. <span data-ttu-id="02c63-123">Sąraše raskite ir pasirinkite naujinamos parduotuvės dimensijos reikšmę.</span><span class="sxs-lookup"><span data-stu-id="02c63-123">In the list, find and select the dimension value for the store being updated.</span></span>
18. <span data-ttu-id="02c63-124">Sąraše spustelėkite saitą pasirinktoje eilutėje.</span><span class="sxs-lookup"><span data-stu-id="02c63-124">In the list, click the link in the selected row.</span></span>
19. <span data-ttu-id="02c63-125">Lauke „CostCenter“ spustelėkite išplečiamąjį mygtuką, kad atidarytumėte peržvalgą.</span><span class="sxs-lookup"><span data-stu-id="02c63-125">In the CostCenter field, click the drop-down button to open the lookup.</span></span>
20. <span data-ttu-id="02c63-126">Sąraše raskite ir pasirinkite norimą įrašą.</span><span class="sxs-lookup"><span data-stu-id="02c63-126">In the list, find and select the desired record.</span></span>
21. <span data-ttu-id="02c63-127">Sąraše spustelėkite saitą pasirinktoje eilutėje.</span><span class="sxs-lookup"><span data-stu-id="02c63-127">In the list, click the link in the selected row.</span></span>
22. <span data-ttu-id="02c63-128">Lauke Skyrius spustelėkite išplečiamąjį mygtuką, kad atidarytumėte peržvalgą.</span><span class="sxs-lookup"><span data-stu-id="02c63-128">In the Department field, click the drop-down button to open the lookup.</span></span>
23. <span data-ttu-id="02c63-129">Sąraše raskite ir pasirinkite norimą įrašą.</span><span class="sxs-lookup"><span data-stu-id="02c63-129">In the list, find and select the desired record.</span></span>
24. <span data-ttu-id="02c63-130">Sąraše spustelėkite saitą pasirinktoje eilutėje.</span><span class="sxs-lookup"><span data-stu-id="02c63-130">In the list, click the link in the selected row.</span></span>
25. <span data-ttu-id="02c63-131">Spustelėkite Įrašyti.</span><span class="sxs-lookup"><span data-stu-id="02c63-131">Click Save.</span></span>
