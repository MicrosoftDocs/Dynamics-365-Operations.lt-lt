# <a name="create-new-users"></a>Naujų vartotojų kūrimas

[!include [task guide banner](../../includes/task-guide-banner.md)]

Vartotojai yra vidiniai jūsų organizacijos darbuotojai arba išoriniai klientai bei tiekėjai, kuriems reikalinga prieiga prie sistemos, kad galėtų atlikti savo užduotis. Sistemos administratoriai gali atlikitti šią procedūrą, kad įtrauktų vartotojus į sistemą. Kuriant šią procedūrą naudojama demonstracinių duomenų įmonė yra USMF. 


## <a name="add-a-new-user"></a>Naujo vartotojo įtraukimas
1. Pasirinkite Sistemos administravimas > Vartotojai > Vartotojai.
2. Spustelėkite Naujas.
3. Lauke „Vartotojo ID“ įveskite reikšmę.
    * Įveskite unikalų vartotojo identifikatorių. Reikės vartotojo ID.  
4. Lauke „Vartotojo vardas“ įveskite reikšmę.
    * Įveskite vartotojo vardą.  
5. Lauke „Domenas“ įveskite reikšmę.
    * Įveskite vartotojo domeną.  
6. Lauke „Pseudonimas“ įveskite reikšmę.
    * Įveskite vartotojo pseudonimą.  
7. Lauke „Įmonė“ spustelėkite išplečiamąjį mygtuką, kad atidarytumėte peržvalgą.
8. Sąraše raskite ir pasirinkite norimą įrašą.
9. Sąraše spustelėkite saitą pasirinktoje eilutėje.
    * Pasirinkite vartotoją įmonę  
10. Spustelėkite „Priskirti vaidmenis“.
11. Sąraše raskite ir pasirinkite norimą įrašą.
12. Spustelėkite GERAI.
13. Spustelėkite Įrašyti.

## <a name="import-users"></a>Importuoti vartotojus
1. Spustelėkite „Importuoti vartotojus“.
2. Sąraše pažymėkite pasirinktą eilutę.
3. Spustelėkite „Importuoti vartotojus“.
4. Spustelėkite Uždaryti.
