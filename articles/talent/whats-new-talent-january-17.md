# <a name="whats-new-or-changed-in-dynamics-365-talent-core-hr-january-17-2019"></a>Kas nauja ar pasikeitė sistemoje „Dynamics 365 Talent: Core HR“ (2019 m. sausio 17 d.)

[!include [banner](includes/banner.md)]

**8.1.2107 versija**

Šioje temoje aprašomos naujos ir pakeistos „Core HR“ funkcijos.

## <a name="minor-changes-and-bug-fixes-included-in-this-release"></a>Į šį leidimą įtraukti smulkūs pakeitimai ir klaidų ištaisymai

### <a name="new-position-assignment-start-datetime-default-has-been-corrected"></a>Naujų pareigų priskyrimo pradžios dato / laiko numatytoji reikšmė patikslinta
Perkeliant darbuotojus sistema nustatys datą / laiką į pasirinktą datą ir dienos pradžios laiką, atsižvelgdama į vartotojo laiko juostą.

### <a name="all-employees-have-the-same-position-description-in-the-exited-worker-task-management-list"></a>Visų darbuotojų pareigų aprašymai išėjusių darbuotojų užduočių valdymo sąraše sutampa
Atliktas darbo srities **Užduočių valdymas** atnaujinimas, kad būtų rodomi tinkami visų darbuotojų, kurie neseniai išėjo iš įmonės, pareigų aprašymai.

### <a name="action-requested-by-field-is-now-populated-on-workers-action-page"></a>Dabar laukas Veiksmo reikalauja užpildytas puslapyjeje Darbuotojų veiksmai
Atlikus šį pakeitimą, numatytoji lauko **Veiksmo reikalauja** reikšmė yra vartotojas, kuris prašo pakeitimo.

### <a name="ideas-portal-updated"></a>Atnaujintas idėjų portalas
Programoje pateikti saitai, skirti pateikti idėją į „Talent“, atnaujinti nurodant naują idėjų portalą. Visos idėjos, įvestos sename portale, buvo perkeltos, todėl visos jūsų puikios idėjos neišnyko ir kiti už jas gali balsuoti.
