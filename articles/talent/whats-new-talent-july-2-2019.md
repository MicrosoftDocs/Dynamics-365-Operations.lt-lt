# <a name="whats-new-or-changed-in-dynamics-365-for-talent-july-02-2019"></a>Kas nauja ar pasikeitė „Dynamics 365 for Talent” (2019 m. liepos mėn. 2 d.)

[!include [banner](includes/banner.md)]

Šioje temoje aprašomos naujos ir pakeistos „Microsoft Dynamics 365 for Talent“ funkcijos.

## <a name="changes-in-attract"></a>„Attract“ pakeitimai

### <a name="use-organization-service-accounts-to-send-e-mails"></a>Organizacijos tarnybos paskyrų naudojimas siunčiant el. laiškus

Dabar galite sukonfigūruoti el. pašto parametrus, kad el. laiškai kandidatams, kalbintojams ir samdos vadovams būtų siunčiami iš jūsų organizacijos el. pašto tarnybos paskyros. Daugiau informacijos žr. [El. pašto tarnybos paskyros prijungimas](./attract-configure-email-settings.md#connect-an-email-service-account).

### <a name="offer-email-templates"></a>Pasiūlymų el. laiškų šablonai 

Dabar galite konfigūruoti visų etapų organizacijos el. laiškų šablonus ruošdami pasiūlymus, juos tvirtindami ir priimdami. Prieš siųsdami el. laiškus vartotojai gali naudoti numatytąjį šabloną arba jį tinkinti. Norėdami gauti daugiau informacijos, žr. [El. pašto šablonų parametrų konfigūravimas](./attract-configure-email-settings.md#configure-email-template-settings).

### <a name="candidate-consent-on-application"></a>Kandidatų sutikimas teikiant prašymus

Dabar galite įjungti organizacijos sąlygų saitą. Įjungus šį parametrą kandidatai prieš pateikdami prašymą įdarbinti turės sutikti su sąlygomis. 

### <a name="coming-soon-in-attract"></a>Jau greitai „Attract“
#### <a name="job-approvals-appear-on-the-home-page"></a>Užduočių patvirtinimai rodomi pagrindiniame puslapyje

Patvirtinimai rodomi ataskaitų srities skyriuje **Patvirtinimai**. Tvirtintojai gali peržiūrėti savo patvirtinimus dalyje **Priskirta jums**, kurioje rodomas užduoties ID, pavadinimas, kiti tvirtintojai ir užduoties priskyrimo data. Vartotojai, pateikę užduotį patvirtinti, gali peržiūrėti savo užduotis dalyje **Jūsų užklausos**, kurioje rodomi pateiktą užduotį dar turintys patvirtinti tvirtintojai.

## <a name="changes-in-onboard"></a>Supažindinimo pakeitimai

Šiame leidime įtraukti smulkūs klaidų ištaisymai, skirti „Dynamics 365 Talent: Onboard“

## <a name="changes-in-core-hr"></a>„Core HR“ pakeitimai

Šiame leidime įtraukti smulkūs klaidų ištaisymai, skirti „Dynamics 365 Talent: Core HR“.
