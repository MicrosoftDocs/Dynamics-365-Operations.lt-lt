# <a name="whats-new-or-changed-in-dynamics-365-talent-december-17-2019"></a>Kas nauja ar pasikeitė sistemoje „Dynamics 365 Talent“ (2019 m. gruodžio 17 d.)

Šiame straipsnyje aprašomos naujos ir pakeistos „Dynamics 365 Talent“ funkcijos.

## <a name="changes-in-attract"></a>„Attract“ pakeitimai

Šiame leidime pataisytos nežymios klaidos programoje „Dynamics 365 Talent: Attract“.

## <a name="changes-in-onboard"></a>Supažindinimo pakeitimai

Šiame leidime pataisytos nežymios klaidos programoje „Dynamics 365 Talent: Onboard“.

## <a name="changes-in-core-hr"></a>„Core HR“ pakeitimai

Šiame skyriuje aprašyti pakeitimai taikomi 8.1.2674 komponavimo versijai. Kai kurių antraščių skaičiai skliausteliuose nurodo palaikymo numerius „Microsoft Dynamics Lifecycle Services“ (LCS).

### <a name="print-performance-reviews"></a>Veiklos efektyvumo vertinimų spausdinimas

Spausdinimo našumo apžvalgos dabar pasiekiamos visose aplinkose. Norėdami įgalinti šią funkciją, naudokite darbo sritį **Funkcijų valdymas**. Daugiau informacijos apie funkcijų valdymą žr. [Funkcijos valdymo apžvalga](https://docs.microsoft.com/dynamics365/fin-ops-core/fin-ops/get-started/feature-management/feature-management-overview). Daugiau informacijos apie spausdinimo našumo apžvalgas ieškokite skyriuje [Našumo apžvalgos](https://docs.microsoft.com/dynamics365/talent/performance-management-overview#performance-reviews).

### <a name="retired-position-with-direct-reports-floats-in-org-diagram-as-an-open-position-385436"></a>Panaikintos pareigos su tiesioginėmis ataskaitomis nefiksuojamos organizacijos diagramoje kaip atviros pareigos (385436)

Šios savaitės leidime pataisoma, kaip panaikintos pareigos su tiesioginėmis ataskaitomis rodomos ataskaitų hierarchijoje.
 
### <a name="system-unable-to-generate-calendar-days-381195"></a>Sistemoje nepavyksta generuoti kalendorinių dienų (381195) 

Dėl šios savaitės pakeitimų ištaisoma problema, kai nepavykdavo sugeneruoti kalendorinių dienų.

### <a name="current-balance-report-in-leave-pbi-doubles-employee-balance-385581"></a>Dabartinėje balanso ataskaitoje atostogų PBI dvigubinamas darbuotojų balansas (385581)

Dėl šio pakeitimo pataisoma dabartinė balanso ataskaita šio scenarijaus atveju.
