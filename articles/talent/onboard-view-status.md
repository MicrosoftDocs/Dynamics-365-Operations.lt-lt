# <a name="view-the-status-of-onboarding-employees-and-your-tasks"></a>Darbuotojų supažindinimo ir užduočių būsenos peržiūra

[!include [banner](includes/banner.md)]

„Microsoft Dynamics 365 for Talent: Onboard” leidžia jums peržiūrėti, kokią pažangą daro jūsų nauji darbuotojai savo supažindinimo veiklose. Taip pat galite peržiūrėti visas užduotis, kurios jums buvo priskirtos iš supažindinimo vadovų.

## <a name="view-the-status-of-onboarding-employees"></a>Darbuotojų supažindinimo būsenos peržiūra

1. Kairiajame meniu pasirinkite **Vadovai**. Matote visų išsiųstų supažindinimo vadovų eigos apžvalgą.

    ![[Supažindinimo vadovų būsenos peržiūra](./media/onboard-guide-status.png)](./media/onboard-guide-status.png)

2. Norėdami detalizuoti, pasirinkite vadovą, kurį norite peržiūrėti.
3. Pasirinkite skirtuką **Veiklos** pamatyti, kokia veiklas jūsų naujas darbuotojas jau atliko.

    ![[Veiklų, kurias baigė naujas darbuotojas, peržiūrėjimas](./media/onboard-status-activities.png)](./media/onboard-status-activities.png)

## <a name="view-onboarding-tasks-that-are-assigned-to-you"></a>Jums priskirtų supažindinimo užduočių peržiūra

1. Kairiajame meniu pasirinkite **Užduotys**. Jūs matote visų supažindinimo užduočių, kurios buvo jums priskirtos iš supažindinimo vadovų ir nusiųstos darbuotojams, apžvalgą.
2. Norėdami detalizuoti, pasirinkite pageidaujamą užduotį.
3. Tolesnę veiklą su nauju darbuotoju tęskite pagal reikalavimą.
4. Pabaigę užduotį, pažymėkite ją kaip atliktą, pridėdami varnelę apskritime.
