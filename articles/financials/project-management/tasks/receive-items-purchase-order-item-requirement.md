# <a name="receive-items-on-purchase-order-from-item-requirement"></a><span data-ttu-id="8a7b1-103">Gauti pirkimo užsakyme pateiktas prekes pagal prekės poreikį</span><span class="sxs-lookup"><span data-stu-id="8a7b1-103">Receive items on purchase order from item requirement</span></span>

[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="8a7b1-104">Šioje procedūroje nurodoma, kaip gauti pirkimo užsakyme pateiktas prekes pagal prekės poreikį.</span><span class="sxs-lookup"><span data-stu-id="8a7b1-104">This procedure shows how to receive items on a purchase order from an item requirement.</span></span>

<span data-ttu-id="8a7b1-105">Naudodami prekės poreikį vietoje prekės operacijos, galite planuoti pristatymą prieš faktiškai naudojant prekę, kurti pirkimo užsakymą, įtraukti prekę į prekybos sutarties sistemą bei įtraukti prekės poreikį į gamybos planavimą.</span><span class="sxs-lookup"><span data-stu-id="8a7b1-105">By using an item requirement instead of an item transaction, you can plan for delivery just before the item is actually used, create a purchase order, include the item in the trade-agreement framework, and include the item requirement in production planning.</span></span> 

<span data-ttu-id="8a7b1-106">Šioje užduotyje naudojamas USSI duomenų rinkinys.</span><span class="sxs-lookup"><span data-stu-id="8a7b1-106">This task uses the USSI data set.</span></span>

1. <span data-ttu-id="8a7b1-107">Eikite į Projektų valdymas ir apskaita > Projektai > Visi projektai.</span><span class="sxs-lookup"><span data-stu-id="8a7b1-107">Go to Project management and accounting > Projects > All projects.</span></span>
2. <span data-ttu-id="8a7b1-108">Sąraše spustelėkite saitą pasirinktoje eilutėje.</span><span class="sxs-lookup"><span data-stu-id="8a7b1-108">In the list, click the link in the selected row.</span></span>
3. <span data-ttu-id="8a7b1-109">Veiksmų srityje spustelėkite Planuoti.</span><span class="sxs-lookup"><span data-stu-id="8a7b1-109">On the Action Pane, click Plan.</span></span>
4. <span data-ttu-id="8a7b1-110">Spustelėkite Prekių poreikis.</span><span class="sxs-lookup"><span data-stu-id="8a7b1-110">Click Item requirements.</span></span>
5. <span data-ttu-id="8a7b1-111">Spustelėkite Naujas.</span><span class="sxs-lookup"><span data-stu-id="8a7b1-111">Click New.</span></span>
6. <span data-ttu-id="8a7b1-112">Sąraše pažymėkite pasirinktą eilutę.</span><span class="sxs-lookup"><span data-stu-id="8a7b1-112">In the list, mark the selected row.</span></span>
7. <span data-ttu-id="8a7b1-113">Lauke Prekės numeris įveskite arba pasirinkite reikšmę.</span><span class="sxs-lookup"><span data-stu-id="8a7b1-113">In the Item number field, enter or select a value.</span></span>
8. <span data-ttu-id="8a7b1-114">Lauke Kiekis įveskite skaičių.</span><span class="sxs-lookup"><span data-stu-id="8a7b1-114">In the Quantity field, enter a number.</span></span>
9. <span data-ttu-id="8a7b1-115">Spustelėkite Įrašyti.</span><span class="sxs-lookup"><span data-stu-id="8a7b1-115">Click Save.</span></span>
10. <span data-ttu-id="8a7b1-116">Veiksmų srityje spustelėkite Valdyti.</span><span class="sxs-lookup"><span data-stu-id="8a7b1-116">On the Action Pane, click Manage.</span></span>
11. <span data-ttu-id="8a7b1-117">Spustelėkite Funkcijos.</span><span class="sxs-lookup"><span data-stu-id="8a7b1-117">Click Functions.</span></span>
12. <span data-ttu-id="8a7b1-118">Spustelėkite Kurti pirkimo užsakymą.</span><span class="sxs-lookup"><span data-stu-id="8a7b1-118">Click Create purchase order.</span></span>
13. <span data-ttu-id="8a7b1-119">Pažymėkite žymės langelį Įtraukti.</span><span class="sxs-lookup"><span data-stu-id="8a7b1-119">Select the Include check box.</span></span>
14. <span data-ttu-id="8a7b1-120">Lauke Tiekėjo sąskaita įveskite arba pasirinkite vertę.</span><span class="sxs-lookup"><span data-stu-id="8a7b1-120">In the Vendor account field, enter or select a value.</span></span>
15. <span data-ttu-id="8a7b1-121">Spustelėkite GERAI.</span><span class="sxs-lookup"><span data-stu-id="8a7b1-121">Click OK.</span></span>
16. <span data-ttu-id="8a7b1-122">Eikite į Mokėtinos sumos > Pirkimo užsakymai > Visi pirkimo užsakymai.</span><span class="sxs-lookup"><span data-stu-id="8a7b1-122">Go to Accounts payable > Purchase orders > All purchase orders.</span></span>
17. <span data-ttu-id="8a7b1-123">Sąraše spustelėkite saitą pasirinktoje eilutėje.</span><span class="sxs-lookup"><span data-stu-id="8a7b1-123">In the list, click the link in the selected row.</span></span>
18. <span data-ttu-id="8a7b1-124">Veiksmų srityje spustelėkite Pirkti.</span><span class="sxs-lookup"><span data-stu-id="8a7b1-124">On the Action Pane, click Purchase.</span></span>
19. <span data-ttu-id="8a7b1-125">Spustelėkite Patvirtinti.</span><span class="sxs-lookup"><span data-stu-id="8a7b1-125">Click Confirm.</span></span>
20. <span data-ttu-id="8a7b1-126">Veiksmų srityje spustelėkite Gauti.</span><span class="sxs-lookup"><span data-stu-id="8a7b1-126">On the Action Pane, click Receive.</span></span>
21. <span data-ttu-id="8a7b1-127">Spustelėkite Produkto gavimo kvitas.</span><span class="sxs-lookup"><span data-stu-id="8a7b1-127">Click Product receipt.</span></span>
22. <span data-ttu-id="8a7b1-128">Sąraše pažymėkite pasirinktą eilutę.</span><span class="sxs-lookup"><span data-stu-id="8a7b1-128">In the list, mark the selected row.</span></span>
23. <span data-ttu-id="8a7b1-129">Lauke Produkto gavimo kvitas surinkite reikšmę.</span><span class="sxs-lookup"><span data-stu-id="8a7b1-129">In the Product receipt field, type a value.</span></span>
24. <span data-ttu-id="8a7b1-130">Spustelėkite GERAI.</span><span class="sxs-lookup"><span data-stu-id="8a7b1-130">Click OK.</span></span>
