<span data-ttu-id="9aada-106">Ši procedūra taikoma visoms Europos šalims / regionams.</span><span class="sxs-lookup"><span data-stu-id="9aada-106">This procedure applies to all European countries/regions.</span></span> <span data-ttu-id="9aada-107">Ši procedūra buvo sukurta naudojant demonstracinių duomenų įmonę DEMF, kurios pirminis adresas yra Vokietijoje.</span><span class="sxs-lookup"><span data-stu-id="9aada-107">The procedure was created using the demo data company DEMF with a primary address in Germany.</span></span> <span data-ttu-id="9aada-108">Ši procedūra yra skirta mokėtinų sumų vadovui arba gautinų sumų vadovui.</span><span class="sxs-lookup"><span data-stu-id="9aada-108">This procedure is intended for an accounts payable manager or accounts receivable manager.</span></span> <span data-ttu-id="9aada-109">Ši procedūra yra skirta funkcijai, įtrauktai į „Dynamics 365 for Operations“ 1611 versiją.</span><span class="sxs-lookup"><span data-stu-id="9aada-109">This procedure is for a feature that was added in Dynamics 365 for Operations version 1611.</span></span>
