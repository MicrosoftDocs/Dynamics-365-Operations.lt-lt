<span data-ttu-id="2ab3f-104">Šioje procedūroje parodoma, kaip generuoti mokesčių suderinimo ataskaitą, kurią gali naudoti Islandijos ir Norvegijos juridiniai subjektai.</span><span class="sxs-lookup"><span data-stu-id="2ab3f-104">This procedure shows how to generate the tax reconciliation report that is available for Icelandic and Norwegian legal entities.</span></span> <span data-ttu-id="2ab3f-105">Ši užduotis buvo sukurta naudojant demonstracinių duomenų įmonę DEMF, o juridinio subjekto pirminio adreso šalis / regionas atnaujintas į Norvegiją.</span><span class="sxs-lookup"><span data-stu-id="2ab3f-105">This task was created using the demo data company DEMF with the country/region of legal entity primary address updated to be Norway.</span></span>

    * <span data-ttu-id="2ab3f-110">Pasirinkite reikšmę 600180.</span><span class="sxs-lookup"><span data-stu-id="2ab3f-110">Select the value 600180.</span></span>
