# <a name="vat-reconciliation-report"></a>PVM derinimo ataskaita

[!include [task guide banner](../../includes/task-guide-banner.md)]

Šioje procedūroje parodoma, kaip generuoti mokesčių suderinimo ataskaitą, kurią gali naudoti Islandijos ir Norvegijos juridiniai subjektai. Ši užduotis buvo sukurta naudojant demonstracinių duomenų įmonę DEMF, o juridinio subjekto pirminio adreso šalis / regionas atnaujintas į Norvegiją.

1. Pasirinkite Mokestis > Užklausos ir ataskaitos > PVM ataskaitos > Mokesčių suderinimo ataskaita.
2. Lauke Iš sąskaitos įveskite arba pasirinkite reikšmę.
    * Pasirinkite reikšmę 200140.  
3. Lauke Į sąskaitą įveskite arba pasirinkite reikšmę.
    * Pasirinkite reikšmę 600180.  
4. Lauke Sudengimo laikotarpis įveskite arba pasirinkite reikšmę.
    * Pasirinkite reikšmę Pirmadienis.  
5. Lauke Pradžios data įveskite datą.
    * Pasirinkite reikšmę 2015-11-01.  
6. Lauke „Rodyti išsamią informaciją“ pasirinkite „Taip“.
7. Spustelėkite GERAI.
