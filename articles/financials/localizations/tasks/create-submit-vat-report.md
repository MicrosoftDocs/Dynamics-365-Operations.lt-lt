# <a name="create-and-submit-vat-report"></a>PVM ataskaitos kūrimas ir teikimas

[!include [task guide banner](../../includes/task-guide-banner.md)]

Ši procedūra padės kurti ir teikti PVM ataskaitas Vokietijos valdžios institucijoms. Ši procedūra buvo sukurta naudojant demonstracinių duomenų įmonės DEMF. Šią funkciją gali naudoti juridiniai subjektai, kurių pagrindinis adresas yra Vokietijoje.

1. Eikite į Mokestis > Deklaracijos > PVM > Pateikti PVM per sudengimo laikotarpį.
2. Lauke Sudengimo laikotarpis įveskite arba pasirinkite reikšmę.
3. Lauke Pradžios data įveskite datą.
4. Spustelėkite Gerai.
5. Lauke Kurti elektroninį mokesčių dokumentą pasirinkite Taip.
6. Spustelėkite GERAI.
