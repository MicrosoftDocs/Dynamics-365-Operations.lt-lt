# <a name="deposit-customer-payments"></a><span data-ttu-id="d7b81-103">Mokėti kliento mokėjimus</span><span class="sxs-lookup"><span data-stu-id="d7b81-103">Deposit customer payments</span></span>

[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="d7b81-104">Deponuokite kliento mokėjimus.</span><span class="sxs-lookup"><span data-stu-id="d7b81-104">Deposit customer payments.</span></span> <span data-ttu-id="d7b81-105">Šioje užduotyje naudojama demonstracinė įmonė USMF.</span><span class="sxs-lookup"><span data-stu-id="d7b81-105">This task uses the USMF demo company.</span></span>

1. <span data-ttu-id="d7b81-106">Pasirinkite Gautinos sumos > Mokėjimai > Mokėjimų žurnalas.</span><span class="sxs-lookup"><span data-stu-id="d7b81-106">Go to Accounts receivable > Payments > Payment journal.</span></span>
2. <span data-ttu-id="d7b81-107">Spustelėkite Naujas.</span><span class="sxs-lookup"><span data-stu-id="d7b81-107">Click New.</span></span>
3. <span data-ttu-id="d7b81-108">Lauke Pavadinimas spustelėkite išplečiamąjį mygtuką, kad atidarytumėte peržvalgą.</span><span class="sxs-lookup"><span data-stu-id="d7b81-108">In the Name field, click the drop-down button to open the lookup.</span></span>
4. <span data-ttu-id="d7b81-109">Pasirinkite mokėjimų žurnalą.</span><span class="sxs-lookup"><span data-stu-id="d7b81-109">Select the payment journal.</span></span> 
5. <span data-ttu-id="d7b81-110">Spustelėkite Eilutės.</span><span class="sxs-lookup"><span data-stu-id="d7b81-110">Click Lines.</span></span>
6. <span data-ttu-id="d7b81-111">Lauke Sąskaita pasirinkite klientą, kuriam įrašote mokėjimą.</span><span class="sxs-lookup"><span data-stu-id="d7b81-111">In the Account field, select the Customer for whom you are recording the payment.</span></span>
7. <span data-ttu-id="d7b81-112">Lauke Kreditas įveskite mokėjimo sumą.</span><span class="sxs-lookup"><span data-stu-id="d7b81-112">In the Credit field, enter the amount of the payment.</span></span>
    * <span data-ttu-id="d7b81-113">Pasirinkdami SF, kurios buvo sumokėtos, galite sumą palikti tuščią, kad ją apskaičiuotų sistema.</span><span class="sxs-lookup"><span data-stu-id="d7b81-113">You can choose to leave the amount blank, and have the system calculate it by selecting the invoices which were paid.</span></span>  
8. <span data-ttu-id="d7b81-114">Lauke Mokėjimo nuoroda surinkite reikšmę.</span><span class="sxs-lookup"><span data-stu-id="d7b81-114">In the Payment reference field, type a value.</span></span>
    * <span data-ttu-id="d7b81-115">Mokėjimo nuoroda galėtų būti įvedamas mokėjimo čekio numeris.</span><span class="sxs-lookup"><span data-stu-id="d7b81-115">The payment reference could be the check number for the payment you are entering.</span></span> <span data-ttu-id="d7b81-116">Mokėjimo nuorodos reikia tam, kad mokėjimą būtų galima įtraukti į depozito kvitą,</span><span class="sxs-lookup"><span data-stu-id="d7b81-116">The payment reference is required in order to include the payment on a deposit slip.</span></span>  
9. <span data-ttu-id="d7b81-117">Pažymėkite langelį Naudoti depozito kvitą.</span><span class="sxs-lookup"><span data-stu-id="d7b81-117">Mark the box Use a deposit slip.</span></span>
    * <span data-ttu-id="d7b81-118">Jei mokėjimas turėtų būti įtrauktas į depozitą, šią nuostatą pakeiskite į Taip.</span><span class="sxs-lookup"><span data-stu-id="d7b81-118">If the payment should be included in the deposit, change this setting to Yes.</span></span>  
10. <span data-ttu-id="d7b81-119">Spustelėkite Naujas.</span><span class="sxs-lookup"><span data-stu-id="d7b81-119">Click New.</span></span>
11. <span data-ttu-id="d7b81-120">Lauke Sąskaita pasirinkite kito mokėjimo klientą.</span><span class="sxs-lookup"><span data-stu-id="d7b81-120">In the Account field, select the Customer for the next payment.</span></span>
12. <span data-ttu-id="d7b81-121">Lauke Kreditas įveskite mokėjimo sumą.</span><span class="sxs-lookup"><span data-stu-id="d7b81-121">In the Credit field, enter the payment amount.</span></span>
13. <span data-ttu-id="d7b81-122">Lauke Mokėjimo nuoroda surinkite reikšmę.</span><span class="sxs-lookup"><span data-stu-id="d7b81-122">In the Payment reference field, type a value.</span></span>
14. <span data-ttu-id="d7b81-123">Pažymėkite langelį Naudoti depozito kvitą.</span><span class="sxs-lookup"><span data-stu-id="d7b81-123">Mark the box Use a deposit slip.</span></span>
15. <span data-ttu-id="d7b81-124">Spustelėkite Registruoti.</span><span class="sxs-lookup"><span data-stu-id="d7b81-124">Click Post.</span></span>
    * <span data-ttu-id="d7b81-125">Prieš generuojant depozito kvitą, reikia registruoti mokėjimus.</span><span class="sxs-lookup"><span data-stu-id="d7b81-125">Payments must be posted before the deposit slip can be generated.</span></span> <span data-ttu-id="d7b81-126">Tuo siekiama užtikrinti, kad, sugeneravus depozito kvitą, nepasikeistų mokėjimai.</span><span class="sxs-lookup"><span data-stu-id="d7b81-126">This is to ensure that the payments don't change after the deposit slip is generated.</span></span>  
16. <span data-ttu-id="d7b81-127">Spustelėkite Funkcijos.</span><span class="sxs-lookup"><span data-stu-id="d7b81-127">Click Functions.</span></span>
17. <span data-ttu-id="d7b81-128">Spustelėkite Depozito kvitas.</span><span class="sxs-lookup"><span data-stu-id="d7b81-128">Click Deposit slip.</span></span>
18. <span data-ttu-id="d7b81-129">Spustelėkite GERAI.</span><span class="sxs-lookup"><span data-stu-id="d7b81-129">Click OK.</span></span>
    * <span data-ttu-id="d7b81-130">Pirmasis puslapis naudojamas kurti depozito kvitui.</span><span class="sxs-lookup"><span data-stu-id="d7b81-130">The first page is used to create the deposit slip.</span></span>  
19. <span data-ttu-id="d7b81-131">Spustelėkite GERAI.</span><span class="sxs-lookup"><span data-stu-id="d7b81-131">Click OK.</span></span>
    * <span data-ttu-id="d7b81-132">Antrasis veiksmas yra spausdinti depozito kvitą, tačiau šis veiksmas nebūtinas.</span><span class="sxs-lookup"><span data-stu-id="d7b81-132">The second step is to print the deposit slip, but this step is not required.</span></span>
