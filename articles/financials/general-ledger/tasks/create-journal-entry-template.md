12. <span data-ttu-id="cce67-119">Įveskite sumą lauke Debetas.</span><span class="sxs-lookup"><span data-stu-id="cce67-119">Enter an amount in the Debit field.</span></span>
13. <span data-ttu-id="cce67-120">Spustelėkite Naujas.</span><span class="sxs-lookup"><span data-stu-id="cce67-120">Click New.</span></span>
14. <span data-ttu-id="cce67-121">Lauke Sąskaita nustatykite norimas reikšmes.</span><span class="sxs-lookup"><span data-stu-id="cce67-121">In the Account field, specify the desired values.</span></span>

    * <span data-ttu-id="cce67-129">• Procentas: kvito sumos konvertuojamos į procentų koeficientus, o tai, pasirinkus kvito šabloną, leidžia taikyti bet kokią sumą.</span><span class="sxs-lookup"><span data-stu-id="cce67-129">• Percent: The amounts in the voucher are converted into percentage factors, which allows any amount to be applied when the Voucher template is selected.</span></span>  <span data-ttu-id="cce67-130">• Suma: bus saugomos ir taikomos faktinės sumos.</span><span class="sxs-lookup"><span data-stu-id="cce67-130">• Amount: The actual amounts will be stored and applied.</span></span>  
21. <span data-ttu-id="cce67-131">Spustelėkite Pagrindiniai žurnalai.</span><span class="sxs-lookup"><span data-stu-id="cce67-131">Click General journals.</span></span>
22. <span data-ttu-id="cce67-132">Spustelėkite Naujas.</span><span class="sxs-lookup"><span data-stu-id="cce67-132">Click New.</span></span>
23. <span data-ttu-id="cce67-133">Lauke Pavadinimas spustelėkite išplečiamąjį mygtuką, kad atidarytumėte peržvalgą.</span><span class="sxs-lookup"><span data-stu-id="cce67-133">In the Name field, click the drop-down button to open the lookup.</span></span>
24. <span data-ttu-id="cce67-134">Sąraše spustelėkite saitą pasirinktoje eilutėje.</span><span class="sxs-lookup"><span data-stu-id="cce67-134">In the list, click the link in the selected row.</span></span>
25. <span data-ttu-id="cce67-135">Spustelėkite Eilutės.</span><span class="sxs-lookup"><span data-stu-id="cce67-135">Click Lines.</span></span>
26. <span data-ttu-id="cce67-136">Spustelėkite Funkcijos.</span><span class="sxs-lookup"><span data-stu-id="cce67-136">Click Functions.</span></span>
