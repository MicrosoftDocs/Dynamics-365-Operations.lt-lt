# <a name="set-up-withholding-tax"></a>Nustatyti išskaitomą mokestį

[!include[task guide banner](../../includes/task-guide-banner.md)]

Išskaitomas mokestis yra tiekėjams taikomas mokestis, kurį taikant nesukuriama PVM operacijų. Tiekėjas įsipareigoja sumokėti jo mokėjimams priskiriamą išskaitomą mokestį. Todėl išskaitomą mokestį galima registruoti tik balanso arba įsipareigojimų sąskaitose. Šis užduočių vadovas parodo, kaip nustatyti išskaitomą mokestį.

1. Pasirinkite Mokestis > Netiesioginiai mokesčiai > Išskaitomas mokestis > Išskaitomo mokesčio kodai.
2. Spustelėkite Naujas.
3. Lauke Išskaitomo mokesčio kodas surinkite reikšmę.
4. Lauke Išskaitomo mokesčio pavadinimas įveskite išskaitomo mokesčio kodo pavadinimą.
5. Lauke Pagrindinė sąskaita pasirinkite pagrindinę sąskaitą, kurioje registruoti išskaitomo mokesčio įsipareigojimus.
6. Spustelėkite Įrašyti.
7. Spustelėkite Reikšmės.
8. Sąraše pažymėkite pasirinktą eilutę.
9. Lauke Reikšmė įveskite procentinę dalį, naudojamą skaičiuoti išskaitomam mokesčiui.
10. Spustelėkite Įrašyti.
11. Uždarykite puslapį.
12. Spustelėkite Įrašyti.
13. Uždarykite puslapį.
14. Pasirinkite Mokestis > Netiesioginiai mokesčiai > Išskaitomas mokestis > Išskaitomo mokesčio grupės.
15. Spustelėkite Naujas.
16. Lauke Išskaitomo mokesčio grupė įveskite išskaitomo mokesčio grupės identifikatorių.
17. Lauke Aprašas įveskite išskaitomo mokesčio grupės pavadinimą.
18. Sąraše pažymėkite pasirinktą eilutę.
19. Lauke Išskaitomo mokesčio kodas pasirinkite išskaitomo mokesčio kodą.
20. Sąraše spustelėkite saitą pasirinktoje eilutėje.
21. Spustelėkite Įrašyti.
