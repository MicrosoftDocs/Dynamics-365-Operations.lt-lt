# <a name="fixed-asset-mass-update"></a><span data-ttu-id="4166d-103">Visiškas ilgalaikio turto atnaujinimas</span><span class="sxs-lookup"><span data-stu-id="4166d-103">Fixed asset mass update</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="4166d-104">Jei naudojate knygas, galite keisti turto, įtraukto į tą pačią knygą, grupių nusidėvėjimo konvencijas.</span><span class="sxs-lookup"><span data-stu-id="4166d-104">If you use books, you can change the depreciation conventions for groups of assets that are part of the same book.</span></span>

<span data-ttu-id="4166d-105">Pavyzdžiui, jei esate Jungtinėse Amerikos Valstijose ir daugiau nei 40 procentų savo turto pradėjote naudoti ketvirtąjį metų ketvirtį, turite naudoti ketvirčio vidurio nusidėvėjimo konvenciją.</span><span class="sxs-lookup"><span data-stu-id="4166d-105">For example, if you are in the United States, and you put more than 40 percent of your assets in service during the fourth quarter of the year, you must use the mid-quarter depreciation convention.</span></span> <span data-ttu-id="4166d-106">Norėdami keisti visą turtą, kuriam reikia naujos nusidėvėjimo konvencijos, galite naudoti masinio atnaujinimo procesą.</span><span class="sxs-lookup"><span data-stu-id="4166d-106">You can use the process for a mass update to change all assets that require the new depreciation convention.</span></span> 

<span data-ttu-id="4166d-107">Atnaujinant turto nusidėvėjimo konvenciją, panaikinamos visos egzistuojančios šio turto nusidėvėjimo operacijos.</span><span class="sxs-lookup"><span data-stu-id="4166d-107">When you update the depreciation convention for assets, you delete all depreciation transactions that exist for those assets.</span></span> <span data-ttu-id="4166d-108">Be to, panaikinamos ir visos to turto nusidėvėjimo koregavimo operacijos, papildomo nusidėvėjimo operacijos ir specialiojo nusidėvėjimo operacijos.</span><span class="sxs-lookup"><span data-stu-id="4166d-108">You also delete all transactions for depreciation adjustments, transactions for bonus depreciation, and transactions for extraordinary depreciation for those assets.</span></span> 

<span data-ttu-id="4166d-109">Norėdami atnaujinti jau likviduoto turto nusidėvėjimo konvenciją, pirmiausiai turite panaikinti esamas likvidavimo operacijas.</span><span class="sxs-lookup"><span data-stu-id="4166d-109">To update the depreciation convention for assets that have already been disposed of, you must first delete the existing disposal transactions.</span></span> <span data-ttu-id="4166d-110">Taip pat turite panaikinti visas operacijas, sugeneruotas dėl likvidavimo proceso.</span><span class="sxs-lookup"><span data-stu-id="4166d-110">You must also delete all transactions that were generated because of the disposal process.</span></span> 

<span data-ttu-id="4166d-111">Atnaujinę turto nusidėvėjimo konvenciją, galite vykdyti kiekvieno turto vieneto nusidėvėjimo ir specialiojo nusidėvėjimo operacijas.</span><span class="sxs-lookup"><span data-stu-id="4166d-111">After you update the depreciation convention for assets, you can process depreciation and extraordinary depreciation for each asset.</span></span> <span data-ttu-id="4166d-112">Be to, jei reikia atlikti koregavimą, galite rankiniu būdu atlikti nusidėvėjimo koregavimą.</span><span class="sxs-lookup"><span data-stu-id="4166d-112">You can also make manual depreciation adjustments, if any adjustments are required.</span></span>
