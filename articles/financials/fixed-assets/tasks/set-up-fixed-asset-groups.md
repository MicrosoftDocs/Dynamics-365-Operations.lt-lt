# <a name="set-up-fixed-asset-groups"></a><span data-ttu-id="2a0ae-103">Nustatyti ilgalaikio turto grupes</span><span class="sxs-lookup"><span data-stu-id="2a0ae-103">Set up fixed asset groups</span></span>

[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="2a0ae-104">Ši procedūra parodo, kaip sukurti naują ilgalaikio turto grupę.</span><span class="sxs-lookup"><span data-stu-id="2a0ae-104">This procedure shows how to create a new fixed asset group.</span></span> <span data-ttu-id="2a0ae-105">Jis naudoja vaidmenį Buhalteris ir USMF juridinio subjekto demonstracinius duomenis.</span><span class="sxs-lookup"><span data-stu-id="2a0ae-105">It uses the Accountant role and demo data for the USMF legal entity.</span></span>

1. <span data-ttu-id="2a0ae-106">Pasirinkite Ilgalaikis turtas > Nustatymas > Ilgalaikio turto grupės.</span><span class="sxs-lookup"><span data-stu-id="2a0ae-106">Go to Fixed assets > Setup > Fixed asset groups.</span></span>
2. <span data-ttu-id="2a0ae-107">Spustelėkite Naujas.</span><span class="sxs-lookup"><span data-stu-id="2a0ae-107">Click New.</span></span>
3. <span data-ttu-id="2a0ae-108">Lauke Ilgalaikio turto grupė surinkite reikšmę.</span><span class="sxs-lookup"><span data-stu-id="2a0ae-108">In the Fixed asset group field, type a value.</span></span>
4. <span data-ttu-id="2a0ae-109">Lauke Pavadinimas surinkite reikšmę.</span><span class="sxs-lookup"><span data-stu-id="2a0ae-109">In the Name field, type a value.</span></span>
    * <span data-ttu-id="2a0ae-110">Grupės Ilgalaikis turtas funkcijos Automatiškai numeruoti ilgalaikį turtą ir Numeracijos kodas bus viršesni už nuostatas ilgalaikio turto parametruose.</span><span class="sxs-lookup"><span data-stu-id="2a0ae-110">Autonumber fixed assets and Number sequence code on the Fixed asset group will override the settings on the Fixed assets parameters.</span></span> <span data-ttu-id="2a0ae-111">Jei šios ilgalaikio turto grupės turtas bus numeruojamas kitaip nei kitose grupėse, čia numeravimą galite keisti.</span><span class="sxs-lookup"><span data-stu-id="2a0ae-111">You can change it here if the assets in this fixed asset group will have different numbering from other groups.</span></span>  
5. <span data-ttu-id="2a0ae-112">Spustelėkite Knygos.</span><span class="sxs-lookup"><span data-stu-id="2a0ae-112">Click Books.</span></span>
6. <span data-ttu-id="2a0ae-113">Lauke Knyga įveskite arba pasirinkite reikšmę.</span><span class="sxs-lookup"><span data-stu-id="2a0ae-113">In the Book field, enter or select a value.</span></span>
    * <span data-ttu-id="2a0ae-114">Laukas Skaičiuoti nusidėvėjimą nustatytas į Taip, todėl turto knyga bus įtraukta į nusidėvėjimo pasiūlymus.</span><span class="sxs-lookup"><span data-stu-id="2a0ae-114">The Calculate depreciation field is set to Yes, so the asset book will be included in depreciation proposals.</span></span> <span data-ttu-id="2a0ae-115">Jei laukas Skaičiuoti nusidėvėjimą nustatytas į Ne, turtas nebus automatiškai nudėvimas.</span><span class="sxs-lookup"><span data-stu-id="2a0ae-115">If Calculate depreciation is set to No, the asset will not be automatically depreciated.</span></span>  
7. <span data-ttu-id="2a0ae-116">Nustatykite turto dėvėjimo laiką metais.</span><span class="sxs-lookup"><span data-stu-id="2a0ae-116">Set the Service life of the asset, in years.</span></span>
    * <span data-ttu-id="2a0ae-117">Atkreipkite dėmesį, kad lauko Nusidėvėjimo laikotarpiai reikšmė apskaičiuojama nustačius dėvėjimo laiką.</span><span class="sxs-lookup"><span data-stu-id="2a0ae-117">Note that the Depreciation periods field value is calculated after setting the Service life.</span></span>  
8. <span data-ttu-id="2a0ae-118">Lauke Nusidėvėjimo konvencija pasirinkite parinktį.</span><span class="sxs-lookup"><span data-stu-id="2a0ae-118">In the Depreciation convention field, select an option.</span></span>
9. <span data-ttu-id="2a0ae-119">Uždarykite puslapį.</span><span class="sxs-lookup"><span data-stu-id="2a0ae-119">Close the page.</span></span>
