[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="8b184-104">Šioje procedūroje nurodoma, kaip likviduoti ilgalaikį turtą naudojant laisvos formos SF.</span><span class="sxs-lookup"><span data-stu-id="8b184-104">This procedure shows how to dispose of a fixed asset using the free text invoice.</span></span>

1. <span data-ttu-id="8b184-105">Pasirinkite Gautinos sumos > Sąskaitos faktūros > Visos laisvos formos SF.</span><span class="sxs-lookup"><span data-stu-id="8b184-105">Go to Accounts receivable > Invoices > All free text invoices.</span></span>
2. <span data-ttu-id="8b184-106">Spustelėkite Naujas.</span><span class="sxs-lookup"><span data-stu-id="8b184-106">Click New.</span></span>
3. <span data-ttu-id="8b184-107">Lauke Kliento sąskaita įveskite arba pasirinkite reikšmę.</span><span class="sxs-lookup"><span data-stu-id="8b184-107">In the Customer account field, enter or select a value.</span></span>
4. <span data-ttu-id="8b184-108">Patikrinkite numatytąją SF datą ir, jei įmanoma, pakoreguokite.</span><span class="sxs-lookup"><span data-stu-id="8b184-108">Validate the default Invoice date and edit if applicable.</span></span>
5. <span data-ttu-id="8b184-109">Patikrinkite kitus numatytuosius antraštės laukus, pvz., Valiuta ir, jei įmanoma, pakoreguotikte.</span><span class="sxs-lookup"><span data-stu-id="8b184-109">Validate remaining default header fields, such as Currency and edit if applicable.</span></span>
6. <span data-ttu-id="8b184-110">SF eilutėje įveskite aprašymą.</span><span class="sxs-lookup"><span data-stu-id="8b184-110">Enter a Description into the invoice line.</span></span>
7. <span data-ttu-id="8b184-111">Įveskite arba pasirinkite SF eilutės pagrindinę sąskaitą.</span><span class="sxs-lookup"><span data-stu-id="8b184-111">Enter or select the Main account for the invoice line.</span></span>
8. <span data-ttu-id="8b184-112">Patikrinkite numatytąją PVM grupę ir Prekės PVM grupę.</span><span class="sxs-lookup"><span data-stu-id="8b184-112">Validate the default Sales tax group and Item sales tax group.</span></span>
9. <span data-ttu-id="8b184-113">Įveskite ilgalaikio turto vieneto kainą arba hte pardavimo sumą.</span><span class="sxs-lookup"><span data-stu-id="8b184-113">Enter the Unit price or hte Amount of the sale of the fixed asset.</span></span>
10. <span data-ttu-id="8b184-114">Spustelėkite Eilutės informacija.</span><span class="sxs-lookup"><span data-stu-id="8b184-114">Click Line details.</span></span>  
11. <span data-ttu-id="8b184-115">Pasirinkite parduodamo ilgalaikio turto numerį.</span><span class="sxs-lookup"><span data-stu-id="8b184-115">Select the Fixed asset number to be sold.</span></span>
12. <span data-ttu-id="8b184-116">Spustelėkite Registruoti.</span><span class="sxs-lookup"><span data-stu-id="8b184-116">Click Post.</span></span>
