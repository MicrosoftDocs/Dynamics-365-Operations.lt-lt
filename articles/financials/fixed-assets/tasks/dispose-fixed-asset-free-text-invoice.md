# <a name="dispose-of-a-fixed-asset-using-a-free-text-invoice"></a><span data-ttu-id="53607-103">Ilgalaikio turto likvidavimas naudojant laisvos formos SF</span><span class="sxs-lookup"><span data-stu-id="53607-103">Dispose of a fixed asset using a free text invoice</span></span>

[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="53607-104">Šioje temoje aiškinama, kaip likviduoti ilgalaikį turtą naudojant laisvos formos sąskaitą-faktūrą.</span><span class="sxs-lookup"><span data-stu-id="53607-104">This topic explains how to dispose of a fixed asset using the free text invoice.</span></span>

1. <span data-ttu-id="53607-105">Naršymo srityje eikite į **Moduliai > Gautinos sumos > Sąskaita-faktūra > Visos laisvos formos sąskaitos-faktūros**.</span><span class="sxs-lookup"><span data-stu-id="53607-105">In the navigation pane, go to **Modules > Accounts receivable > Invoices > All free text invoices**.</span></span>
2. <span data-ttu-id="53607-106">Pasirinkite **Naujas**.</span><span class="sxs-lookup"><span data-stu-id="53607-106">Select **New**.</span></span>
3. <span data-ttu-id="53607-107">Lauke **Kliento sąskaita** įveskite arba pasirinkite reikšmę.</span><span class="sxs-lookup"><span data-stu-id="53607-107">In the **Customer account** field, enter or select a value.</span></span>
4. <span data-ttu-id="53607-108">Patikrinkite numatytąją **Sąskaita-faktūra** datą ir ją redaguokite, jei taikoma.</span><span class="sxs-lookup"><span data-stu-id="53607-108">Validate the default **Invoice** date and edit if applicable.</span></span>
5. <span data-ttu-id="53607-109">Patikrinkite likusius numatytuosius antraštės laukus, pvz., **Valiuta** ir redaguokite, jei taikoma.</span><span class="sxs-lookup"><span data-stu-id="53607-109">Validate remaining default header fields, such as **Currency** and edit if applicable.</span></span>
6. <span data-ttu-id="53607-110">Skyriuje **Sąskaitos-faktūros eilutės** patikrinkite laukus **Aprašas** ir **Pagrindinė sąskaita**.</span><span class="sxs-lookup"><span data-stu-id="53607-110">In the **Invoice lines** section, validate the **Description** and **Main account** fields.</span></span>
7. <span data-ttu-id="53607-111">Patikrinkite numatytuosius laukus **PVM grupė** ir **Prekės PVM grupė**.</span><span class="sxs-lookup"><span data-stu-id="53607-111">Validate the default **Sales tax group** and **Item sales tax group** fields.</span></span>
8. <span data-ttu-id="53607-112">Įveskite ilgalaikio turto pardavimo **Vieneto kaina** arba **Suma**.</span><span class="sxs-lookup"><span data-stu-id="53607-112">Enter the **Unit price** or the **Amount** of the sale of the fixed asset.</span></span>
9. <span data-ttu-id="53607-113">Pasirinkite skyrių **Eilutės informacija**.</span><span class="sxs-lookup"><span data-stu-id="53607-113">Select the **Line details** section.</span></span>  
10. <span data-ttu-id="53607-114">Lauke **Ilgalaikis turtas** įveskite arba pasirinkite reikšmę.</span><span class="sxs-lookup"><span data-stu-id="53607-114">Enter or select a value in the **Fixed asset** field.</span></span>
11. <span data-ttu-id="53607-115">Pasirinkite **Registruoti**.</span><span class="sxs-lookup"><span data-stu-id="53607-115">Select **Post**.</span></span>
