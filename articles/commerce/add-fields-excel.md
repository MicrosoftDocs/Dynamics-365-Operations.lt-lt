[!include [banner](../includes/banner.md)]

<span data-ttu-id="bb8d8-104">Šioje temoje aprašoma, kaip įtraukti laukus į „Microsoft Excel“ darbaknygę, kad galėtumėte redaguoti mažmeninės prekybos operacijas „Microsoft Dynamics 365 Commerce“.</span><span class="sxs-lookup"><span data-stu-id="bb8d8-104">This topic describes how to add fields to a Microsoft Excel workbook so that you can edit retail transactions in Microsoft Dynamics 365 Commerce.</span></span>

## <a name="overview"></a><span data-ttu-id="bb8d8-105">Apžvalga</span><span class="sxs-lookup"><span data-stu-id="bb8d8-105">Overview</span></span>

<span data-ttu-id="bb8d8-106">Kai sugeneruojate „Excel“ failą, kad galėtumėte redaguoti mažmeninės prekybos operacijas, į failą įtraukiami kai kurie numatytieji laukai.</span><span class="sxs-lookup"><span data-stu-id="bb8d8-106">When you generate an Excel file so that you can edit retail transactions, the file is filled with some default fields.</span></span> <span data-ttu-id="bb8d8-107">Jeigu laukas, kurį reikia atnaujinti, pagal numatytuosius parametrus nėra matomas sugeneruotame „Excel“ faile, galite jį įtraukti.</span><span class="sxs-lookup"><span data-stu-id="bb8d8-107">If a field that must be updated isn't visible by default in the generated Excel file, you can add it.</span></span>

## <a name="add-fields-to-a-worksheet-in-an-excel-workbook"></a><span data-ttu-id="bb8d8-108">Laukų įtraukimas į darbalapį „Excel“ darbaknygėje</span><span class="sxs-lookup"><span data-stu-id="bb8d8-108">Add fields to a worksheet in an Excel workbook</span></span>

<span data-ttu-id="bb8d8-109">Norėdami įtraukti laukus į „Excel“ darbaknygę, kad galėtumėte redaguoti mažmeninės prekybos operacijas, atlikite nurodytus veiksmus.</span><span class="sxs-lookup"><span data-stu-id="bb8d8-109">To add fields to an Excel workbook so that you can edit retail transactions, follow these steps.</span></span>

1. <span data-ttu-id="bb8d8-110">Atsisiųskite ir atidarykite „Excel“ failą iš puslapio **Išrašai**, puslapio **Mažmeninės prekybos operacijos** arba plytelės **Operacijų tikrinimo klaidos** darbo srityje **Parduotuvės finansai**.</span><span class="sxs-lookup"><span data-stu-id="bb8d8-110">Download and open the Excel file from the **Statements** page, the **Retail transactions** page, or the **Transaction validation failures** tile in the **Store financials** workspace.</span></span>
1. <span data-ttu-id="bb8d8-111">Pasirinkti **Dizainas**.</span><span class="sxs-lookup"><span data-stu-id="bb8d8-111">Select **Design**.</span></span>
1. <span data-ttu-id="bb8d8-112">Norimai lentelei pasirinkite pieštuko simbolį, tada galimų laukų sąraše suraskite ir pasirinkite lauką, kurį norite įtraukti.</span><span class="sxs-lookup"><span data-stu-id="bb8d8-112">Select the pencil symbol for the desired table, and then, in the list of available fields, find and select the field that you want to add.</span></span>
1. <span data-ttu-id="bb8d8-113">Pasirinkite **Įtraukti** ir tada pasirinkite **Naujinti**.</span><span class="sxs-lookup"><span data-stu-id="bb8d8-113">Select **Add**, and then select **Update**.</span></span> <span data-ttu-id="bb8d8-114">Galite pertvarkyti laukus.</span><span class="sxs-lookup"><span data-stu-id="bb8d8-114">You can reorder fields.</span></span>
1. <span data-ttu-id="bb8d8-115">Kai naujinimas bus atliktas, pasirinkite **Atnaujinti**, kad būtų gauti duomenys naujam stulpeliui.</span><span class="sxs-lookup"><span data-stu-id="bb8d8-115">After the update is completed, select **Refresh** to fetch the data for the new column.</span></span>

<span data-ttu-id="bb8d8-116">Dabar naują lauką ir jam skirtus duomenis turi būti galima redaguoti „Excel“.</span><span class="sxs-lookup"><span data-stu-id="bb8d8-116">The new field and data for it should now be available for editing in Excel.</span></span>
