# <a name="browser-requirements-for-dynamics-365-commerce-sites"></a>„Dynamics 365 Commerce“ svetainių naršyklės reikalavimai

[!include [banner](includes/banner.md)]

Šiame straipsnyje minimi "Sites" naršyklės Microsoft Dynamics 365 Commerce reikalavimai.

„Dynamics 365 Commerce“ saitai geriausiai palaikomi tolesnių žiniatinklio naršyklių. Norėdami surasti paskutinę kiekvienos žiniatinklio naršyklės versiją, eikite į programinės įrangos gamintojo svetainę.

- [„Microsoft Edge“](https://www.microsoft.com/edge) (iki esamos versijos minus 1)
- [„Mozilla Firefox“](https://www.mozilla.org/firefox/new) (iki esamos versijos minus 1)
- [„Google Chrome“](https://www.google.com/chrome) (iki esamos versijos minus 1)
- [„Apple Safari“](https://support.apple.com/downloads/safari) (iki esamos versijos minus 1)

> [!NOTE]
> „Internet Explorer“ nepalaikomas. Jei naudojate „Internet Explorer“, kai kurios funkcijos gali neveikti ir vartotojai gali turėti papildomą standartinę patirtį.
