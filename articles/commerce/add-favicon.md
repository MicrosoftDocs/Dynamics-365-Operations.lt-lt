# <a name="add-a-favicon"></a><span data-ttu-id="55056-103">Įtraukti parankinių piktogramą</span><span class="sxs-lookup"><span data-stu-id="55056-103">Add a favicon</span></span>

[!include [banner](includes/banner.md)]

<span data-ttu-id="55056-104">Šioje temoje paaiškinama, kaip į savo svetainę įtraukti parankinių piktogramą.</span><span class="sxs-lookup"><span data-stu-id="55056-104">This topic explains how to add a favicon to your site.</span></span>

## <a name="overview"></a><span data-ttu-id="55056-105">Peržiūrėti</span><span class="sxs-lookup"><span data-stu-id="55056-105">Overview</span></span>

<span data-ttu-id="55056-106">Parankinių piktogrma yra mažas grafikos failas, rodomas žiniatinklio naršyklės skirtuke, adresų juostoje, naršymo retrospektyvoje, žymelėse, parankiniuose ir kitose vietose.</span><span class="sxs-lookup"><span data-stu-id="55056-106">A favicon is a small graphics file that is shown on a web browser tab, in the Address bar, in the browsing history, and in bookmarks or favorites, among other places.</span></span> <span data-ttu-id="55056-107">Rekomenduojame parankinių piktogramą įtraukti į savo svetainę, nes ji vaizduoja ir sustiprina jūsų prekės ženklą bei jūsų svetainę padeda atskirti nuo kitų svetainių, kuriose lankosi jūsų klientai.</span><span class="sxs-lookup"><span data-stu-id="55056-107">We recommend that you add a favicon to your site, because it represents and reinforces your brand, and helps distinguish your site from other sites that your customers visit.</span></span>

<span data-ttu-id="55056-108">Nors į svetainę galima įtraukti kelias įvairių dydžių ir failų tipų parankinių piktogramas, šioje temoje parodyta, kaip įtraukti vieną parankinių piktogramą.</span><span class="sxs-lookup"><span data-stu-id="55056-108">Although you can add multiple favicons of various sizes and file types to your site, this topic shows how to add a single favicon.</span></span> <span data-ttu-id="55056-109">Tačiau tas pats procesas ir vieta naudojami norint įtraukti daugiau parankinių piktogramų.</span><span class="sxs-lookup"><span data-stu-id="55056-109">However, the same process and location are used to add more favicons.</span></span>

## <a name="upload-a-favicon-to-your-sites-asset-collection"></a><span data-ttu-id="55056-110">Parankinių piktogramos nusiuntimas į svetainės turto rinkinį</span><span class="sxs-lookup"><span data-stu-id="55056-110">Upload a favicon to your site's asset collection</span></span>

<span data-ttu-id="55056-111">Norėdami parankinių piktogramą nusiųsti į savo svetainės turto rinkinį, atlikite tolesnius veiksmus.</span><span class="sxs-lookup"><span data-stu-id="55056-111">To upload a favicon to your site's asset collection, follow these steps.</span></span>

1. <span data-ttu-id="55056-112">Kairiojoje naršymo srityje pasirinkite **Medijos biblioteka**.</span><span class="sxs-lookup"><span data-stu-id="55056-112">In the left navigation pane, select **Media Library**.</span></span>
1. <span data-ttu-id="55056-113">Komandų juostoje pasirinkite **įkelti \> Įkelti medijos elementus**.</span><span class="sxs-lookup"><span data-stu-id="55056-113">On the command bar, select **Upload \> Upload Media Items**.</span></span>
1. <span data-ttu-id="55056-114">Failų naršyklės lange eikite į parankinių piktogramos vaizdo failą, kurį norite įkelti, pasirinkite jį ir tada pasirinkite **Atidaryti**.</span><span class="sxs-lookup"><span data-stu-id="55056-114">In the File Explorer window, browse to the favicon image file that you want to upload, select it, and then select **Open**.</span></span>
1. <span data-ttu-id="55056-115">Dialogo lange **Įkelti medijos elementą** įveskite reikiamą pavadinimą ir alternatyvųjį tekstą.</span><span class="sxs-lookup"><span data-stu-id="55056-115">In the **Upload Media Item** dialog box, enter the required title and alt text.</span></span>
1. <span data-ttu-id="55056-116">Norėdami iš karto po nusiuntimo publikuoti vaizdą, pažymėkite žymės langelį **Publikuoti medijos elementus po įkėlimo**.</span><span class="sxs-lookup"><span data-stu-id="55056-116">If you want to publish the image immediately after upload, select the **Publish media items after upload** check box.</span></span>

    > [!NOTE]
    > <span data-ttu-id="55056-117">Jei nepasirenkate žymės langelio **Publikuoti medijos elementus po įkėlimo**, turite grįžti į puslapį **Medijos elementai** ir parankinių piktogramą patys publikuoti vėliau.</span><span class="sxs-lookup"><span data-stu-id="55056-117">If you don't select the **Publish media items after upload** check box, you must return to **Media items** page and manually publish the favicon later.</span></span>

1. <span data-ttu-id="55056-118">Pasirinkite **Gerai**.</span><span class="sxs-lookup"><span data-stu-id="55056-118">Select **OK**.</span></span>
1. <span data-ttu-id="55056-119">Dešinėje esančioje ypatybių srityje nukopijuokite viešąjį parankinių piktogramos URL.</span><span class="sxs-lookup"><span data-stu-id="55056-119">In the property pane on the right, copy the public URL of the favicon.</span></span> <span data-ttu-id="55056-120">Šį URL naudosite vėliau.</span><span class="sxs-lookup"><span data-stu-id="55056-120">You will use this URL later.</span></span>

## <a name="create-the-html-for-your-favicon"></a><span data-ttu-id="55056-121">Jūsų parankinių piktogramos HTML kūrimas</span><span class="sxs-lookup"><span data-stu-id="55056-121">Create the HTML for your favicon</span></span>

<span data-ttu-id="55056-122">Norėdami sukurti parankinių piktogramos HTML, naudokite tolesnę HTML eilutę.</span><span class="sxs-lookup"><span data-stu-id="55056-122">To create the HTML for the favicon, use the following HTML string.</span></span> <span data-ttu-id="55056-123">Atribute **href** **Public\_URL\_for\_your\_favicon** („viešasis parankinių piktogramos URL“) pakeiskite anksčiau nukopijuotu viešuoju URL.</span><span class="sxs-lookup"><span data-stu-id="55056-123">For the **href** attribute, replace **Public\_URL\_for\_your\_favicon** with the public URL that you copied earlier.</span></span>

## <a name="create-a-page-fragment-that-contains-a-metatag-for-your-favicon"></a><span data-ttu-id="55056-124">Puslapio fragmento, kuriame yra jūsų parankinių piktogramos metažymių, kūrimas</span><span class="sxs-lookup"><span data-stu-id="55056-124">Create a page fragment that contains a metatag for your favicon</span></span>

<span data-ttu-id="55056-125">Norėdami sukurti puslapio fragmentą, kuriame yra jūsų parankinių piktogramos metažymių, atlikite tolesnius veiksmus.</span><span class="sxs-lookup"><span data-stu-id="55056-125">To create a page fragment that contains a metatag for your favicon, follow these steps.</span></span>

1. <span data-ttu-id="55056-126">Eikite į **Fragmentai** ir tuomet pasirinkite **Naujas**.</span><span class="sxs-lookup"><span data-stu-id="55056-126">Go to **Fragments**, and select **New**.</span></span>
1. <span data-ttu-id="55056-127">**Naujo puslapio fragmentas** teksto laukelyje, pasirinkite **Meta skirtukai**, kaip modulį, kuriuo remiasi puslapio fragmentas.</span><span class="sxs-lookup"><span data-stu-id="55056-127">In the **New page fragment** dialog box, select **Metatags** as the module that the page fragment is based on.</span></span>
1. <span data-ttu-id="55056-128">Įveskite puslapio fragmento pavadinimą, tada pasirinkite **Gerai**.</span><span class="sxs-lookup"><span data-stu-id="55056-128">Enter a name for the page fragment, and then select **OK**.</span></span>
1. <span data-ttu-id="55056-129">Fragmento hierarchijos medyje pasirinkite antrinį elementą **Numatytosios metažymės**.</span><span class="sxs-lookup"><span data-stu-id="55056-129">In the fragment hierarchy tree, select the **Default metatags** child.</span></span>
1. <span data-ttu-id="55056-130">Dešiniosios srities dalyje **Metažymės** pasirinkite **Įtraukti** ir įveskite anksčiau sukurtą HTML eilutę, skirtą parankinių piktogramai.</span><span class="sxs-lookup"><span data-stu-id="55056-130">In the right pane, under **Meta Tags**, select **Add**, and then enter the HTML string that you created earlier for the favicon.</span></span> 
1. <span data-ttu-id="55056-131">Pasirinkite **Baigti redagavimą**, o tada pasirinkite **Publikuoti**, kad publikuotumėte puslapio fragmentą.</span><span class="sxs-lookup"><span data-stu-id="55056-131">Select **Finish editing**, and then select **Publish** to publish the page fragment.</span></span>

## <a name="add-the-metatag-page-fragment-to-the-html-head-section-of-your-pages"></a><span data-ttu-id="55056-132">Metažymės puslapio fragmento įtraukimas į jūsų puslapių HTML antraštės sekciją</span><span class="sxs-lookup"><span data-stu-id="55056-132">Add the metatag page fragment to the HTML head section of your pages</span></span>

<span data-ttu-id="55056-133">Norėdami įtraukti metažymės puslapio fragmentą į jūsų puslapių HTML **antraštės** sekciją, atlikite tolesnius veiksmus.</span><span class="sxs-lookup"><span data-stu-id="55056-133">To add the metatag page fragment to the HTML **head** section of your pages, follow these steps.</span></span>

1. <span data-ttu-id="55056-134">Eikite į **Šablonai**, atidarykite puslapių, į kuriuos norite įtraukti parankinių piktogramą, šabloną ir pasirinkite **Redaguoti**.</span><span class="sxs-lookup"><span data-stu-id="55056-134">Go to **Templates**, open the template for the pages that you want to add your favicon to, and then select **Edit**.</span></span>
1. <span data-ttu-id="55056-135">Šablonų hierarchijos medyje pasirinkite daugtaškio (**...**) mygtuką, esantį dešinėje konteinerio **HTML antraštė** pusėje, ir pasirinkite **Įtraukti puslapio fragmentą**.</span><span class="sxs-lookup"><span data-stu-id="55056-135">In the template hierarchy tree, select the ellipsis (**...**) button to the right of the **HTML head** container, and then select **Add page fragment**.</span></span>
1. <span data-ttu-id="55056-136">**Pasirinkti puslapio fragmentą** teksto laukelyje, pasirinkite meta skirtuko puslapio fragmentą, kurį sukūrėte anksčiau ir tuomet pasirinkite **Gerai**.</span><span class="sxs-lookup"><span data-stu-id="55056-136">In the **Select page fragment** dialog box, select the metatag page fragment that you created earlier, and then select **OK**.</span></span>
1. <span data-ttu-id="55056-137">Pasirinkite **Baigti redagavimą**, o tada pasirinkite **Publikuoti**, kad publikuotumėte šabloną.</span><span class="sxs-lookup"><span data-stu-id="55056-137">Select **Finish editing**, and then select **Publish** to publish the template.</span></span>

> [!NOTE]
> <span data-ttu-id="55056-138">Jei jūsų svetainė naudoja daugiau nei vieną šabloną, turite įtraukti metažymės puslapio fragmentą į visus juos.</span><span class="sxs-lookup"><span data-stu-id="55056-138">If your site uses more than one template, you must add the metatags page fragment to all of them.</span></span>

<span data-ttu-id="55056-139">Kai peržiūrite puslapius, pagrįstus šablonu, į kurį įtraukėte metažymės puslapio fragmentą, matysite parankinių piktogramą naršyklės skirtuke.</span><span class="sxs-lookup"><span data-stu-id="55056-139">When you preview pages that are based on the template that you added the metatags page fragment to, you should now see the favicon on the browser tab.</span></span>
