<span data-ttu-id="3735d-106">Pagrindinis puslapis yra numatytasis puslapis, į kurį pirkėjai įeina, kai lankosi el. prekybos svetainėje.</span><span class="sxs-lookup"><span data-stu-id="3735d-106">The home page is the default page that shoppers go to when they visit an e-Commerce site.</span></span> <span data-ttu-id="3735d-107">Paprastai šiame puslapyje demonstruojami produktai ir akcijos, naudojant rinkodaros modulių derinį.</span><span class="sxs-lookup"><span data-stu-id="3735d-107">Typically, this page showcases products and promotions by using a combination of marketing modules.</span></span> <span data-ttu-id="3735d-108">Pagrindiniame puslapyje turi būti daug atvaizdų ir teksto, kad sudomintų pirkėjus.</span><span class="sxs-lookup"><span data-stu-id="3735d-108">The home page should be rich with images and text to keep shoppers engaged.</span></span>

<span data-ttu-id="3735d-109">Toliau pateiktame paveikslėlyje parodytas pagrindinio puslapio pavyzdys, kuris buvo sukurtas naudojant internetinį darbo pradžios rinkinį ir „Fabrikam“ temą.</span><span class="sxs-lookup"><span data-stu-id="3735d-109">The following illustration shows an example of a home page that was built by using the online starter kit and the "Fabrikam" theme.</span></span>

<span data-ttu-id="3735d-111">Pagrindinio puslapio viršuje yra antraštė, rodanti visas produktų kategorijas ir kitus puslapius, kuriuose pardavėjas norėtų, kad klientai naršytų.</span><span class="sxs-lookup"><span data-stu-id="3735d-111">The top of the home page has a header that shows all the product categories and other pages that the retailer wants customers to browse.</span></span> <span data-ttu-id="3735d-112">Pagrindinio puslapio apačioje yra poraštė, kurioje yra spartieji saitai su įvairiomis temomis, kurios gali sudominti klientus.</span><span class="sxs-lookup"><span data-stu-id="3735d-112">The bottom of the home page has a footer that contains quick links to various topics that might interest customers.</span></span>

<span data-ttu-id="3735d-113">Pagrindinėje pagrindinio puslapio sekcijoje galima išskirti produktus, kategorijas arba reklaminius gaminius naudojant įvairius „Dynamics 365 Commerce“ modulius:</span><span class="sxs-lookup"><span data-stu-id="3735d-113">The main section of the home page can highlight products, categories, or promotions by using various Dynamics 365 Commerce modules:</span></span>

- <span data-ttu-id="3735d-114">**Pagrindinė reklaminė juosta** – paprastai pirmasis pagrindinio skyriaus viršuje esanti elementas nurodo vieną ar daugiau reklaminės juostos atvaizdų, kuriuose išryškinami nauji produktai ir akcijos parduotuvėje.</span><span class="sxs-lookup"><span data-stu-id="3735d-114">**Hero** – Typically, the first item at the top of the main section shows one or more "hero" images that highlight new products and promotions in the store.</span></span> <span data-ttu-id="3735d-115">Jei yra keli pagrindinės reklaminės juostos atvaizdai, jie nuomojami karuseliniame modulyje, kad vartotojai galėtų juos naršyti.</span><span class="sxs-lookup"><span data-stu-id="3735d-115">If there are multiple hero images, they are hosted in a carousel module so that users can browse them.</span></span>

    <span data-ttu-id="3735d-116">Šioje iliustracijoje pateiktas pagrindinio puslapio pavyzdys, kuriame pagrindinis elementas pagrindiniame skyriuje yra turinio bloko modulio, pavadinimu „Naujas“, pagrindinės reklaminės juostos maketas.</span><span class="sxs-lookup"><span data-stu-id="3735d-116">The following illustration shows an example of a home page where the first item in the main section is a hero layout of a content block module that is named "New Arrival."</span></span>

    ![Pagrindinės reklaminės juostos modulio pavyzdys](./media/Hero.PNG)

- <span data-ttu-id="3735d-118">**Funkcija** – turinio bloko modulio funkcijų maketas, naudojamas produktams ar reklamai parduoti, naudojant atvaizdų ir teksto derinį.</span><span class="sxs-lookup"><span data-stu-id="3735d-118">**Feature** – A feature layout of a content block module is used to market products or promotions by using a combination of images and text.</span></span> <span data-ttu-id="3735d-119">Funkcijų maketai gali būti naudojami atskirai arba gali būti talpinami karuselės modulyje.</span><span class="sxs-lookup"><span data-stu-id="3735d-119">Features layouts can be used independently, or they can be hosted in a carousel module.</span></span>

    <span data-ttu-id="3735d-120">Toliau parodytame paveikslėlyje pateiktas turinio bloko modulio funkcijų maketo pagrindiniame puslapyje pavyzdys.</span><span class="sxs-lookup"><span data-stu-id="3735d-120">The following illustration shows an example of feature layout of a content block module on a home page.</span></span>

    ![Reklamuojamų produktų modulių pavyzdžiai](./media/Feature.PNG)

- <span data-ttu-id="3735d-122">**Išklotinė** – turinio bloko modulio išklotinės maketas, kuris parodo kelis produktus ar produktų kategorijas, naudojant vaizdų ir teksto derinį kelių stulpelių makete.</span><span class="sxs-lookup"><span data-stu-id="3735d-122">**Tile** – A tile layout of a content block module is used to showcase multiple products or category of products by using a combination of images and text in a multicolumn layout.</span></span> <span data-ttu-id="3735d-123">Šioje temoje esančio pagrindinio puslapio ankstesnėje iliustracijoje išklotinės maketas atvaizduoja trijų stulpelių prekes **Apsipirkti moterų skyriuje**, **Apsipirkti vyrų skyriuje** ir **Pirkti aksesuarus**.</span><span class="sxs-lookup"><span data-stu-id="3735d-123">In the illustration of a home page that appears earlier in this topic, a tile  layout is used for the three-column rendering of the **Shop Women**, **Shop Men**, and **Shop Accessories** items.</span></span>
- <span data-ttu-id="3735d-124">**Vaizdo įrašų leistuvas** – vaizdo įrašų leistuvo modulis gali būti naudojamas vaizdo įrašų turiniui pagrindiniame puslapyje rodyti.</span><span class="sxs-lookup"><span data-stu-id="3735d-124">**Video player** – A video player module can be used to showcase video content on the home page.</span></span> <span data-ttu-id="3735d-125">Anksčiau šioje temoje esančioje pagrindinio puslapio iliustracijoje galima rasti vaizdo įrašų leistuvo modulį.</span><span class="sxs-lookup"><span data-stu-id="3735d-125">The illustration of a home page that appears earlier in this topic includes a video player module.</span></span>
- <span data-ttu-id="3735d-126">**Teksto blokas** – turinio bloko modulis gali būti naudojamas teksto turiniui pagrindiniame puslapyje pateikti, vieno stulpelio arba kelių stulpelių maketo forma.</span><span class="sxs-lookup"><span data-stu-id="3735d-126">**Text block** – A content rich block module can be used to present text content on the home page in a single-column or multicolumn layout.</span></span>
- <span data-ttu-id="3735d-127">**Produktų rekomendacijos** – produktų rekomendacijų moduliai naudojami sąrašams, tokiems kaip **Naujas**, **Populiariausi** ir **Geriausiai parduodami**, rodyti pagrindiniame puslapyje.</span><span class="sxs-lookup"><span data-stu-id="3735d-127">**Product recommendations** – Product recommendations modules are used to show lists, such as **New**, **Trending**, and **Best Selling** on the home page.</span></span> <span data-ttu-id="3735d-128">Šiuose sąrašuose pateikiami produktai pagal apsipirkimo tendencijas ir jie gali būti algoritmiškai sugeneruoti arba rankiniu būdu kuruojami.</span><span class="sxs-lookup"><span data-stu-id="3735d-128">These lists showcase products based on shopping trends, and they can be algorithmically generated or manually curated.</span></span> <span data-ttu-id="3735d-129">Taip klientai gali greitai atrasti svarbiausius produktus ir tęsti apsipirkimą.</span><span class="sxs-lookup"><span data-stu-id="3735d-129">They help customers quickly discover top products and then continue to shop.</span></span>

    <span data-ttu-id="3735d-130">Toliau pateikiamoje iliustracijoje rodomas produktų rekomendacijų modulių pagrindiniame puslapyje pavyzdys.</span><span class="sxs-lookup"><span data-stu-id="3735d-130">The following illustration shows an example of product recommendations modules on a home page.</span></span>

    ![Produktų rekomendacijų modulių pavyzdžiai](./media/Recommendations.PNG)
