[!include [banner](includes/banner.md)]

<span data-ttu-id="b38e8-104">Šioje temoje aprašomas užduočių valdymas „Microsoft Dynamics 365 Commerce“ elektroninio kasos aparato (EKA) programoje.</span><span class="sxs-lookup"><span data-stu-id="b38e8-104">This topic describes task management in the Microsoft Dynamics 365 Commerce point of sale (POS) application.</span></span>

<span data-ttu-id="b38e8-105">„Dynamics 365 Commerce“ EKA programoje yra užduočių valdymo funkcijų, kurios leidžia parduotuvės vadybininkams ir darbuotojams tvarkyti užduotis ir naujinti užduočių būseną.</span><span class="sxs-lookup"><span data-stu-id="b38e8-105">The Dynamics 365 Commerce POS application has task management features that let store managers and workers manage tasks and update task status.</span></span> <span data-ttu-id="b38e8-106">Parduotuvės darbuotojai gali pasiekti užduotis, pasirinkdami EKA pagrindiniame puslapyje esančią plytelę **Užduotys** arba pasirinkdami užduočių pranešimus.</span><span class="sxs-lookup"><span data-stu-id="b38e8-106">Store workers can access tasks either by selecting the **Tasks** tile on the POS home page or by selecting task notifications.</span></span> <span data-ttu-id="b38e8-107">Pagal numatytuosius nustatymus parduotuvės darbuotojai perkeliami į skirtuką **Mano užduotys**, kur gali peržiūrėti jiems priskirtas užduotis.</span><span class="sxs-lookup"><span data-stu-id="b38e8-107">By default, store workers are taken to the **My tasks** tab, where they can view the tasks that are assigned to them.</span></span> <span data-ttu-id="b38e8-108">Tačiau jie gali lengvai pereiti prie skirtukų **Pradelstos užduotys**, **Atviros užduotys** ir **Užduočių sąrašai**.</span><span class="sxs-lookup"><span data-stu-id="b38e8-108">However, they can easily switch to the **Overdue tasks**, **Open tasks**, and **Task lists** tabs.</span></span>

## <a name="task-operations-for-store-managers"></a><span data-ttu-id="b38e8-109">Parduotuvių vadybininkams skirtos užduočių operacijos</span><span class="sxs-lookup"><span data-stu-id="b38e8-109">Task operations for store managers</span></span>

<span data-ttu-id="b38e8-110">Parduotuvės vadybininkai gali atlikti šias užduočių operacijas EKA programoje, naudodami komandų juostos mygtukus:</span><span class="sxs-lookup"><span data-stu-id="b38e8-110">Store managers can perform the following task operations in the POS application by using the buttons on the command bar:</span></span>

- <span data-ttu-id="b38e8-112">**Užduoties būsena** – pažymėtų užduočių būsenos keitimas.</span><span class="sxs-lookup"><span data-stu-id="b38e8-112">**Task status** – Change the status of selected tasks.</span></span>
- <span data-ttu-id="b38e8-113">**Filtruoti** – pagal numatytuosius nustatymus rodomos tik aktyvios užduotys.</span><span class="sxs-lookup"><span data-stu-id="b38e8-113">**Filter** – By default, only active tasks are shown.</span></span> <span data-ttu-id="b38e8-114">Tačiau taikydami filtrus vadybininkai gali peržiūrėti visas užduotis, net užduotis, kurios yra baigtos arba atšauktos.</span><span class="sxs-lookup"><span data-stu-id="b38e8-114">However, by applying filters, managers can view all tasks, even tasks that have been completed or canceled.</span></span>
- <span data-ttu-id="b38e8-115">**Nauja užduotis** – užduoties kūrimas pagal esamą užduočių sąrašą arba vienos paskirties užduoties kūrimas.</span><span class="sxs-lookup"><span data-stu-id="b38e8-115">**New task** – Create a task under an existing task list, or create an single-purpose task.</span></span>

<span data-ttu-id="b38e8-116">Parduotuvės darbuotojai gali atlikti šias užduočių operacijas EKA programoje, naudodami komandų juostos mygtukus:</span><span class="sxs-lookup"><span data-stu-id="b38e8-116">Store workers can perform the following task operations in the POS application by using the buttons on the command bar:</span></span>

- <span data-ttu-id="b38e8-117">**Užduoties būsena** – pažymėtų užduočių būsenos keitimas.</span><span class="sxs-lookup"><span data-stu-id="b38e8-117">**Task status** – Change the status of selected tasks.</span></span>
- <span data-ttu-id="b38e8-118">**Filtruoti** – pagal numatytuosius nustatymus rodomos tik aktyvios užduotys.</span><span class="sxs-lookup"><span data-stu-id="b38e8-118">**Filter** – By default, only active tasks are shown.</span></span> <span data-ttu-id="b38e8-119">Tačiau taikydami filtrus darbuotojai gali peržiūrėti visas užduotis, net užduotis, kurios yra baigtos arba atšauktos.</span><span class="sxs-lookup"><span data-stu-id="b38e8-119">However, by applying filters, workers can view all tasks, even tasks that have been completed or canceled.</span></span>
