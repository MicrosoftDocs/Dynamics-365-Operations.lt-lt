## <a name="prerequisites"></a><span data-ttu-id="57765-105">Būtinieji komponentai</span><span class="sxs-lookup"><span data-stu-id="57765-105">Prerequisites</span></span>

<span data-ttu-id="57765-106">Kad į svetainę būtų galima įtraukti informaciją apie autorių teises, reikia turėti tolesnius elementus.</span><span class="sxs-lookup"><span data-stu-id="57765-106">Before you can add a copyright notice to your site, you must have the following items:</span></span>

- <span data-ttu-id="57765-107">Šabloną su bendrai naudojamu poraštės fragmentu.</span><span class="sxs-lookup"><span data-stu-id="57765-107">A template that includes a shared footer fragment.</span></span>
- <span data-ttu-id="57765-108">Puslapį, kuriame naudojamas tas šablonas.</span><span class="sxs-lookup"><span data-stu-id="57765-108">A page that uses that template.</span></span>

## <a name="add-a-copyright-notice"></a><span data-ttu-id="57765-109">Įtraukti informaciją apie autorių teises</span><span class="sxs-lookup"><span data-stu-id="57765-109">Add a copyright notice</span></span>

<span data-ttu-id="57765-110">Norėdami kiekvieno puslapio, kuriame naudojamas konkretus šablonas, apačioje įtraukti informaciją apie autorių teises, atlikite tolesnius veiksmus.</span><span class="sxs-lookup"><span data-stu-id="57765-110">To add a copyright notice to the bottom of every page that uses a specific template, follow these steps.</span></span>

1. <span data-ttu-id="57765-111">Nueikite į **Fragmentai** ir pasirinkite **Naujas puslapio fragmentas**.</span><span class="sxs-lookup"><span data-stu-id="57765-111">Go to **Fragments**, and then select **New Page Fragment**.</span></span>
1. <span data-ttu-id="57765-112">Dialogo lange pasirinkite modulį **Poraštė** ir pavadinkite fragmentą.</span><span class="sxs-lookup"><span data-stu-id="57765-112">In the dialog box, select the **Footer** module, and name the fragment.</span></span> <span data-ttu-id="57765-113">Pavyzdžiui, įveskite **Poraštė-autorių teisės**.</span><span class="sxs-lookup"><span data-stu-id="57765-113">For example, enter **Footer-Copyright**.</span></span>
1. <span data-ttu-id="57765-114">Pasirinkite **Gerai**.</span><span class="sxs-lookup"><span data-stu-id="57765-114">Select **OK**.</span></span>
1. <span data-ttu-id="57765-115">Naršymo srityje pasirinkite prie elemento **Poraštė** esantį daugtaškio mygtuką (**...**), tada – **Įtraukti modulį**.</span><span class="sxs-lookup"><span data-stu-id="57765-115">In the navigation pane, select the ellipsis button (**...**) next to **Footer**, and then select **Add Module**.</span></span>
1. <span data-ttu-id="57765-116">Dialogo lange pasirinkite **Poraštės kategorija** ir **Gerai**.</span><span class="sxs-lookup"><span data-stu-id="57765-116">In the dialog box, select **Footer category**, and then select **OK**.</span></span>
1. <span data-ttu-id="57765-117">Naršymo srityje pasirinkite prie elemento **Poraštės kategorija** esantį daugtaškio mygtuką, tada – **Įtraukti modulį**.</span><span class="sxs-lookup"><span data-stu-id="57765-117">In the navigation pane, select the ellipsis button next to **Footer category**, and then select **Add Module**.</span></span>
1. <span data-ttu-id="57765-118">Dialogo lange pasirinkite **Teksto blokas** ir **Gerai**.</span><span class="sxs-lookup"><span data-stu-id="57765-118">In the dialog box, select **Text block**, and then select **OK**.</span></span>
1. <span data-ttu-id="57765-119">Naršymo srityje pasirinkite **Teksto blokas**.</span><span class="sxs-lookup"><span data-stu-id="57765-119">In the navigation pane, select **Text block**.</span></span>
1. <span data-ttu-id="57765-120">Dešinėje esančios ypatybių srities lauke **Pastraipa** įtraukite pranešimą apie autorių teises.</span><span class="sxs-lookup"><span data-stu-id="57765-120">In the properties pane on the right, in the **Paragraph** field, add your copyright message.</span></span> <span data-ttu-id="57765-121">Pavyzdžiui, įveskite **Autorių teisės priklauso „Fabrikam“, 2019 m.**</span><span class="sxs-lookup"><span data-stu-id="57765-121">For example, enter **(C) Fabrikam 2019**.</span></span>
1. <span data-ttu-id="57765-122">Pasirinkite **Įrašyti**, **Įrašyti ir atrakinti**, tada – **Publikuoti**.</span><span class="sxs-lookup"><span data-stu-id="57765-122">Select **Save**, select **Check In**, and then select **Publish**.</span></span>
1. <span data-ttu-id="57765-123">Nueikite į **Šablonai**, pasirinkite šabloną, tada – **Paimti ir užrakinti**.</span><span class="sxs-lookup"><span data-stu-id="57765-123">Go to **Templates**, select the template, and then select **Check Out**.</span></span>
1. <span data-ttu-id="57765-124">Dalyje **Puslapio struktūa** išplėskite **Pagrindinis tekstas**, tada – **Numatytasis puslapis**.</span><span class="sxs-lookup"><span data-stu-id="57765-124">Under **Page Outline**, expand **Body**, and then expand **Default Page**.</span></span>
1. <span data-ttu-id="57765-125">Pasirinkite prie elemento **Poraštės vieta** esantį daugtaškio mygtuką, tada – **Įtraukti fragmentą**.</span><span class="sxs-lookup"><span data-stu-id="57765-125">Select the ellipsis button next to **Footer Slot**, and then select **Add Fragment**.</span></span>
1. <span data-ttu-id="57765-126">Pasirinkite anksčiau sukurtą fragmentą, tada – **Pasirinkti**.</span><span class="sxs-lookup"><span data-stu-id="57765-126">Select the fragment that you created earlier, and then select **Select**.</span></span>
1. <span data-ttu-id="57765-127">Šabloną įrašykite ir atrakinkite bei publikuokite.</span><span class="sxs-lookup"><span data-stu-id="57765-127">Check in the template, and publish it.</span></span>

<span data-ttu-id="57765-128">Poraštė su informacija apie autorių teises automatiškai atsiranda visų puslapių, kuriuose naudojamas pasirinktas šablonas, apačioje.</span><span class="sxs-lookup"><span data-stu-id="57765-128">The footer that contains the copyright notice automatically appears at the bottom of all pages that use the selected template.</span></span>
