## <a name="add-a-copyright-notice"></a><span data-ttu-id="787d4-109">Įtraukti informaciją apie autorių teises</span><span class="sxs-lookup"><span data-stu-id="787d4-109">Add a copyright notice</span></span>

<span data-ttu-id="787d4-110">Norėdami kiekvieno puslapio, kuriame naudojamas konkretus šablonas, apačioje įtraukti informaciją apie autorių teises, atlikite tolesnius veiksmus.</span><span class="sxs-lookup"><span data-stu-id="787d4-110">To add a copyright notice to the bottom of every page that uses a specific template, follow these steps.</span></span>

1. <span data-ttu-id="787d4-111">Nueikite į **Fragmentai** ir pasirinkite **Naujas puslapio fragmentas**.</span><span class="sxs-lookup"><span data-stu-id="787d4-111">Go to **Fragments**, and then select **New Page Fragment**.</span></span>
1. <span data-ttu-id="787d4-112">Dialogo lange pasirinkite modulį **Poraštė** ir pavadinkite fragmentą.</span><span class="sxs-lookup"><span data-stu-id="787d4-112">In the dialog box, select the **Footer** module, and name the fragment.</span></span> <span data-ttu-id="787d4-113">Pavyzdžiui, įveskite **Poraštė-autorių teisės**.</span><span class="sxs-lookup"><span data-stu-id="787d4-113">For example, enter **Footer-Copyright**.</span></span>
1. <span data-ttu-id="787d4-114">Pasirinkite **Gerai**.</span><span class="sxs-lookup"><span data-stu-id="787d4-114">Select **OK**.</span></span>
1. <span data-ttu-id="787d4-115">Naršymo srityje pasirinkite prie elemento **Poraštė** esantį daugtaškio mygtuką (**...**), tada – **Įtraukti modulį**.</span><span class="sxs-lookup"><span data-stu-id="787d4-115">In the navigation pane, select the ellipsis button (**...**) next to **Footer**, and then select **Add Module**.</span></span>
1. <span data-ttu-id="787d4-116">Dialogo lange pasirinkite **Poraštės kategorija** ir **Gerai**.</span><span class="sxs-lookup"><span data-stu-id="787d4-116">In the dialog box, select **Footer category**, and then select **OK**.</span></span>
1. <span data-ttu-id="787d4-117">Naršymo srityje pasirinkite prie elemento **Poraštės kategorija** esantį daugtaškio mygtuką, tada – **Įtraukti modulį**.</span><span class="sxs-lookup"><span data-stu-id="787d4-117">In the navigation pane, select the ellipsis button next to **Footer category**, and then select **Add Module**.</span></span>
1. <span data-ttu-id="787d4-118">Dialogo lange pasirinkite **Raiškiojo turinio bloko elementas** ir **Gerai**.</span><span class="sxs-lookup"><span data-stu-id="787d4-118">In the dialog box, select **Content rich block item**, and then select **OK**.</span></span>
1. <span data-ttu-id="787d4-119">Naršymo srityje pasirinkite **Raiškiojo turinio bloko elementas**.</span><span class="sxs-lookup"><span data-stu-id="787d4-119">In the navigation pane, select **Content rich block item**.</span></span>
1. <span data-ttu-id="787d4-120">Dešinėje esančios ypatybių srities lauke **Pastraipa** įtraukite pranešimą apie autorių teises.</span><span class="sxs-lookup"><span data-stu-id="787d4-120">In the properties pane on the right, in the **Paragraph** field, add your copyright message.</span></span> <span data-ttu-id="787d4-121">Pavyzdžiui, įveskite **Autorių teisės priklauso „Fabrikam“, 2019 m.**</span><span class="sxs-lookup"><span data-stu-id="787d4-121">For example, enter **(C) Fabrikam 2019**.</span></span>
1. <span data-ttu-id="787d4-122">Pasirinkite **Įrašyti**, **Įrašyti ir atrakinti**, tada – **Publikuoti**.</span><span class="sxs-lookup"><span data-stu-id="787d4-122">Select **Save**, select **Check In**, and then select **Publish**.</span></span>
1. <span data-ttu-id="787d4-123">Nueikite į **Šablonai**, pasirinkite šabloną, tada – **Paimti ir užrakinti**.</span><span class="sxs-lookup"><span data-stu-id="787d4-123">Go to **Templates**, select the template, and then select **Check Out**.</span></span>
1. <span data-ttu-id="787d4-124">Dalyje **Puslapio struktūa** išplėskite **Pagrindinis tekstas**, tada – **Numatytasis puslapis**.</span><span class="sxs-lookup"><span data-stu-id="787d4-124">Under **Page Outline**, expand **Body**, and then expand **Default Page**.</span></span>
1. <span data-ttu-id="787d4-125">Pasirinkite prie elemento **Poraštės vieta** esantį daugtaškio mygtuką, tada – **Įtraukti fragmentą**.</span><span class="sxs-lookup"><span data-stu-id="787d4-125">Select the ellipsis button next to **Footer Slot**, and then select **Add Fragment**.</span></span>
1. <span data-ttu-id="787d4-126">Pasirinkite anksčiau sukurtą fragmentą, tada – **Pasirinkti**.</span><span class="sxs-lookup"><span data-stu-id="787d4-126">Select the fragment that you created earlier, and then select **Select**.</span></span>
1. <span data-ttu-id="787d4-127">Šabloną įrašykite ir atrakinkite bei publikuokite.</span><span class="sxs-lookup"><span data-stu-id="787d4-127">Check in the template, and publish it.</span></span>

<span data-ttu-id="787d4-128">Poraštė su informacija apie autorių teises automatiškai atsiranda visų puslapių, kuriuose naudojamas pasirinktas šablonas, apačioje.</span><span class="sxs-lookup"><span data-stu-id="787d4-128">The footer that contains the copyright notice automatically appears at the bottom of all pages that use the selected template.</span></span>
