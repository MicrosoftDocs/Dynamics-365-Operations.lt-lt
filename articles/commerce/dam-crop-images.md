[!include [banner](includes/banner.md)]

<span data-ttu-id="8fed2-104">Šioje temoje aprašoma, kaip apkarpyti vaizdus „Microsoft Dynamics 365 Commerce“ svetainės generatoriuje.</span><span class="sxs-lookup"><span data-stu-id="8fed2-104">This topic describes how to crop images in Microsoft Dynamics 365 Commerce site builder.</span></span>

## <a name="overview"></a><span data-ttu-id="8fed2-105">Peržiūra</span><span class="sxs-lookup"><span data-stu-id="8fed2-105">Overview</span></span>

<span data-ttu-id="8fed2-106">„Commerce“ svetainės generatoriaus medijos biblioteka leidžia apkarpyti vaizdus, kad juos būtų galima optimizuoti skirtingi modulių tipams ir peržiūros sritims.</span><span class="sxs-lookup"><span data-stu-id="8fed2-106">The Commerce site builder Media Library allows you to crop images to optimize them for different module types and viewports.</span></span>

## <a name="crop-an-image"></a><span data-ttu-id="8fed2-107">Vaizdo apkarpymas</span><span class="sxs-lookup"><span data-stu-id="8fed2-107">Crop an image</span></span>

<span data-ttu-id="8fed2-108">Norėdami apkarpyti vaizdą svetainės generatoriuje, atlikite šiuos veiksmus.</span><span class="sxs-lookup"><span data-stu-id="8fed2-108">To crop an image in site builder, follow these steps.</span></span>

1. <span data-ttu-id="8fed2-109">Kairiojoje „Commerce“ svetainės generatoriaus naršymo srityje pasirinkite **Medijos biblioteka**.</span><span class="sxs-lookup"><span data-stu-id="8fed2-109">In the left navigation pane of Commerce site builder, select **Media Library**.</span></span>
1. <span data-ttu-id="8fed2-110">Pagrindiniame lange pasirinkite vaizdą, kurį norite modifikuoti.</span><span class="sxs-lookup"><span data-stu-id="8fed2-110">In the main window, select the image you want to modify.</span></span>
1. <span data-ttu-id="8fed2-111">Komandų juostoje pasirinkite **Redaguoti**.</span><span class="sxs-lookup"><span data-stu-id="8fed2-111">On the command bar, select **Edit**.</span></span>
1. <span data-ttu-id="8fed2-112">Pasirinkite vaizdą, kad atsidarytų **Redaguoti režimą** langas.</span><span class="sxs-lookup"><span data-stu-id="8fed2-112">Select the image to enter **Edit Mode**.</span></span>
1. <span data-ttu-id="8fed2-113">Po pasirinktimi **Redaguoti režimą** pasirinkite **Redaguoti rodinį pagal modulį**.</span><span class="sxs-lookup"><span data-stu-id="8fed2-113">Under **Edit Mode**, select **Edit View by Module**.</span></span>
1. <span data-ttu-id="8fed2-114">Pasirinkite modulio tipą išplečiamajame meniu **Modulis**.</span><span class="sxs-lookup"><span data-stu-id="8fed2-114">From the **Module** drop-down menu, select the module type.</span></span>
1. <span data-ttu-id="8fed2-115">Pasirinkite rodinio tipą išplečiamajame meniu **Rodinio tipas**.</span><span class="sxs-lookup"><span data-stu-id="8fed2-115">From the **View type** drop-down menu, select the view type.</span></span>
1. <span data-ttu-id="8fed2-116">Išplečiamajame meniu **Išdėstymas** pasirinkite vaizdo išdėstymą.</span><span class="sxs-lookup"><span data-stu-id="8fed2-116">From the **Placement** drop-down menu, select the image placement.</span></span>
1. <span data-ttu-id="8fed2-117">Pasirinkite peržiūros srities dydį išplečiamajame meniu **Peržiūros sritis**.</span><span class="sxs-lookup"><span data-stu-id="8fed2-117">From the **Viewport** drop-down menu, select the viewport size.</span></span>
1. <span data-ttu-id="8fed2-118">Vaizdas persidengia su sritimi, nurodančia apkarpymo vietą.</span><span class="sxs-lookup"><span data-stu-id="8fed2-118">The image is overlaid with the area representing the crop region.</span></span> <span data-ttu-id="8fed2-119">Perkelkite ir keiskite apkarpymo vietos dydį pagal poreikį.</span><span class="sxs-lookup"><span data-stu-id="8fed2-119">Move and resize the crop region as needed.</span></span> <span data-ttu-id="8fed2-120">Proporcijos bus išlaikytos automatiškai.</span><span class="sxs-lookup"><span data-stu-id="8fed2-120">The aspect ratio will be maintained automatically.</span></span>
1. <span data-ttu-id="8fed2-121">Kai baigsite, komandų juostoje pasirinkite **Įrašyti** ir pasirinkite **Baigti redagavimą**.</span><span class="sxs-lookup"><span data-stu-id="8fed2-121">When you're done, on the command bar, select **Save**, and then select **Finish editing**.</span></span> 

<span data-ttu-id="8fed2-122">Baigus pasirinktinį apkarpymą, vaizdo pakeitimai įsigalios beveik iš karto.</span><span class="sxs-lookup"><span data-stu-id="8fed2-122">After custom cropping is completed, image modifications will take effect almost immediately.</span></span>
