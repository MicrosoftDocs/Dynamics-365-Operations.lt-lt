# <a name="task-management-overview"></a><span data-ttu-id="b310f-103">Užduočių valdymo apžvalga</span><span class="sxs-lookup"><span data-stu-id="b310f-103">Task management overview</span></span>

[!include [banner](includes/banner.md)]

<span data-ttu-id="b310f-104">Šioje temoje pateikiama užduočių valdymo programoje „Microsoft Dynamics 365 Commerce“ apžvalga, skirta vadovams ir darbuotojams.</span><span class="sxs-lookup"><span data-stu-id="b310f-104">This topic provides an overview of task management for managers and workers in Microsoft Dynamics 365 Commerce.</span></span>

<span data-ttu-id="b310f-105">Mažmeninės prekybos aplinkoje visada sunku užtikrinti, kad užduotys būtų atliekamos tinkamo žmogaus tinkamu laiku.</span><span class="sxs-lookup"><span data-stu-id="b310f-105">In a retail environment, it's always difficult to make sure that tasks are performed by the right person at the right time.</span></span> <span data-ttu-id="b310f-106">Mažmenininkai turi sugebėti pranešti darbuotojams apie būsimas užduotis ir pateikti susijusį verslo kontekstą, kad būtų galima atlikti užduotis tinkamai ir laiku.</span><span class="sxs-lookup"><span data-stu-id="b310f-106">Retailers must be able to notify workers about upcoming tasks and provide related business context, so that the tasks can be completed correctly and on time.</span></span>

<span data-ttu-id="b310f-107">Užduočių valdymas yra „Dynamics 365 Commerce“ produktyvumo funkcija, leidžianti vadovams ir darbuotojams kurti užduočių sąrašus, valdyti priskyrimo kriterijus, sekti užduočių būseną ir integruoti šias operacijas į „Commerce“ tarnybinį biurą ir elektroninio kasos aparato (EKA) programas.</span><span class="sxs-lookup"><span data-stu-id="b310f-107">Task management is a productivity feature in Dynamics 365 Commerce that lets managers and workers create task lists, manage assignment criteria, track task status, and integrate these operations between Commerce back office and point of sale (POS) applications.</span></span>

<span data-ttu-id="b310f-108">Pardavimo valdymo asmenys gali naudoti užduočių valdymo funkciją, kad sukurtų užduočių sąrašus, skirtus mažmeninės prekybos parduotuvėms, ir sektų būseną pagal parduotuvę ar darbuotoją.</span><span class="sxs-lookup"><span data-stu-id="b310f-108">Headquarters personas can use task management to create task lists for retail stores, and to track status by store or worker.</span></span> <span data-ttu-id="b310f-109">Jie taip pat gali kurti pasikartojančių užduočių (pvz., „Ketvirtadienio vakaro uždarymo kontrolinis sąrašas“).</span><span class="sxs-lookup"><span data-stu-id="b310f-109">They can also create recurrent tasks (for example, "Thursday night closing checklist").</span></span>

<span data-ttu-id="b310f-110">Parduotuvių vadybininkai gali naudoti užduočių valdymo funkciją, kad priskirtų užduočių atskiriems darbuotojams, siųstų pranešimus apie būsimas arba pradelstas užduotis, atnaujintų užduoties būseną ir kurtų vienos paskirties užduotis EKA programoje.</span><span class="sxs-lookup"><span data-stu-id="b310f-110">Store managers can use task management to assign tasks to individual workers, send notifications about upcoming tasks or tasks that are past due, update task status, and create single-purpose tasks in the POS application.</span></span> <span data-ttu-id="b310f-111">Tada darbuotojai matys pranešimus, galės peržiūrėti užduoties informaciją ir atnaujinti užduoties būseną EKA programoje.</span><span class="sxs-lookup"><span data-stu-id="b310f-111">Workers can then see notifications, view task details, and update task status at the POS.</span></span>
