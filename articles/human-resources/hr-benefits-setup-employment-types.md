# <a name="set-up-employment-types"></a>Įdarbinimo tipų nustatymas

[!include [banner](includes/preview-feature.md)]

Įdarbinimo tipų nustatymas „Dynamics 365 Human Resources“, kad jie atitiktų jūsų darbuotojus.

1. Darbo srities **Išmokų valdymas** dalyje **Sąranka** pasirinkite **Įdarbinimo tipai**.

2. Pasirinkite **Naujas**.

3. Nurodyti šių laukų vertes:

   | Laukas | Aprašymas |
   | --- | --- |
   | Įdarbinimo tipas | Unikalus įdarbinimo tipo identifikavimas. |
   | Aprašymas | Įdarbinimo tipo aprašas. |

4. Pasirinkite **Įrašyti**.
