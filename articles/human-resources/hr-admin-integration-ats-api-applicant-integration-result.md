Fizinis pavadinimas: mshr_hcmapplicantintegrationresult

Toks numeravimas suteikia būseną kandidato įrašui.

| Reikšmė | Žyma | Aprašas |
| --- | --- | --- |
| 200000000 | Neapdorota | Kandidatas vis dar svarstomas. |
| 200000001 | Pasamdyta | Kandidatas buvo pasamdytas. |
| 200000002 | Nepasamdyta | Sprendimas buvo padarytas nesamdyti kandidato. |
| 200000003 | Atmesta | Kandidatas buvo pašalintas iš svarstymų. |
