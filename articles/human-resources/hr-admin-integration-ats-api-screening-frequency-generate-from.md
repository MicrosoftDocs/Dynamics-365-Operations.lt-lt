# <a name="screening-frequency-generate-from"></a>Atrankos dažnumas sukuriamas pagal

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Šioje temoje aprašoma atrankos dažnumo sukūrimą pagal parinktį nustatytą „Dynamics 365 Human Resources“.

Fizinis pavadinimas: mshr_hcmfrequencygeneratefrom

Toks numeravimas suteikia parinktį nustatytą būsenos vertėms siekiant nustatyti skaičiavimo pradžios datą kitai būtinai atrankai.

| Reikšmė | Žyma:  | aprašymas |
| --- | --- | --- |
| 200000000 Nepasirinkta | Nebuvo parinkta jokios vertės. |
| 200000001 Duomenys užbaigti | Skaičiavimas atliekamas nuo paskutinės užbaigtos atrankos datos. |
| 200000002 Būtina data | Skaičiavimas atliekamas nuo paskutinės būtinos atrankos datos. |
