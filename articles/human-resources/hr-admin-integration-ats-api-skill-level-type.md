# <a name="skill-level-type"></a>Įgūdžių lygio tipas


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Šioje temoje aprašoma įgūdžių lygio tipo parinktis nustatyta „Dynamics 365 Human Resources“.

Fizinis pavadinimas: mshr_hrmskillleveltype

Toks numeravimas suteikia parinktį nustatytą būsenos vertėms, kurios kategorizuoja lygio priskyrimą kandidato įgūdžiams.

| Reikšmė | Žyma | Aprašas |
| --- | --- | --- |
| 200000000 | Faktinis | Esamas lygių lygis. |
| 200000001 | Uždavinys | Tikslinė vertė įgūdžių lygiui. |
