## <a name="option-set-process-type"></a>Parinkties nustatymas: proceso tipas

Faktinis pavadinimas: „mshr_hrmCompProcessType”

Numeravimas pateikia parinktį, nustatytą proceso tipui „Human Resources“ virtualiose objektuose.

| Reikšmė | Žyma:  | Aprašas |
| --- | --- | --- |
| 200000000 | None | Nėra. |
| 200000001 | Pasirinkta | Pasirinkta. |
| 200000002 | Atsisakyta | Atleista. |
