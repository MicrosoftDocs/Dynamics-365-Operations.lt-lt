# <a name="custom-fields"></a>Pasirinktiniai laukai

[!include [rename-banner](~/includes/cc-data-platform-banner.md)]

Galite įtraukti pasirinktinius laukus į „Dynamics 365 Human Resources“ puslapius. Daugiau informacijos apie pasirinktinių laukų kūrimą žr. [Pasirinktinių laukų kūrimas ir darbas su jais](https://docs.microsoft.com/dynamics365/unified-operations/fin-and-ops/get-started/user-defined-fields).

Dauguma Common Data Service objektų palaiko pasirinktinius laukus. Išsamų Common Data Service objektų sąrašą žr. [Common Data Service objektai](https://docs.microsoft.com/dynamics365/human-resources/hr-developer-entities). 

Įsitikinkite, kad įgalinote visus sukurtus pasirinktinius laukus, kad galėtumėte juos naudoti „Common Data Service“. Daugiau informacijos žr. skyrių [Pasirinktinių laukų rodymas duomenų objektuose](https://docs.microsoft.com/dynamics365/unified-operations/fin-and-ops/get-started/user-defined-fields#exposing-custom-fields-on-data-entities).
