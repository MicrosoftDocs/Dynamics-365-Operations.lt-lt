# <a name="payroll---gender-option-set"></a>Algalapis – Lyties parinkties nustatymas

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Šioje temoje aprašoma lyties parinktis nustatyta „Dynamics 365 Human Resources“.

## <a name="option-set-gender-option"></a>Parinkties nustatymas: Lyties parinktis

Toks numeravimas suteikia parinkčių rinkinį darbuotojų lytims. Ji yra prieinama mshr_hcmpersongender parinkties nustatyme.

| Reikšmė | Žyma | Aprašas |
| --- | --- | --- |
| 200000000 | None | Nebuvo nurodyta jokia lytis. |
| 200000001 | Vyras | Vyras. |
| 200000002 | Moteris | Moteris. |
| 200000003 | Neapibrėžta | Nenurodytos lyties pasirinkimas. |
