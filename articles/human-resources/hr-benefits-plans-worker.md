# <a name="create-worker-benefit-plans"></a>Darbininko išmokų planų kūrimas

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Galite sukurti darbininkų išmokų planus programoje „Microsoft Dynamics 365 Human Resources“, norėdami pasirinkti darbuotojams skirtus išmokų planus ir patvirtinti išmokų planų pasirinkimus. Paprastai darbuotojai patys pasirenka išmokų planus naudodami darbuotojų savitarnos paslaugą, o tada išmokų administratorius patvirtina pasirinkimus. 

1. Darbo srities **Išmokų valdymas** dalyje **Planai** pasirinkite **Darbininkų išmokų planai**.

2. Pasirinkite **Naujas**.

3. Nurodyti šių laukų vertes:

   | Laukas | Aprašymas |
   | --- | --- |
   | Laikotarpis | Nurodo išmokų laikotarpį, naudotiną planams filtruoti sparčiajame skirtuke Planai. Filtruokite planus, kad būtų lengviau pasirinkti visų plano įrašų pogrupį, kad galėtumėte patvirtinti pogrupį. Pavyzdžiui, pasirinkite laikotarpį, kurį sukūrėte, pavadintą 2015 m., kad patvirtintumėte visus 2015 m. registracijos išmokoms pasirinkimus. |
   | Darbininkas | Nurodo darbininką, naudotiną planams filtruoti sparčiajame skirtuke Planai. Filtruokite planus, kad būtų lengviau pasirinkti visų plano įrašų pogrupį ir galėtumėte patvirtinti pogrupį. |
   | Juridinis subjektas | Nurodo juridinį subjektą, naudotiną planams filtruoti sparčiajame skirtuke Planai. Filtruokite planus, kad būtų lengviau pasirinkti visų plano įrašų pogrupį ir galėtumėte patvirtinti pogrupį. |
   | Padengimo parinktis | Nurodo draudimo parinktį, naudotiną planams filtruoti sparčiajame skirtuke Planai. Filtruokite planus, kad būtų lengviau pasirinkti visų plano įrašų pogrupį ir galėtumėte patvirtinti pogrupį. |
   | Numatytoji | Išmokų planai filtruojami pagal tai, ar jie yra numatytieji planai. Filtruokite planus, kad būtų lengviau pasirinkti visų plano įrašų pogrupį ir galėtumėte patvirtinti pogrupį. |
   | Būsena | Išmokų planai filtruojami pagal jų būseną. Filtruokite planus, kad būtų lengviau pasirinkti visų plano įrašų pogrupį ir galėtumėte patvirtinti pogrupį. |
   | Patvirtinimas | Nurodo patvirtinimo būseną, naudotiną planams filtruoti sparčiajame skirtuke Planai. Filtruokite planus, kad būtų lengviau pasirinkti visų plano įrašų pogrupį ir galėtumėte patvirtinti pogrupį. |
   | Atšaukimas | Nurodo atšaukimo būseną, naudotiną planams filtruoti sparčiajame skirtuke Planai. Filtruokite planus, kad būtų lengviau pasirinkti visų plano įrašų pogrupį ir galėtumėte patvirtinti pogrupį. |
   | Galiojančios datos filtras | Filtruoja planus pagal tai, ar jie nebegaliojantys, aktyvūs, ar bus aktyvūs ateityje. Pažymėkite žymės langelius, atitinkančius planus, kuriuos norite matyti sparčiajame skirtuke Planai. |
   | Planai | Sparčiajame skirtuke Planai pateikiami planai, atitinkantys nurodytus filtro kriterijus. Atitinkamos konfigūracijos parinktys, kurias nustatė HR darbuotojai ir registracijos pasirinkimai, kuriuos pasirinko darbuotojai, yra įtraukti į kiekvieną eilutę. Apibrėžtas laukas nurodo, ar yra tikrinimo nesutapimų dėl plano pasirinkimo. |

4. Pasirinkite **Įrašyti**.
