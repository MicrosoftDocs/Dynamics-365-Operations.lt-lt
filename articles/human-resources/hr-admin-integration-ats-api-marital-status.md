# <a name="marital-status"></a>Šeiminė padėtis

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Šioje temoje aprašoma šeiminės padėties parinktį nustatytą „Dynamics 365 Human Resources“.

Fizinis pavadinimas: mshr_dirpersonmaritalstatus

Toks numeravimas suteikia parinktį nustatytą asmens įrašų šeiminę padėtį.

| Reikšmė | Žyma | Aprašas |
| --- | --- | --- |
| 200000000 | Joks | Nebuvo parinkta jokios vertės.
| 200000001 | Nevedęs / neištekėjusi | Nesusituokęs ar nesusituokusi. |
| 200000002 | Vedęs / ištekėjusi | Susituokęs ar susituokusi. |
| 200000003 | Išsiskyręs (-usi) | Išsiskyręs / išsiskyrusi. |
| 200000004 | Našlys (-ė) | Našlys ar našlė. |
