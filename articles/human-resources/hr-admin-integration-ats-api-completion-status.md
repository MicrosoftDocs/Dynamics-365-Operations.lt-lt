# <a name="completion-status"></a>Užbaigimo būsena

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Šioje temoje aprašoma užbaigimo būsenos parinktį nustatytą „Dynamics 365 Human Resources“.

Fizinis pavadinimas: mshr_hcmcompletionstatus

Toks numeravimas suteikia parinktį nustatytą būsenos vertėms pretendento atrankai. 

| Reikšmė | Žyma:  | aprašymas |
| --- | --- | --- |
| 200000000 | Nebaigtas | Pretendentas dar neužbaigė atrankos. |
| 200000001 | Praleisti | Pretendentas dar užbaigė atranką. |
| 200000002 | Triktis | Pretendentas nepavyko atranka. |
