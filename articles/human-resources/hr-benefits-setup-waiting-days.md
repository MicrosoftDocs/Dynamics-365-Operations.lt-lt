# <a name="configure-waiting-days"></a>Laukimo dienų konfigūravimas


[!INCLUDE [PEAP](../includes/peap-2.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Programoje „Microsoft Dynamics 365 Human Resources“, laukimo dienos nustato mėnesinį etapą, kuris bus taikomas išmokų planams. Pavyzdžiui, 15 laukimo diena yra kiekvieno mėnesio 15 diena. 

1. Darbo srities **Išmokų valdymas** dalyje **Sąranka** pasirinkite **Laukimo dienos**.

2. Pasirinkite **Naujas**.

3. Nurodyti šių laukų vertes:

   | Laukas | Aprašymas |
   | --- | --- |
   | **Laukimo diena** | Laukimo dienos unikalusis identifikatorius. |
   | **Aprašymas** | Laukimo dienos aprašas. |
   | **Diena** | Mėnesio diena, kuriai sistema turėtų priskirti laukimo laikotarpio datą. |
   
4. Pasirinkite **Įrašyti**.
