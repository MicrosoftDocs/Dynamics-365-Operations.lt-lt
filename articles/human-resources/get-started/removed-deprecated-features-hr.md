# <a name="removed-or-deprecated-features-in-dynamics-365-human-resources"></a>Pašalintos arba pasenusios "Dynamics 365" personalo funkcijos

Šiame straipsnyje aprašomos priemonės, kurios buvo pašalintos arba suplanuotos pašalinti iš Microsoft Dynamics 365 personalo.

- *Pašalinta* funkcija nebėra įtraukta į produktą.
- *Nebenaudojama* funkcija nebėra aktyviai tobulinama ir gali būti pašalinta ateityje išleidus naujinį.

Šis sąrašas skirtas suteikti jums informacijos apie pašalintas ir nebenaudojamas funkcijas, kad galėtumėte geriau planuoti savo darbą.

## <a name="features-removed-or-deprecated-in-human-resources"></a>Personalo funkcijos pašalintos arba pasenusios 

### <a name="new-lcs-project-creation-in-human-resources-stand-alone"></a>Naujas LCS projekto kūrimas personalo valdymo srityje (atskiras) 

[Pateikti personalo ištekliams](/hr-admin-setup-provision)

| &nbsp; | &nbsp; |
|------------|--------------------|
| **Nerekomendavimo/pašalinimo priežastis** | Pakeista projekto sukūrimu finansų ir operacijų infrastruktūrose. Norėdami daugiau sužinoti apie infrastruktūros suliejimą, žr. " [Dynamics 365" žmogiškųjų išteklių infrastruktūros suliejimo DUK](/hr-infrastructure-merge-faq). |
| **Pakeitė kita funkcija?**   | Taip |
| **Paveiktos produkto sritys**         | Prašymas |
| **Visuotinio diegimo parinktis**              | Viskas |
| **Būsena**                         | Pasenusi: nuo 2022 m. liepos mėn. mes nebepalaikome naujų projektų personalo srityje (autonominis). |
