# <a name="generate-payroll-entities"></a>Algalapio objektų generavimas

Naudokite šią „OData” funkciją generuoti objektams, reikalingiems algalapio integravimui. Jei Personalo valdyme atlikti bet kokie šių objektų pakeitimai, pavyzdžiui, įtraukti pasirinktiniai laukai, šią funkciją galima iškviesti dar kartą kiekvieno objekto metaduomenų atnaujinimui. Atsake yra operacijos ID, kurį galite stebėti, kad žinotumėte, kada generavimo procesas bus baigtas.

**Užklausa**

```http
GET [Organizaton URI]/api/data/v9.1/RefreshHumanResourcesVirtualEntities
```

**pagrindinis tekstas**

```json
{
    "PhysicalNames" : ["PayrollEmployeeEntity", "HcmWorkerBaseEntity", "PayrollPositionEntity", "PayrollPositionJobEntity", "PayrollWorkerAddressEntity", "HcmJobDetailEntity", "HcmCompFixedPlanTableEntity", "PayrollFixedCompensationPlanEntity", "HcmEmploymentDetailEntity"]
}
```

**Atsakas**

```json
{
    "AsyncOperationId": "8b98d338-f939-4c86-9a91-80b76b6ab2ea"
}
```
