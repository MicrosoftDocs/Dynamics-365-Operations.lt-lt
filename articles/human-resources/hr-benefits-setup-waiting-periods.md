# <a name="configure-waiting-periods"></a>Laukimo laikotarpių konfigūravimas

[!include [banner](includes/preview-feature.md)]

Programoje „Microsoft Dynamics 365 Human Resources“, laukimo dienos nustato etapą, kuris bus taikomas išmokų planams. Pavyzdžiui, trys mėnesiai nuo samdos dienos, pirma kiekvieno mėnesio diena arba šeši mėnesiai.   

1. Darbo srities **Išmokų valdymas** dalyje **Sąranka** pasirinkite **Laukimo laikotarpiai**.

2. Pasirinkite **Naujas**.

3. Nurodyti šių laukų vertes:

   | Laukas | Aprašymas |
   | --- | --- |
   | Laukimo kodas | Laukimo laikotarpio unikalusis identifikatorius. |
   | Aprašymas | Laukimo laikotarpio aprašymas. |
   | Laukimo metodas | Išplečiamajame verčių sąraše pasirinkite tinkamą laukimo metodą. Pasirinktys yra tokios: grynasis, šis mėnuo, šis ketvirtis, šie metai ir ši savaitė. |
   | Mėnesiai | Įvesti mėnesių, pridedamų prie laukimo metodo skaičiuojant laukimo datą, skaičių. |
   | Days | Įvesti dienų, pridedamų prie laukimo metodo skaičiuojant laukimo datą, skaičių. |
   | Laukimo diena | Pasirinkti laukimo dieną, naudojamą laukimo datai skaičiuoti. |

4. Pasirinkite **Įrašyti**.
