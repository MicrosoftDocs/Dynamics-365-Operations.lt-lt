| 200000003 | Profesinis | Darbuotojai. |
| 200000004 | Technikas | Technikai. |
| 200000005 | Prekybos darbuotojai | Prekybos darbuotojai. |
| 200000006 | Administravimas | Administracinės pagalbos darbuotojai. |
| 200000007 | Amatininkai | Amatininkai. |
| 200000008 | Operatoriai | Operatoriai. |
