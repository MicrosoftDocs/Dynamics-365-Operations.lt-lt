# <a name="set-up-rounding-rules"></a>Apvalinimo taisyklių nustatymas

Siekdami atitikti savo organizacijos poreikius, programoje  „Dynamics 365 Human Resources“ galite nustatyti apvalinimo taisykles.

1. Darbo srities **Išmokų valdymas** dalyje **Sąranka** pasirinkite **Apvalinimo taisyklės**.

2. Pasirinkite **Naujas**.

3. Nurodykite laukų **Apvalinti iki artimiausios**, **Scenarijus** ir **Apvalinimo forma** reikšmes.

4. Pasirinkite **Įrašyti**.
