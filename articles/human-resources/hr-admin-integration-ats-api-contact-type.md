Fizinis pavadinimas: mshr_logisticselectronicaddressmethodtype

Toks numeravimas suteikia parinktį nustatytą būsenos vertėms pretendento kontakto tipams. 

| Reikšmė | Žyma:  | aprašymas |
| --- | --- | --- |
| 200000000 | None | Nepasirinktas joks tipas. |
| 200000001 | Telefonas | Telefono numeris. |
| 200000002 | El. pašto adresas | El. pašto adresas. |
| 200000003 | URL | Žiniatinklio svetainės URL. |
| 200000004 | Teleksas | Telekso numeris. |
| 200000005 | Faksas | Fakso numeris. |
| 200000006 | „Facebook“ | „Facebook“ paskyra. Identifikuotas pagal vartotojo ID. |
| 200000007 | Twitter | „Twitter“ paskyra. Identifikuotas pagal @username. |
| 200000008 | „LinkedIn“ | „Linkedin“ paskyra. Identifikuotas pagal vartotojo vardą. |
