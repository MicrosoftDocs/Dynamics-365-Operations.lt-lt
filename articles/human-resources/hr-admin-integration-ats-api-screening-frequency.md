# <a name="screening-frequency"></a>Atrankos dažnumas

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Šioje temoje aprašoma atrankos dažnumo parinktis nustatyta „Dynamics 365 Human Resources“.

Fizinis pavadinimas: mshr_hcmfrequencyunit

Toks numeravimas suteikia parinktį nustatytą būsenos vertėms atrankos dažnumui. 

| Reikšmė | Žyma | Aprašas |
| --- | --- | --- |
| 200000000 Tik vieną kartą | Atranka būtina tik kartą. |
| 200000001 Kasdien | Atrankos dažnumas skaičiuojamas dienomis. |
| 200000002 Kas savaitę | Atrankos dažnumas skaičiuojamas savaitėmis. |
| 200000003 Kas mėnesį | Atrankos dažnumas skaičiuojamas mėnesiais. |
| 200000004 Kas metus | Atrankos dažnumas skaičiuojamas metais. |
