# <a name="award-type"></a>Premijos tipas


[!INCLUDE [PEAP](../includes/peap-1.md)]

[!include [Applies to Human Resources](../includes/applies-to-hr.md)]

Šiame straipsnyje aprašoma premijos tipo parinktis, nustatyta Dynamics 365 Human Resources.

## <a name="option-set-award-type"></a>Parinkties nustatymas: premijos tipas

Faktinis pavadinimas: „mshr_HrmCompVarAwardEmplType”

Numeravimas pateikia parinkčių nustatymą kintamosios atlyginimo dalies premijos tipui „Human Resources“ virtualiose objektuose.

| Reikšmė | Žyma: | Aprašas |
| --- | --- | --- |
| 200000000 | Bendroji premija | Bendroji premija. |
| 200000001 | Visa fiksuoto užmokesčio suma | Visa fiksuoto užmokesčio suma. |
