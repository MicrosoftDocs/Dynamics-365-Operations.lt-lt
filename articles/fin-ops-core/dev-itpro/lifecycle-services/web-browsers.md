# <a name="supported-web-browsers-for-lifecycle-services"></a>Palaikomos ciklo tarnybų žiniatinklio naršyklės 

[!include[banner](../includes/banner.md)]

Vartotojai gali pasiekti Microsoft Dynamics "Lifecyle Services" (LCS) naudodami naujausią šių populiariausių naršyklės versijų: 

- Microsoft Edge(rekomenduojama: [Chromiumi pagrįstas edge](https://support.microsoft.com/microsoft-edge/download-the-new-microsoft-edge-based-on-chromium-0f4a3dd7-55df-60f5-739f-00010dba52cf))
- Chromuotasis
- Arba turi būti iš dalies.

Norėdami surasti paskutinę kiekvienos žiniatinklio naršyklės versiją, eikite į programinės įrangos gamintojo svetainę. Optimaliam našumui ir optimaliai patirtimii, rekomenduojame naudoti naujausią modernios naršyklės versiją, ypač Microsoft Edge.
