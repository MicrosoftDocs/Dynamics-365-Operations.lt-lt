# <a name="assign-users-to-security-roles"></a><span data-ttu-id="a3726-103">Vartotojų priskyrimas saugos vaidmenims</span><span class="sxs-lookup"><span data-stu-id="a3726-103">Assign users to security roles</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="a3726-104">Norėdami „Finance and Operations” programose naudoti bet ką, išskyrus bendrąsias galimybes, vartotojai turi būti priskirti saugos vaidmenims.</span><span class="sxs-lookup"><span data-stu-id="a3726-104">To use anything other than common capabilities in Finance and Operations apps, users must be assigned to security roles.</span></span> <span data-ttu-id="a3726-105">Galite priskirti vartotojus vaidmenims automatiškai, remiantis taisyklėmis ir verslo duomenimis, pašalinti vartotojus iš automatinio vaidmenų priskyrimo arba įtraukti vartotojus į vaidmenis neautomatiniu būdu.</span><span class="sxs-lookup"><span data-stu-id="a3726-105">You can assign users to roles automatically, based on rules and business data, exclude users from automatic role assignment, or add users to roles manually.</span></span>

## <a name="automatically-assign-users-to-roles"></a><span data-ttu-id="a3726-106">Automatinis vartotojų priskyrimas vaidmenims</span><span class="sxs-lookup"><span data-stu-id="a3726-106">Automatically assign users to roles</span></span>

2. <span data-ttu-id="a3726-109">Medyje pasirinkite Apskaitos prižiūrėtojas.</span><span class="sxs-lookup"><span data-stu-id="a3726-109">In the tree, select 'Accounting supervisor'.</span></span> <span data-ttu-id="a3726-110">Pasirinkite vaidmenį, kurio taisyklę norite sukonfigūruoti.</span><span class="sxs-lookup"><span data-stu-id="a3726-110">Select the role that you want to configure the rule for.</span></span> <span data-ttu-id="a3726-111">Šiame pavyzdyje pasirinkite Apskaitos prižiūrėtojas.</span><span class="sxs-lookup"><span data-stu-id="a3726-111">In this example, select Accounting supervisor.</span></span> 
3. <span data-ttu-id="a3726-112">Pasirinkite **Įtraukti taisyklę**, kad atidarytumėte dialogo meniu.</span><span class="sxs-lookup"><span data-stu-id="a3726-112">Select **Add rule** to open the dialog menu.</span></span>
4. <span data-ttu-id="a3726-113">Sąraše **Pasirinkti užklausą** raskite ir pasirinkite norimą įrašą.</span><span class="sxs-lookup"><span data-stu-id="a3726-113">In the **Select a query** list, find and select the desired record.</span></span> <span data-ttu-id="a3726-114">Pasirinkite su šia taisykle naudotiną užklausą.</span><span class="sxs-lookup"><span data-stu-id="a3726-114">Select the query to use for this rule.</span></span>  
5. <span data-ttu-id="a3726-115">Sąraše **Narystės taisyklės pavadinimas** spustelėkite saitą pasirinktoje eilutėje.</span><span class="sxs-lookup"><span data-stu-id="a3726-115">In the **Membership rule name** list, click the link in the selected row.</span></span>
6. <span data-ttu-id="a3726-116">Pasirinkite **Redaguoti užklausą**.</span><span class="sxs-lookup"><span data-stu-id="a3726-116">Select **Edit query**.</span></span> <span data-ttu-id="a3726-117">Pagal poreikį užklausą redaguokite.</span><span class="sxs-lookup"><span data-stu-id="a3726-117">Edit the query, as needed.</span></span>  
7. <span data-ttu-id="a3726-118">Pasirinkite **Gerai**.</span><span class="sxs-lookup"><span data-stu-id="a3726-118">Select **OK**.</span></span>
8. <span data-ttu-id="a3726-119">Pasirinkite **Automatiškai priskirti vaidmenį**.</span><span class="sxs-lookup"><span data-stu-id="a3726-119">Select **Run automatic role assignment**.</span></span>
9. <span data-ttu-id="a3726-120">Eikite į **Naršymo sritis > Moduliai > Sistemos administravimas > Vartotojai > Vartotojai** (geriausia – atskiroje naršyklės kortelėje).</span><span class="sxs-lookup"><span data-stu-id="a3726-120">Go to **Navigation pane > Modules > System administration > Users > Users** (ideally in a separate browser tab).</span></span>
10. <span data-ttu-id="a3726-121">Peržiūrėkite vaidmenis, priskirtus skirtingiems vartotojams, kad patvirtintumėte, jog vaidmenų priskyrimo užklausa yra teisinga.</span><span class="sxs-lookup"><span data-stu-id="a3726-121">Review the roles assigned to various users to confirm that the role assignment query was correct.</span></span> <span data-ttu-id="a3726-122">Jei reikia, pakoreguokite ir vykdykite iš naujo.</span><span class="sxs-lookup"><span data-stu-id="a3726-122">Adjust and re-run if needed.</span></span>

3. <span data-ttu-id="a3726-126">Medyje pasirinkite Apskaitos prižiūrėtojas.</span><span class="sxs-lookup"><span data-stu-id="a3726-126">In the tree, select 'Accounting supervisor'.</span></span> <span data-ttu-id="a3726-127">Pasirinkite vaidmenį.</span><span class="sxs-lookup"><span data-stu-id="a3726-127">Select a role.</span></span> <span data-ttu-id="a3726-128">Šiam pavyzdžiui pasirinkite Apskaitos prižiūrėtojas.</span><span class="sxs-lookup"><span data-stu-id="a3726-128">For this example, select Accounting supervisor.</span></span>

5. <span data-ttu-id="a3726-130">Sąraše **Priskirti vartotojus vaidmeniui arba iš jo pašalinti** pažymėkite pasirinktą eilutę.</span><span class="sxs-lookup"><span data-stu-id="a3726-130">In the **Assign users to or exclude users from role** list, mark the selected row.</span></span> <span data-ttu-id="a3726-131">Pasirinkite vartotoją.</span><span class="sxs-lookup"><span data-stu-id="a3726-131">Select a user.</span></span>

## <a name="manually-assign-users-to-roles"></a><span data-ttu-id="a3726-138">Vartotojų priskyrimas vaidmenims neautomatiniu būdu</span><span class="sxs-lookup"><span data-stu-id="a3726-138">Manually assign users to roles</span></span>
<span data-ttu-id="a3726-139">Vartotojus, kurie neautomatiniu būdu priskirti saugos vaidmenims, administratorius taip pat turi pašalinti neautomatiniu būdu.</span><span class="sxs-lookup"><span data-stu-id="a3726-139">Users who are manually assigned to security roles must also be manually removed by the administrator.</span></span> <span data-ttu-id="a3726-140">Šie vartotojai nėra pašalinami iš vaidmenų pagal automatinio vaidmenų priskyrimo taisykles.</span><span class="sxs-lookup"><span data-stu-id="a3726-140">These users are not removed from roles by rules for automatic role assignment.</span></span>

1. <span data-ttu-id="a3726-141">Eikite į **Naršymo sritis > Moduliai > Sistemos administravimas > Sauga > Priskirti vartotojus vaidmenims**.</span><span class="sxs-lookup"><span data-stu-id="a3726-141">Go to **Navigation pane > Modules > System administration > Security > Assign users to roles**.</span></span>
2. <span data-ttu-id="a3726-142">Medyje pasirinkite vaidmenį, tada meniu **Vaidmeniui priskirti vartotojai** pasirinkite **Rankiniu būdu priskirti / pašalinti vartotojus**.</span><span class="sxs-lookup"><span data-stu-id="a3726-142">In the tree, select a role, and in the **Users assigned to role** menu, select **Manually assign / exclude users**.</span></span>
4. <span data-ttu-id="a3726-143">Dalyje **Priskirti vartotojus vaidmeniui arba iš jo pašalinti** išvardyti vartotojai, kuriems nebuvo priskirtas vaidmuo, ir laukas **Priskyrimo režimas** nustatytas į **Nėra**.</span><span class="sxs-lookup"><span data-stu-id="a3726-143">In the **Assign users to or exclude users from role**, users that have not been assigned the role are listed with the **Assignment mode** set to **None**.</span></span> <span data-ttu-id="a3726-144">Pasirinkite vieną ar daugiau vartotojų, kuriems reikia priskirti vaidmenį.</span><span class="sxs-lookup"><span data-stu-id="a3726-144">Select one or more users that should be assigned the role.</span></span>
5. <span data-ttu-id="a3726-145">Dalyje **Veiksmų sritis** pasirinkite **Priskirti vaidmeniui**.</span><span class="sxs-lookup"><span data-stu-id="a3726-145">On the **Action pane**, select **Assign to role**.</span></span> <span data-ttu-id="a3726-146">Laukas **Priskyrimo režimas** atnaujinamas į **Rankinis**, o vartotojams priskirtas naujas vaidmuo.</span><span class="sxs-lookup"><span data-stu-id="a3726-146">The **Assignment mode** is updated to **Manual** and the users now have a new role assigned.</span></span>
