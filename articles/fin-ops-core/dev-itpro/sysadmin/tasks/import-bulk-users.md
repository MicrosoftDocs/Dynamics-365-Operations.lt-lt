# <a name="import-users-in-bulk"></a>Masinis vartotojų importavimas

[!include [task guide banner](../../includes/task-guide-banner.md)]

Šią procedūrą gali naudoti sistemos administratoriai, norėdami importuoti didelį skaičių vartotojų iš „Azure Active Directory“.


## <a name="run-as-a-batch-job"></a>Vykdyti kaip paketinę užduotį
1. Pasirinkite Sistemos administravimas > Vartotojai > Vartotojai.
2. Spustelėkite Paketo importavimas.
3. Išplėskite dalį Vykdyti fone.
4. Lauke Paketinis vykdymas pasirinkite Taip.
5. Lauke Užduoties aprašas įrašykite reikšmę.
6. Lauke Paketo grupė įveskite arba pasirinkite vertę.
    * Tai pasirinktinis veiksmas.  
7. Lauke Privatus pasirinkite Taip.
    * Tai pasirinktinis veiksmas.  
8. Lauke Kritinis darbas pasirinkite Taip.
    * Tai pasirinktinis veiksmas.  
9. Lauke Stebėjimo kategorija pasirinkite parinktį.
10. Spustelėkite GERAI.

## <a name="run-in-a-sandbox-environment"></a>Paleiskite smėlio dėžės aplinkoje
1. Spustelėkite Paketo importavimas.
2. Spustelėkite GERAI.
