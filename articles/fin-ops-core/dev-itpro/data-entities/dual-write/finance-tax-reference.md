# <a name="access-to-finance-and-tax-reference-data"></a>Prieiga prie finansinių ir mokesčių nuorodos duomenų

[!include [banner](../../includes/banner.md)]

[!include [rename-banner](~/includes/cc-data-platform-banner.md)]

Kiekviena įmonė dirba su pagrindiniu finansinių duomenų rinkiniu, tarp šių duomenų gali būti: finansinis kalendorius; valiuta, kuria atliekamos finansinės operacijos; sąskaitos, į kurias arba iš kurių pervedami verslo veiklai vykdyti reikalingi pinigai; mokesčių tarifai ir pavedimai. Šie duomenys laikomi programėlėse Finance and Operations. Tačiau jie perduodami „Dataverse” tam, kad „Customer Engagement“ programos turėtų vieną finansinių ir mokesčių duomenų šaltinį. Tokiu būdu užtikrinamas duomenų vienodumas visoje verslo ekosistemoje.

Finansiniai ir mokesčių duomenys yra integruoti naudojant šiuos susiejimus:

+ [Integruota didžioji knyga](ledger-mapping.md)
+ [Bendrieji integruotų mokesčių duomenys](tax-mapping.md)
