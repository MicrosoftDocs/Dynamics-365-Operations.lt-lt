# <a name="list-of-er-functions-in-the-mathematical-category"></a>Matematinės kategorijos ER funkcijų sąrašas

[!include [banner](../includes/banner.md)]

Modulio Elektroninės ataskaitos (ER) matematines funkcijas galima naudoti norint atlikti daug įprastų matematinių skaičiavimų. Šiame straipsnyje pateikiama šių funkcijų suvestinė.

## <a name="list-of-supported-functions"></a>Palaikomų funkcijų sąrašas

| Funkcija | Aprašymas |
|----------|-------------|
| [Abs](er-functions-mathematical-abs.md)             | Ši funkcija kaip tipo *Realusis skaičius* reikšmę pateikia absoliučiąją nurodyto skaičiaus reikšmę (modulį). Kitaip tariant, grąžinamas skaičius be grotelių. |
| [Galia](er-functions-mathematical-power.md)         | Ši funkcija pateikia tipo *Realusis skaičius* reikšmę, nurodančią rezultatą, kai nurodytas teigiamas skaičius pakeliamas nurodytu laipsniu. |
| [Round](er-functions-mathematical-round.md)         | Ši funkcija nurodytą skaičių pateikia kaip tipo *Realusis skaičius* reikšmę, suapvalintą iki nurodyto skaičiaus po kablelio. |
| [RoundDown](er-functions-mathematical-rounddown.md) | Ši funkcija nurodytą skaičių pateikia kaip tipo *Realusis skaičius* reikšmę, suapvalintą į mažesnę pusę iki nurodyto skaičiaus po kablelio. |
| [RoundUp](er-functions-mathematical-roundup.md)     | Ši funkcija nurodytą skaičių pateikia kaip tipo *Realusis skaičius* reikšmę, suapvalintą į didesnę pusę iki nurodyto skaičiaus po kablelio. |
