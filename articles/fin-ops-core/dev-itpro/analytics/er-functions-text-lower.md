# <a name="LOWER">LOWER ER funkcija</a>

[!include [banner](../includes/banner.md)]

`LOWER` funkcija pateikia nurodytą teksto eilutę kaip *Eilutės* reikšmę, kai ji buvo konvertuota į mažąsias raides.

## <a name="syntax"></a>Sintaksė

```
LOWER (text)
```

## <a name="arguments"></a>Argumentai

`text`: *Eilutė*

*Eilutės* reikšmė, nurodanti tekstą.

## <a name="return-values"></a>Grįžties vertės

*Eilutė*

Gaunama tekstinė reikšmė.

## <a name="example"></a>Pavyzdys

`LOWER ("Sample")` grąžina **„pavyzdys“**.
