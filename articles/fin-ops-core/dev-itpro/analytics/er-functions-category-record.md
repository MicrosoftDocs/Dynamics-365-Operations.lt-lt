# <a name="list-of-er-functions-in-the-record-category"></a>Įrašų kategorijos ER funkcijų sąrašas

[!include [banner](../includes/banner.md)]

Naudojant modulio Elektroninės ataskaitos (ER) įrašų funkcijas, galima išgauti informaciją iš duomenų tipo *Konteineris (įrašas)* duomenų šaltinių bei su jais atlikti operacijas. Šiame straipsnyje pateikiama šių funkcijų suvestinė.

## <a name="list-of-supported-functions"></a>Palaikomų funkcijų sąrašas

| Funkcija | Aprašymas |
|----------|-------------|
| [NullContainer](er-functions-record-nullcontainer.md) | Ši funkcija pateikia neapibrėžtą tipo *Konteineris (įrašas)* reikšmę, kurios struktūra yra tokia pati, kaip nurodyto įrašų sąrašo arba įrašo. |
| [EmptyRecord](er-functions-record-emptyrecord.md) | Ši funkcija pateikia neapibrėžtą tipo *Konteineris (įrašas)* reikšmę, kurios struktūra yra tokia pati, kaip nurodyto įrašų sąrašo arba įrašo. |
