# <a name="UPPER">UPPER ER funkcija</a>

[!include [banner](../includes/banner.md)]

`UPPER` funkcija pateikia nurodytą teksto eilutę kaip *Eilutės* reikšmę, kai ji buvo konvertuota į didžiąsias raides.

## <a name="syntax"></a>Sintaksė

```
UPPER (text )
```

## <a name="arguments"></a>Argumentai

`text`: *Eilutė*

Tinkamas *Eilutės* tipo duomenų šaltinio maršrutas.

## <a name="return-values"></a>Grįžties vertės

*Eilutė*

Gaunama tekstinė reikšmė.

## <a name="example"></a>Pavyzdys

`UPPER ("Sample")` grąžina **„PAVYZDYS“**.
