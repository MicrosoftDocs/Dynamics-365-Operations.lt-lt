# <a name="newguid-er-function"></a>NEWGUID ER funkcija

[!include [banner](../includes/banner.md)]

Ši `NEWGUID` funkcija sugeneruoja naują visuotinai unikalų identifikatorių (GUID) ir grąžina jį kaip *[GUID](er-formula-supported-data-types-primitive.md#guid)* vertę.

## <a name="syntax"></a>Sintaksė

```vb
NEWGUID ()
```

## <a name="return-values"></a>Grįžimo vertės

*GUID*

Gauta GUID reikšmė.

## <a name="example"></a>Pavyzdys

`NEWGUID()` visada grąžina naują *GUID* vertę, pvz., **3afcf7ff-af10-ec11-b6e6-000d3a13209e**.
