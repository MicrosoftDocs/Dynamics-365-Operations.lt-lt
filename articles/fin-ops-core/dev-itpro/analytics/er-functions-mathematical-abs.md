# <a name="ABS">ABS ER funkcija</a>

[!include [banner](../includes/banner.md)]

`ABS` funkcija grąžina nurodyto skaičiaus absoliučiąją reikšmę (modulį) kaip *Bulio logikos* reikšmę. Kitaip tariant, grąžinamas skaičius be grotelių.

## <a name="syntax"></a>Sintaksė

```
ABS (number)
```

## <a name="arguments"></a>Argumentai

`number`: *Realusis*

Skaitinė reikšmė, kurios modulį norite gauti.

## <a name="return-values"></a>Grįžties vertės

*Tikrasis*

Gaunama skaitinė reikšmė.

## <a name="example"></a>Pavyzdys

`ABS (-1)` grąžina **1**.
