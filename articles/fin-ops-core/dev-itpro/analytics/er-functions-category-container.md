# <a name="list-of-er-functions-in-the-container-category"></a>Konteinerio kategorijos ER funkcijų sąrašas

[!include [banner](../includes/banner.md)]

[Elektroninės ataskaitos (ER)](general-electronic-reporting.md) konteinerio [funkcijas](er-formula-language.md#functions) galima naudoti operacijoms su duomenų tipo *Konteineris* duomenų šaltiniais. Šios operacijos įvyksta, kai apdorojimo duomenis atspindi dvejetainių duomenų rinkinį dvejetainio didelio objekto (BLOB) formatu. Šioje temoje pateikiama šių funkcijų suvestinė.

## <a name="list-of-supported-functions"></a>Palaikomų funkcijų sąrašas

| Funkcija | Aprašymas |
|----------|-------------|
| [„Base64StringToContainer”](er-functions-container-base64stringtocontainer.md) | Ši funkcija grąžina *Konteineris* vertę, kurią sudaro dvejetainis turinys, iššifruotas iš nurodytos ASCII eilutės, kuri atitinka dvejetainių-teksto kodavimo schemų „Base64” grupę. |
