[!include [task guide banner](../../includes/task-guide-banner.md)]

1. <span data-ttu-id="e375b-109">Parinktyje **Naršymo sritis** eikite į **Moduliai > Projekto valdymas ir apskaita > Grafikai > Mano grafikai**.</span><span class="sxs-lookup"><span data-stu-id="e375b-109">In the **Navigation pane**, go to **Modules > Project management and accounting > Timesheets > My timesheets**.</span></span>
2. <span data-ttu-id="e375b-110">Norėdami įvesti naują grafiką, spustelėkite **Naujas**.</span><span class="sxs-lookup"><span data-stu-id="e375b-110">To enter a new timesheet, click **New**.</span></span>
    - <span data-ttu-id="e375b-111">Pagal numatytuosius parametrus išplečiamajame sąraše Ištekliai rodomas darbuotojas, priskirtas dabartinio vartotojui.</span><span class="sxs-lookup"><span data-stu-id="e375b-111">The Resource drop-down list shows the worker assigned to the current user, by default.</span></span>  
    - <span data-ttu-id="e375b-112">Jei vartotojas yra nustatomas kaip perėmėjas, bus pateiktos pavardės, kad vartotojas pats galėtų įvesti grafiką.</span><span class="sxs-lookup"><span data-stu-id="e375b-112">If the user is designated as a delegate, this will list the names so that a user can enter a timesheet on their behalf.</span></span>  
3. <span data-ttu-id="e375b-113">Lauke **Data** įveskite datą.</span><span class="sxs-lookup"><span data-stu-id="e375b-113">In the **Date** field, enter a date.</span></span> <span data-ttu-id="e375b-114">Jei ši parinktis pasirinkta, naujos grafiko eilutės bus sukurtos naudojant grafiko parametrus, kurie buvo sukonfigūruoti kaip parankiniai.</span><span class="sxs-lookup"><span data-stu-id="e375b-114">If this option is selected, new timesheet lines will be created by using the timesheet settings that were configured as favorites.</span></span>  
4. <span data-ttu-id="e375b-115">Spustelėkite **Gerai**.</span><span class="sxs-lookup"><span data-stu-id="e375b-115">Click **OK**.</span></span>
5. <span data-ttu-id="e375b-116">Spustelėkite **Nauja eilutė**.</span><span class="sxs-lookup"><span data-stu-id="e375b-116">Click **New line**.</span></span>
6. <span data-ttu-id="e375b-117">Sąraše pažymėkite pasirinktą eilutę.</span><span class="sxs-lookup"><span data-stu-id="e375b-117">In the list, mark the selected row.</span></span> <span data-ttu-id="e375b-118">Lauke **Juridinis subjektas** rodomas dabartinis juridinis subjektas, nustatytas pagal numatytuosius nustatymus.</span><span class="sxs-lookup"><span data-stu-id="e375b-118">The **Legal Entity** field displays the current Legal entity by default.</span></span>

8. <span data-ttu-id="e375b-120">Sąraše raskite ir pasirinkite norimą įrašą.</span><span class="sxs-lookup"><span data-stu-id="e375b-120">In the list, find and select the desired record.</span></span>
9. <span data-ttu-id="e375b-121">Sąraše spustelėkite saitą pasirinktoje eilutėje.</span><span class="sxs-lookup"><span data-stu-id="e375b-121">In the list, click the link in the selected row.</span></span>
10. <span data-ttu-id="e375b-122">Lauke **Veiklos numeris** spustelėkite išplečiamąjį mygtuką, kad atidarytumėte peržvalgą.</span><span class="sxs-lookup"><span data-stu-id="e375b-122">In the **Activity number** field, click the drop-down button to open the lookup.</span></span>
11. <span data-ttu-id="e375b-123">Sąraše raskite ir pasirinkite norimą įrašą.</span><span class="sxs-lookup"><span data-stu-id="e375b-123">In the list, find and select the desired record.</span></span>
12. <span data-ttu-id="e375b-124">Sąraše spustelėkite saitą pasirinktoje eilutėje.</span><span class="sxs-lookup"><span data-stu-id="e375b-124">In the list, click the link in the selected row.</span></span>
13. <span data-ttu-id="e375b-125">Lauke **Kategorija** spustelėkite išplečiamąjį mygtuką, kad atidarytumėte peržvalgą.</span><span class="sxs-lookup"><span data-stu-id="e375b-125">In the **Category** field, click the drop-down button to open the lookup.</span></span>
14. <span data-ttu-id="e375b-126">Sąraše raskite ir pasirinkite norimą įrašą.</span><span class="sxs-lookup"><span data-stu-id="e375b-126">In the list, find and select the desired record.</span></span>
15. <span data-ttu-id="e375b-127">Sąraše spustelėkite saitą pasirinktoje eilutėje.</span><span class="sxs-lookup"><span data-stu-id="e375b-127">In the list, click the link in the selected row.</span></span>
16. <span data-ttu-id="e375b-128">Įveskite valandų skaičių, dirbtą kiekvieną dieną.</span><span class="sxs-lookup"><span data-stu-id="e375b-128">Enter the number of hours worked each day.</span></span> <span data-ttu-id="e375b-129">Įveskite valandas dešimtainių formatu.</span><span class="sxs-lookup"><span data-stu-id="e375b-129">Enter the hours in decimal format.</span></span> <span data-ttu-id="e375b-130">Pvz., jei dirbote dvi valandas ir penkiolika minčių, įveskite 2,25.</span><span class="sxs-lookup"><span data-stu-id="e375b-130">For example, if you worked for two hours and fifteen minutes, enter 2.25.</span></span>

21. <span data-ttu-id="e375b-135">Spustelėkite **Pateikti**.</span><span class="sxs-lookup"><span data-stu-id="e375b-135">Click **Submit**.</span></span>
22. <span data-ttu-id="e375b-136">Spustelėkite **Pateikti**.</span><span class="sxs-lookup"><span data-stu-id="e375b-136">Click **Submit**.</span></span>
