# <a name="set-a-users-preferred-time-zone"></a>Vartotojo pageidaujamos laiko zonos nustatymas

[!include [banner](../../includes/banner.md)]


[!INCLUDE [PEAP](../../../../includes/peap-1.md)]

Šiame straipsnyje paaiškinama, kaip sistemos administratoriaus vaidmens vartotojas gali nustatyti laiko juostą vartotojui.

1. Eikite į **Naršymo sritis > Moduliai > Sistemos administravimas > Užklausos > Paketinė užduotis**.
2. Sąraše raskite ir pasirinkite norimą įrašą.
3. Pasirinkite **Vartotojo parinktys**.
4. Pasirinkti skirtuką **Nuostatos**.
5. Lauke **Laiko juosta** pasirinkite parinktį iš išplečiamojo meniu.
6. Pasirinkite **Įrašyti**.
