# <a name="create-an-operating-unit"></a><span data-ttu-id="7f890-103">Valdymo vieneto kūrimas</span><span class="sxs-lookup"><span data-stu-id="7f890-103">Create an operating unit</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="7f890-104">Valdymo vienetas yra organizacija, kuri yra naudojama ekonominiams ištekliams valdyti ir verslo veiklos procesams skirstyti.</span><span class="sxs-lookup"><span data-stu-id="7f890-104">An operating unit is an organization that is used to divide the control of economic resources and operational processes in a business.</span></span> <span data-ttu-id="7f890-105">Valdymo vienete žmonės turi maksimaliai išnaudoti negausius išteklius, pagerinti procesus ir atsiskaityti už savo našumą.</span><span class="sxs-lookup"><span data-stu-id="7f890-105">People in an operating unit have a duty to maximize the use of scarce resources, improve processes, and account for their performance.</span></span> <span data-ttu-id="7f890-106">Į valdymo vienetų tipus įeina išlaidų centrai, verslo struktūrų vienetai, padaliniai ir vertės srautai.</span><span class="sxs-lookup"><span data-stu-id="7f890-106">The types of operating units include cost centers, business units, departments, and value streams.</span></span> <span data-ttu-id="7f890-107">Norėdami sukurti valdymo vienetą, naudokite nurodytą procedūrą.</span><span class="sxs-lookup"><span data-stu-id="7f890-107">Use the following procedure to create an operating unit.</span></span> <span data-ttu-id="7f890-108">Kuriant šią procedūrą naudojama demonstracinių duomenų įmonė yra USMF.</span><span class="sxs-lookup"><span data-stu-id="7f890-108">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="7f890-109">Eikite į **Naršymo sritis > Moduliai > Organizacijos administravimas > Organizacijos > Juridiniai subjektai**.</span><span class="sxs-lookup"><span data-stu-id="7f890-109">Go to **Navigation pane > Modules > Organization administration > Organizations > Operating units.**</span></span>
2. <span data-ttu-id="7f890-110">Spustelėdami **Naujas** atidarykite išplečiamąjį dialogo langą.</span><span class="sxs-lookup"><span data-stu-id="7f890-110">Click **New** to open the drop dialog.</span></span>
3. <span data-ttu-id="7f890-111">Sąraše raskite ir pasirinkite norimą įrašą.</span><span class="sxs-lookup"><span data-stu-id="7f890-111">In the list, find and select the desired record.</span></span> <span data-ttu-id="7f890-112">Pasirinkite valdymo vieneto, kurį norite sukurti, tipą.</span><span class="sxs-lookup"><span data-stu-id="7f890-112">Select the type of operating unit you want to create.</span></span>  
4. <span data-ttu-id="7f890-113">Sąraše spustelėkite saitą pasirinktoje eilutėje.</span><span class="sxs-lookup"><span data-stu-id="7f890-113">In the list, click the link in the selected row.</span></span>
5. <span data-ttu-id="7f890-114">Lauke **Pavadinimas** įveskite reikšmę.</span><span class="sxs-lookup"><span data-stu-id="7f890-114">In the **Name** field, type a value.</span></span>
    + <span data-ttu-id="7f890-115">Jei reikia, išplėskite skyrių **Bendra**.</span><span class="sxs-lookup"><span data-stu-id="7f890-115">Expand the **General** section, if necessary.</span></span>  
    + <span data-ttu-id="7f890-116">Pateikite bendrąją informaciją apie valdymo vienetą, pvz., identifikacijos numerį, DUNS numerį ir vadovą.</span><span class="sxs-lookup"><span data-stu-id="7f890-116">Provide general information about the operating unit, such as an identification number, DUNS number, and manager.</span></span>    
    + <span data-ttu-id="7f890-117">Jei reikia, išplėskite skyrių **Adresai**.</span><span class="sxs-lookup"><span data-stu-id="7f890-117">Expand the **Addresses** section, if necessary.</span></span>  
    + <span data-ttu-id="7f890-118">Įveskite adreso informaciją, pvz., gatvės pavadinimą ir numerį, pašto kodą ir miestą.</span><span class="sxs-lookup"><span data-stu-id="7f890-118">Enter address information, such as the street name and number, postal code, and city.</span></span> <span data-ttu-id="7f890-119">Spustelėkite **Įtraukti**, kad įvestumėte naują adreso įrašą, arba spustelėkite „Redaguoti“, kad modifikuotumėte esamą adreso įrašą.</span><span class="sxs-lookup"><span data-stu-id="7f890-119">Click **Add** to enter a new address record, or click Edit to modify an existing address record.</span></span>   
    + <span data-ttu-id="7f890-120">Jei reikia, išplėskite skyrių **Kontaktinė informacija**.</span><span class="sxs-lookup"><span data-stu-id="7f890-120">Expand the **Contact information** section, if necessary.</span></span>  
    + <span data-ttu-id="7f890-121">Įveskite informaciją apie komunikacijos priemones, pvz., elektroninio pašto adresus, URL ir telefono numerius.</span><span class="sxs-lookup"><span data-stu-id="7f890-121">Enter information about methods of communication, such as email addresses, URLs, and telephone numbers.</span></span> <span data-ttu-id="7f890-122">Norėdami įvesti naują komunikacijos įrašą, spustelėkite „Naujas“.</span><span class="sxs-lookup"><span data-stu-id="7f890-122">To enter a new communication record, click New.</span></span> <span data-ttu-id="7f890-123">Norėdami modifikuoti esamą komunikacijos įrašą, spustelėkite **Daugiau parinkčių > Išplėstiniai nustatymai**.</span><span class="sxs-lookup"><span data-stu-id="7f890-123">To modify an existing communication record, click **More options > Advanced**.</span></span>   
6. <span data-ttu-id="7f890-124">Arba, jei reikia, pakeiskite **Valdymo vieneto numeris**.</span><span class="sxs-lookup"><span data-stu-id="7f890-124">Optionally, change the **Operating unit number** as needed.</span></span> <span data-ttu-id="7f890-125">Atkreipkite dėmesį, kad šis numeris yra atitinkamos **Šalies** unikalus identifikatorius ir negali būti toks pats, kaip bet kuris kitas valdymo vienetas.</span><span class="sxs-lookup"><span data-stu-id="7f890-125">Note that this number is a unique idenitifier for the correspondng **Party** record and cannot be the same as any other operating unit.</span></span>
7. <span data-ttu-id="7f890-126">Pasirinkite **Įrašyti**.</span><span class="sxs-lookup"><span data-stu-id="7f890-126">Select **Save**.</span></span>
