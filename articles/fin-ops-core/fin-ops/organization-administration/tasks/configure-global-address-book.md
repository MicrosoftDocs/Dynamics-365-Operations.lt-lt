# <a name="configure-the-global-address-book"></a><span data-ttu-id="0d611-103">Visuotinės adresų knygelės konfigūravimas</span><span class="sxs-lookup"><span data-stu-id="0d611-103">Configure the global address book</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="0d611-104">Naudokite šią procedūrą norėdami nustatyti numatytąsias vertes ir visuotinės adresų knygelės saugos strategijas.</span><span class="sxs-lookup"><span data-stu-id="0d611-104">Use this procedure to set the default values and security policies for the global address book.</span></span> 

<span data-ttu-id="0d611-105">Kuriant šią užduotį naudojama demonstracinių duomenų įmonė yra USMF.</span><span class="sxs-lookup"><span data-stu-id="0d611-105">The demo data company used to create this task is USMF.</span></span> <span data-ttu-id="0d611-106">Ši užduotis yra skirta planavimo ir konfigūracijos komandai.</span><span class="sxs-lookup"><span data-stu-id="0d611-106">This task is intended for the Planning and configuration team.</span></span>

1. <span data-ttu-id="0d611-107">Eikite į **Moduliai > Organizacijos administravimas > Visuotinė adresų knygelė > Visuotinės adresų knygelės parametrai**.</span><span class="sxs-lookup"><span data-stu-id="0d611-107">In the Navigation pane, go to **Modules > Organization administration > Global address book > Global address book parameters**.</span></span>
2. <span data-ttu-id="0d611-108">Lauke **Pavadinimo seka** pažymėkite, kaip turi būti rodomi pavadinimai.</span><span class="sxs-lookup"><span data-stu-id="0d611-108">In the **Name sequence** field, select how names should be shown.</span></span>
3. <span data-ttu-id="0d611-109">Lauke **Naikinti įrašus, kurie neturi vaidmenų** pasirinkite, ar naikinti įrašus, kuriems nebuvo priskirtas vaidmuo.</span><span class="sxs-lookup"><span data-stu-id="0d611-109">In **Delete parties with no roles**, select whether to delete parties with that have not been assigned a role.</span></span>
4. <span data-ttu-id="0d611-110">Lauke **Naudoti dublikatų tikrinimą** pasirinkite, ar norite patikrinti pasikartojančius įrašus.</span><span class="sxs-lookup"><span data-stu-id="0d611-110">In **Use duplicate check**, select whether to check for duplicate records.</span></span>
5. <span data-ttu-id="0d611-111">Lauke **Rodyti DUNS numerį adresuose** pasirinkite, ar norite rodyti adresų DUNS numerį.</span><span class="sxs-lookup"><span data-stu-id="0d611-111">In **Display DUNS number on addresses**, select whether to display the DUNS number on addresses.</span></span>
6. <span data-ttu-id="0d611-112">Lauke **Tikrinti unikalųjį DUNS numerį** pasirinkite, ar norite patikrinti unikalius DUNS numerius.</span><span class="sxs-lookup"><span data-stu-id="0d611-112">In **Check for unique DUNS number**, select whether to check for unique DUNS numbers.</span></span>
7. <span data-ttu-id="0d611-113">Lauke **Šalis** pasirinkite parinktį.</span><span class="sxs-lookup"><span data-stu-id="0d611-113">In the **Party** field, select an option.</span></span>
8. <span data-ttu-id="0d611-114">Lauke **Klientas** pasirinkite parinktį.</span><span class="sxs-lookup"><span data-stu-id="0d611-114">In the **Customer** field, select an option.</span></span>
9. <span data-ttu-id="0d611-115">Lauke **Tiekėjas** pasirinkite parinktį.</span><span class="sxs-lookup"><span data-stu-id="0d611-115">In the **Vendor** field, select an option.</span></span>
10. <span data-ttu-id="0d611-116">Lauke **Potencialus klientas** pasirinkite parinktį.</span><span class="sxs-lookup"><span data-stu-id="0d611-116">In the **Prospect** field, select an option.</span></span>
11. <span data-ttu-id="0d611-117">Lauke **Konkurentas** pasirinkite parinktį.</span><span class="sxs-lookup"><span data-stu-id="0d611-117">In the **Competitor** field, select an option.</span></span>
12. <span data-ttu-id="0d611-118">Spustelėkite skirtuką **Privačios vietos sauga**.</span><span class="sxs-lookup"><span data-stu-id="0d611-118">Click the **Private location security** tab.</span></span>
13. <span data-ttu-id="0d611-119">Sąraše raskite ir pasirinkite norimą įrašą.</span><span class="sxs-lookup"><span data-stu-id="0d611-119">In the list, find and select the desired record.</span></span> <span data-ttu-id="0d611-120">Paspauskite klavišą Shift, jei norite pasirinkti kelis vaidmenis ir įtraukti į sritį **Pasirinkti vaidmenys**, tada spustelėkite rodyklę ir įtraukite pasirinktus vaidmenis.</span><span class="sxs-lookup"><span data-stu-id="0d611-120">Press the Shift key to select multiple roles to add to the **Selected roles** pane and then click the arrow to add the selected roles.</span></span>  
14. <span data-ttu-id="0d611-121">Spustelėkite **Įrašyti**.</span><span class="sxs-lookup"><span data-stu-id="0d611-121">Click **Save**.</span></span>
