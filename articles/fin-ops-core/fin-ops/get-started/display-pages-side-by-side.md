# <a name="show-pages-side-by-side-using-the-open-in-new-window-feature"></a><span data-ttu-id="e4693-103">Vieno šalia kito puslapių rodymas naudojant funkciją Atidaryti naujame lange</span><span class="sxs-lookup"><span data-stu-id="e4693-103">Show pages side-by-side using the Open in new window feature</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="e4693-104">Šiame straipsnyje paaiškinta, kaip puslapius rodyti vieną šalia kito.</span><span class="sxs-lookup"><span data-stu-id="e4693-104">This article explains how to display pages side by side.</span></span>

<span data-ttu-id="e4693-105">Tam, kad greitai atliktumėte užduotis, galbūt norėsite kelis puslapius peržiūrėti vieną šalia kito.</span><span class="sxs-lookup"><span data-stu-id="e4693-105">You may want to view multiple pages side by side to complete tasks quickly.</span></span> <span data-ttu-id="e4693-106">Pavyzdžiui, galbūt norėsite patikrinti eilutes arba jų įvesti daugiau nei viename žurnale.</span><span class="sxs-lookup"><span data-stu-id="e4693-106">As an example, you might want to validate or enter lines in more than one journal.</span></span> <span data-ttu-id="e4693-107">Paprastai, norėdami eilutes patikrinti ar įvesti daugiau nei viename žurnale, turėtumėte eiti pirmyn ir atgal tarp puslapio, kuriame rodomas žurnalų sąrašas ir puslapio, kuriame rodomos nurodyto žurnalo eilutės.</span><span class="sxs-lookup"><span data-stu-id="e4693-107">Typically, to validate or enter lines in more than one journal, you would have to go back and forth between the page that displays a list of journals, and the page that displays lines for a given journal.</span></span> <span data-ttu-id="e4693-108">Tačiau funkcija **Atidaryti naujame lange** leidžia šiuos puslapius rodyti vieną šalia kito, kad galėtumėte greitai atlikti savo užduotis.</span><span class="sxs-lookup"><span data-stu-id="e4693-108">However, the **Open in new window** feature enables you to display these pages side-by-side so that you can perform your tasks quickly.</span></span>

<span data-ttu-id="e4693-109">Tęsiant pirmiau minėtą pavyzdį, peržiūrėdami eilutes, galite spustelėti piktogramą **Atidaryti naujame lange**.</span><span class="sxs-lookup"><span data-stu-id="e4693-109">Continuing with the example mentioned above, when viewing the lines, you can click the **Open in new window** icon.</span></span>

<span data-ttu-id="e4693-110">[![Spustelėkite piktogramą Atidaryti naujame lange.](./media/open-in-new-window-icon.png)](./media/open-in-new-window-icon.png)</span><span class="sxs-lookup"><span data-stu-id="e4693-110">[![Click the Open in new window icon.](./media/open-in-new-window-icon.png)](./media/open-in-new-window-icon.png)</span></span>

<span data-ttu-id="e4693-111">Spustelėjus piktogramą **Atidaryti naujame lange**, eilučių puslapis atidaromas naujoje iššokančioje naršyklėje, o pradinėje naršyklėje per istoriją grįžtama į puslapį, kuriame buvo rodomas žurnalų sąrašas.</span><span class="sxs-lookup"><span data-stu-id="e4693-111">Clicking the **Open in new window** icon opens the lines page in a new, pop-up browser, and then navigates the original browser back in history to the page that displayed the list of journals.</span></span> <span data-ttu-id="e4693-112">Tada jūs galite puslapius rodyti vieną šalia kito.</span><span class="sxs-lookup"><span data-stu-id="e4693-112">You can then display both pages side by side.</span></span> <span data-ttu-id="e4693-113">Peržiūrėję žurnalą, galite pakeisti pasirinktą žurnalą iš žurnalų sąrašo puslapio ir iššokusio lango eilučių puslapyje automatiškai bus rodomos naujai pasirinkto žurnalo eilutės.</span><span class="sxs-lookup"><span data-stu-id="e4693-113">After viewing a journal, you can change the selected journal on the journal list page, and the lines page in the pop-up window will automatically display the lines of the newly selected journal.</span></span>

<span data-ttu-id="e4693-114">[![Galite puslapius rodyti vieną šalia kito.](./media/pages-show-side-by-side.png)](./media/pages-show-side-by-side.png)</span><span class="sxs-lookup"><span data-stu-id="e4693-114">[![You can display pages side-by-side.](./media/pages-show-side-by-side.png)](./media/pages-show-side-by-side.png)</span></span>

<span data-ttu-id="e4693-115">Dinamiškai susiejama ir atnaujinama dėl ryšių, esančių tarp duomenų, kurie naudojami šiuose puslapiuose.</span><span class="sxs-lookup"><span data-stu-id="e4693-115">The dynamic linking and refreshing happens due to the relations that exist between the data that is backing these pages.</span></span> <span data-ttu-id="e4693-116">Jei sistema nežino apie ryšį tarp duomenų, lange atlikus pakeitimą, iš jo iššokęs langas automatiškai neatsinaujins.</span><span class="sxs-lookup"><span data-stu-id="e4693-116">If the system is not aware of the relation between the data, the pop-up window will not refresh automatically in response to a change in the window it originated from.</span></span>

<span data-ttu-id="e4693-117">Kai kuriuose puslapiuose yra keli rodiniai, pvz., Tinklelio rodinys, Antraštės rodinys ir Išsamios informacijos rodinys.</span><span class="sxs-lookup"><span data-stu-id="e4693-117">Some pages have multiple views such as the Grid view, Header view, and Details view.</span></span> <span data-ttu-id="e4693-118">Dėl **Atidaryti naujame lange** piktogramos, naujame naršyklės lange bus atidarytas visas puslapis.</span><span class="sxs-lookup"><span data-stu-id="e4693-118">The **Open in new window** icon causes the entire page to open in the new browser window.</span></span> <span data-ttu-id="e4693-119">Todėl, naudodami **Atidaryti naujame lange** funkciją, negalite dviejų to paties puslapio rodinių laikyti vieną šalia kito.</span><span class="sxs-lookup"><span data-stu-id="e4693-119">Therefore, you cannot keep two views of the same page side by side using the **Open in new window** feature.</span></span> <span data-ttu-id="e4693-120">Beveik visuose tokiuose puslapiuose yra naršymo sąrašas, kurį naudodami galite perjungti įrašus ir pasiekti panašią patirtį.</span><span class="sxs-lookup"><span data-stu-id="e4693-120">Almost all such pages have a navigation list that you can use to switch between records and achieve a similar experience.</span></span>

<span data-ttu-id="e4693-121">Prieš naudodami funkciją **Atidaryti naujame lange**, turėtumėte sukonfigūruoti naršyklės iššokančiųjų langų blokavimo funkciją, kad ji leistų iššokančiuosius langus iš svetainės URL.</span><span class="sxs-lookup"><span data-stu-id="e4693-121">Before using the **Open in new window** feature, you should configure your browser's pop-up blocker to allow pop-ups from the URL of the site.</span></span> <span data-ttu-id="e4693-122">Pvz., galite leisti iššokančiuosius langus iš \*.dynamics.com.</span><span class="sxs-lookup"><span data-stu-id="e4693-122">As an example, you could allow pop-ups from "\*.dynamics.com".</span></span>

<span data-ttu-id="e4693-123">Funkciją **Atidaryti naujame lange** galima naudoti tik kai lange atidarytas daugiau nei vienas puslapis.</span><span class="sxs-lookup"><span data-stu-id="e4693-123">The **Open in new window** feature is only available when there is more than one page open in the window.</span></span> <span data-ttu-id="e4693-124">Be to, kai nebėra atidarytų puslapių (tai yra, kai uždarote to lango paskutinį puslapį), iššokantysis langas automatiškai užsidaro.</span><span class="sxs-lookup"><span data-stu-id="e4693-124">Also, the pop-up window automatically closes when there are no more pages open (that is, when you close the last page in that window).</span></span> <span data-ttu-id="e4693-125">Sistema taip pat uždaro atidarytus puslapius, kai pereinate į kitą programos sritį.</span><span class="sxs-lookup"><span data-stu-id="e4693-125">The system also closes open pages when you navigate to a different area in the application.</span></span> <span data-ttu-id="e4693-126">Todėl, jei yra atidarytų iššokančiųjų langų ir jūs pereinate į kitą programos sritį, iššokantieji langai uždaromi automatiškai, nes sistema uždarė tų langų puslapius.</span><span class="sxs-lookup"><span data-stu-id="e4693-126">Therefore, if you have pop-up windows open and navigate to a different area in the application, the pop-up windows close automatically because the system closed the pages in those windows.</span></span>

<span data-ttu-id="e4693-127">Iššokančiųjų langų viršutinėje juostoje rodoma informacija apie įmonę, kurioje puslapis buvo atidarytas, ir juostą galima tik skaityti.</span><span class="sxs-lookup"><span data-stu-id="e4693-127">The top bar in the pop-up windows displays information about the company the page was opened in and is read-only.</span></span> <span data-ttu-id="e4693-128">Iššokantieji langai taip pat priklauso nuo pagrindinio naršyklės lango.</span><span class="sxs-lookup"><span data-stu-id="e4693-128">The pop-up windows also rely on the main browser window.</span></span> <span data-ttu-id="e4693-129">Jei pagrindinis langas uždaromas ar atnaujinamas, visus atidarytus iššokančiuosius langus bus galima tik skaityti.</span><span class="sxs-lookup"><span data-stu-id="e4693-129">If the main window is closed or refreshed, all open pop-up windows will become read only.</span></span> <span data-ttu-id="e4693-130">Jei susiklosto tokia situacija, tai reiškia, kad vis dar galite peržiūrėti tų langų informaciją, bet negalėsite su ja sąveikauti.</span><span class="sxs-lookup"><span data-stu-id="e4693-130">If this situation occurs, you can still view the information in these windows, but you will not be able to interact with it.</span></span>
