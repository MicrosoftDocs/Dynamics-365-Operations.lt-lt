# <a name="change-the-banner-or-logo"></a><span data-ttu-id="23ddd-103">Keisti reklaminę juostą arba logotipą</span><span class="sxs-lookup"><span data-stu-id="23ddd-103">Change the banner or logo</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="23ddd-104">Toliau pateiktoje procedūroje išvardijami veiksmai, kuriuos sistemos administratoriai gali naudoti, norėdami atnaujinti reklaminės juostos arba logotipo vaizdą, kuris rodomas juridiniam subjektui.</span><span class="sxs-lookup"><span data-stu-id="23ddd-104">The following procedure lists the steps that system administrators can use to update the banner or logo image that is displayed for a legal entity.</span></span> <span data-ttu-id="23ddd-105">Kuriant šią procedūrą naudojama demonstracinių duomenų įmonė yra USMF.</span><span class="sxs-lookup"><span data-stu-id="23ddd-105">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="23ddd-106">Eikite į **Naršymo sritis > Moduliai > Organizacijos administravimas > Organizacijos > Juridiniai subjektai**.</span><span class="sxs-lookup"><span data-stu-id="23ddd-106">Go to **Navigation pane > Modules > Organization administration > Organizations > Legal entities**.</span></span>
2. <span data-ttu-id="23ddd-107">Sąrašo kairėje pusėje pasirinkite juridinį subjektą, kurio reklaminę juostą arba logotipą norite naujinti.</span><span class="sxs-lookup"><span data-stu-id="23ddd-107">In the list on the left, select the legal entity for which you want to update the banner or logo.</span></span> <span data-ttu-id="23ddd-108">Jei vaizdas jau pasirinktas, pereikite prie kito veiksmo.</span><span class="sxs-lookup"><span data-stu-id="23ddd-108">If it is already selected, go to the next step.</span></span>
3. <span data-ttu-id="23ddd-109">Išplėskite skirtuką **Ataskaitų srities vaizdas**.</span><span class="sxs-lookup"><span data-stu-id="23ddd-109">Expand **Dashboard image** tab.</span></span>
4. <span data-ttu-id="23ddd-110">Pakeisti **Keisti**.</span><span class="sxs-lookup"><span data-stu-id="23ddd-110">Select **Change**.</span></span>
    
    <span data-ttu-id="23ddd-111">Tinkamiausia reklaminės juostos vaizdo skiriamoji geba yra 1920 x 281 pikselių.</span><span class="sxs-lookup"><span data-stu-id="23ddd-111">The ideal resolution for a banner image is 1920 x 281 pixels.</span></span> <span data-ttu-id="23ddd-112">Tinkamiausias logotipo vaizdo plotis yra 350 pikselių.</span><span class="sxs-lookup"><span data-stu-id="23ddd-112">The ideal width for a logo image is 350 pixels.</span></span>
    
5. <span data-ttu-id="23ddd-113">Pasirinkite **Įrašyti**.</span><span class="sxs-lookup"><span data-stu-id="23ddd-113">Select **Save**.</span></span>
6. <span data-ttu-id="23ddd-114">Eikite į **Naršymo sritis > Moduliai > Bendras > Bendras > Numatytoji ataskaitų sritis**.</span><span class="sxs-lookup"><span data-stu-id="23ddd-114">Go to **Navigation pane > Modules > Common > Common > Default dashboard**.</span></span> <span data-ttu-id="23ddd-115">Naujas reklaminės juostos arba logotipo vaizdas turėtų būti rodomas ataskaitų srityje.</span><span class="sxs-lookup"><span data-stu-id="23ddd-115">You should see the new banner or logo image on the dashboard.</span></span>  
    
    <span data-ttu-id="23ddd-116">Jums gali reikėti pakeisti įmonę naudojant įmonės parinkiklį į tą, kurios reklaminę juostą nusiuntėte.</span><span class="sxs-lookup"><span data-stu-id="23ddd-116">You may need to change your company, using the company picker, to the one you uploaded the banner for.</span></span>
