[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="a2e8c-104">Kartais laisvas darbo vietas gali būti užimti kandidatai, kurie jau yra jūsų organizacijos darbuotojai.</span><span class="sxs-lookup"><span data-stu-id="a2e8c-104">Sometimes open positions can be filled by candidates who are already employees in your organization.</span></span> <span data-ttu-id="a2e8c-105">Ši procedūra padeda žingsnis po žingsnio atlikti esamo darbuotojo samdymo per įdarbinimo procesą veiksmus.</span><span class="sxs-lookup"><span data-stu-id="a2e8c-105">This procedure walks through the steps of hiring an existing employee through the recruiting process.</span></span> <span data-ttu-id="a2e8c-106">Šioje procedūroje įdarbinimo projektas jau nustatytas, o esamas darbuotojas jau pateikė prašymą dalyvauti įdarbinimo projekte.</span><span class="sxs-lookup"><span data-stu-id="a2e8c-106">In this procedure, a recruitment project has already been set up, and an existing employee has already submitted an application for the recruitment project.</span></span> <span data-ttu-id="a2e8c-107">Kuriant šią procedūrą naudojama demonstracinių duomenų įmonė yra USMF.</span><span class="sxs-lookup"><span data-stu-id="a2e8c-107">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="a2e8c-108">Pasirinkite Personalas > Įdarbinimas > Prašymai > Prašymai.</span><span class="sxs-lookup"><span data-stu-id="a2e8c-108">Go to Human resources > Recruitment > Applications > Applications.</span></span>

3. <span data-ttu-id="a2e8c-111">Spustelėkite „Prašymo būsena“.</span><span class="sxs-lookup"><span data-stu-id="a2e8c-111">Click Application status.</span></span>
    * <span data-ttu-id="a2e8c-112">Prašymo būsena rodo, kokiame įdarbinimo proceso etape yra šis prašymas.</span><span class="sxs-lookup"><span data-stu-id="a2e8c-112">The application status indicates where an application is at in the recruitment process.</span></span>  <span data-ttu-id="a2e8c-113">Nė vienas iš šių veiksmų nėra būtinas.</span><span class="sxs-lookup"><span data-stu-id="a2e8c-113">Each of these steps is optional.</span></span> <span data-ttu-id="a2e8c-114">Paprastai prašymo būsenos keistųsi tokia tvarka: gauta, patvirtinta ir pokalbis.</span><span class="sxs-lookup"><span data-stu-id="a2e8c-114">Typically, an application would move the statuses in the following order:  received, confirmed, and interviewed.</span></span> <span data-ttu-id="a2e8c-115">Po pokalbio proceso būtų priimtas sprendimas dėl įdarbinimo.</span><span class="sxs-lookup"><span data-stu-id="a2e8c-115">After the interview process, a hiring decision would be made.</span></span>  
4. <span data-ttu-id="a2e8c-116">Spustelėkite „Pakeisti pareigas“.</span><span class="sxs-lookup"><span data-stu-id="a2e8c-116">Click Change position.</span></span>
5. <span data-ttu-id="a2e8c-117">Pasirinkite pareigas, kurioms samdote darbuotoją.</span><span class="sxs-lookup"><span data-stu-id="a2e8c-117">Select the position that you are hiring the employee into.</span></span>
6. <span data-ttu-id="a2e8c-118">Lauke „Naujo priskyrimo pradžios data“ įveskite datą, nuo kurios darbuotojas pradės dirbti naujose pareigose.</span><span class="sxs-lookup"><span data-stu-id="a2e8c-118">In the New Assignment Start Date field, enter the date that the employee will begin working in the new position.</span></span>

    * <span data-ttu-id="a2e8c-120">Naujų pareigų pradžios data ir senų pareigų pabaigos data gali persidengti.</span><span class="sxs-lookup"><span data-stu-id="a2e8c-120">The starting date for the new position and the ending date of the old position may overlap.</span></span> <span data-ttu-id="a2e8c-121">Tai gali įvykti, kai asmuo eina abi pareigas vykstant pereinamajam laikotarpiui.</span><span class="sxs-lookup"><span data-stu-id="a2e8c-121">This can happen when a person is performing duties for both positions during a transition period.</span></span>

9. <span data-ttu-id="a2e8c-124">Spustelėkite „Pakeisti pareigas“.</span><span class="sxs-lookup"><span data-stu-id="a2e8c-124">Click Change position.</span></span>
    * <span data-ttu-id="a2e8c-125">Šiuo metu taip pat galite pakeisti atlyginimą.</span><span class="sxs-lookup"><span data-stu-id="a2e8c-125">You may also change compensation at this time.</span></span> <span data-ttu-id="a2e8c-126">Jei šiuo metu atlyginimo nepriskirsite, galite jį pakeisti darbuotojo formoje atidarę skirtuką „Atlyginimas“ ir pasirinkę „Fiksuotas planas“.</span><span class="sxs-lookup"><span data-stu-id="a2e8c-126">If you do not assign compensation at this time, you can change it by going to the worker form, selecting the Compensation tab, and choosing 'Fixed Plan'.</span></span> <span data-ttu-id="a2e8c-127">Kai pažymėsite „Keisti pareigas“, prašymo būsena bus atnaujinta į „Įdarbintas“.</span><span class="sxs-lookup"><span data-stu-id="a2e8c-127">After you select 'Change position', the status on the application will be updated to 'Employed'.</span></span>
