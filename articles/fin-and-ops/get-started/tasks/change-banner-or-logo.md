# <a name="change-the-banner-or-logo"></a><span data-ttu-id="f2c4f-103">Keisti reklaminę juostą arba logotipą</span><span class="sxs-lookup"><span data-stu-id="f2c4f-103">Change the banner or logo</span></span>

[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="f2c4f-104">Toliau pateiktoje procedūroje išvardijami veiksmai, kuriuos sistemos administratoriai gali naudoti, norėdami atnaujinti reklaminės juostos arba logotipo vaizdą, kuris rodomas juridiniam subjektui.</span><span class="sxs-lookup"><span data-stu-id="f2c4f-104">The following procedure lists the steps that system administrators can use to update the banner or logo image that is displayed for a legal entity.</span></span> <span data-ttu-id="f2c4f-105">Kuriant šią procedūrą naudojama demonstracinių duomenų įmonė yra USMF.</span><span class="sxs-lookup"><span data-stu-id="f2c4f-105">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="f2c4f-106">Pasirinkite Organizacijos administravimas > Organizacijos > Juridiniai subjektai.</span><span class="sxs-lookup"><span data-stu-id="f2c4f-106">Go to Organization administration > Organizations > Legal entities.</span></span>
2. <span data-ttu-id="f2c4f-107">Sąrašo kairėje pusėje pasirinkite juridinį subjektą, kurio reklaminę juostą arba logotipą norite naujinti.</span><span class="sxs-lookup"><span data-stu-id="f2c4f-107">In the list on the left, select the legal entity for which you want to update the banner or logo.</span></span> <span data-ttu-id="f2c4f-108">Jei vaizdas jau pasirinktas, pereikite prie kito veiksmo.</span><span class="sxs-lookup"><span data-stu-id="f2c4f-108">If it is already selected, go to the next step.</span></span>
3. <span data-ttu-id="f2c4f-109">Išplėskite sekciją Vaizdai.</span><span class="sxs-lookup"><span data-stu-id="f2c4f-109">Expand the Images section.</span></span> <span data-ttu-id="f2c4f-110">Jei ji jau išplėsta, pereikite prie kito veiksmo.</span><span class="sxs-lookup"><span data-stu-id="f2c4f-110">If it is already expanded, go to the next step.</span></span>
4. <span data-ttu-id="f2c4f-111">Spustelėkite Redaguoti.</span><span class="sxs-lookup"><span data-stu-id="f2c4f-111">Click Edit.</span></span>
5. <span data-ttu-id="f2c4f-112">Lauke Ataskaitų srities įmonės vaizdo tipas pasirinkite „Reklaminė juosta“.</span><span class="sxs-lookup"><span data-stu-id="f2c4f-112">In the Dashboard company image type field, select 'Banner'.</span></span>
6. <span data-ttu-id="f2c4f-113">Spustelėkite Keisti.</span><span class="sxs-lookup"><span data-stu-id="f2c4f-113">Click Change.</span></span>
    * <span data-ttu-id="f2c4f-114">Patarimas: tinkamiausia reklaminės juostos vaizdo skiriamoji geba yra 1920 x 281 pikselių.</span><span class="sxs-lookup"><span data-stu-id="f2c4f-114">Tip: The ideal resolution for a banner image is 1920 x 281 pixels.</span></span> <span data-ttu-id="f2c4f-115">Tinkamiausias logotipo vaizdo plotis yra 350 pikselių.</span><span class="sxs-lookup"><span data-stu-id="f2c4f-115">The ideal width for a logo image is 350 pixels.</span></span>  
    * <span data-ttu-id="f2c4f-116">Kai nusiuntimas baigtas, naujas reklaminės juostos arba logotipo vaizdas turėtų būti rodomas sekcijoje Ataskaitų srities vaizdas.</span><span class="sxs-lookup"><span data-stu-id="f2c4f-116">When the upload is complete, you should see your new banner or logo image in the Dashboard Image section.</span></span>  
7. <span data-ttu-id="f2c4f-117">Spustelėkite Įrašyti.</span><span class="sxs-lookup"><span data-stu-id="f2c4f-117">Click Save.</span></span>
8. <span data-ttu-id="f2c4f-118">Pasirinkite Bendra > Bendra > Numatytoji ataskaitų sritis.</span><span class="sxs-lookup"><span data-stu-id="f2c4f-118">Go to Common > Common > Default dashboard.</span></span>
    * <span data-ttu-id="f2c4f-119">Naujas reklaminės juostos arba logotipo vaizdas turėtų būti rodomas ataskaitų srityje.</span><span class="sxs-lookup"><span data-stu-id="f2c4f-119">You should see the new banner or logo image on the dashboard.</span></span>  
    * <span data-ttu-id="f2c4f-120">Jums gali reikėti pakeisti įmonę naudojant įmonės parinkiklį į tą, kurios reklaminę juostą nusiuntėte.</span><span class="sxs-lookup"><span data-stu-id="f2c4f-120">You may need to change your company, using the company picker, to the one you uploaded the banner for.</span></span>
