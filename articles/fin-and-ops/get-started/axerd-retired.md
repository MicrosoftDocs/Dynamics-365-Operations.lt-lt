# <a name="axerd-retired"></a>AxErd panaikintas

Deja, teko panaikinti AxErd tinklalapį, kuriame buvo pateiktos daugelio „Microsoft Dynamics AX 2012“ lentelių objektų ryšių diagramos (ERD). Mums pavyko užregistruoti projekto šaltinio kodą ir tinklalapio HTML failus „GitHub“ saugykloje: https://github.com/Microsoft/ax-2012-doc-tools

HTML puslapiai laikomi saugykloje adresu [https://github.com/Microsoft/ax-2012-doc-tools/tree/master/AxErd/AxErd/_RelatedFiles_AxErd/Deploy/Default.htm](https://github.com/Microsoft/ax-2012-doc-tools/tree/master/AxErd/AxErd/_RelatedFiles_AxErd/Deploy/Default.htm)

Aprašymas, kaip sukurti savo ERD, pateiktas šiuo adresu: [https://github.com/Microsoft/ax-2012-doc-tools/blob/master/AxErd/AxErd/_RelatedFiles_AxErd/Deploy/Help-HowToUse-AxErd.htm](https://github.com/Microsoft/ax-2012-doc-tools/blob/master/AxErd/AxErd/_RelatedFiles_AxErd/Deploy/Help-HowToUse-AxErd.htm)

Sprendimas yra šakniniame AxErd aplanke [https://github.com/Microsoft/ax-2012-doc-tools/tree/master/AxErd](https://github.com/Microsoft/ax-2012-doc-tools/tree/master/AxErd).
