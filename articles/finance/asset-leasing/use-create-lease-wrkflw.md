# <a name="use-lease-approval-workflows"></a><span data-ttu-id="71fd2-103">Nuomos patvirtinimo darbo eigų naudojimas</span><span class="sxs-lookup"><span data-stu-id="71fd2-103">Use lease approval workflows</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="71fd2-104">Šioje temoje paaiškinama, kaip naudoti darbo eigas, norint patvirtinti turto nuomą ir kaip stebėti darbo eigos būseną ir retrospektyvą.</span><span class="sxs-lookup"><span data-stu-id="71fd2-104">This topic explains how to use workflows to approve asset leases, and how to track the status and history of the workflows.</span></span> <span data-ttu-id="71fd2-105">Darbo eigos padeda suderinti nuomos patvirtinimų valdymą, suteikiant standartinį patvirtinimo priemonių rinkinį ir priskiriant konkrečius vartotojus, tvirtinančius kiekvieną proceso žingsnį.</span><span class="sxs-lookup"><span data-stu-id="71fd2-105">Workflows help bring consistency to the management of lease approvals by providing a standard set of approval steps and assigning specific users who approve each step of the process.</span></span> <span data-ttu-id="71fd2-106">Tvirtintojas gali patvirtinti nuomą, jį atmesti, prašyti pakeisti arba priskirti jį kitam vartotojui patvirtinti.</span><span class="sxs-lookup"><span data-stu-id="71fd2-106">An approver can approve a lease, reject it, request a change to it, or assign it to another user for approval.</span></span> <span data-ttu-id="71fd2-107">Be to, darbo eigos gali būti geriau matomos patvirtinimo procese, nes jūs sekate savo būseną ir retrospektyvą.</span><span class="sxs-lookup"><span data-stu-id="71fd2-107">Workflows can also bring more visibility into the approval process by letting you track their status and history.</span></span> <span data-ttu-id="71fd2-108">Be to, galite peržiūrėti centralizuotą užduočių sąrašą, kuriame išvardijamos užduotys ir patvirtinimai, priskiriami konkretiems tvirtintojams.</span><span class="sxs-lookup"><span data-stu-id="71fd2-108">Additionally, you can view a centralized worklist that lists the tasks and approvals that are assigned to specific approvers.</span></span>

<span data-ttu-id="71fd2-109">Prieš pradėdami naudoti šią procedūrą, įsitikinkite, kad buvo sukurta bent jau nuomos patvirtinimo darbo eiga.</span><span class="sxs-lookup"><span data-stu-id="71fd2-109">Before you use this procedure, make sure that at least on lease approval workflow has been created.</span></span> <span data-ttu-id="71fd2-110">Jei nėra darbo eigos, sukurkite ją.</span><span class="sxs-lookup"><span data-stu-id="71fd2-110">If no workflow exists, create one.</span></span> <span data-ttu-id="71fd2-111">Informacijos apie tai, kaip nustatyti darbo eigą, ieškokite [Nuomos tvirtinimo darbo eigų nustatymas](set-up-lease-wrkflw.md).</span><span class="sxs-lookup"><span data-stu-id="71fd2-111">For information about how to set up a workflow, see [Set up lease approval workflows](set-up-lease-wrkflw.md).</span></span>

1. <span data-ttu-id="71fd2-112">Pateikite nuomą patvirtinti.</span><span class="sxs-lookup"><span data-stu-id="71fd2-112">Submit a lease for approval.</span></span> <span data-ttu-id="71fd2-113">Nuomos puslapyje **Knygos informacija** pasirinkite **Darbo eiga**, tada pasirinkite **Pateikti**.</span><span class="sxs-lookup"><span data-stu-id="71fd2-113">On the **Book details** page for the lease, select **Workflow**, and then select **Submit**.</span></span>
2. <span data-ttu-id="71fd2-114">Pasirodžiusiame dialogo lange galite pridėti komentarą.</span><span class="sxs-lookup"><span data-stu-id="71fd2-114">In the dialog box that appears, you can add a comment.</span></span> <span data-ttu-id="71fd2-115">Nurodytas tvirtintojas matys šį komentarą kartu su nuoma.</span><span class="sxs-lookup"><span data-stu-id="71fd2-115">The designated approver will see this comment together with the lease.</span></span> <span data-ttu-id="71fd2-116">Baigę įvesti komentarą pasirinkite **Pateikti**.</span><span class="sxs-lookup"><span data-stu-id="71fd2-116">When you've finished entering the comment, select **Submit**.</span></span> <span data-ttu-id="71fd2-117">Nuoma pateikiama darbo eigos sistemai, o tvirtintojas jį gauna tvirtinti.</span><span class="sxs-lookup"><span data-stu-id="71fd2-117">The lease is submitted to the workflow system, and the approver receives it for approval.</span></span>
3. <span data-ttu-id="71fd2-118">Norėdami peržiūrėti nuomos sutartis, kurias jie paskirti tvirtinti, pasirinkite **Moduliai \> Bendra \> Darbo elementai \> Man priskirti darbo elementai**.</span><span class="sxs-lookup"><span data-stu-id="71fd2-118">To view the leases that they are designated for approval, go to **Modules \> Common \> Work items \> Work items assigned to me**.</span></span>
4. <span data-ttu-id="71fd2-119">Puslapyje **Man priskirti darbo elementai** pasirinkite nuomos, kurią norite peržiūrėti, saitą **Nuomos ID**.</span><span class="sxs-lookup"><span data-stu-id="71fd2-119">On the **Work items assigned to me** page, select the **Lease ID** link for the lease that you want to view.</span></span> <span data-ttu-id="71fd2-120">Rodomas puslapis priklauso nuo nuomos knygų ir nuomos informacijos, su kuria jūs turite prieigą.</span><span class="sxs-lookup"><span data-stu-id="71fd2-120">The page that appears depends on the lease books and lease details that you have access to.</span></span>
5. <span data-ttu-id="71fd2-121">Pabaigę peržiūrėti nuomą, pasirinkite **Darbo eiga** ir nuspręskite, kokių veiksmų reikia imtis.</span><span class="sxs-lookup"><span data-stu-id="71fd2-121">When you've finish viewing the lease, select **Workflow**, and decide what action should be taken.</span></span> <span data-ttu-id="71fd2-122">Parinktys: **Tvirtinti**, **Atmesti**, **Prašyti pakeisti**, **Perduoti** ir **Atšaukti**.</span><span class="sxs-lookup"><span data-stu-id="71fd2-122">The options include **Approve**, **Rejected**, **Request change**, **Delegate**, and **Canceled**.</span></span> <span data-ttu-id="71fd2-123">Taip pat galite pasirinkti **Peržiūrėti retrospektyvą**, kad peržiūrėtumėte pasirinktos nuomos patvirtinimo retrospektyvą.</span><span class="sxs-lookup"><span data-stu-id="71fd2-123">You can also select **View history** to view the approval history for the selected lease.</span></span>
6. <span data-ttu-id="71fd2-124">Pasirinkę veiksmą, įveskite komentarą, kuris aprašys veiksmą.</span><span class="sxs-lookup"><span data-stu-id="71fd2-124">After you've selected an action, enter a comment to describe the action.</span></span> <span data-ttu-id="71fd2-125">Kai baigiate rašyti komentarą, sąraše pasirinkite veiksmą **Patvirtinta**.</span><span class="sxs-lookup"><span data-stu-id="71fd2-125">When you've finished entering the comment, select the **Approved** action in the list.</span></span>
7. <span data-ttu-id="71fd2-126">Norėdami peržiūrėti patvirtinimo veiksmus, vėl atidarykite puslapį **Nuomos informacija** iš puslapio **Nuomos suvestinė** ir pasirinkite **Darbo eiga \> Rodinio retrospektyva**.</span><span class="sxs-lookup"><span data-stu-id="71fd2-126">To view the approval actions, go back to the **Lease details** page from the **Lease summary** page, and then select **Workflow \> View history**.</span></span>

    <span data-ttu-id="71fd2-127">Galite peržiūrėti darbo eigos veiklą darbo puslapyje **Darbo eigos retrospektyva**.</span><span class="sxs-lookup"><span data-stu-id="71fd2-127">You can view the workflow activities on the **Workflow history** page.</span></span> <span data-ttu-id="71fd2-128">Šiame puslapyje parodomi darbo eigos veiksmai, kurių buvo imtasi konkrečioje nuomos sutartyje.</span><span class="sxs-lookup"><span data-stu-id="71fd2-128">This page shows the workflow steps that have been taken on the specific lease.</span></span> <span data-ttu-id="71fd2-129">Norėdami peržiūrėti priskirtų darbo elementų būsenas, taip pat galite naudoti lauką **Darbo elementai**.</span><span class="sxs-lookup"><span data-stu-id="71fd2-129">You can also use the **Work items** field to view the status of the assigned work items.</span></span>

8. <span data-ttu-id="71fd2-130">Norėdami išjungti darbo eigą, puslapyje **Darbo eigos retrospektyva** pasirinkite **Atšaukti**.</span><span class="sxs-lookup"><span data-stu-id="71fd2-130">To stop a workflow, on the **Workflow history** page, select **Recall**.</span></span> <span data-ttu-id="71fd2-131">Pasirodžiusiame dialogo lange įveskite komentarą, tada pasirinkite **Gerai**.</span><span class="sxs-lookup"><span data-stu-id="71fd2-131">In the dialog box that appears, enter a comment, and then select **OK**.</span></span>
9. <span data-ttu-id="71fd2-132">Norėdami išjungti darbo eigą arba suaktyvinti anksčiau sukurtą darbo eigą, pasirinkite **Turto nuoma \> Sąranka \> Nuomos darbo eiga**.</span><span class="sxs-lookup"><span data-stu-id="71fd2-132">To inactivate a workflow, or to activate a workflow that was previously created, go to **Asset leasing \> Setup \> Lease workflow**.</span></span> <span data-ttu-id="71fd2-133">Tada puslapyje **Nuomos darbo eiga** pasirinkite **Darbo eiga \> Versijos**.</span><span class="sxs-lookup"><span data-stu-id="71fd2-133">Then, on the **Lease workflow** page, select **Workflow \> Versions**.</span></span> <span data-ttu-id="71fd2-134">Norėdami, kad dabartinė darbo eiga būtų neaktyvi, pasirinkite aktyvią nuomos ir nuomos versijos dialogo langą ir pasirinkite **Išjungti**.</span><span class="sxs-lookup"><span data-stu-id="71fd2-134">To make a current workflow inactive, select the active lease in the lease version dialog box, and then select **Make inactive**.</span></span> <span data-ttu-id="71fd2-135">Norėdami suaktyvinti esamą darbo eigą, pasirinkite darbo eigą ir pasirinkite **Suaktyvinti**.</span><span class="sxs-lookup"><span data-stu-id="71fd2-135">To make an existing workflow active, select the workflow, and then select **Make active**.</span></span>
