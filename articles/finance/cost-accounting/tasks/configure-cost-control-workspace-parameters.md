# <a name="configure-cost-control-workspace-parameters"></a><span data-ttu-id="ef03f-103">Konfigūruoti savikainos kontrolės darbo srities parametrus</span><span class="sxs-lookup"><span data-stu-id="ef03f-103">Configure cost control workspace parameters</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="ef03f-104">Naudokite šią procedūrą norėdami konfigūruoti savikainos kontrolės darbo sritį, kad vadybininkai įvairiuose lygiuose organizacijoje galėtų suprasti savo savikainos objektus, pvz., savikainos centrus ir produktų grupes.</span><span class="sxs-lookup"><span data-stu-id="ef03f-104">Use this procedure to configure the Cost control workspace so that managers at various levels in an organization can gain insight into their cost objects, such as cost centers and product groups.</span></span> <span data-ttu-id="ef03f-105">Kuriant šį įrašą buvo naudojama demonstracinė įmonė USP2.</span><span class="sxs-lookup"><span data-stu-id="ef03f-105">The USP2 demo company was used to create this recording.</span></span>

1. <span data-ttu-id="ef03f-106">Eikite į Savikainos apskaita > Sąranka > Savikainos kontrolės darbo srities konfigūracija.</span><span class="sxs-lookup"><span data-stu-id="ef03f-106">Go to Cost accounting > Setup > Cost control workspace configuration.</span></span>
2. <span data-ttu-id="ef03f-107">Spustelėkite Naujas.</span><span class="sxs-lookup"><span data-stu-id="ef03f-107">Click New.</span></span>
3. <span data-ttu-id="ef03f-108">Lauke Pavadinimas surinkite reikšmę.</span><span class="sxs-lookup"><span data-stu-id="ef03f-108">In the Name field, type a value.</span></span>
4. <span data-ttu-id="ef03f-109">Lauke Aprašas įveskite reikšmę.</span><span class="sxs-lookup"><span data-stu-id="ef03f-109">In the Description field, type a value.</span></span>
5. <span data-ttu-id="ef03f-110">Lauke Paskelbta pasirinkite Taip.</span><span class="sxs-lookup"><span data-stu-id="ef03f-110">Select Yes in the Published field.</span></span>
    * <span data-ttu-id="ef03f-111">Jei nustatote šią pasirinktį Taip, vartotojai, kuriems priskirtas vienas iš šių vaidmenų, gali matyti ataskaitą išlaidų kontrolės darbo srityje: savikainos apskaitos vadovas, savikainos apskaitininkas, savikainos apskaitininko klerkas ar išlaidų objekto kontrolierius.</span><span class="sxs-lookup"><span data-stu-id="ef03f-111">If you set this option to Yes, users who are assigned one of these roles can see the report in the Cost control workspace: cost accounting manager, cost accountant, cost accountant clerk, or cost object controller.</span></span> <span data-ttu-id="ef03f-112">Jei nustatote šią pasirinktį Ne, tik vartotojai, kuriems priskirtas vienas iš šių vaidmenų, gali matyti ataskaitą išlaidų kontrolės darbo srityje: savikainos apskaitos vadovas, savikainos apskaitininkas ar savikainos apskaitininko klerkas.</span><span class="sxs-lookup"><span data-stu-id="ef03f-112">If you set this option to No, only users who are assigned one of these roles can see the report in the Cost control workspace: cost accounting manager, cost accountant, or cost accountant clerk.</span></span>  
6. <span data-ttu-id="ef03f-113">Išplėskite duomenų filtravimo skyrių.</span><span class="sxs-lookup"><span data-stu-id="ef03f-113">Expand the Data filtering section.</span></span>
7. <span data-ttu-id="ef03f-114">Lauke Savikainos kontrolės įtaisas įveskite arba pasirinkite reikšmę.</span><span class="sxs-lookup"><span data-stu-id="ef03f-114">In the Cost control unit field, enter or select a value.</span></span>
8. <span data-ttu-id="ef03f-115">Lauke Biudžeto originali versija įveskite arba pasirinkite vertę.</span><span class="sxs-lookup"><span data-stu-id="ef03f-115">In the Budget original version field, enter or select a value.</span></span>
9. <span data-ttu-id="ef03f-116">Lauke Savikainos elementų dimensijų hierarchija įveskite arba pasirinkite reikšmę.</span><span class="sxs-lookup"><span data-stu-id="ef03f-116">In the Cost element dimension hierarchy field, enter or select a value.</span></span>
10. <span data-ttu-id="ef03f-117">Lauke Savikainos objekto dimensijų hierarchija įveskite arba pasirinkite reikšmę.</span><span class="sxs-lookup"><span data-stu-id="ef03f-117">In the Cost object dimension hierarchy field, enter or select a value.</span></span>
11. <span data-ttu-id="ef03f-118">Išplėskite skyrių Priskirti skaičiavimo įrašus.</span><span class="sxs-lookup"><span data-stu-id="ef03f-118">Expand the Assign calculation records section.</span></span>
12. <span data-ttu-id="ef03f-119">Spustelėkite Naujas.</span><span class="sxs-lookup"><span data-stu-id="ef03f-119">Click New.</span></span>
13. <span data-ttu-id="ef03f-120">Sąraše pažymėkite pasirinktą eilutę.</span><span class="sxs-lookup"><span data-stu-id="ef03f-120">In the list, mark the selected row.</span></span>
14. <span data-ttu-id="ef03f-121">Lauke Finansinio kalendoriaus laikotarpis įveskite arba pasirinkite reikšmę.</span><span class="sxs-lookup"><span data-stu-id="ef03f-121">In the Fiscal calendar period field, enter or select a value.</span></span>
15. <span data-ttu-id="ef03f-122">Lauke Aktuali versija įveskite arba pasirinkite reikšmę.</span><span class="sxs-lookup"><span data-stu-id="ef03f-122">In the Actual version field, enter or select a value.</span></span>
16. <span data-ttu-id="ef03f-123">Išplėskite kiekvieno stulpelio skyriaus ataskaitinius laikotarpius.</span><span class="sxs-lookup"><span data-stu-id="ef03f-123">Expand the Fiscal periods per column section.</span></span>
17. <span data-ttu-id="ef03f-124">Lauke Dabartinis periodas pasirinkite Taip.</span><span class="sxs-lookup"><span data-stu-id="ef03f-124">Select Yes in the Current period field.</span></span>
18. <span data-ttu-id="ef03f-125">Išplėskite stulpelius, kad būtų rodomas savikainos skyrius.</span><span class="sxs-lookup"><span data-stu-id="ef03f-125">Expand the Columns to display for costs section.</span></span>
19. <span data-ttu-id="ef03f-126">Lauke Fiksuota savikaina pasirinkite Taip.</span><span class="sxs-lookup"><span data-stu-id="ef03f-126">Select Yes in the Fixed cost field.</span></span>
20. <span data-ttu-id="ef03f-127">Lauke Kintama savikaina pasirinkite Taip.</span><span class="sxs-lookup"><span data-stu-id="ef03f-127">Select Yes in the Variable cost field.</span></span>
21. <span data-ttu-id="ef03f-128">Lauke Visa savikaina pasirinkite Taip.</span><span class="sxs-lookup"><span data-stu-id="ef03f-128">Select Yes in the Total cost field.</span></span>
22. <span data-ttu-id="ef03f-129">Spustelėkite Įrašyti.</span><span class="sxs-lookup"><span data-stu-id="ef03f-129">Click Save.</span></span>
23. <span data-ttu-id="ef03f-130">Uždarykite puslapį.</span><span class="sxs-lookup"><span data-stu-id="ef03f-130">Close the page.</span></span>
24. <span data-ttu-id="ef03f-131">Eikite į Savikainos apskaita > Darbo sritys > Savikainos kontrolė.</span><span class="sxs-lookup"><span data-stu-id="ef03f-131">Go to Cost accounting > Workspaces > Cost control.</span></span>
25. <span data-ttu-id="ef03f-132">Pasirinkite formuluotę, kad pamatytumėte fiksuotą, kintamą, visą ir neklasifikuotą savikainą pasirinktiems ataskaitiniams laikotarpiams.</span><span class="sxs-lookup"><span data-stu-id="ef03f-132">Select a statement to see fixed, variable, total, and unclassified costs for the selected fiscal periods.</span></span>
26. <span data-ttu-id="ef03f-133">Lauke Finansinio kalendoriaus laikotarpis įveskite arba pasirinkite reikšmę.</span><span class="sxs-lookup"><span data-stu-id="ef03f-133">In the Fiscal calendar period field, enter or select a value.</span></span>
27. <span data-ttu-id="ef03f-134">Lauke Savikainos objekto dimensijų hierarchijos mazgas įveskite arba pasirinkite reikšmę.</span><span class="sxs-lookup"><span data-stu-id="ef03f-134">In the Cost object dimension hierarchy node field, enter or select a value.</span></span>
    * <span data-ttu-id="ef03f-135">Pasirinkę savikainos objekto dimensijų hierarchiją, išplėskite savikainos elemento dimensijų hierarchiją, kad pamatytumėte pageidaujamas savikainos vertes.</span><span class="sxs-lookup"><span data-stu-id="ef03f-135">After you've selected a Cost object dimension hierarchy, expand the Cost element dimension hierarchy to see the desired cost values.</span></span> <span data-ttu-id="ef03f-136">Pavyzdžiui, galite išplėsti hierarchiją iki gamybos pridėtinių išlaidų, kad pamatytumėte vertę.</span><span class="sxs-lookup"><span data-stu-id="ef03f-136">For example, you can expand the hierarchy to Manufacturing overhead to see the value.</span></span>
