# <a name="create-a-cost-accounting-ledger"></a><span data-ttu-id="a95d4-103">Kurti savikainos apskaitos didžiąją knygą</span><span class="sxs-lookup"><span data-stu-id="a95d4-103">Create a cost accounting ledger</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="a95d4-104">Išlaidų apskaitos didžioji knyga atitinka visą atsiskaitantį vienetą.</span><span class="sxs-lookup"><span data-stu-id="a95d4-104">A cost accounting ledger represents the overall reporting unit.</span></span> <span data-ttu-id="a95d4-105">Apibrėžia išlaidų elemento dimensija, statistikos dimensija, finansinis kalendorius ir valiuta.</span><span class="sxs-lookup"><span data-stu-id="a95d4-105">It is defined by a cost element dimension, statistical dimension, fiscal calendar, and currency.</span></span> <span data-ttu-id="a95d4-106">Sąvoka juridiniai subjektai nesuprantama.</span><span class="sxs-lookup"><span data-stu-id="a95d4-106">It is agnostic to the concept of legal entities.</span></span> <span data-ttu-id="a95d4-107">Juridinis subjektas ir jo duomenys gali būti susieti su daugeliu savikainos apskaitos didžiųjų knygų.</span><span class="sxs-lookup"><span data-stu-id="a95d4-107">A legal entity and its data can be associated with many cost accounting ledgers.</span></span> <span data-ttu-id="a95d4-108">Šiame įraše naudojama demonstracinių duomenų įmonė USP2.</span><span class="sxs-lookup"><span data-stu-id="a95d4-108">This recording uses the USP2 demo data company.</span></span>

1. <span data-ttu-id="a95d4-109">Eikite į Savikainos apskaita > Didžiosios knygos nustatymas > Savikainos apskaitos didžiosios knygos.</span><span class="sxs-lookup"><span data-stu-id="a95d4-109">Go to Cost accounting > Ledger setup > Cost accounting ledgers.</span></span>
2. <span data-ttu-id="a95d4-110">Spustelėkite Naujas.</span><span class="sxs-lookup"><span data-stu-id="a95d4-110">Click New.</span></span>
3. <span data-ttu-id="a95d4-111">Lauke Pavadinimas surinkite reikšmę.</span><span class="sxs-lookup"><span data-stu-id="a95d4-111">In the Name field, type a value.</span></span>
4. <span data-ttu-id="a95d4-112">Lauke Savikainos elementų dimensija įveskite arba pasirinkite reikšmę.</span><span class="sxs-lookup"><span data-stu-id="a95d4-112">In the Cost element dimension field, enter or select a value.</span></span>
5. <span data-ttu-id="a95d4-113">Lauke Finansinis kalendorius įveskite arba pasirinkite reikšmę.</span><span class="sxs-lookup"><span data-stu-id="a95d4-113">In the Fiscal calendar field, enter or select a value.</span></span>
6. <span data-ttu-id="a95d4-114">Lauke Apskaitos valiuta įveskite arba pasirinkite reikšmę.</span><span class="sxs-lookup"><span data-stu-id="a95d4-114">In the Accounting currency field, enter or select a value.</span></span>
7. <span data-ttu-id="a95d4-115">Lauke Keitimo kurso tipas įveskite arba pasirinkite reikšmę.</span><span class="sxs-lookup"><span data-stu-id="a95d4-115">In the Exchange rate type field, enter or select a value.</span></span>
8. <span data-ttu-id="a95d4-116">Lauke Statistikos dimensija įveskite arba pasirinkite reikšmę.</span><span class="sxs-lookup"><span data-stu-id="a95d4-116">In the Statistical dimension field, enter or select a value.</span></span>
9. <span data-ttu-id="a95d4-117">Spustelėkite Įrašyti.</span><span class="sxs-lookup"><span data-stu-id="a95d4-117">Click Save.</span></span>
