# <a name="establish-customer-payment-fees"></a><span data-ttu-id="96cbd-103">Nustatyti kliento mokėjimo mokesčius</span><span class="sxs-lookup"><span data-stu-id="96cbd-103">Establish customer payment fees</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="96cbd-104">Sukurkite mokėjimų klientams mokesčius.</span><span class="sxs-lookup"><span data-stu-id="96cbd-104">Create payment fees for customer payments.</span></span>

<span data-ttu-id="96cbd-105">Šioje užduotyje naudojama demonstracinė įmonė USMF.</span><span class="sxs-lookup"><span data-stu-id="96cbd-105">This task uses the USMF demo company.</span></span>

1. <span data-ttu-id="96cbd-106">**Naršymo srityje** eikite į **Moduliai> Gautinos sumos > Mokėjimų sąranka > Mokėjimo mokestis**.</span><span class="sxs-lookup"><span data-stu-id="96cbd-106">In the **Navigation pane**, go to **Modules > Accounts receivable > Payments setup > Payment fee**.</span></span>
2. <span data-ttu-id="96cbd-107">Spustelėkite **Naujas**.</span><span class="sxs-lookup"><span data-stu-id="96cbd-107">Click **New**.</span></span>
3. <span data-ttu-id="96cbd-108">Lauke **Mokesčio ID** įveskite mokesčio ID.</span><span class="sxs-lookup"><span data-stu-id="96cbd-108">In the **Fee ID** field, enter a Fee ID.</span></span> <span data-ttu-id="96cbd-109">Mokesčio ID rodomas mokėjimų žurnaluose, todėl nurodykite jį aprašomąjį, kad suprastumėte, koks mokestis nustatomas.</span><span class="sxs-lookup"><span data-stu-id="96cbd-109">The Fee ID displays on payment journals, so make it descriptive to understand what fee is being assessed.</span></span>  
4. <span data-ttu-id="96cbd-110">Lauke **Pavadinimas** įveskite mokesčio pavadinimą.</span><span class="sxs-lookup"><span data-stu-id="96cbd-110">In the **Name** field, enter a fee name.</span></span>
5. <span data-ttu-id="96cbd-111">Lauke **Mokesčio aprašas** įveskite mokesčio aprašą.</span><span class="sxs-lookup"><span data-stu-id="96cbd-111">In the **Fee description** field, enter a description for the fee.</span></span>
6. <span data-ttu-id="96cbd-112">Lauke **Mokestis** pasirinkite ar mokestis bus taikomas kliento ar Didžiosios knygos sąskaitai.</span><span class="sxs-lookup"><span data-stu-id="96cbd-112">In the **Charge** field, select whether the fee will be charged to the Customer or a Ledger account.</span></span> <span data-ttu-id="96cbd-113">Jei mokesčiu apmokestinamas klientas, pasirinkite Klientas.</span><span class="sxs-lookup"><span data-stu-id="96cbd-113">If the customer is assessed the fee, select Customer.</span></span> <span data-ttu-id="96cbd-114">Jei mokesčiu, kaip išlaidomis, bus apmokestinta jūsų organizacija, pasirinkite Didžioji knyga.</span><span class="sxs-lookup"><span data-stu-id="96cbd-114">If the fee will be assess to your organization as an expense, select Ledger.</span></span> <span data-ttu-id="96cbd-115">Šiai užduočiai atlikti pasirinkite Klientas.</span><span class="sxs-lookup"><span data-stu-id="96cbd-115">For this task, select Customer.</span></span>  
7. <span data-ttu-id="96cbd-116">Lauke **Žurnalo tipas** pasirinkite žurnalo tipą, kuris gali naudoti šį mokėjimo mokestį.</span><span class="sxs-lookup"><span data-stu-id="96cbd-116">In the **Journal type** field, select the type of journal that can use this payment fee.</span></span> <span data-ttu-id="96cbd-117">Jei šie mokesčiai naudojami klientų mokėjimams, žurnalo tipas greičiausiai bus Kliento mokėjimas.</span><span class="sxs-lookup"><span data-stu-id="96cbd-117">If these fees are used for customer payments, the journal type will likely be Customer payment.</span></span>  
8. <span data-ttu-id="96cbd-118">Spustelėkite **Įrašyti**.</span><span class="sxs-lookup"><span data-stu-id="96cbd-118">Click **Save**.</span></span>
9. <span data-ttu-id="96cbd-119">Spustelėkite **Mokėjimo mokesčio sąranka**.</span><span class="sxs-lookup"><span data-stu-id="96cbd-119">Click **Payment fee setup**.</span></span> <span data-ttu-id="96cbd-120">Mokėjimo mokesčio sąranka naudojama apibrėžti kriterijams, kada mokėjimo mokesčiu bus apmokestinama.</span><span class="sxs-lookup"><span data-stu-id="96cbd-120">The Payment fee setup is used to define the criteria for when the payment fee will be assessed.</span></span>  <span data-ttu-id="96cbd-121">Pavyzdžiui, galite apibrėžti, kad mokestis bus skaičiuojamas, jei banko sąskaita bus USMF OPER, o mokėjimo būdas bus čekis.</span><span class="sxs-lookup"><span data-stu-id="96cbd-121">For example, you can define that the fee will be calculated if the bank account is USMF OPER, and the method of payment is check.</span></span>  
10. <span data-ttu-id="96cbd-122">Lauke **Grupavimai** pasirinkite Lentelė, Grupė arba Visi, kad apibrėžtumėte, kurios banko sąskaitos bus apmokestintos šiuo mokesčiu.</span><span class="sxs-lookup"><span data-stu-id="96cbd-122">In the **Groupings** field, select either Table, Group or All to define which bank accounts will be assessed this fee.</span></span> <span data-ttu-id="96cbd-123">Jei pasirenkate Visos, šiuo mokesčiu galėtų būti apmokestintos visos banko sąskaitos.</span><span class="sxs-lookup"><span data-stu-id="96cbd-123">If you select All, all bank accounts could be assessed this fee.</span></span>  <span data-ttu-id="96cbd-124">Jei pasirenkate Lentelė, šiuo mokesčiu gali būti apmokestinta tik jūsų pasirinkta banko sąskaita.</span><span class="sxs-lookup"><span data-stu-id="96cbd-124">If you select Table, only the bank account you select could be assessed this fee.</span></span> <span data-ttu-id="96cbd-125">Jei pasirenkate Grupė, šiuo mokesčiu gali būti apmokestintos tik pasirinktos bankų grupės sąskaitos.</span><span class="sxs-lookup"><span data-stu-id="96cbd-125">If you select Group, only the bank accounts in the selected bank group could be assessed this fee.</span></span>  
11. <span data-ttu-id="96cbd-126">Lauke **Banko ryšys** pasirinkite banko grupę arba banko sąskaitą.</span><span class="sxs-lookup"><span data-stu-id="96cbd-126">In the **Bank relation** field, select either a bank group or a bank account.</span></span> <span data-ttu-id="96cbd-127">Jei pasirinkote Lentelė, peržvalgoje bus rodomos banko sąskaitos.</span><span class="sxs-lookup"><span data-stu-id="96cbd-127">If you selected Table, the lookup will display bank accounts.</span></span> <span data-ttu-id="96cbd-128">Jei pasirinkote Grupė, peržvalgoje bus rodomos bankų grupės.</span><span class="sxs-lookup"><span data-stu-id="96cbd-128">If you selected Group, the lookup will display bank groups.</span></span>  
12. <span data-ttu-id="96cbd-129">Sąraše spustelėkite saitą pasirinktoje eilutėje.</span><span class="sxs-lookup"><span data-stu-id="96cbd-129">In the list, click the link in the selected row.</span></span>
13. <span data-ttu-id="96cbd-130">Lauke **Mokėjimo būdas** pasirinkite mokėjimo būdą, kuris bus apmokestintas šiuo mokesčiu.</span><span class="sxs-lookup"><span data-stu-id="96cbd-130">In the **Method of payment** field, select the Method of payment for which this fee will be assessed.</span></span> <span data-ttu-id="96cbd-131">Pavyzdžiui., mokesčiu galite apmokestinti savo klientus, jei jie mokėjimus siunčia kaip čekį, o ne kaip elektroninį mokėjimą.</span><span class="sxs-lookup"><span data-stu-id="96cbd-131">For example, you may assess a fee to your customers if they send payments as a check, rather than as an electronic payment.</span></span>  
14. <span data-ttu-id="96cbd-132">Sąraše raskite ir pasirinkite norimą įrašą.</span><span class="sxs-lookup"><span data-stu-id="96cbd-132">In the list, find and select the desired record.</span></span>
15. <span data-ttu-id="96cbd-133">Jei taikytina, lauke **Mokėjimo valiuta** įveskite mokėjimo valiutą.</span><span class="sxs-lookup"><span data-stu-id="96cbd-133">If relevant, in the **Payment currency** field, enter a payment currency.</span></span> <span data-ttu-id="96cbd-134">Mokėjimo valiuta naudojama kaip papildomas kriterijus nustatyti, ar bus taikomas mokestis.</span><span class="sxs-lookup"><span data-stu-id="96cbd-134">The payment currency is used as an additional criteria for whether the fee will be assessed.</span></span>  <span data-ttu-id="96cbd-135">Pavyzdžiui, jūsų bankas gali taikyti papildomą mokestį už mokėjimus, gautus JAV doleriais, nes jie paprastai operuoja tik eurais.</span><span class="sxs-lookup"><span data-stu-id="96cbd-135">For example, your bank may charge an extra fee for payments received in USD currency, since they normally only transact in EUR currency.</span></span>  
16. <span data-ttu-id="96cbd-136">Pasirinkite, ar mokestis bus procentas, suma, ar intervalas.</span><span class="sxs-lookup"><span data-stu-id="96cbd-136">Select whether the fee will be a percent, amount or interval.</span></span>
17. <span data-ttu-id="96cbd-137">Lauke **Procentai / suma** įveskite mokesčio procentus arba sumą.</span><span class="sxs-lookup"><span data-stu-id="96cbd-137">In the **Percentage/Amount field**, enter either percentage or amount of the fee.</span></span> <span data-ttu-id="96cbd-138">Jei laukas Procentas / suma yra Procentas, tada čia įvesta reikšmė bus procentas.</span><span class="sxs-lookup"><span data-stu-id="96cbd-138">If the Percentage/Amount field is Percent, then the value enter here will be a percentage.</span></span> <span data-ttu-id="96cbd-139">Jei laukas Procentas / suma yra Suma, tada čia jūsų įvesta reikšmė bus suma.</span><span class="sxs-lookup"><span data-stu-id="96cbd-139">If the Percentage/Amount field is Amount, then the value you enter here will be an amount.</span></span> <span data-ttu-id="96cbd-140">Jei laukas Procentas / suma yra intervalas, naudokite skirtuką Intervalas, kad apibrėžtumėte pakopas.</span><span class="sxs-lookup"><span data-stu-id="96cbd-140">If the Percentage/Amount field is Interval, use the Interval tab to define the tiers.</span></span>  
18. <span data-ttu-id="96cbd-141">Lauke **Mokesčio valiuta** pasirinkite mokesčio valiutą.</span><span class="sxs-lookup"><span data-stu-id="96cbd-141">In the **Fee currency** field, select the currency of the fee.</span></span> <span data-ttu-id="96cbd-142">Tai valiuta, kuria bus sukurta mokestis.</span><span class="sxs-lookup"><span data-stu-id="96cbd-142">This is the currency in which the fee will be created.</span></span>  
19. <span data-ttu-id="96cbd-143">Spustelėkite **Įrašyti**.</span><span class="sxs-lookup"><span data-stu-id="96cbd-143">Click **Save**.</span></span>
