# <a name="create-a-credit-note-on-project-invoices"></a><span data-ttu-id="5e54a-103">Kurti projekto SF kredito pažymą</span><span class="sxs-lookup"><span data-stu-id="5e54a-103">Create a credit note on project invoices</span></span>

[!include [task guide banner](../../includes/task-guide-banner.md)]

1. <span data-ttu-id="5e54a-104">Eikite į Projektų valdymas ir apskaita > Projektai > Visi projektai.</span><span class="sxs-lookup"><span data-stu-id="5e54a-104">Go to Project management and accounting > Projects > All projects.</span></span> 
2. <span data-ttu-id="5e54a-105">Sąraše raskite ir pasirinkite norimą įrašą.</span><span class="sxs-lookup"><span data-stu-id="5e54a-105">In the list, find and select the desired record.</span></span> 
3. <span data-ttu-id="5e54a-106">Pasirinkite projekto ID, naudojamą kredito pažymai.</span><span class="sxs-lookup"><span data-stu-id="5e54a-106">Select the Project ID that is used for the Credit Note.</span></span> <span data-ttu-id="5e54a-107">Sąraše spustelėkite saitą pasirinktoje eilutėje.</span><span class="sxs-lookup"><span data-stu-id="5e54a-107">In the list, click the link in the selected row.</span></span> 
4. <span data-ttu-id="5e54a-108">Veiksmų srityje spustelėkite Valdyti.</span><span class="sxs-lookup"><span data-stu-id="5e54a-108">On the Action Pane, click Manage.</span></span> 
5. <span data-ttu-id="5e54a-109">Spustelėkite Sąskaitų faktūrų žurnalai.</span><span class="sxs-lookup"><span data-stu-id="5e54a-109">Click Invoice journals.</span></span> 
6. <span data-ttu-id="5e54a-110">Pasirinkite kredito pažymos operaciją.</span><span class="sxs-lookup"><span data-stu-id="5e54a-110">Select the transaction for credit note.</span></span> <span data-ttu-id="5e54a-111">Sąraše pažymėkite pasirinktą eilutę.</span><span class="sxs-lookup"><span data-stu-id="5e54a-111">In the list, mark the selected row.</span></span> 
7. <span data-ttu-id="5e54a-112">Spustelėkite Funkcijos.</span><span class="sxs-lookup"><span data-stu-id="5e54a-112">Click Functions.</span></span> 
8. <span data-ttu-id="5e54a-113">Spustelėkite Kredito pažymos pasirinkimas.</span><span class="sxs-lookup"><span data-stu-id="5e54a-113">Click Select for credit note.</span></span> 
9. <span data-ttu-id="5e54a-114">Pažymėdami žymės langelį pasirinkite kredito žymos operaciją.</span><span class="sxs-lookup"><span data-stu-id="5e54a-114">Select the transaction for credit note by marking the check box.</span></span>
10. <span data-ttu-id="5e54a-115">Spustelėkite GERAI.</span><span class="sxs-lookup"><span data-stu-id="5e54a-115">Click OK.</span></span> 

<span data-ttu-id="5e54a-116">Atlikus šį veiksmą bus pasiekiama SF pasiūlymo kredito pažyma.</span><span class="sxs-lookup"><span data-stu-id="5e54a-116">After this step, the Credit Note will be available for Invoice Proposal.</span></span>
