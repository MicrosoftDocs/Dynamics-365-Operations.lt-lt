<span data-ttu-id="fd18b-104">Po to, kai sukuriate ir išsiunčiate projekto pasiūlymą klientui, prieš atnaujindami pasiūlymo būseną į **Išsiųsta** turite gauti kliento patvirtinimą.</span><span class="sxs-lookup"><span data-stu-id="fd18b-104">After you create and send a project quotation to a customer, you must get confirmation from the customer before you update the quotation status to **Sent**.</span></span> <span data-ttu-id="fd18b-105">Bet kokie kliento pageidaujami pakeitimai gali būti atlikti pasiūlyme.</span><span class="sxs-lookup"><span data-stu-id="fd18b-105">Any modifications requested by the customer can be updated in the quotation.</span></span> <span data-ttu-id="fd18b-106">Atnaujinę pasiūlymo būseną į **Išsiųsta**, jo modifikuoti nebegalėsite.</span><span class="sxs-lookup"><span data-stu-id="fd18b-106">After you update the quotation status to **Sent**, you can’t modify it.</span></span> <span data-ttu-id="fd18b-107">Toliau pateiktoje procedūroje aprašomos parinktys, naudojamos siunčiant pasiūlymą patvirtinimui gauti, atnaujinant remiantis grįžtamuoju ryšiu ir siunčiant pasiūlymą.</span><span class="sxs-lookup"><span data-stu-id="fd18b-107">The following procedure describes the options to send a quotation for confirmation, make updates based on feedback, and then send the quotation.</span></span>

## <a name="send-a-project-quotation-confirmation"></a><span data-ttu-id="fd18b-108">Projekto pasiūlymo patvirtinimo siuntimas</span><span class="sxs-lookup"><span data-stu-id="fd18b-108">Send a project quotation confirmation</span></span>  

<span data-ttu-id="fd18b-109">Galite siųsti esamą projekto pasiūlymą patvirtinimui gauti elektroniniu paštu arba kaip išspausdintą kopiją.</span><span class="sxs-lookup"><span data-stu-id="fd18b-109">You can send an existing project quotation for confirmation through email or as a printed copy.</span></span>

2. <span data-ttu-id="fd18b-111">Puslapyje **Projekto pasiūlymas** pasirinkite pasiūlymą, kurį norite išsiųsti patvirtinimui gauti.</span><span class="sxs-lookup"><span data-stu-id="fd18b-111">On the **Project quotation** page, select the quotation you want to send for confirmation.</span></span>

6. <span data-ttu-id="fd18b-116">Puslapyje **Spausdinimo paskirties vietos parametrai** pasirinkite **Gerai**.</span><span class="sxs-lookup"><span data-stu-id="fd18b-116">On the **Print destination settings** page, select **OK**.</span></span>
