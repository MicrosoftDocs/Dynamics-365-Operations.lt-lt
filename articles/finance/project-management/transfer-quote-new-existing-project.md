[!include [banner](../includes/banner.md)]

<span data-ttu-id="be30b-104">Kai projekto kainos būsena yra **Patvirtinta**, galite perkelti kai kurias arba visas kainos eilutes į naują arba esamą projektą naudodami **Perkelti projektą** vedlį.</span><span class="sxs-lookup"><span data-stu-id="be30b-104">When a project quotation has a status of **Confirmed**, you can transfer some or all the quotation lines to a new or existing project by using the **Transfer to project** wizard.</span></span> 

<span data-ttu-id="be30b-105">Norėdami pasinaudoti vedliu, atlikite šiuos veiksmus.</span><span class="sxs-lookup"><span data-stu-id="be30b-105">To use the wizard, follow these steps.</span></span>
