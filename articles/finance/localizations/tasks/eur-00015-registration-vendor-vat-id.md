# <a name="eur-00015-registration-of-vendor-vat-id"></a>EUR-00015: tiekėjo PVM ID registravimas

[!include [banner](../../includes/banner.md)]

Šioje procedūroje parodoma, kaip įtraukti PVM registracijos ID ir neapmokestinimo kodą į tiekėjo kodą. Šis procesas yra panašus ir kai naudojami juridiniai subjektai bei klientai. 

Prieš atlikdami šią procedūrą, turite nustatyti PVM ID. Ši procedūra taikoma visoms Europos šalims / regionams. Ši procedūra buvo sukurta naudojant demonstracinių duomenų įmonę DEMF, kurios pirminis adresas yra Vokietijoje. Ši procedūra yra skirta duomenų valdymo administratoriui, mokėtinų sumų vadovui arba gautinų sumų vadovui. Ši procedūra yra skirta į 1611 „Dynamics 365 for Operations“ versiją įtrauktai funkcijai aprašyti.

1. Pasirinkite Mokėtinos sumos > Tiekėjai > Visi tiekėjai.
2. Sąraše raskite ir pasirinkite tiekėją DE-01001
3. Spustelėkite Registracijos ID.
4. Spustelėkite Pridėti.
5. Pasirinkite PVM ID.
6. Lauke Registracijos numeris įveskite reikšmę.
    * Nurodykite pasirinkto tiekėjo PVM ID Vokietijoje. ID turi atitikti nurodytą registracijos tipo formatą.  
7. Spustelėkite skirtuką Bendra.
8. Lauke Įsigalioja įveskite datą.
9. Spustelėkite Įrašyti.
10. Spustelėkite Naujas.
11. Lauke Pavadinimas arba aprašas surinkite reikšmę.
    * Pavyzdžiui, įveskite ITA.  
12. Lauke Šalis / regionas įveskite arba pasirinkite reikšmę.
    * Pavyzdžiui, pasirinkite ITA.  
13. Lauke Pagrindinis šalies pasirinkite Taip.
14. Spustelėkite Įrašyti.
15. Spustelėkite skirtuką Peržiūra.
16. Spustelėkite Pridėti.
17. Lauke Registracijos tipas įveskite arba pasirinkite reikšmę.
    * Pavyzdžiui, pasirinkite PVM ID.  
18. Lauke Registracijos numeris įveskite reikšmę.
    * Pavyzdžiui, nurodykite PVM ID Italijoje.  ID turi atitikti nurodytą registracijos tipo formatą.  
19. Spustelėkite Įrašyti.
20. Uždarykite puslapį.
21. Sąraše raskite ir pasirinkite norimą įrašą.
    * Pavyzdžiui, pasirinkite DE-01001.  
22. Sąraše spustelėkite saitą pasirinktoje eilutėje.
23. Išplėskite skyrių SF ir pristatymas.
24. Spustelėkite Redaguoti.
25. Lauke Neapmokestinimo kodas įveskite arba pasirinkite vertę.
26. Spustelėkite Įrašyti.
