# <a name="set-up-tds-ledger-accounts"></a><span data-ttu-id="711b3-103">TDS knygos sąskaitų nustatymas</span><span class="sxs-lookup"><span data-stu-id="711b3-103">Set up TDS ledger accounts</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="711b3-104">Šioje temoje paaiškinama, kaip nustatyti šaltinio (TDS) priemonės mokesčių išskaitomos DK sąskaitas.</span><span class="sxs-lookup"><span data-stu-id="711b3-104">This topic explains how to set up ledger accounts for the Tax Deducted at Source (TDS) feature.</span></span> <span data-ttu-id="711b3-105">Norėdami nustatyti **Sąskaitų grafiką** naudokite sąskaitų plano puslapį.</span><span class="sxs-lookup"><span data-stu-id="711b3-105">You use the **Chart of accounts** page to set up ledger accounts for TDS.</span></span>

1. <span data-ttu-id="711b3-106">Eikite į **Didžioji knyga \> Sąskaitų planas \> Sąskaitos \> Pagrindinės sąskaitos**.</span><span class="sxs-lookup"><span data-stu-id="711b3-106">Go to **General ledger \> Chart of accounts \> Accounts \> Main accounts**.</span></span>
2. <span data-ttu-id="711b3-107">Skirtuke **Peržiūra** pasirinkite **Alt+N** norėdami sukurti TDS DK sąskaitą ir įveskite reikiamą informaciją.</span><span class="sxs-lookup"><span data-stu-id="711b3-107">On the **Overview** tab, select **Alt+N** to create a TDS ledger account, and enter the required details.</span></span>
3. <span data-ttu-id="711b3-108">Lauko **Registravimo** tipas skirtuke **Nustatymas** pasirinkite **Išskaitomas mokestis**.</span><span class="sxs-lookup"><span data-stu-id="711b3-108">On the **Setup** tab, in the **Posting type** field, select **Withholding tax**.</span></span>     

    > [!NOTE]
    > <span data-ttu-id="711b3-109">Knygis sąskaitos, kurių publikavimo tipas **Išskaitomas mokestis** gali būti apibrėžtas tik kaip gautinų sąskaitų, kurios naudojamos TDS gautinų sumų registravimui pagal TDS mokesčio kodą.</span><span class="sxs-lookup"><span data-stu-id="711b3-109">Ledger accounts that have a posting type of **Withholding tax** can be defined only as receivable accounts that are used to post the TDS receivable amount for a TDS tax code.</span></span>

4. <span data-ttu-id="711b3-110">Uždarykite puslapį.</span><span class="sxs-lookup"><span data-stu-id="711b3-110">Close the page.</span></span>
