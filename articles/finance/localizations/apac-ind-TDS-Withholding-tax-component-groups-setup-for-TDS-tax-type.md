# <a name="set-up-withholding-tax-component-groups-for-the-tds-tax-type"></a><span data-ttu-id="df2d0-103">Nustatykite išskaitomo mokesčio komponento grupių TDS mokesčių tipui</span><span class="sxs-lookup"><span data-stu-id="df2d0-103">Set up withholding tax component groups for the TDS tax type</span></span>

[!include [banner](../includes/banner.md)]

<span data-ttu-id="df2d0-104">Šioje temoje paaiškinama, kaip nustatyti išskaitomo mokesčio komponentų grupės, tokios kaip **Nuomos** ir **Rangovo**, šskaitomo mokesčio šaltinio mokesčių tipui (TDS).</span><span class="sxs-lookup"><span data-stu-id="df2d0-104">This topic explains how to set up withholding tax component groups, such as **Rent** and **Contractor**, for the Tax Deducted at Source (TDS) tax type.</span></span>

1. <span data-ttu-id="df2d0-105">Eikite į **Mokestis \> Nustatymai \> Išskaitomas mokestis \> Išskaitomo mokesčio komponentų grupės**.</span><span class="sxs-lookup"><span data-stu-id="df2d0-105">Go to **Tax \> Setup \> Withholding tax \> Withholding tax component groups**.</span></span>

    <span data-ttu-id="df2d0-106">[![Išskaitomo mokesčio komponentų grupių puslapis](./media/apac-ind-TDS-8.png)](./media/apac-ind-TDS-8.png)</span><span class="sxs-lookup"><span data-stu-id="df2d0-106">[![Withholding tax component groups page](./media/apac-ind-TDS-8.png)](./media/apac-ind-TDS-8.png)</span></span>

2. <span data-ttu-id="df2d0-107">Mokesčio **tipo lauke** pasirinkite **TDS** norėdami nustatyti TDS mokesčio komponento grupių išskaitomo mokesčio komponentus.</span><span class="sxs-lookup"><span data-stu-id="df2d0-107">In the **Tax type** field, select **TDS** to set up withholding tax component groups for the TDS tax type.</span></span>
3. <span data-ttu-id="df2d0-108">Veiksmų srityje pasirinkite **Nauja** eilutei sukurti.</span><span class="sxs-lookup"><span data-stu-id="df2d0-108">On the Action Pane, select **New** to create a line.</span></span>
4. <span data-ttu-id="df2d0-109">Lauke **Išskaitomo mokesčio komponento grupės** įveskite TDS komponento grupės pavadinimą.</span><span class="sxs-lookup"><span data-stu-id="df2d0-109">In the **Withholding tax component group** field, enter a name for the TDS component group.</span></span>
5. <span data-ttu-id="df2d0-110">Lauke **Aprašas** įveskite aprašą.</span><span class="sxs-lookup"><span data-stu-id="df2d0-110">In the **Description** field, enter a description.</span></span>
6. <span data-ttu-id="df2d0-111">Skirtuko **Bendra** „FastTab“ lauke **būsena** pasirinkite TDS komponentų grupės gyvenamosios vietos būseną.</span><span class="sxs-lookup"><span data-stu-id="df2d0-111">On the **General** FastTab, in the **Status** field, select the residential status of the TDS component group.</span></span> <span data-ttu-id="df2d0-112">Pasirinktys yra **Nuolatinis** gyventojas ir **Nenuolatinis** gyventojas.</span><span class="sxs-lookup"><span data-stu-id="df2d0-112">The options are **Resident** and **Non-resident**.</span></span>
7. <span data-ttu-id="df2d0-113">Skyriaus **kodo lauke** įveskite skyriaus kodą, kuris taikomas TDS komponentų grupei.</span><span class="sxs-lookup"><span data-stu-id="df2d0-113">In the **Section code** field, enter the section code that is applied to the TDS component group.</span></span>
8. <span data-ttu-id="df2d0-114">Uždarykite puslapį.</span><span class="sxs-lookup"><span data-stu-id="df2d0-114">Close the page.</span></span>
