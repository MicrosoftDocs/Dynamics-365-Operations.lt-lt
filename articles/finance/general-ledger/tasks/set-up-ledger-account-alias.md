# <a name="set-up-a-ledger-account-alias"></a><span data-ttu-id="7f750-103">Nustatyti didžiosios knygos pseudonimą</span><span class="sxs-lookup"><span data-stu-id="7f750-103">Set up a ledger account alias</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="7f750-104">Ši procedūra nurodo, kaip sukurti sąskaitos pseudonimą, kuriame pateikiama nuoroda, kur galima įvesti sąskaitos numerį.</span><span class="sxs-lookup"><span data-stu-id="7f750-104">This procedure shows how to create an account alias that provides a shortcut for entering an account number.</span></span> <span data-ttu-id="7f750-105">Šioje procedūroje naudojami demonstracinių duomenų įmonės USMF.</span><span class="sxs-lookup"><span data-stu-id="7f750-105">This procedure uses demo data company USMF.</span></span>

1. <span data-ttu-id="7f750-106">Pasirinkite Didžioji knyga > Sąskaitų planas > Sąskaitos > Didžiosios knygos sąskaitos pseudonimas.</span><span class="sxs-lookup"><span data-stu-id="7f750-106">Go to General ledger > Chart of accounts > Accounts > Ledger account alias.</span></span>
2. <span data-ttu-id="7f750-107">Spustelėkite Naujas.</span><span class="sxs-lookup"><span data-stu-id="7f750-107">Click New.</span></span>
3. <span data-ttu-id="7f750-108">Lauke Didžiosios knygos sąskaitos pseudonimas įveskite reikšmę.</span><span class="sxs-lookup"><span data-stu-id="7f750-108">In the Ledger account alias field, type a value.</span></span>
4. <span data-ttu-id="7f750-109">Lauke Sąskaitos struktūra pasirinkite sąskaitai ir dimensijoms priklausančią struktūrą.</span><span class="sxs-lookup"><span data-stu-id="7f750-109">In the Account structure field, select the structure the account and dimensions belong to.</span></span>
5. <span data-ttu-id="7f750-110">Lauke „Įmonė“ spustelėkite išplečiamąjį mygtuką, kad atidarytumėte peržvalgą.</span><span class="sxs-lookup"><span data-stu-id="7f750-110">In the Company field, click the drop-down button to open the lookup.</span></span>
6. <span data-ttu-id="7f750-111">Sąraše raskite ir pasirinkite įmonę, kuriai taikomas pseudonimas.</span><span class="sxs-lookup"><span data-stu-id="7f750-111">In the list, find and select the company that the alias applies to.</span></span>
7. <span data-ttu-id="7f750-112">Sąraše spustelėkite saitą pasirinktoje eilutėje.</span><span class="sxs-lookup"><span data-stu-id="7f750-112">In the list, click the link in the selected row.</span></span>
8. <span data-ttu-id="7f750-113">Lauke DK sąskaitos pseudonimo aprašas nurodykite sąskaitą ir dimensijas.</span><span class="sxs-lookup"><span data-stu-id="7f750-113">In the Ledger account alias definition field, specify the account and dimensions.</span></span>
    * <span data-ttu-id="7f750-114">Sąskaita ir dimensijos bus nustatomos naudojant nuorodą.</span><span class="sxs-lookup"><span data-stu-id="7f750-114">The account and dimensions will be populated when using the shortcut.</span></span>  
9. <span data-ttu-id="7f750-115">Lauke Pradinis židinys pasirinkite dimensiją, kuri turės židinį, kai bus naudojamas pseudonimas.</span><span class="sxs-lookup"><span data-stu-id="7f750-115">In the Initial focus field, select the dimension that will have focus when the alias is used.</span></span>
    * <span data-ttu-id="7f750-116">Po to, kai įvesite nuorodą ir bus nustatyta sąskaita ir dimensijos, žymeklis arba židinys persikels į lauką Pradinis židinys.</span><span class="sxs-lookup"><span data-stu-id="7f750-116">After you type the shortcut, and the account and dimensions are populated, the Initial focus field is where the cursor or focus will move to.</span></span>
