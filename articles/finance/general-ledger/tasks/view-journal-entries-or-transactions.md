# <a name="view-journal-entries-or-transactions"></a><span data-ttu-id="fe322-103">Žurnalo įrašai arba operacijos</span><span class="sxs-lookup"><span data-stu-id="fe322-103">View journal entries or transactions</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="fe322-104">Šioje procedūroje parodoma, kaip, naudojant užklausą Kvitų operacijos, ieškoti žurnalų įrašų ar operacijų.</span><span class="sxs-lookup"><span data-stu-id="fe322-104">This procedure shows how to use the Voucher transactions inquiry to search for journal entries or transactions.</span></span>

1. <span data-ttu-id="fe322-105">Eikite į **Naršymo sritis > Moduliai > Didžioji knyga > Užklausos ir ataskaitos > Kvitų operacijos**.</span><span class="sxs-lookup"><span data-stu-id="fe322-105">Go to **Navigation pane > Modules > General ledger > Inquiries and reports > Voucher transactions**.</span></span>
2. <span data-ttu-id="fe322-106">Pasirinkite lauką, kuriame norite nustatyti filtro kriterijus.</span><span class="sxs-lookup"><span data-stu-id="fe322-106">Select the field for which you want to define a filter criteria.</span></span>
3. <span data-ttu-id="fe322-107">Įveskite pasirinkto lauko filtro kriterijus.</span><span class="sxs-lookup"><span data-stu-id="fe322-107">Enter your filter critieria for the selected field.</span></span> <span data-ttu-id="fe322-108">Galite filtruoti pagal vieną reikšmę arba intervalą.</span><span class="sxs-lookup"><span data-stu-id="fe322-108">You could filter on a single value or a range.</span></span> <span data-ttu-id="fe322-109">Apibrėždami intervalą patikrinkite, ar naudojama teisinga sintaksė.</span><span class="sxs-lookup"><span data-stu-id="fe322-109">When defining a range, make sure the correct syntax is used.</span></span> <span data-ttu-id="fe322-110">Reikšmės turi būti atskirtos dviem taškais (..).</span><span class="sxs-lookup"><span data-stu-id="fe322-110">The values should be separated by a double period (..).</span></span>  
4. <span data-ttu-id="fe322-111">Spustelėkite skirtuką **Sujungimai**, kad įtrauktumėte papildomų lentelių, kurias norite filtruoti.</span><span class="sxs-lookup"><span data-stu-id="fe322-111">Click the **Joins** tab to add additional tables from which to filter.</span></span>
5. <span data-ttu-id="fe322-112">Medyje pasirinkite **Lentelės / bendrojo žurnalo įrašas**.</span><span class="sxs-lookup"><span data-stu-id="fe322-112">In the tree, select **Tables/General journal entry**.</span></span>
6. <span data-ttu-id="fe322-113">Spustelėkite **Įtraukti lentelės sujungimą**.</span><span class="sxs-lookup"><span data-stu-id="fe322-113">Click **Add table join**.</span></span>
7. <span data-ttu-id="fe322-114">Spustelėkite **Atšaukti**, jei papildomos lentelės įtraukti nenorite.</span><span class="sxs-lookup"><span data-stu-id="fe322-114">Click **Cancel** if you decide not to add an additional table.</span></span>
8. <span data-ttu-id="fe322-115">Spustelėkite skirtuką **Intervalas**.</span><span class="sxs-lookup"><span data-stu-id="fe322-115">Click the **Range** tab.</span></span>
9. <span data-ttu-id="fe322-116">Spustelėdami **Gerai** įvykdykite užklausą.</span><span class="sxs-lookup"><span data-stu-id="fe322-116">Click **OK** to run the query.</span></span>
10. <span data-ttu-id="fe322-117">Veiksmų srityje spustelėkite **Operacijos kilmė**.</span><span class="sxs-lookup"><span data-stu-id="fe322-117">On the Action pane, click **Transaction origin**.</span></span> <span data-ttu-id="fe322-118">Galima naudoti įvairius tinklelio mygtukus, norint peržiūrėti papildomą informaciją apie pasirinktą kvito įrašą.</span><span class="sxs-lookup"><span data-stu-id="fe322-118">Various buttons about the grid can be used to research additional information about the selected record of the voucher.</span></span> <span data-ttu-id="fe322-119">Kai kurie mygtukai gali neveikti, atsižvelgiant į operacijos tipą ir charakteristiką.</span><span class="sxs-lookup"><span data-stu-id="fe322-119">Some buttons may not be available, depending on the type of transaction and characteristics of the transaction.</span></span>
11. <span data-ttu-id="fe322-120">Uždarykite puslapį.</span><span class="sxs-lookup"><span data-stu-id="fe322-120">Close the page.</span></span>
12. <span data-ttu-id="fe322-121">Veiksmų srityje spustelėkite **Originalus dokumentas**.</span><span class="sxs-lookup"><span data-stu-id="fe322-121">On the Action pane, Click **Original document**.</span></span>
13. <span data-ttu-id="fe322-122">Uždarykite puslapį.</span><span class="sxs-lookup"><span data-stu-id="fe322-122">Close the page.</span></span>
