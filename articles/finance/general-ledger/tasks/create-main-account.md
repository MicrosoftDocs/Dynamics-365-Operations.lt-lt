# <a name="create-a-main-account"></a><span data-ttu-id="c4adb-103">Kurti pagrindinę sąskaitą</span><span class="sxs-lookup"><span data-stu-id="c4adb-103">Create a main account</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="c4adb-104">Ši užduotis padeda įtraukti pagrindinę sąskaitą į esamą sąskaitų planą.</span><span class="sxs-lookup"><span data-stu-id="c4adb-104">This task guide steps through adding a main account to an existing chart of accounts.</span></span> <span data-ttu-id="c4adb-105">Šiame įraše naudojama demonstracinė įmonė USMF.</span><span class="sxs-lookup"><span data-stu-id="c4adb-105">This recording uses the USMF demo company.</span></span>  

1. <span data-ttu-id="c4adb-106">Pasirinkite **Naršymo sritis > Moduliai > Didžioji knyga > Sąskaitų planas > Sąskaitos > Pagrindinės sąskaitos**.</span><span class="sxs-lookup"><span data-stu-id="c4adb-106">Go to **Navigation pane > Modules > General ledger > Chart of accounts > Accounts > Main accounts**.</span></span>
2. <span data-ttu-id="c4adb-107">Spustelėkite **Naujas**.</span><span class="sxs-lookup"><span data-stu-id="c4adb-107">Click **New**.</span></span>
3. <span data-ttu-id="c4adb-108">Lauke **Pagrindinė sąskaita** įveskite reikšmę.</span><span class="sxs-lookup"><span data-stu-id="c4adb-108">In the **Main account** field, type a value.</span></span>
4. <span data-ttu-id="c4adb-109">Lauke **Pavadinimas** įveskite reikšmę.</span><span class="sxs-lookup"><span data-stu-id="c4adb-109">In the **Name** field, type a value.</span></span>
5. <span data-ttu-id="c4adb-110">Lauke **Pagrindinės sąskaitos tipas** pasirinkite tipą, kuris geriausiai nurodo sąskaitų balansą ir vietą finansinėse ataskaitose.</span><span class="sxs-lookup"><span data-stu-id="c4adb-110">In the **Main account type** field, select the type that best represents the accounts balance and location on financial statements.</span></span>
6. <span data-ttu-id="c4adb-111">Sąraše pasirinkite sąskaitos kategoriją, kuriai priklauso pagrindinė sąskaita.</span><span class="sxs-lookup"><span data-stu-id="c4adb-111">In the list, select the account category the main account belongs to.</span></span> <span data-ttu-id="c4adb-112">Sąskaitų kategorija naudojama numatytosioms finansinėms ataskaitoms ir „Power BI“ ataskaitų srities turiniui.</span><span class="sxs-lookup"><span data-stu-id="c4adb-112">Account category is used for default financial reports and Power BI dashboard content.</span></span>  
7. <span data-ttu-id="c4adb-113">Sąraše spustelėkite saitą pasirinktoje eilutėje.</span><span class="sxs-lookup"><span data-stu-id="c4adb-113">In the list, click the link in the selected row.</span></span> <span data-ttu-id="c4adb-114">Pakeiskite numatytąjį debeto arba kredito balansą.</span><span class="sxs-lookup"><span data-stu-id="c4adb-114">Change the default debit or credit balance.</span></span>  
8. <span data-ttu-id="c4adb-115">Lauke **Numatytoji valiuta** iš valiutų sąrašo pasirinkite reikšmę.</span><span class="sxs-lookup"><span data-stu-id="c4adb-115">In the **Default currency** field, select a value from the list of currencies.</span></span>
9. <span data-ttu-id="c4adb-116">Sąraše raskite ir pasirinkite norimą įrašą.</span><span class="sxs-lookup"><span data-stu-id="c4adb-116">In the list, find and select the desired record.</span></span>
10. <span data-ttu-id="c4adb-117">Sąraše spustelėkite saitą pasirinktoje eilutėje.</span><span class="sxs-lookup"><span data-stu-id="c4adb-117">In the list, click the link in the selected row.</span></span>
11. <span data-ttu-id="c4adb-118">Perjunkite skyriaus **Juridinio subjekto nepaisymai** išplėtimą.</span><span class="sxs-lookup"><span data-stu-id="c4adb-118">Toggle the expansion of the **Legal entity overrides** section.</span></span>
12. <span data-ttu-id="c4adb-119">Spustelėkite **Įtraukti** ir pasirinkite juridinį subjektą.</span><span class="sxs-lookup"><span data-stu-id="c4adb-119">Click **Add** to select a legal entity.</span></span>
13. <span data-ttu-id="c4adb-120">Sąraše pasirinkite Juridinis subjektas.</span><span class="sxs-lookup"><span data-stu-id="c4adb-120">In the list, select the Legal entity.</span></span>
14. <span data-ttu-id="c4adb-121">Spustelėkite **Pridėti**.</span><span class="sxs-lookup"><span data-stu-id="c4adb-121">Click **Add**.</span></span>
15. <span data-ttu-id="c4adb-122">Sąraše pažymėkite pasirinktą eilutę.</span><span class="sxs-lookup"><span data-stu-id="c4adb-122">In the list, mark the selected row.</span></span>
16. <span data-ttu-id="c4adb-123">Pažymėkite arba atžymėkite žymės langelį **Sustabdytas**.</span><span class="sxs-lookup"><span data-stu-id="c4adb-123">Check or uncheck the **Suspended** checkbox.</span></span>
17. <span data-ttu-id="c4adb-124">Išplėskite skyrių **Finansinės ataskaitos**.</span><span class="sxs-lookup"><span data-stu-id="c4adb-124">Expand the **Financial reporting** section.</span></span>
18. <span data-ttu-id="c4adb-125">Lauke **Valiutos kurso tipas** spustelėkite išplečiamąjį mygtuką, kad atidarytumėte peržvalgą.</span><span class="sxs-lookup"><span data-stu-id="c4adb-125">In the **Exchange rate type** field, click the drop-down button to open the lookup.</span></span>
19. <span data-ttu-id="c4adb-126">Sąraše pasirinkite**Sąskaitos valiutos kurso tipas**.</span><span class="sxs-lookup"><span data-stu-id="c4adb-126">In the list, select the **Exchange rate type for the account**.</span></span>
20. <span data-ttu-id="c4adb-127">Sąraše spustelėkite saitą pasirinktoje eilutėje.</span><span class="sxs-lookup"><span data-stu-id="c4adb-127">In the list, click the link in the selected row.</span></span>
21. <span data-ttu-id="c4adb-128">Lauke **Valiutos konvertavimo tipas** pasirinkite sąskaitos valiutos kursų apskaičiavimo metodą.</span><span class="sxs-lookup"><span data-stu-id="c4adb-128">In the **Currency translation type** field, select the method for calculating exchange rates for the account.</span></span>
22. <span data-ttu-id="c4adb-129">Uždarykite puslapį.</span><span class="sxs-lookup"><span data-stu-id="c4adb-129">Close the page.</span></span>
