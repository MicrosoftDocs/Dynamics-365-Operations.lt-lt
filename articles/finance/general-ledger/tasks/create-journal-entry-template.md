[!include [banner](../../includes/banner.md)]

<span data-ttu-id="c3837-104">Užregistruoti žurnalo kvitai gali būti įrašyti kaip kvitų šablonai ir taikomi naujame žurnalo kvite.</span><span class="sxs-lookup"><span data-stu-id="c3837-104">Posted journal vouchers can be saved as Voucher templates and applied in a new journal voucher.</span></span> <span data-ttu-id="c3837-105">Šioje procedūroje naudojama demonstracinė įmonė USMF.</span><span class="sxs-lookup"><span data-stu-id="c3837-105">This procedure uses the USMF demo company.</span></span>

1. <span data-ttu-id="c3837-106">Eikite į **Naršymo sritis > Moduliai > Bendroji didžioji knyga > Žurnalo įrašai > Bendrieji žurnalai**.</span><span class="sxs-lookup"><span data-stu-id="c3837-106">Go to **Navigation pane > Modules > General ledger > Journal entries > General journals**.</span></span>

5. <span data-ttu-id="c3837-111">Sąraše spustelėkite saitą pasirinktoje eilutėje.</span><span class="sxs-lookup"><span data-stu-id="c3837-111">In the list, click the link in the selected row.</span></span>

9. <span data-ttu-id="c3837-115">Lauke **Debetas** įveskite reikšmę.</span><span class="sxs-lookup"><span data-stu-id="c3837-115">In the **Debit** field, type a value.</span></span>
10. <span data-ttu-id="c3837-116">Spustelėkite **Naujas**.</span><span class="sxs-lookup"><span data-stu-id="c3837-116">Click **New**.</span></span>
11. <span data-ttu-id="c3837-117">Lauke **Sąskaitos tipas** įveskite reikšmę.</span><span class="sxs-lookup"><span data-stu-id="c3837-117">In the **Account type** field, type a value.</span></span>
12. <span data-ttu-id="c3837-118">Lauke **Aprašo laukas** surinkite reikšmę.</span><span class="sxs-lookup"><span data-stu-id="c3837-118">In the **Description** field, type a value.</span></span>
13. <span data-ttu-id="c3837-119">Lauke **Debetas** įveskite reikšmę.</span><span class="sxs-lookup"><span data-stu-id="c3837-119">In the **Debit** field, type a value.</span></span>
14. <span data-ttu-id="c3837-120">Spustelėkite **Naujas**.</span><span class="sxs-lookup"><span data-stu-id="c3837-120">Click **New**.</span></span>
14. <span data-ttu-id="c3837-121">Lauke **Sąskaita** nustatykite norimas reikšmes.</span><span class="sxs-lookup"><span data-stu-id="c3837-121">In the **Account** field, specify the desired values.</span></span>
15. <span data-ttu-id="c3837-122">Lauke **Aprašo laukas** surinkite reikšmę.</span><span class="sxs-lookup"><span data-stu-id="c3837-122">In the **Description** field, type a value.</span></span>

19. <span data-ttu-id="c3837-126">Spustelėkite šabloną **Įrašyti kvitą**.</span><span class="sxs-lookup"><span data-stu-id="c3837-126">Click **Save voucher** template.</span></span>

    - <span data-ttu-id="c3837-129">Procentas: sumos kvituose yra konvertuojamos į procentų veiksnius, kurie leidžia taikyti bet kurią sumą, kai pasirenkamas kvito šablonas.</span><span class="sxs-lookup"><span data-stu-id="c3837-129">Percent: The amounts in the voucher are converted into percentage factors, which allows any amount to be applied when the Voucher template is selected.</span></span>
    - <span data-ttu-id="c3837-130">Suma: faktinė suma bus saugoma ir taikoma.</span><span class="sxs-lookup"><span data-stu-id="c3837-130">Amount: The actual amounts will be stored and applied.</span></span>  
21. <span data-ttu-id="c3837-131">Spustelėkite **Bendrieji žurnalai**.</span><span class="sxs-lookup"><span data-stu-id="c3837-131">Click **General journals**.</span></span>
22. <span data-ttu-id="c3837-132">Spustelėkite **Naujas**.</span><span class="sxs-lookup"><span data-stu-id="c3837-132">Click **New**.</span></span>
23. <span data-ttu-id="c3837-133">Lauke **Pavadinimas** spustelėkite išplečiamąjį mygtuką, kad atidarytumėte peržvalgą.</span><span class="sxs-lookup"><span data-stu-id="c3837-133">In the **Name** field, click the drop-down button to open the lookup.</span></span>
24. <span data-ttu-id="c3837-134">Sąraše spustelėkite saitą pasirinktoje eilutėje.</span><span class="sxs-lookup"><span data-stu-id="c3837-134">In the list, click the link in the selected row.</span></span>
25. <span data-ttu-id="c3837-135">Spustelėkite **Eilutės**.</span><span class="sxs-lookup"><span data-stu-id="c3837-135">Click **Lines**.</span></span>
26. <span data-ttu-id="c3837-136">spustelėkite **Funkcijos**.</span><span class="sxs-lookup"><span data-stu-id="c3837-136">Click **Functions**.</span></span>
27. <span data-ttu-id="c3837-137">Spustelėkite **Pasirinkti kvito šabloną**.</span><span class="sxs-lookup"><span data-stu-id="c3837-137">Click **Select voucher template**.</span></span>
