# <a name="create-a-journal-entry-using-template"></a><span data-ttu-id="f4348-103">Kurti žurnalo įrašą naudojant šabloną</span><span class="sxs-lookup"><span data-stu-id="f4348-103">Create a journal entry using template</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="f4348-104">Užregistruoti žurnalo kvitai gali būti įrašyti kaip kvitų šablonai ir taikomi naujame žurnalo kvite.</span><span class="sxs-lookup"><span data-stu-id="f4348-104">Posted journal vouchers can be saved as Voucher templates and applied in a new journal voucher.</span></span> <span data-ttu-id="f4348-105">Šioje procedūroje naudojama demonstracinė įmonė USMF.</span><span class="sxs-lookup"><span data-stu-id="f4348-105">This procedure uses the USMF demo company.</span></span>

1. <span data-ttu-id="f4348-106">Eikite į **Naršymo sritis > Moduliai > Bendroji didžioji knyga > Žurnalo įrašai > Bendrieji žurnalai**.</span><span class="sxs-lookup"><span data-stu-id="f4348-106">Go to **Navigation pane > Modules > General ledger > Journal entries > General journals**.</span></span>
2. <span data-ttu-id="f4348-107">**Veiksmų sritis** spustelėkite **Naujas**.</span><span class="sxs-lookup"><span data-stu-id="f4348-107">On the **Action pane**, click **New**.</span></span> <span data-ttu-id="f4348-108">Ši procedūra prasideda sukuriant ir registruojant žurnalo kvitą, tačiau visus anksčiau užregistruotus žurnalo kvitus galima įrašyti kaip šabloną.</span><span class="sxs-lookup"><span data-stu-id="f4348-108">This procedure starts by creating and posting a journal voucher, but any previously posted journal voucher can be saved as a template.</span></span>

4. <span data-ttu-id="f4348-110">Sąraše raskite ir pasirinkite norimą įrašą.</span><span class="sxs-lookup"><span data-stu-id="f4348-110">In the list, find and select the desired record.</span></span>
5. <span data-ttu-id="f4348-111">Sąraše spustelėkite saitą pasirinktoje eilutėje.</span><span class="sxs-lookup"><span data-stu-id="f4348-111">In the list, click the link in the selected row.</span></span>
6. <span data-ttu-id="f4348-112">Spustelėkite **Eilutės**.</span><span class="sxs-lookup"><span data-stu-id="f4348-112">Click **Lines**.</span></span>
7. <span data-ttu-id="f4348-113">Lauke **Sąskaitos tipas** įveskite reikšmę.</span><span class="sxs-lookup"><span data-stu-id="f4348-113">In the **Account type** field, type a value.</span></span>
8. <span data-ttu-id="f4348-114">Lauke **Aprašo laukas** surinkite reikšmę.</span><span class="sxs-lookup"><span data-stu-id="f4348-114">In the **Description** field, type a value.</span></span>
9. <span data-ttu-id="f4348-115">Lauke **Debetas** įveskite reikšmę.</span><span class="sxs-lookup"><span data-stu-id="f4348-115">In the **Debit** field, type a value.</span></span>
10. <span data-ttu-id="f4348-116">Spustelėkite **Naujas**.</span><span class="sxs-lookup"><span data-stu-id="f4348-116">Click **New**.</span></span>
11. <span data-ttu-id="f4348-117">Lauke **Sąskaitos tipas** įveskite reikšmę.</span><span class="sxs-lookup"><span data-stu-id="f4348-117">In the **Account type** field, type a value.</span></span>
12. <span data-ttu-id="f4348-118">Lauke **Aprašo laukas** surinkite reikšmę.</span><span class="sxs-lookup"><span data-stu-id="f4348-118">In the **Description** field, type a value.</span></span>
13. <span data-ttu-id="f4348-119">Lauke **Debetas** įveskite reikšmę.</span><span class="sxs-lookup"><span data-stu-id="f4348-119">In the **Debit** field, type a value.</span></span>
14. <span data-ttu-id="f4348-120">Spustelėkite **Naujas**.</span><span class="sxs-lookup"><span data-stu-id="f4348-120">Click **New**.</span></span>
14. <span data-ttu-id="f4348-121">Lauke **Sąskaita** nustatykite norimas reikšmes.</span><span class="sxs-lookup"><span data-stu-id="f4348-121">In the **Account** field, specify the desired values.</span></span>
15. <span data-ttu-id="f4348-122">Lauke **Aprašo laukas** surinkite reikšmę.</span><span class="sxs-lookup"><span data-stu-id="f4348-122">In the **Description** field, type a value.</span></span>
16. <span data-ttu-id="f4348-123">Lauke **Kreditas** įveskite reikšmę, subalansuojančią kvitą.</span><span class="sxs-lookup"><span data-stu-id="f4348-123">In the **Credit** field, type a value to balance the voucher.</span></span>
17. <span data-ttu-id="f4348-124">**Veiksmų sritis** spustelėkite **Registruoti**.</span><span class="sxs-lookup"><span data-stu-id="f4348-124">On the **Action pane**, click **Post**.</span></span>
18. <span data-ttu-id="f4348-125">spustelėkite **Funkcijos**.</span><span class="sxs-lookup"><span data-stu-id="f4348-125">Click **Functions**.</span></span>
19. <span data-ttu-id="f4348-126">Spustelėkite šabloną **Įrašyti kvitą**.</span><span class="sxs-lookup"><span data-stu-id="f4348-126">Click **Save voucher** template.</span></span>
20. <span data-ttu-id="f4348-127">Ši procedūra remiasi tipu **Procentų šablonas**.</span><span class="sxs-lookup"><span data-stu-id="f4348-127">This procedure assumes a **Percent Template** type.</span></span> <span data-ttu-id="f4348-128">Spustelėkite Gerai.</span><span class="sxs-lookup"><span data-stu-id="f4348-128">Click OK.</span></span>
    - <span data-ttu-id="f4348-129">Procentas: sumos kvituose yra konvertuojamos į procentų veiksnius, kurie leidžia taikyti bet kurią sumą, kai pasirenkamas kvito šablonas.</span><span class="sxs-lookup"><span data-stu-id="f4348-129">Percent: The amounts in the voucher are converted into percentage factors, which allows any amount to be applied when the Voucher template is selected.</span></span>
    - <span data-ttu-id="f4348-130">Suma: faktinė suma bus saugoma ir taikoma.</span><span class="sxs-lookup"><span data-stu-id="f4348-130">Amount: The actual amounts will be stored and applied.</span></span>  
21. <span data-ttu-id="f4348-131">Spustelėkite **Bendrieji žurnalai**.</span><span class="sxs-lookup"><span data-stu-id="f4348-131">Click **General journals**.</span></span>
22. <span data-ttu-id="f4348-132">Spustelėkite **Naujas**.</span><span class="sxs-lookup"><span data-stu-id="f4348-132">Click **New**.</span></span>
23. <span data-ttu-id="f4348-133">Lauke **Pavadinimas** spustelėkite išplečiamąjį mygtuką, kad atidarytumėte peržvalgą.</span><span class="sxs-lookup"><span data-stu-id="f4348-133">In the **Name** field, click the drop-down button to open the lookup.</span></span>
24. <span data-ttu-id="f4348-134">Sąraše spustelėkite saitą pasirinktoje eilutėje.</span><span class="sxs-lookup"><span data-stu-id="f4348-134">In the list, click the link in the selected row.</span></span>
25. <span data-ttu-id="f4348-135">Spustelėkite **Eilutės**.</span><span class="sxs-lookup"><span data-stu-id="f4348-135">Click **Lines**.</span></span>
26. <span data-ttu-id="f4348-136">spustelėkite **Funkcijos**.</span><span class="sxs-lookup"><span data-stu-id="f4348-136">Click **Functions**.</span></span>
27. <span data-ttu-id="f4348-137">Spustelėkite **Pasirinkti kvito šabloną**.</span><span class="sxs-lookup"><span data-stu-id="f4348-137">Click **Select voucher template**.</span></span>

29. <span data-ttu-id="f4348-141">Lauke **Suma** įveskite piniginę sumą, kuri bus taikoma kvite.</span><span class="sxs-lookup"><span data-stu-id="f4348-141">In the **Amount** field, enter the amount to be applied to the voucher.</span></span> <span data-ttu-id="f4348-142">Laukas **Suma** rodomas, tik jei kvito šablonas yra procento tipo.</span><span class="sxs-lookup"><span data-stu-id="f4348-142">The **Amount** field is only displayed if the voucher template is of type Percent.</span></span>  
30. <span data-ttu-id="f4348-143">Spustelėkite **Gerai**.</span><span class="sxs-lookup"><span data-stu-id="f4348-143">Click **OK**.</span></span>
