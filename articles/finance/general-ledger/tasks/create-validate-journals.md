# <a name="create-and-validate-journals"></a><span data-ttu-id="6061d-103">Žurnalų kūrimas ir tvirtinimas</span><span class="sxs-lookup"><span data-stu-id="6061d-103">Create and validate journals</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="6061d-104">Šioje procedūroje sukuriami ir tikrinami žurnalai ir žurnalo eilutės.</span><span class="sxs-lookup"><span data-stu-id="6061d-104">This procedure creates and validates journals and journal lines.</span></span> <span data-ttu-id="6061d-105">Šią procedūrą galite išbandyti naudodami USMF demonstracinę įmonę.</span><span class="sxs-lookup"><span data-stu-id="6061d-105">You can try this procedure using the USMF demo company.</span></span>  

1. <span data-ttu-id="6061d-106">Eikite į **Didžio knyga > Žurnalo įrašai > Bendrieji žurnalai**.</span><span class="sxs-lookup"><span data-stu-id="6061d-106">Go to **General ledger > Journal entries > General journals**.</span></span>
2. <span data-ttu-id="6061d-107">Spustelėkite **Naujas**.</span><span class="sxs-lookup"><span data-stu-id="6061d-107">Click **New**.</span></span>
3. <span data-ttu-id="6061d-108">Lauke **Pavadinimas** spustelėkite išplečiamąjį mygtuką, kad atidarytumėte peržvalgą.</span><span class="sxs-lookup"><span data-stu-id="6061d-108">In the **Name** field, click the drop-down button to open the lookup.</span></span>
4. <span data-ttu-id="6061d-109">Sąraše raskite ir pasirinkite norimą įrašą.</span><span class="sxs-lookup"><span data-stu-id="6061d-109">In the list, find and select the desired record.</span></span>
5. <span data-ttu-id="6061d-110">Spustelėkite **Eilutės**.</span><span class="sxs-lookup"><span data-stu-id="6061d-110">Click **Lines**.</span></span>
6. <span data-ttu-id="6061d-111">Lauke **Sąskaita** įveskite atitinkamą sąskaitą pagal sąskaitos tipą.</span><span class="sxs-lookup"><span data-stu-id="6061d-111">In the **Account** field enter an appropriate account based on the Account type.</span></span>
7. <span data-ttu-id="6061d-112">Lauke **Aprašo laukas** surinkite reikšmę.</span><span class="sxs-lookup"><span data-stu-id="6061d-112">In the **Description** field, type a value.</span></span>
8. <span data-ttu-id="6061d-113">Įveskite sąskaitos **debeto** arba **kredito** sumą.</span><span class="sxs-lookup"><span data-stu-id="6061d-113">Enter an amount for the account as either a **Debit** or **Credit**.</span></span> 
9. <span data-ttu-id="6061d-114">Lauke **Korespondentinė sąskaita** įveskite atitinkamą sąskaitą pagal korespondentinės sąskaitos tipą.</span><span class="sxs-lookup"><span data-stu-id="6061d-114">In the **Offset account** field, enter an appropriate account based on the Offset account type.</span></span>
10. <span data-ttu-id="6061d-115">Spustelėkite **Tikrinti**.</span><span class="sxs-lookup"><span data-stu-id="6061d-115">Click **Validate**.</span></span>
11. <span data-ttu-id="6061d-116">Spustelėkite **Tikrinti**.</span><span class="sxs-lookup"><span data-stu-id="6061d-116">Click **Validate**.</span></span>
12. <span data-ttu-id="6061d-117">Spustelėkite **Registruoti.**</span><span class="sxs-lookup"><span data-stu-id="6061d-117">Click **Post**.</span></span>
13. <span data-ttu-id="6061d-118">Spustelėkite **Kvitas**.</span><span class="sxs-lookup"><span data-stu-id="6061d-118">Click **Voucher**.</span></span>
