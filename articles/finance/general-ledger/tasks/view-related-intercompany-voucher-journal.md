# <a name="view-related-intercompany-voucher-from-journal"></a>Peržiūrėti susijusį vidinės įmonės kvitą iš žurnalo

[!include [banner](../../includes/banner.md)]

Susijusiame kvito lange kvitas iš korespondentinės įmonės rodomas registruojant vidinės įmonės operaciją iš bendrojo žurnalo.


## <a name="post-an-intercompany-journal"></a>Registruoti vidinės įmonės žurnalą
1. Eiti į **bendruosius žurnalus**, spustelėkite **Naujas**.
2. Sąraše pažymėkite pasirinktą eilutę.
3. Lauke Pavadinimas **įveskite** arba pasirinkite vidinės įmonės žurnalo pavadinimą.
4. Sąraše **spustelėkite** Eilutės, pažymėkite pasirinktą eilutę.
5. Lauke **Sąskaita** nustatykite norimas reikšmes.
6. **Lauke Aprašymas** įveskite arba pasirinkite vertę.
7. Uždarykite puslapį.
8. Lauke **Debetas** įveskite skaičių.
9. Lauke Korespondentinė **įmonė** įveskite arba pasirinkite korespondentinę įmonę.
10. Lauke **Korespondentinė sąskaita** nustatykite norimas reikšmes.
11. Spustelėkite **Registruoti**.

## <a name="view-related-intercompany-voucher"></a>Peržiūrėti susijusį vidinės įmonės kvitą
1. Spustelėkite **Kvitas**.
2. Sąraše **spustelėkite Susiję** kvitai, pažymėkite pasirinktą eilutę.
3. Spustelėkite **Kvitas**.
