# <a name="maintain-financial-dimension-default-templates"></a><span data-ttu-id="1954d-103">Finansinių dimensijų numatytųjų šablonų tvarkymas</span><span class="sxs-lookup"><span data-stu-id="1954d-103">Maintain financial dimension default templates</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="1954d-104">Šioje temoje parodyta, kaip nustatyti finansinės dimensijos numatytuosius šablonus.</span><span class="sxs-lookup"><span data-stu-id="1954d-104">This topic shows how to set up financial dimension default templates.</span></span> <span data-ttu-id="1954d-105">Šablonus galima naudoti norint greitai paskirstyti sumas šaltinio dokumentuose.</span><span class="sxs-lookup"><span data-stu-id="1954d-105">The templates can be used to quickly distribute amounts on source documents.</span></span>

1. <span data-ttu-id="1954d-106">Naršymo srityje eikite į **Moduliai > Didžioji knyga > Sąskaitų planas > Dimensijos > Finansinės dimensijos numatytieji šablonai**.</span><span class="sxs-lookup"><span data-stu-id="1954d-106">In the navigation pane, go to **Modules > General ledger > Chart of accounts > Dimensions > Financial dimension default templates**.</span></span>
2. <span data-ttu-id="1954d-107">Pasirinkite **Naujas**.</span><span class="sxs-lookup"><span data-stu-id="1954d-107">Select **New**.</span></span>
3. <span data-ttu-id="1954d-108">Lauke **Šablono ID** įveskite Šablono ID.</span><span class="sxs-lookup"><span data-stu-id="1954d-108">In the **Template ID** field, enter a Template ID.</span></span>
4. <span data-ttu-id="1954d-109">Įveskite pirmųjų dimensijų reikšmių procentinę dalį.</span><span class="sxs-lookup"><span data-stu-id="1954d-109">Enter the percentage for the first dimension values.</span></span>
5. <span data-ttu-id="1954d-110">Įveskite finansinių dimensijų reikšmes, kurios atitinka įvestą procentinę dalį.</span><span class="sxs-lookup"><span data-stu-id="1954d-110">Enter the financial dimension values that correspond to the percent entered.</span></span>
6. <span data-ttu-id="1954d-111">Pasirinkite **Įtraukti**.</span><span class="sxs-lookup"><span data-stu-id="1954d-111">Select **Add**.</span></span>
7. <span data-ttu-id="1954d-112">Įveskite kitų dimensijų reikšmių procentinę dalį.</span><span class="sxs-lookup"><span data-stu-id="1954d-112">Enter another percent for the next dimension values.</span></span> <span data-ttu-id="1954d-113">Šiame pavyzdyje įveskite procentinę dalį, kad būtų paskirstyta mažiau nei 100 % viso kiekio.</span><span class="sxs-lookup"><span data-stu-id="1954d-113">For this example, enter a percentage so you still have less than 100% allocated.</span></span> 
8. <span data-ttu-id="1954d-114">Įveskite finansinių dimensijų reikšmes, kurios atitinka įvestą procentinę dalį.</span><span class="sxs-lookup"><span data-stu-id="1954d-114">Enter the financial dimension values that correspond to the percent entered.</span></span>
9. <span data-ttu-id="1954d-115">Pasirinkite **Įtraukti**.</span><span class="sxs-lookup"><span data-stu-id="1954d-115">Select **Add**.</span></span>
10. <span data-ttu-id="1954d-116">Įveskite kitą procentinę dalį, kurią pridėjus bendras procentinis dydis būtų lygus 100 %.</span><span class="sxs-lookup"><span data-stu-id="1954d-116">Enter another percent, which will make the total percent equal to 100%.</span></span>
11. <span data-ttu-id="1954d-117">Įveskite finansinių dimensijų reikšmes, kurios atitinka įvestą procentinę dalį.</span><span class="sxs-lookup"><span data-stu-id="1954d-117">Enter the financial dimension values that correspond to the percent entered.</span></span>
12. <span data-ttu-id="1954d-118">Pasirinkite **Įrašyti**.</span><span class="sxs-lookup"><span data-stu-id="1954d-118">Select **Save**.</span></span>
