# <a name="set-up-main-account-categories"></a>Nustatyti pagrindinės sąskaitos kategorijas

[!include [banner](../../includes/banner.md)]

Šioje temoje paaiškinta, kaip nustatyti pagrindinės sąskaitos kategorijas. Pagrindinės sąskaitos kategorijos naudojamos finansinių ataskaitų ir „Power BI“ numatytosioms ataskaitoms. Pagrindines sąskaitos kategorijas, sukurtas pagal numatytuosius nustatymus, galima pervardyti, bet jų negalima panaikinti. Ataskaitų ir analizės tikslams galima sukurti ir naudoti papildomas sąskaitos kategorijas. Šioje temoje aprašoma demonstravimo įmonė USMF.

## <a name="create-a-main-account-category"></a>Pagrindinės sąskaitos kategorijos kūrimas
1. Naršymo srityje eikite į **Moduliai > Didžioji knyga > Sąskaitų planas > Sąskaitos > Pagrindinės sąskaitos kategorijos**.
2. Pasirinkite **Naujas**.
3. Lauke **Pagrindinės sąskaitos kategorija** įveskite unikalų pavadinimą.
4. Lauke **Aprašas** įveskite pagrindinės sąskaitos kategorijos aprašą.
5. Lauke **Pagrindinės sąskaitos tipas** pasirinkite pagrindinės sąskaitos tipą, kuris bus susietas su kategorija.

## <a name="link-main-accounts-to-account-category"></a>Pagrindinių sąskaitų susiejimas su sąskaitos kategorija
1. Spustelėkite **Susieti pagrindines sąskaitas**.
2. Sąraše pasirinkite pagrindines sąskaitas, kurias norite priskirti pagrindinės sąskaitos kategorijai, pažymėdami langelius stulpelyje **Susieta**. Pagrindines sąskaitas priskyrus pagrindinės sąskaitos kategorijai ir naudojant tą kategoriją finansinėms ataskaitoms ir analizei, bus sujungti sąskaitų balansai.  
3. Norėdami pasirinkti pagrindines sąskaitas pažymėkite arba išvalykite parinktį **Susieta**.
4. Pasirinkite **Gerai**.
5. Pasirinkite **Taip**.
