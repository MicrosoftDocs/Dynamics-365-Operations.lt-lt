# <a name="create-consolidation-groups-and-additional-consolidation-accounts"></a>Konsolidavimo grupių ir papildomų konsolidavimo sąskaitų kūrimas

[!include [banner](../../includes/banner.md)]

Ši procedūra nurodo, kaip kurti konsolidacijos sąskaitų grupę ir įtraukti į ją sąskaitas. Šioje procedūroje naudojama demonstracinių duomenų įmonė USMF.


## <a name="create-a-consolidation-account-group"></a>Konsoliduotų sąskaitų grupės kūrimas
1. Pasirinkite Didžioji knyga > Sąskaitų planas > Sąskaitos > Konsolidavimo sąskaitų grupės.
2. Spustelėkite Naujas.
3. Lauke Konsolidavimo sąskaitų grupė įveskite unikalų konsolidavimo sąskaitų grupės identifikatorių.
4. Lauke Pavadinimas surinkite reikšmę.

## <a name="add-accounts-to-consolidation-account-group"></a>Sąskaitų įtraukimas į konsolidavimo sąskaitų grupę
1. Uždarykite puslapį.
2. Pasirinkite Didžioji knyga > Sąskaitų planas > Sąskaitos > Papildomos konsolidavimo sąskaitos.
3. Spustelėkite Naujas.
4. Lauke Pagrindinė sąskaita spustelėkite išplečiamąjį mygtuką, kad atidarytumėte peržvalgą.
5. Sąraše spustelėkite pagrindinę sąskaitą, kurią norite susieti.
6. Lauke Konsolidavimo sąskaitų grupė spustelėkite išplečiamąjį mygtuką, kad atidarytumėte peržvalgą.
7. Sąraše spustelėkite konsolidavimo sąskaitų grupę.
8. Lauke Konsolidavimo sąskaita įveskite reikšmę.
9. Lauke Konsolidavimo sąskaitos pavadinimas įveskite reikšmę.
