# <a name="invoice-capture-received-files"></a>SF fiksavimo gauti failai

[!include [banner](../includes/banner.md)]
[!include [preview banner](../includes/preview-banner.md)]

SF fiksavime gautų failų **puslapis yra** centrinė vieta, kurioje SF failai gaunami iš skirtingų šaltinių.

## <a name="upload-invoice-files"></a>Nusiųsti SF failus

Mokėtinų sumų (AP) tarnautojai gali įkelti SF vaizdus **pasirinkdami Nusiųsti** failą ir atsidarę **dialogo** langą Įkelti. Kai AP klerkas pasirenka failą, mygtukas **Įkelti** tampa galimi.

## <a name="include-or-obsolete-invoice-files"></a>Įtraukti arba nebenaudojami SF failai

Vartotojai gali pasirinkti SF, kuriuose yra klaidų arba įspėjimų, tada įtraukti arba nebenaudojami SF, pasirinkdami atitinkamą mygtuką veiksmų srityje. Gautą failą (nebenaudojamas) rodinyje **nebus rodoma SF, kuri pažymėta kaip pasenusi**.

## <a name="view-captured-invoices"></a>Peržiūrėti užfiksuotas SF

Sėkmingai atpažinę gautą failą, užfiksuota SF informacija grąžinama AI modelyje ir įvedama Microsoft Dataverse į. AP klerkas gali patikrinti SF informaciją, pasirinkdamas **Peržiūrėti užfiksuotas SF**. Vartotojai gali, jei reikia, atsisiųsti originalios SF failą.
