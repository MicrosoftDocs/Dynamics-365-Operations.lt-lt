# <a name="assign-a-product-lifecycle-state-to-a-released-product"></a>Produkto ciklo būsenos priskyrimas išleistam produktui

[!include [banner](../../includes/banner.md)]

Ši procedūra nurodo, kaip priskirti produkto ciklo būseną išleistam produktui. Būtina sąlyga: pirmiausia turite paleisti užduočių vedlį „Kurti naują produkto ciklo būseną“, kad įsitikintumėte, jog prieš paleidžiant šį užduočių vedlį yra sukurta bent viena produkto ciklo būsena.


## <a name="find-a-released-product-that-you-want-to-assign-a-product-lifcycle-state"></a>Rasti išleistą produktą, kuriam norima priskirti produkto ciklo būseną
1. Eikite į Produkto informacijos valdymas > Produktai > Patvirtinti produktai.
2. Norėdami rasti įrašus, naudokite spartųjį filtrą. Pvz., filtruokite lauką Prekės numeris, įvesdami reikšmę „d“.
3. Sąraše spustelėkite saitą pasirinktoje eilutėje.
4. Spustelėkite Redaguoti.
5. Lauke Produkto ciklo būsena įveskite arba pasirinkite reikšmę.
6. Spustelėkite Įrašyti.
