# <a name="create-a-bar-code-for-a-product"></a><span data-ttu-id="2f4d6-103">Produkto brūkšninio kodo kūrimas</span><span class="sxs-lookup"><span data-stu-id="2f4d6-103">Create a bar code for a product</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="2f4d6-104">Šioje procedūroje parodoma, kaip neautomatiniu būdu kurti brūkšninį kodą naudojant prekės numerį M0001 kaip pavyzdį.</span><span class="sxs-lookup"><span data-stu-id="2f4d6-104">This topic shows how to manually create a bar code using the item number M0001 as an example.</span></span> <span data-ttu-id="2f4d6-105">Kuriant šią procedūrą naudojama demonstracinių duomenų įmonė yra USMF.</span><span class="sxs-lookup"><span data-stu-id="2f4d6-105">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="2f4d6-106">Eikite į **Produkto informacijos valdymas \> Produktai \> Išleisti produktai**.</span><span class="sxs-lookup"><span data-stu-id="2f4d6-106">Go to **Product information management \> Products \> Released products**.</span></span>
1. <span data-ttu-id="2f4d6-107">Sąraše raskite ir pasirinkite norimą įrašą.</span><span class="sxs-lookup"><span data-stu-id="2f4d6-107">In the list, find and select the desired record.</span></span> <span data-ttu-id="2f4d6-108">Šiame pavyzdyje spustelėkite prekės numerį **M0001**.</span><span class="sxs-lookup"><span data-stu-id="2f4d6-108">For this example, select item number **M0001**.</span></span>
1. <span data-ttu-id="2f4d6-109">Veiksmų srityje spustelėkite **Valdyti atsargas**.</span><span class="sxs-lookup"><span data-stu-id="2f4d6-109">On the Action Pane, select **Manage inventory**.</span></span>
1. <span data-ttu-id="2f4d6-110">Pasirinkite **Brūkšniniai kodai**.</span><span class="sxs-lookup"><span data-stu-id="2f4d6-110">Select **Bar codes**.</span></span>
1. <span data-ttu-id="2f4d6-111">Pasirinkite **Naujas**.</span><span class="sxs-lookup"><span data-stu-id="2f4d6-111">Select **New**.</span></span>
1. <span data-ttu-id="2f4d6-112">Pažymėkite pasirinktą eilutę, sukurtą toliau pateiktame sąraše.</span><span class="sxs-lookup"><span data-stu-id="2f4d6-112">Mark the selected row that is created in the list below.</span></span>
1. <span data-ttu-id="2f4d6-113">Lauke **Brūkšninio kodo nustatymas** įveskite arba pasirinkite reikšmę.</span><span class="sxs-lookup"><span data-stu-id="2f4d6-113">In the **Barcode setup** field, enter or select a value.</span></span>
1. <span data-ttu-id="2f4d6-114">Lauke **Brūkšninis kodas** įveskite arba pasirinkite reikšmę.</span><span class="sxs-lookup"><span data-stu-id="2f4d6-114">In the **Bar code** field, enter or select a value.</span></span>
1. <span data-ttu-id="2f4d6-115">Lauke **Brūkšninis kodas** įveskite reikšmę.</span><span class="sxs-lookup"><span data-stu-id="2f4d6-115">In the **Bar code** field, type a value.</span></span>  
1. <span data-ttu-id="2f4d6-116">Lauke **Kiekis** įveskite skaičių.</span><span class="sxs-lookup"><span data-stu-id="2f4d6-116">In the **Quantity** field, enter a number.</span></span>
1. <span data-ttu-id="2f4d6-117">Pasirinkite **Įrašyti**.</span><span class="sxs-lookup"><span data-stu-id="2f4d6-117">Select **Save**.</span></span>
1. <span data-ttu-id="2f4d6-118">Uždarykite puslapį.</span><span class="sxs-lookup"><span data-stu-id="2f4d6-118">Close the page.</span></span>
