# <a name="create-a-bar-code-for-a-product"></a><span data-ttu-id="8259c-103">Produkto brūkšninio kodo kūrimas</span><span class="sxs-lookup"><span data-stu-id="8259c-103">Create a bar code for a product</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="8259c-104">Šioje procedūroje parodoma, kaip neautomatiniu būdu kurti brūkšninį kodą naudojant prekės numerį M0001 kaip pavyzdį.</span><span class="sxs-lookup"><span data-stu-id="8259c-104">This topic shows how to manually create a bar code using the item number M0001 as an example.</span></span> <span data-ttu-id="8259c-105">Kuriant šią procedūrą naudojama demonstracinių duomenų įmonė yra USMF.</span><span class="sxs-lookup"><span data-stu-id="8259c-105">The demo data company used to create this procedure is USMF.</span></span>

1. <span data-ttu-id="8259c-106">Pagrindiniame puslapyje pasirinkite **Išleisto produkto priežiūra**.</span><span class="sxs-lookup"><span data-stu-id="8259c-106">Select **Released product maintenance** on the homepage.</span></span>
2. <span data-ttu-id="8259c-107">Skyriuje **Saitai** eikite į **Produktai > Patvirtinti produktai**.</span><span class="sxs-lookup"><span data-stu-id="8259c-107">Go to **Products > Released products** under the **Links** section.</span></span>
3. <span data-ttu-id="8259c-108">Sąraše raskite ir pasirinkite norimą įrašą.</span><span class="sxs-lookup"><span data-stu-id="8259c-108">In the list, find and select the desired record.</span></span> <span data-ttu-id="8259c-109">Šiame pavyzdyje spustelėkite prekės numerį **M0001**.</span><span class="sxs-lookup"><span data-stu-id="8259c-109">For this example, select item number **M0001**.</span></span>
4. <span data-ttu-id="8259c-110">Veiksmų srityje spustelėkite **Valdyti atsargas**.</span><span class="sxs-lookup"><span data-stu-id="8259c-110">On the Action Pane, select **Manage inventory**.</span></span>
5. <span data-ttu-id="8259c-111">Pasirinkite **Brūkšniniai kodai**.</span><span class="sxs-lookup"><span data-stu-id="8259c-111">Select **Bar codes**.</span></span>
6. <span data-ttu-id="8259c-112">Pasirinkite **Naujas**.</span><span class="sxs-lookup"><span data-stu-id="8259c-112">Select **New**.</span></span>
7. <span data-ttu-id="8259c-113">Pažymėkite pasirinktą eilutę, sukurtą toliau pateiktame sąraše.</span><span class="sxs-lookup"><span data-stu-id="8259c-113">Mark the selected row that is created in the list below.</span></span>
8. <span data-ttu-id="8259c-114">Lauke **Brūkšninio kodo nustatymas** įveskite arba pasirinkite reikšmę.</span><span class="sxs-lookup"><span data-stu-id="8259c-114">In the **Barcode setup** field, enter or select a value.</span></span>
9. <span data-ttu-id="8259c-115">Lauke **Brūkšninis kodas** įveskite arba pasirinkite reikšmę.</span><span class="sxs-lookup"><span data-stu-id="8259c-115">In the **Bar code** field, enter or select a value.</span></span>
10. <span data-ttu-id="8259c-116">Lauke **Brūkšninis kodas** įveskite reikšmę.</span><span class="sxs-lookup"><span data-stu-id="8259c-116">In the **Bar code** field, type a value.</span></span>  
11. <span data-ttu-id="8259c-117">Lauke **Kiekis** įveskite skaičių.</span><span class="sxs-lookup"><span data-stu-id="8259c-117">In the **Quantity** field, enter a number.</span></span>
12. <span data-ttu-id="8259c-118">Pasirinkite **Įrašyti**.</span><span class="sxs-lookup"><span data-stu-id="8259c-118">Select **Save**.</span></span>
13. <span data-ttu-id="8259c-119">Uždarykite puslapį.</span><span class="sxs-lookup"><span data-stu-id="8259c-119">Close the page.</span></span>
