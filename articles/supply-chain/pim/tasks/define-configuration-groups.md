# <a name="define-configuration-groups"></a>Apibrėžti konfigūracijos grupes

[!include [banner](../../includes/banner.md)]

Ši procedūra nurodo, kaip nustatyti produkto konfigūracijos grupes sukuriant dvi grupes demonstracinių duomenų įmonėje „USMF Company“. Tai yra ketvirtoji iš aštuonių procedūrų, kuriomis paaiškinama, kaip kurti konfigūravimo pagal dimensijas kombinacijas.

1. Pasirinkite Produkto informacijos valdymas > Nustatymas > Komplektavimo specifikacijos ir formulės > Konfigūracijos grupės.
2. Spustelėkite Naujas.
3. Lauke „Konfigūracijos grupė“ įveskite reikšmę.
    * Įveskite konfigūracijos grupės pavadinimą, kuris bus naudojama KS medžiagų eilutėse.  
4. Lauke Pavadinimas surinkite reikšmę.
5. Spustelėkite Naujas.
6. Lauke „Konfigūracijos grupė“ įveskite reikšmę.
    * Įveskite konfigūracijos grupės pavadinimą, kuris bus naudojama KS medžiagų eilutėse.  
7. Lauke Pavadinimas surinkite reikšmę.
8. Uždarykite puslapį.
