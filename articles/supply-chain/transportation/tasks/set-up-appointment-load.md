# <a name="set-up-an-appointment-for-a-load"></a>Apkrovos paskyros nustatymas

[!include [banner](../../includes/banner.md)]

Šioje procedūroje parodoma, kaip nustatyti ir planuoti krovinio rampos paskyrą. Paprastai tą daro transportavimo koordinatorius. Šią procedūrą galite naudoti USMF demonstracinių duomenų įmonėje.


## <a name="select-the-load"></a>Krovinio pasirinkimas
1. Pasirinkite Transportavimo valdymas > Planavimas > Krovinio planavimo darbo sritis.
2. Išvalykite žymės langelį Slėpti išsiųstus ir gautus.
3. Sąraše pasirinkite krovinį, kurio būsena Išsiųsta.
4. Spustelėkite Transportavimas.
5. Spustelėkite Paskyros planavimas.

## <a name="create-an-appointment"></a>Paskyros kūrimas
1. Spustelėkite Naujas.
2. Lauke Paskyros taisyklė įveskite arba pasirinkite reikšmę.
3. Spustelėkite Įrašyti.
4. Spustelėkite Atnaujinti būseną.
5. Spustelėkite Patvirtinti.
6. Spustelėkite Įrašyti.
7. Uždarykite puslapį.
