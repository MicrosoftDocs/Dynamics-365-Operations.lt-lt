[!include [banner](../../includes/banner.md)]

<span data-ttu-id="f10ed-104">Laikui bėgant atsargų operacijų lentelė (`InventTrans`) toliau plečiasi ir sunaudoja daugiau duomenų bazės vietos.</span><span class="sxs-lookup"><span data-stu-id="f10ed-104">Over time, the inventory transactions table (`InventTrans`) will continue to grow and consume more database space.</span></span> <span data-ttu-id="f10ed-105">Todėl dėl lentelės pateiktos užklausos palaipsniui veiks lėčiau.</span><span class="sxs-lookup"><span data-stu-id="f10ed-105">Therefore, queries that are made against the table will gradually become slower.</span></span> <span data-ttu-id="f10ed-106">Šioje temoje aprašoma, kaip galima naudoti atsargų operacijų archyvo priemonę *duomenims apie atsargų operacijas archyvuoti* siekiant pagerinti sistemos našumą.</span><span class="sxs-lookup"><span data-stu-id="f10ed-106">This topic describes how you can use the *Inventory transactions archive* feature to archive data about inventory transactions to help improve system performance.</span></span>

> [!NOTE]
