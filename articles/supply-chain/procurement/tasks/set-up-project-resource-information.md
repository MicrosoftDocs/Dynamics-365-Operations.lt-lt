# <a name="set-up-project-resource-information"></a>Nustatyti projekto išteklių informaciją

[!INCLUDE [task guide banner](../../includes/task-guide-banner.md)]

Šioje procedūroje parodoma, kaip nustatyti projekto išteklius. Šioje užduotyje naudojamas USMF duomenų rinkinys.

1. Eikite į Projektų valdymas ir apskaita > Projekto ištekliai > Išteklių sąrašas.
2. Spustelėkite Nauji ištekliai.
3. Lauke Darbuotojas įveskite arba pasirinkite reikšmę.
4. Lauke Kalendorius įveskite arba pasirinkite reikšmę.
5. Lauke Laikotarpio kodas įveskite arba pasirinkite reikšmę.
6. Spustelėkite Kurti.
7. Spustelėkite Savikainos.
8. Spustelėkite Savikaina – valandos.
9. Spustelėkite Naujas.
10. Lauke Įsigaliojimo data įveskite datą.
11. Lauke Savikaina įveskite skaičių.
12. Spustelėkite Įrašyti.
13. Uždarykite puslapį.
14. Spustelėkite Pardavimo kainos.
15. Spustelėkite Pardavimo kaina – valanda.
16. Spustelėkite Naujas.
17. Lauke Įsigaliojimo data įveskite datą.
18. Lauke Kainos įveskite skaičių.
19. Lauke Galioja pasirinkite parinktį.
20. Spustelėkite Įrašyti.
21. Uždarykite puslapį.
