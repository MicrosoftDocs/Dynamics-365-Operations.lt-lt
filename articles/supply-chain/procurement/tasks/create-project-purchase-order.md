# <a name="create-project-purchase-order"></a><span data-ttu-id="203bc-103">Kurti projekto pirkimo užsakymą</span><span class="sxs-lookup"><span data-stu-id="203bc-103">Create project purchase order</span></span>

[!include [banner](../../includes/banner.md)]

<span data-ttu-id="203bc-104">Šioje procedūroje parodoma, kaip kurti projekto pirkimo užsakymą.</span><span class="sxs-lookup"><span data-stu-id="203bc-104">This procedure shows you how to create a project purchase order.</span></span> <span data-ttu-id="203bc-105">Šioje užduotyje naudojamas USSI duomenų rinkinys.</span><span class="sxs-lookup"><span data-stu-id="203bc-105">This task uses the USSI data set.</span></span>

1. <span data-ttu-id="203bc-106">Eikite į Projektų valdymas ir apskaita > Projektai > Visi projektai.</span><span class="sxs-lookup"><span data-stu-id="203bc-106">Go to Project management and accounting > Projects > All projects.</span></span>
2. <span data-ttu-id="203bc-107">Sąraše spustelėkite saitą pasirinktoje eilutėje.</span><span class="sxs-lookup"><span data-stu-id="203bc-107">In the list, click the link in the selected row.</span></span>
3. <span data-ttu-id="203bc-108">Veiksmų srityje spustelėkite Valdyti.</span><span class="sxs-lookup"><span data-stu-id="203bc-108">On the Action Pane, click Manage.</span></span>
4. <span data-ttu-id="203bc-109">Spustelėkite „Prekės užduotis“.</span><span class="sxs-lookup"><span data-stu-id="203bc-109">Click Item task.</span></span>
5. <span data-ttu-id="203bc-110">Spustelėkite Pirkimo užsakymas.</span><span class="sxs-lookup"><span data-stu-id="203bc-110">Click Purchase order.</span></span>
6. <span data-ttu-id="203bc-111">Lauke Tiekėjo sąskaita įveskite arba pasirinkite vertę.</span><span class="sxs-lookup"><span data-stu-id="203bc-111">In the Vendor account field, enter or select a value.</span></span>
7. <span data-ttu-id="203bc-112">Lauke Teritorija įveskite arba pasirinkite reikšmę.</span><span class="sxs-lookup"><span data-stu-id="203bc-112">In the Site field, enter or select a value.</span></span>
    * <span data-ttu-id="203bc-113">Šių veiksmų nereikia, bet jie supaprastina pirkimo užsakymą nustatant pirkimo užsakymo eilučių numatytąją vietą ir sandėlį.</span><span class="sxs-lookup"><span data-stu-id="203bc-113">These steps aren't required, but they do simplify the purchase order by setting up a default site and warehouse for the purchase order lines.</span></span>  
8. <span data-ttu-id="203bc-114">Lauke Sandėlis įveskite arba pasirinkite reikšmę.</span><span class="sxs-lookup"><span data-stu-id="203bc-114">In the Warehouse field, enter or select a value.</span></span>
9. <span data-ttu-id="203bc-115">Spustelėkite GERAI.</span><span class="sxs-lookup"><span data-stu-id="203bc-115">Click OK.</span></span>
10. <span data-ttu-id="203bc-116">Sąraše pažymėkite pasirinktą eilutę.</span><span class="sxs-lookup"><span data-stu-id="203bc-116">In the list, mark the selected row.</span></span>
11. <span data-ttu-id="203bc-117">Lauke Prekės numeris įveskite arba pasirinkite reikšmę.</span><span class="sxs-lookup"><span data-stu-id="203bc-117">In the Item number field, enter or select a value.</span></span>
    * <span data-ttu-id="203bc-118">Tai gali būti prekės numeris arba įsigijimo kategorija.</span><span class="sxs-lookup"><span data-stu-id="203bc-118">This can be the item number or a procurement category.</span></span>  
12. <span data-ttu-id="203bc-119">Išplėskite skyrių Eilutės informacija.</span><span class="sxs-lookup"><span data-stu-id="203bc-119">Expand the Line details section.</span></span>
13. <span data-ttu-id="203bc-120">Spustelėkite skirtuką Projektas.</span><span class="sxs-lookup"><span data-stu-id="203bc-120">Click the Project tab.</span></span>
    * <span data-ttu-id="203bc-121">Patikrinkite, ar yra pardavimo kaina ir savikaina.</span><span class="sxs-lookup"><span data-stu-id="203bc-121">Verify that the sales and cost prices are available.</span></span> <span data-ttu-id="203bc-122">Jei jų nėra, bet reikia, įveskite informaciją.</span><span class="sxs-lookup"><span data-stu-id="203bc-122">If they are not available but needed, enter the information.</span></span>  
14. <span data-ttu-id="203bc-123">Spustelėkite Įrašyti.</span><span class="sxs-lookup"><span data-stu-id="203bc-123">Click Save.</span></span>
