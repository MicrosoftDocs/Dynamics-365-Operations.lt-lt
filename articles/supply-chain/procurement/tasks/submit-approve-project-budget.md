[!include [banner](../../includes/banner.md)]

<span data-ttu-id="f615d-104">Šioje procedūroje nurodoma, kaip sukurti ir pateikti projekto biudžetą.</span><span class="sxs-lookup"><span data-stu-id="f615d-104">This procedure shows you how to create and submit the budget for a project.</span></span> 

<span data-ttu-id="f615d-105">Kai kuriate projekto biudžetą, galite įvesti įvertintas projekto įplaukas ir išlaidas, o tada naudoti jas faktinėms projekto operacijoms valdyti.</span><span class="sxs-lookup"><span data-stu-id="f615d-105">When you create a project budget, you can enter estimated revenues and costs for a project, and then use those to control actual project transactions.</span></span> <span data-ttu-id="f615d-106">Sudarant projekto biudžetą visi pradiniai biudžetai ir tikslinimai turi būti siunčiami į projekto darbo eigą patvirtinti.</span><span class="sxs-lookup"><span data-stu-id="f615d-106">In project budgeting, all original budgets and revisions must be sent to project workflow for approval.</span></span> <span data-ttu-id="f615d-107">Darbo eiga suteikia didesnę proceso kontrolę ir sukuria keitimų istorijos įrašą.</span><span class="sxs-lookup"><span data-stu-id="f615d-107">Workflow gives you increased control over the process and creates a change history record.</span></span>

<span data-ttu-id="f615d-108">Ši užduotis buvo sukurta naudojant USSI duomenų rinkinį.</span><span class="sxs-lookup"><span data-stu-id="f615d-108">This task was created using the USSI data set.</span></span>

1. <span data-ttu-id="f615d-109">**Naršymo srityje** eikite į **Moduliai > Projekto valdymas ir apskaita > Projektai > Visi projektai**.</span><span class="sxs-lookup"><span data-stu-id="f615d-109">In the **Navigation pane**, go to **Modules > Project management and accounting > Projects > All projects**.</span></span>
2. <span data-ttu-id="f615d-110">Sąraše raskite ir pasirinkite norimą įrašą.</span><span class="sxs-lookup"><span data-stu-id="f615d-110">In the list, find and select the desired record.</span></span>
3. <span data-ttu-id="f615d-111">Sąraše spustelėkite saitą pasirinktoje eilutėje.</span><span class="sxs-lookup"><span data-stu-id="f615d-111">In the list, click the link in the selected row.</span></span>
4. <span data-ttu-id="f615d-112">Parinktyje **Veiksmų sritis** spustelėkite **Planas**.</span><span class="sxs-lookup"><span data-stu-id="f615d-112">On the **Action Pane**, click **Plan**.</span></span>
5. <span data-ttu-id="f615d-113">Spustelėkite **Projekto biudžetas**.</span><span class="sxs-lookup"><span data-stu-id="f615d-113">Click **Project budget**.</span></span>
6. <span data-ttu-id="f615d-114">Lauke **Aprašo laukas** surinkite reikšmę.</span><span class="sxs-lookup"><span data-stu-id="f615d-114">In the **Description** field, type a value.</span></span>
7. <span data-ttu-id="f615d-115">Išplėskite „fastTab“ **Išlaidos**.</span><span class="sxs-lookup"><span data-stu-id="f615d-115">Expand the **Cost** fastTab.</span></span>
8. <span data-ttu-id="f615d-116">Spustelėkite **Naujas**.</span><span class="sxs-lookup"><span data-stu-id="f615d-116">Click **New**.</span></span>
9. <span data-ttu-id="f615d-117">Lauke **Operacijos tipas** pažymėkite parinktį.</span><span class="sxs-lookup"><span data-stu-id="f615d-117">In the **Transaction type** field, select an option.</span></span>
10. <span data-ttu-id="f615d-118">Lauke **Kategorija** įveskite arba pasirinkite reikšmę.</span><span class="sxs-lookup"><span data-stu-id="f615d-118">In the **Category** field, enter or select a value.</span></span>
11. <span data-ttu-id="f615d-119">Lauke **Pradinis biudžetas** įveskite skaičių.</span><span class="sxs-lookup"><span data-stu-id="f615d-119">In the **Original budget** field, enter a number.</span></span>
12. <span data-ttu-id="f615d-120">Išplėskite „fastTab“ **Įplaukos**.</span><span class="sxs-lookup"><span data-stu-id="f615d-120">Expand the **Revenues** fastTab.</span></span>
13. <span data-ttu-id="f615d-121">Spustelėkite **Naujas**.</span><span class="sxs-lookup"><span data-stu-id="f615d-121">Click **New**.</span></span>
14. <span data-ttu-id="f615d-122">Sąraše pažymėkite pasirinktą eilutę.</span><span class="sxs-lookup"><span data-stu-id="f615d-122">In the list, mark the selected row.</span></span>
15. <span data-ttu-id="f615d-123">Lauke **Operacijos tipas** pažymėkite parinktį.</span><span class="sxs-lookup"><span data-stu-id="f615d-123">In the **Transaction type** field, select an option.</span></span>
16. <span data-ttu-id="f615d-124">Lauke **Kategorija** įveskite arba pasirinkite reikšmę.</span><span class="sxs-lookup"><span data-stu-id="f615d-124">In the **Category** field, enter or select a value.</span></span>
17. <span data-ttu-id="f615d-125">Lauke **Pradinis biudžetas** įveskite skaičių.</span><span class="sxs-lookup"><span data-stu-id="f615d-125">In the **Original budget** field, enter a number.</span></span>
18. <span data-ttu-id="f615d-126">Spustelėkite **Įrašyti**.</span><span class="sxs-lookup"><span data-stu-id="f615d-126">Click **Save**.</span></span>
19. <span data-ttu-id="f615d-127">Spustelėkite **Darbo eiga**.</span><span class="sxs-lookup"><span data-stu-id="f615d-127">Click **Workflow**.</span></span>
20. <span data-ttu-id="f615d-128">Spustelėkite **Pateikti**.</span><span class="sxs-lookup"><span data-stu-id="f615d-128">Click **Submit**.</span></span>
21. <span data-ttu-id="f615d-129">Lauke **Komentaras** įveskite reikšmę.</span><span class="sxs-lookup"><span data-stu-id="f615d-129">In the **Comment** field, type a value.</span></span>
22. <span data-ttu-id="f615d-130">Spustelėkite **Pateikti**.</span><span class="sxs-lookup"><span data-stu-id="f615d-130">Click **Submit**.</span></span>
