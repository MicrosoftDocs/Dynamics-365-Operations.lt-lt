# <a name="set-up-subscription-groups"></a><span data-ttu-id="4a204-103">Nustatyti abonementų grupes</span><span class="sxs-lookup"><span data-stu-id="4a204-103">Set up subscription groups</span></span> 

[!include [banner](../includes/banner.md)]


1.  <span data-ttu-id="4a204-104">Spustelėkite **Aptarnavimo valdymas** \> **Sąranka** \> **Aptarnavimo abonementai** \> **Abonementų grupės**.</span><span class="sxs-lookup"><span data-stu-id="4a204-104">Click **Service management** \> **Setup** \> **Service subscriptions** \> **Subscription groups**.</span></span>

2.  <span data-ttu-id="4a204-105">Norėdami sukurti naują abonementų grupę, paspauskite CTRL+N arba spustelėkite **Nauja**.</span><span class="sxs-lookup"><span data-stu-id="4a204-105">Press Ctrl+N or click **New** to create a new subscription group.</span></span>

3.  <span data-ttu-id="4a204-106">Lauke **Abonementų grupė** įveskite naujos abonementų grupės pavadinimą.</span><span class="sxs-lookup"><span data-stu-id="4a204-106">In the **Subscription group** field, enter a name for the new subscription group.</span></span>

4.  <span data-ttu-id="4a204-107">Lauke **SF išrašymas** pasirinkite abonementų SF išrašymo dažnumą.</span><span class="sxs-lookup"><span data-stu-id="4a204-107">In the **Invoicing** field, select an invoicing frequency for the subscription.</span></span> <span data-ttu-id="4a204-108">SF dažnumas nurodomas pagal laikotarpio kodą, kuris nustatytas formoje **Laikotarpių tipai**.</span><span class="sxs-lookup"><span data-stu-id="4a204-108">The invoicing frequency is indicated by a period code that is set up in the **Period types** form.</span></span>

5.  <span data-ttu-id="4a204-109">Norėdami kaupti įplaukas už abonemento operacijas, pažymėkite žymės langelį **Kaupti įplaukas**, po to lauke **Kaupimas** pasirinkite dažnumą.</span><span class="sxs-lookup"><span data-stu-id="4a204-109">To accrue revenue for subscription transactions, select the **Accrue revenue** check box, and then in the **Accrual** field, select a frequency.</span></span> <span data-ttu-id="4a204-110">Kaupimo dažnumas paremtas laikotarpio kodu, kuris nustatytas formoje **Laikotarpiai**.</span><span class="sxs-lookup"><span data-stu-id="4a204-110">The accrual frequency is based on a period code that is set up in the **Periods** form.</span></span>
