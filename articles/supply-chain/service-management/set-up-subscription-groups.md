[!include [banner](../includes/banner.md)]


1.  <span data-ttu-id="7a72d-104">Spustelėkite **Aptarnavimo valdymas** \> **Sąranka** \> **Aptarnavimo abonementai** \> **Abonementų grupės**.</span><span class="sxs-lookup"><span data-stu-id="7a72d-104">Click **Service management** \> **Setup** \> **Service subscriptions** \> **Subscription groups**.</span></span>

2.  <span data-ttu-id="7a72d-105">Norėdami sukurti naują abonementų grupę, paspauskite CTRL+N arba spustelėkite **Nauja**.</span><span class="sxs-lookup"><span data-stu-id="7a72d-105">Press Ctrl+N or click **New** to create a new subscription group.</span></span>

3.  <span data-ttu-id="7a72d-106">Lauke **Abonementų grupė** įveskite naujos abonementų grupės pavadinimą.</span><span class="sxs-lookup"><span data-stu-id="7a72d-106">In the **Subscription group** field, enter a name for the new subscription group.</span></span>

5.  <span data-ttu-id="7a72d-109">Norėdami kaupti įplaukas už abonemento operacijas, pažymėkite žymės langelį **Kaupti įplaukas**, po to lauke **Kaupimas** pasirinkite dažnumą.</span><span class="sxs-lookup"><span data-stu-id="7a72d-109">To accrue revenue for subscription transactions, select the **Accrue revenue** check box, and then in the **Accrual** field, select a frequency.</span></span> <span data-ttu-id="7a72d-110">Kaupimo dažnumas paremtas laikotarpio kodu, kuris nustatytas formoje **Laikotarpiai**.</span><span class="sxs-lookup"><span data-stu-id="7a72d-110">The accrual frequency is based on a period code that is set up in the **Periods** form.</span></span>
