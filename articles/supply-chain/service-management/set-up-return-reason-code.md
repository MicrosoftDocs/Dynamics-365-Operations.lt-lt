<span data-ttu-id="198cd-110">Ši procedūra suteikia galimybę nustatyti grąžinimo priežasčių kodus, kuriuos galite naudoti, kad nurodytumėte, kodėl klientas grąžino produktą.</span><span class="sxs-lookup"><span data-stu-id="198cd-110">This procedure helps you set up return reason codes that you can use to indicate why a product was returned by the customer.</span></span>
