[!include [banner](../includes/banner.md)]


<span data-ttu-id="34b69-104">Norėdami apdoroti grąžinimą, pirmiausia patikrinkite grąžinimo kiekį, atnaujinkite kiekio lauką prekių gavimo žurnale.</span><span class="sxs-lookup"><span data-stu-id="34b69-104">To process a return, first validate the return quantity, update the quantity field in the item arrival journal.</span></span> <span data-ttu-id="34b69-105">Tada pasirinkite perdavimo kodą arba nurodykite, kad grąžintos prekės turi būti tikrinamos.</span><span class="sxs-lookup"><span data-stu-id="34b69-105">Then select a disposition code or indicate that the returned items have to be inspected.</span></span> <span data-ttu-id="34b69-106">Atlikę šiuos veiksmus, galite registruoti grąžinimo užsakymo prekių gavimo žurnalą.</span><span class="sxs-lookup"><span data-stu-id="34b69-106">After completing these steps, you can post the item arrival journal for the return order.</span></span>

1.  <span data-ttu-id="34b69-107">Spustelėkite **Atsargų valdymas** \> **Laikotarpio** \> **Gavimų apžvalga**.</span><span class="sxs-lookup"><span data-stu-id="34b69-107">Click **Inventory management** \> **Periodic** \> **Arrival overview**.</span></span>

3.  <span data-ttu-id="34b69-109">Jei gavimų sąrašas yra ilgas, norėdami sutrumpinti sąrašą, naudokite srities **Diapazonas** laukus.</span><span class="sxs-lookup"><span data-stu-id="34b69-109">If the list of receipts is long, use the fields in the **Range** area to narrow the list.</span></span>

4.  <span data-ttu-id="34b69-110">Nustatykite grąžinimo užsakymo eilutę, kurią norite registruoti, pažymėkite jos langelį **Pasirinkti gavimui**, tada spustelėkite **Pradėti gavimą**.</span><span class="sxs-lookup"><span data-stu-id="34b69-110">Locate the line of the return order that you want to post, select its **Select for arrival** box, and then click **Start arrival**.</span></span>

    > [!TIP]
    > <P><span data-ttu-id="34b69-112">Norėdami peržiūrėti išsamią informaciją, pasirinkite žurnalą, tada spustelėkite <STRONG>Eilutės</STRONG>.</span><span class="sxs-lookup"><span data-stu-id="34b69-112">To view detailed information, select a journal, and then click <STRONG>Lines</STRONG>.</span></span></P>

<span data-ttu-id="34b69-114">Užregistravus žurnalą, grąžintos prekės registruojamos į atsargas, o formoje **Grąžinimo užsakymai** nurodoma, kad prekės buvo pristatytos į sandėlį.</span><span class="sxs-lookup"><span data-stu-id="34b69-114">After the journal is posted, the returned items are registered in inventory, and the **Return orders** form indicates that the items have arrived at the warehouse.</span></span>
