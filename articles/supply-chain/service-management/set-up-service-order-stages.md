[!include [banner](../includes/banner.md)]


1.  <span data-ttu-id="4c17c-104">Spustelėkite **Aptarnavimo valdymas** \> **Sąranka** \> **Aptarnavimo užsakymai** \> **Aptarnavimo etapai**.</span><span class="sxs-lookup"><span data-stu-id="4c17c-104">Click **Service management** \> **Setup** \> **Service orders** \> **Service stages**.</span></span>

2.  <span data-ttu-id="4c17c-105">Norėdami sukurti naują įrašą, paspauskite CTRL+N.</span><span class="sxs-lookup"><span data-stu-id="4c17c-105">Press CTRL+N to create a new record.</span></span>

3.  <span data-ttu-id="4c17c-106">Lauke **Aptarnavimo etapas** ir **Aprašas** nurodykite aptarnavimo etapo ID ir aprašą.</span><span class="sxs-lookup"><span data-stu-id="4c17c-106">In the **Service stage** and **Description** fields, specify a service stage ID and description.</span></span>

4.  <span data-ttu-id="4c17c-107">Pasirinkite atitinkamus etapo parametrus.</span><span class="sxs-lookup"><span data-stu-id="4c17c-107">Select the appropriate parameters for the stage.</span></span>

5.  <span data-ttu-id="4c17c-108">Pasirinkite esamo etapo pirminį etapą arba lauką **Pirminis** palikite tuščią, jei etapas yra pirminis etapo sąrankos etapas.</span><span class="sxs-lookup"><span data-stu-id="4c17c-108">Select the parent stage for the current stage or leave the **Parent** field blank if the stage is the initial stage in the stage setup.</span></span>
