# <a name="create-service-subscriptions-from-a-project"></a><span data-ttu-id="b5e85-103">Aptarnavimo abonemento pagal projektą kūrimas</span><span class="sxs-lookup"><span data-stu-id="b5e85-103">Create service subscriptions from a project</span></span>    

<span data-ttu-id="b5e85-104">[!įtraukti [reklaminę juostą](../includes/banner.md)</span><span class="sxs-lookup"><span data-stu-id="b5e85-104">[!include [banner](../includes/banner.md)</span></span>


<span data-ttu-id="b5e85-105">Galite sukurti aptarnavimo abonementą pagal projektą dalyje **Projektų valdymas ir apskaita**.</span><span class="sxs-lookup"><span data-stu-id="b5e85-105">You can create a service subscription from a project in **Project management and accounting**.</span></span>

1.  <span data-ttu-id="b5e85-106">Spustelėkite **Projektų valdymas ir apskaita** \> **Bendra** \> **Projektai** \> **Visi projektai**.</span><span class="sxs-lookup"><span data-stu-id="b5e85-106">Click **Project management and accounting** \> **Common** \> **Projects** \> **All projects**.</span></span> <span data-ttu-id="b5e85-107">Atidarykite projektą, kurio abonementą norite sukurti.</span><span class="sxs-lookup"><span data-stu-id="b5e85-107">Open the project that you want to create a subscription for.</span></span>

2.  <span data-ttu-id="b5e85-108">Formos **Projektai**, esančios **Veiksmų sritis**, skirtuke **Valdyti** spustelėkite **Aptarnavimas** \> **Aptarnavimo abonementas**.</span><span class="sxs-lookup"><span data-stu-id="b5e85-108">On the **Projects** form, on the **Action Pane**, on the **Manage** tab, click **Service** \> **Service subscription**.</span></span>

4.  <span data-ttu-id="b5e85-111">Formoje įveskite likusią informaciją.</span><span class="sxs-lookup"><span data-stu-id="b5e85-111">Enter the remaining information in the form.</span></span>

<span data-ttu-id="b5e85-112">Dabar galite kurti abonemento abonementines operacijas.</span><span class="sxs-lookup"><span data-stu-id="b5e85-112">You can now create subscription transactions for the subscription.</span></span>
