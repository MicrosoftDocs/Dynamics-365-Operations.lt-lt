# <a name="create-service-subscriptions-from-a-project"></a><span data-ttu-id="6af21-103">Aptarnavimo abonemento pagal projektą kūrimas</span><span class="sxs-lookup"><span data-stu-id="6af21-103">Create service subscriptions from a project</span></span>    

[!include [banner](../includes/banner.md)]


<span data-ttu-id="6af21-104">Galite sukurti aptarnavimo abonementą pagal projektą dalyje **Projektų valdymas ir apskaita**.</span><span class="sxs-lookup"><span data-stu-id="6af21-104">You can create a service subscription from a project in **Project management and accounting**.</span></span>

1.  <span data-ttu-id="6af21-105">Spustelėkite **Projektų valdymas ir apskaita** \> **Bendra** \> **Projektai** \> **Visi projektai**.</span><span class="sxs-lookup"><span data-stu-id="6af21-105">Click **Project management and accounting** \> **Common** \> **Projects** \> **All projects**.</span></span> <span data-ttu-id="6af21-106">Atidarykite projektą, kurio abonementą norite sukurti.</span><span class="sxs-lookup"><span data-stu-id="6af21-106">Open the project that you want to create a subscription for.</span></span>

2.  <span data-ttu-id="6af21-107">Formos **Projektai**, esančios **Veiksmų sritis**, skirtuke **Valdyti** spustelėkite **Aptarnavimas** \> **Aptarnavimo abonementas**.</span><span class="sxs-lookup"><span data-stu-id="6af21-107">On the **Projects** form, on the **Action Pane**, on the **Manage** tab, click **Service** \> **Service subscription**.</span></span>

4.  <span data-ttu-id="6af21-110">Formoje įveskite likusią informaciją.</span><span class="sxs-lookup"><span data-stu-id="6af21-110">Enter the remaining information in the form.</span></span>

<span data-ttu-id="6af21-111">Dabar galite kurti abonemento abonementines operacijas.</span><span class="sxs-lookup"><span data-stu-id="6af21-111">You can now create subscription transactions for the subscription.</span></span>


> [!NOTE]
> <P><span data-ttu-id="6af21-112">Jei nustatėte abonementų grupės numerių seką, nurodant abonementų grupę abonemento ID generuojamas automatiškai.</span><span class="sxs-lookup"><span data-stu-id="6af21-112">If you have set up a number sequence for the subscription group, a subscription ID is generated automatically for the subscription when you specify the subscription group.</span></span></P>
