# <a name="set-up-service-intervals"></a><span data-ttu-id="5bdde-103">Aptarnavimo intervalų nustatymas</span><span class="sxs-lookup"><span data-stu-id="5bdde-103">Set up service intervals</span></span>  

[!include [banner](../includes/banner.md)]

<span data-ttu-id="5bdde-104">Aptarnavimo intervalas nurodo dažnumą, kokiu kuriamos aptarnavimo sutarčių eilučių aptarnavimo užsakymo eilutės, kai kuriate aptarnavimo užsakymus.</span><span class="sxs-lookup"><span data-stu-id="5bdde-104">Service interval indicates the frequency with which service order lines are created for service agreement lines when you create service orders.</span></span>

1. <span data-ttu-id="5bdde-105">Spustelėkite **Aptarnavimo valdymas** \> **Sąranka** \> **Aptarnavimo sutartys** \> **Aptarnavimo intervalai**.</span><span class="sxs-lookup"><span data-stu-id="5bdde-105">Click **Service management** \> **Setup** \> **Service agreements** \> **Service intervals**.</span></span>
2. <span data-ttu-id="5bdde-106">Sukurkite naują aptarnavimo intervalą.</span><span class="sxs-lookup"><span data-stu-id="5bdde-106">Create a new service interval.</span></span>
3. <span data-ttu-id="5bdde-107">Įveskite aptarnavimo intervalo ID ir aprašymą.</span><span class="sxs-lookup"><span data-stu-id="5bdde-107">Enter the ID and description of the service interval.</span></span>
4. <span data-ttu-id="5bdde-108">Pasirinkite diapazoną lauke **Diapazonas**.</span><span class="sxs-lookup"><span data-stu-id="5bdde-108">In the **Range** field, select the range.</span></span>
5. <span data-ttu-id="5bdde-109">Lauke **Dažnumas** įrašykite dažnumą.</span><span class="sxs-lookup"><span data-stu-id="5bdde-109">In the **Frequency** field, type the frequency.</span></span> <span data-ttu-id="5bdde-110">Dažnumas yra koeficientas, iš kurio turite dauginti diapazoną, kad gautumėte aptarnavimo sutarties intervalą.</span><span class="sxs-lookup"><span data-stu-id="5bdde-110">The frequency is the factor by which you must multiply the range to obtain the interval for a service agreement.</span></span>
6. <span data-ttu-id="5bdde-111">Norėdami įrašyti aptarnavimo intervalą, paspauskite **ALT + S**.</span><span class="sxs-lookup"><span data-stu-id="5bdde-111">Press **Alt+S** to save the service interval.</span></span>
