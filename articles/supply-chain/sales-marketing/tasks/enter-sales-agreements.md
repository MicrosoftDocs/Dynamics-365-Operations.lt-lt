4. <span data-ttu-id="1a86e-110">Lauke **Pardavimo sutarties klasifikacija** pasirinkite norimą įrašą iš išplečiamojo meniu.</span><span class="sxs-lookup"><span data-stu-id="1a86e-110">In the **Sales agreement classification** field, select the desired record from the drop-down menu.</span></span>
5. <span data-ttu-id="1a86e-111">Išplėskite skyrių **Bendra**.</span><span class="sxs-lookup"><span data-stu-id="1a86e-111">Expand the **General** section.</span></span>
6. <span data-ttu-id="1a86e-112">Lauke **Numatytasis įsipareigojimas** pasirinkite **Produkto vertės įsipareigojimas**.</span><span class="sxs-lookup"><span data-stu-id="1a86e-112">In the **Default commitment** field, select **Product value commitment**.</span></span> <span data-ttu-id="1a86e-113">Įsipareigojimo tipas yra privalomas kriterijus, kurį turite priskirti sutarčiai, kad nustatytumėte, kaip sutartis bus įvykdyta.</span><span class="sxs-lookup"><span data-stu-id="1a86e-113">A commitment type is a mandatory criterion that you must assign to the agreement to define how the agreement contract will be fulfilled.</span></span> <span data-ttu-id="1a86e-114">Naudojant keturis iš anksto nustatytus tipus galima nustatyti kliento įsipareigojimo tikslą, kuris išreiškiamas kaip kiekis arba vertė.</span><span class="sxs-lookup"><span data-stu-id="1a86e-114">The four predefined types let you set up the customer's commitment target, expressed as a quantity or a value.</span></span> <span data-ttu-id="1a86e-115">Kiekio įsipareigojimo tipą galima taikyti tik konkrečiam produktui, bet verte grįstus tipus galima taikyti parduodant konkrečius ir nekonkrečius produktus.</span><span class="sxs-lookup"><span data-stu-id="1a86e-115">The quantity commitment type can only be applied to a specific product, but the value-based types are applicable to sales of both specific and non-specific products.</span></span>

## <a name="set-up-product-value-commitment-lines"></a><span data-ttu-id="1a86e-118">Produkto vertės įsipareigojimo eilučių nustatymas</span><span class="sxs-lookup"><span data-stu-id="1a86e-118">Set up product value commitment lines</span></span>
1. <span data-ttu-id="1a86e-119">Pasirinkite **Pridėti eilutę**.</span><span class="sxs-lookup"><span data-stu-id="1a86e-119">Select **Add line**.</span></span>
2. <span data-ttu-id="1a86e-120">Lauke **Elemento numeris** išplečiamajame meniu pasirinkite norimą įrašą.</span><span class="sxs-lookup"><span data-stu-id="1a86e-120">In the **Item number** field, select the desired record from the drop-down menu.</span></span> <span data-ttu-id="1a86e-121">Pasirinktas sutarties įsipareigojimo tipas įtakoja, kokią informaciją galite įvesti sutarties eilutėse.</span><span class="sxs-lookup"><span data-stu-id="1a86e-121">The type of commitment that you have chosen for the agreement affects the kind of information you can enter for the agreement lines.</span></span> <span data-ttu-id="1a86e-122">Pvz., jei sutartis pagrįsta verte, turite nurodyti bendrą grynąją sumą (sutarta valiuta), už kurią klientas įsipareigoja nusipirkti iš jūsų prekių.</span><span class="sxs-lookup"><span data-stu-id="1a86e-122">For example, for a value-based agreement you must specify the total net amount (in the agreed currency) for which the customer commits to buys goods from you.</span></span> <span data-ttu-id="1a86e-123">Šiame pavyzdyje laukai **Kiekis** ir **Vienetas** nepasiekiami eilutėje, nes kuriate sutartį klientui nusipirkti tam tikrą produkto vertę.</span><span class="sxs-lookup"><span data-stu-id="1a86e-123">In this example the **Quantity** and **Unit** fields on the line are unavailable because you're creating an agreement for the customer to buy a specific value of a product.</span></span>   
3. <span data-ttu-id="1a86e-124">Lauke **Grynoji suma** įveskite piniginę sumą, už kurią klientas įsipareigoja nusipirkti.</span><span class="sxs-lookup"><span data-stu-id="1a86e-124">In the **Net amount** field, enter the monetary amount that the customer has committed to buying.</span></span>
4. <span data-ttu-id="1a86e-125">Lauke **Nuolaidos procentas** įveskite procentinę reikšmę, kuri bus taikoma kliento pardavimo užsakymo eilutėms, susietoms su šia sutartimi.</span><span class="sxs-lookup"><span data-stu-id="1a86e-125">In the **Discount percent** field, enter a percentage value that will apply to the customer's sales order lines that are linked to this agreement.</span></span>
5. <span data-ttu-id="1a86e-126">Išplėskite skyrių **Eilutės informacija** section.</span><span class="sxs-lookup"><span data-stu-id="1a86e-126">Expand the **Line details** section.</span></span>
6. <span data-ttu-id="1a86e-127">Pasirinkite **Taip** lauke **Maksimaliai vykdoma**.</span><span class="sxs-lookup"><span data-stu-id="1a86e-127">Select **Yes** in the **Max is enforced** field.</span></span>
    - <span data-ttu-id="1a86e-128">Pasirinkimas **Maksimaliai vykdoma** reiškia, kad bendra visų pardavimo užsakymo eilučių, kurios naudoja specialias įsipareigojimo kainas, nuolaidas ir (arba) mokėjimo sąlygas, suma neturi viršyti įsipareigojime nurodytos sumos.</span><span class="sxs-lookup"><span data-stu-id="1a86e-128">Selecting **Max is enforced** means that the total amount of all the sales order lines that use the commitment's special prices, discounts and/or payment terms must not exceed the amount specified on the commitment.</span></span>  
    - <span data-ttu-id="1a86e-129">Mažiausia ir didžiausia išleidimo sumos nurodo vertės, už kurią reikia parduoti prekių vykdant kiekvieną pasirinktos sutarties pardavimo užsakymą, intervalą.</span><span class="sxs-lookup"><span data-stu-id="1a86e-129">The minimum and maximum release amounts specify a range of values that must be sold on each sales order that uses the selected agreement.</span></span>   
7. <span data-ttu-id="1a86e-130">Išplėskite skyrių **Pardavimo sutarties antraštė**.</span><span class="sxs-lookup"><span data-stu-id="1a86e-130">Expand the **Sales agreement header** section.</span></span> <span data-ttu-id="1a86e-131">Jei sutarties būsena nėra nustatyta į **Galioja**, pardavimo užsakymai negali būti susieti su sutartimi ir dėl to negali prisidėti prie sutarties įvykdymo.</span><span class="sxs-lookup"><span data-stu-id="1a86e-131">Unless the status of the agreement is set to **Effective**, sales orders cannot be associated with the agreement and can therefore not contribute to the fulfilment of that agreement.</span></span> <span data-ttu-id="1a86e-132">Šio etapo metu būseną galite neautomatiškai pakeisti.</span><span class="sxs-lookup"><span data-stu-id="1a86e-132">You can change the status manually at this stage.</span></span> <span data-ttu-id="1a86e-133">Tačiau įprastai būsena keičiama tvirtinant kliento sutartį.</span><span class="sxs-lookup"><span data-stu-id="1a86e-133">However, the status would normally be changed when you confirm the agreement for the customer.</span></span>  
8. <span data-ttu-id="1a86e-134">Veiksmų srityje pasirinkite **Pardavimo sutartis**.</span><span class="sxs-lookup"><span data-stu-id="1a86e-134">On the Action Pane, select **Sales agreement**.</span></span>
9. <span data-ttu-id="1a86e-135">Pasirinkite **Patvirtinimas**.</span><span class="sxs-lookup"><span data-stu-id="1a86e-135">Select **Confirmation**.</span></span> <span data-ttu-id="1a86e-136">Įsitikinkite, kad parinktis **Pažymėti sutartį kaip galiojančią** nustatyta į **Taip**.</span><span class="sxs-lookup"><span data-stu-id="1a86e-136">Make sure that the **Mark agreement as effective** option is set to **Yes**.</span></span>  
10. <span data-ttu-id="1a86e-137">Pasirinkite **Taip** lauke **Spausdinti ataskaitą**.</span><span class="sxs-lookup"><span data-stu-id="1a86e-137">Select **Yes** in the **Print report** field.</span></span>
11. <span data-ttu-id="1a86e-138">Pasirinkite **Gerai**.</span><span class="sxs-lookup"><span data-stu-id="1a86e-138">Select **OK**.</span></span>
12. <span data-ttu-id="1a86e-139">Uždarykite puslapį.</span><span class="sxs-lookup"><span data-stu-id="1a86e-139">Close the page.</span></span> <span data-ttu-id="1a86e-140">Dabar sutartis galioja.</span><span class="sxs-lookup"><span data-stu-id="1a86e-140">The agreement is now effective.</span></span> <span data-ttu-id="1a86e-141">Galite pradėti sieti kliento užsakymus su sutartimi, kad subalansuotumėte atliktą tikslą.</span><span class="sxs-lookup"><span data-stu-id="1a86e-141">You can start linking the customer's orders to the agreement to offset against the committed target.</span></span>
