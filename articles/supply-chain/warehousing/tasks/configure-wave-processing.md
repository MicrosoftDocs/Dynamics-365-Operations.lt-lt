# <a name="configure-wave-processing"></a><span data-ttu-id="b9210-103">Bangos apdorojimo konfigūravimas</span><span class="sxs-lookup"><span data-stu-id="b9210-103">Configure wave processing</span></span>

[!include [task guide banner](../../includes/task-guide-banner.md)]

<span data-ttu-id="b9210-104">Šiame vadove aprašoma, kaip nustatyti kriterijus, pagal kuriuos nustatoma, koks darbas sugeneruojamas sandėliui apdorojus bangą ir ar bangos apdorojamos rankiniu būdu, ar automatiškai.</span><span class="sxs-lookup"><span data-stu-id="b9210-104">This guide describes how to set up the criteria that determine what work is generated for a warehouse when a wave is processed, and whether waves are processed manually or automatically.</span></span> <span data-ttu-id="b9210-105">Kriterijus galite nurodyti nustatydami bangos šablonus ir užklausas, kurios sugretina bangą su išleistais pardavimo užsakymais, gamybos užsakymais arba „kanban“ užsakymais.</span><span class="sxs-lookup"><span data-stu-id="b9210-105">You specify the criteria by setting up wave templates and queries that match a wave with released lines in sales orders, production orders, or kanban orders.</span></span> <span data-ttu-id="b9210-106">Bangos apdorojimas naudojamas tuose sandėliuose, kurie šią funkciją naudoja modulyje Sandėlio valdymas, o tuose, kurie šią funkciją naudoja modulyje Atsargų valdymas, jis nenaudojamas.</span><span class="sxs-lookup"><span data-stu-id="b9210-106">Wave processing is used in warehouses that use the functionality in the Warehouse management module, and not those that use the functionality in the Inventory management module.</span></span> <span data-ttu-id="b9210-107">Šią procedūrą galite vykdyti demonstracinių duomenų įmonėje USMF.</span><span class="sxs-lookup"><span data-stu-id="b9210-107">You can run this procedure in demo data company USMF.</span></span>

1. <span data-ttu-id="b9210-108">Pasirinkite Sandėlio valdymas > Nustatymas > Bangos > Bangų šablonai.</span><span class="sxs-lookup"><span data-stu-id="b9210-108">Go to Warehouse management > Setup > Waves > Wave templates.</span></span>
2. <span data-ttu-id="b9210-109">Spustelėkite Naujas.</span><span class="sxs-lookup"><span data-stu-id="b9210-109">Click New.</span></span>
3. <span data-ttu-id="b9210-110">Lauke Bangos šablono pavadinimas įveskite reikšmę.</span><span class="sxs-lookup"><span data-stu-id="b9210-110">In the Wave template name field, type a value.</span></span>
    * <span data-ttu-id="b9210-111">Kai nustatote bangos šabloną, nurodote seką, kuria šablonai bus gretinami su išleistomis pardavimo užsakymų, gamybos užsakymų ar „kanban‟ eilutėmis.</span><span class="sxs-lookup"><span data-stu-id="b9210-111">When you set up a wave template, you specify the sequence in which the templates will be matched to released lines on sales orders, production orders, or kanbans.</span></span> <span data-ttu-id="b9210-112">Kai eilutė išleidžiama į sandėlį ar į gamybą, ji naudoja pirmąjį bangos šabloną, kurio kriterijus atitinka.</span><span class="sxs-lookup"><span data-stu-id="b9210-112">When a line is released to the warehouse or to production, it uses the first wave template that it meets the criteria for.</span></span> <span data-ttu-id="b9210-113">Rekomenduojame padėti šablonus su specifiškiausiais kriterijais sąrašo viršuje.</span><span class="sxs-lookup"><span data-stu-id="b9210-113">We recommend that you put templates with the most specific criteria at the top of the list.</span></span> <span data-ttu-id="b9210-114">Kuo platesni kriterijai, tuo didesnė tikimybė, kad eilutė atitiks kriterijus, todėl eilutės gali būti priskirtos netinkamai bangai.</span><span class="sxs-lookup"><span data-stu-id="b9210-114">The broader the criteria, the more likely it is for a line to meet the criteria, and this could lead to lines being assigned to the wrong wave.</span></span>  
4. <span data-ttu-id="b9210-115">Lauke Bangos šablono aprašas įveskite reikšmę.</span><span class="sxs-lookup"><span data-stu-id="b9210-115">In the Wave template description field, type a value.</span></span>
5. <span data-ttu-id="b9210-116">Lauke Teritorija įveskite arba pasirinkite reikšmę.</span><span class="sxs-lookup"><span data-stu-id="b9210-116">In the Site field, enter or select a value.</span></span>
    * <span data-ttu-id="b9210-117">Jei naudojate USMF, galite pasirinkti 2-ąją vietą.</span><span class="sxs-lookup"><span data-stu-id="b9210-117">If you’re using USMF, you can select site 2.</span></span>  
6. <span data-ttu-id="b9210-118">Lauke Sandėlis įveskite arba pasirinkite reikšmę.</span><span class="sxs-lookup"><span data-stu-id="b9210-118">In the Warehouse field, enter or select a value.</span></span>
    * <span data-ttu-id="b9210-119">Jei naudojate USMF, galite pasirinkti 24-ąjį sandėlį.</span><span class="sxs-lookup"><span data-stu-id="b9210-119">If you’re using USMF, you can select warehouse 24.</span></span>  
7. <span data-ttu-id="b9210-120">Lauką Automatiškai kurti bangą nustatykite į Taip.</span><span class="sxs-lookup"><span data-stu-id="b9210-120">Set the Automate wave creation field to Yes.</span></span>
    * <span data-ttu-id="b9210-121">Pasirinkite šią parinktį, kad banga būtų automatiškai kuriama į sandėlį išleidus pardavimo užsakymą, gamybos užsakymą arba „kanban“.</span><span class="sxs-lookup"><span data-stu-id="b9210-121">Select this option to automatically create a wave when a sales order, production order, or kanban is released to the warehouse.</span></span>  
8. <span data-ttu-id="b9210-122">Parinktį Apdoroti bangą išleidžiant ją į sandėlį nustatykite į Taip.</span><span class="sxs-lookup"><span data-stu-id="b9210-122">Set the Process wave at release to warehouse option to Yes.</span></span> 
    * <span data-ttu-id="b9210-123">Pasirinkite šią parinktį, kad būtų automatiškai apdorojama banga ir sukuriamas darbas į sandėlį išleidus eilutę.</span><span class="sxs-lookup"><span data-stu-id="b9210-123">Select this option to automatically process the wave and create work when a line is released to the warehouse.</span></span>  
9. <span data-ttu-id="b9210-124">Parinktį Automatizuoti bangos išleidimą nustatykite į Taip.</span><span class="sxs-lookup"><span data-stu-id="b9210-124">Set the Automate wave release option to Yes.</span></span> 
    * <span data-ttu-id="b9210-125">Pasirinkus šią parinktį bus automatiškai išleidžiama banga.</span><span class="sxs-lookup"><span data-stu-id="b9210-125">Select this option to automatically release the wave.</span></span> <span data-ttu-id="b9210-126">Paėmimo darbas sukuriamas ir prieinamas mobiliuosiuose įrenginiuose.</span><span class="sxs-lookup"><span data-stu-id="b9210-126">The picking work is created and made available on mobile devices.</span></span>  
10. <span data-ttu-id="b9210-127">Parinktį Priskirti atidaryti bangas nustatykite į Taip.</span><span class="sxs-lookup"><span data-stu-id="b9210-127">Set the Assign to open waves option to Yes.</span></span> 
    * <span data-ttu-id="b9210-128">Eilutės priskiriamos bangoms pagal bangos šablono užklausos filtrą.</span><span class="sxs-lookup"><span data-stu-id="b9210-128">Lines are assigned to waves based on the query filter for the wave template.</span></span>  
11. <span data-ttu-id="b9210-129">Parinktį Apdoroti bangą automatiškai ties ribine reikšme nustatykite į Taip.</span><span class="sxs-lookup"><span data-stu-id="b9210-129">Set the Process wave automatically at threshold option to Yes.</span></span> 
    * <span data-ttu-id="b9210-130">Pasirinkite šią parinktį, kad banga būtų automatiškai apdorojama jos vertėms pasiekus svorio, siuntos ir laukų grupėje Bangos slenksčiai nurodytų eilučių slenksčius.</span><span class="sxs-lookup"><span data-stu-id="b9210-130">Select this option to automatically process the wave when its values reach the thresholds for weight, shipment, and lines specified in the Wave thresholds field group.</span></span> <span data-ttu-id="b9210-131">Šią parinktį galima naudoti tik lauke Bangos šablono tipas pasirinkus Siunta.</span><span class="sxs-lookup"><span data-stu-id="b9210-131">This option is available only if Shipping is selected in the Wave template type field.</span></span>  
12. <span data-ttu-id="b9210-132">Parinktį Automatiškai leisti papildymo darbą nustatykite į Taip.</span><span class="sxs-lookup"><span data-stu-id="b9210-132">Set the Automate replenishment work release option to Yes.</span></span> 
    * <span data-ttu-id="b9210-133">Pasirinkus šią parinktį bus automatiškai sukuriamas ir išleidžiamas papildymo darbas pagal poreikį.</span><span class="sxs-lookup"><span data-stu-id="b9210-133">Select this option to create demand-based replenishment work and release it automatically.</span></span> <span data-ttu-id="b9210-134">Turite įtraukti papildymo bangos metodą į bangos šabloną ir sukurti tipo Bangos poreikis papildymo šabloną.</span><span class="sxs-lookup"><span data-stu-id="b9210-134">You must add the replenishment wave method to the wave template, and create a replenishment template of the type Wave demand.</span></span>  
13. <span data-ttu-id="b9210-135">Išplėskite dalį Būdai.</span><span class="sxs-lookup"><span data-stu-id="b9210-135">Expand the Methods section.</span></span>
    * <span data-ttu-id="b9210-136">Bangos šablonų būdais galima valdyti veiklų, per kurias praeina kiekviena apdorojama banga, seką.</span><span class="sxs-lookup"><span data-stu-id="b9210-136">Wave template methods allow you to control the sequence of activities that each wave is going through when it’s processed.</span></span> <span data-ttu-id="b9210-137">Pavyzdžiui, galite turėti bangų papildymo būdą.</span><span class="sxs-lookup"><span data-stu-id="b9210-137">For example, you might have a method for wave replenishment.</span></span> <span data-ttu-id="b9210-138">Kai pridedate metodą, jis automatiškai pateikiamas atitinkamoje vietoje pagal sekos veiksmus.</span><span class="sxs-lookup"><span data-stu-id="b9210-138">When you add a method, it’s automatically listed in the appropriate location in the sequence of steps.</span></span> <span data-ttu-id="b9210-139">Jei parinktį Automatiškai leisti papildymo darbą nustatėte į Taip, čia turite įtraukti papildymo būdą.</span><span class="sxs-lookup"><span data-stu-id="b9210-139">If you’ve set the Automate replenishment work release option to Yes, you need to add the replenish method here.</span></span>  
    * <span data-ttu-id="b9210-140">Bangų atributai veikia kaip filtrai – jie riboja, kokios prekės gali naudoti bangą.</span><span class="sxs-lookup"><span data-stu-id="b9210-140">Wave attributes act as filters, to restrict the kind of items that can use the wave.</span></span> <span data-ttu-id="b9210-141">Pavyzdžiui, galite nurodyti prekių grupę.</span><span class="sxs-lookup"><span data-stu-id="b9210-141">For example, you could specify an item group.</span></span>  
14. <span data-ttu-id="b9210-142">Spustelėkite Įrašyti.</span><span class="sxs-lookup"><span data-stu-id="b9210-142">Click Save.</span></span>
15. <span data-ttu-id="b9210-143">Uždarykite puslapį.</span><span class="sxs-lookup"><span data-stu-id="b9210-143">Close the page.</span></span>
16. <span data-ttu-id="b9210-144">Pasirinkite Sandėlio valdymas > Nustatymas > Sandėlio valdymo parametrai.</span><span class="sxs-lookup"><span data-stu-id="b9210-144">Go to Warehouse management > Setup > Warehouse management parameters.</span></span>
17. <span data-ttu-id="b9210-145">Išplėskite dalį Bangos apdorojimas.</span><span class="sxs-lookup"><span data-stu-id="b9210-145">Expand the Wave processing section.</span></span>
18. <span data-ttu-id="b9210-146">Lauke Bangos apdorojimo paketų grupė įveskite arba pasirinkite reikšmę.</span><span class="sxs-lookup"><span data-stu-id="b9210-146">In the Wave processing batch group field, enter or select a value.</span></span>
19. <span data-ttu-id="b9210-147">Parinktį Apdoroti bangas kaip paketinę užduotį nustatykite į Taip.</span><span class="sxs-lookup"><span data-stu-id="b9210-147">Set the Process waves in batch option to Yes.</span></span>
20. <span data-ttu-id="b9210-148">Lauke Laukti užrakto (ms) įveskite skaičių.</span><span class="sxs-lookup"><span data-stu-id="b9210-148">In the Wait for lock (ms) field, enter a number.</span></span>
    * <span data-ttu-id="b9210-149">Įveskite laiką (milisekundėmis), kurį paskirstymo veiksmas lauks sistemos ištekliaus, kuris užrakintas kito paskirstymo veiksmo.</span><span class="sxs-lookup"><span data-stu-id="b9210-149">Enter the time, in milliseconds, that an allocation step will wait for a system resource that is locked by another allocation step.</span></span> <span data-ttu-id="b9210-150">Viršijus laiką, banga neapdorojama ir rodomas klaidos pranešimas.</span><span class="sxs-lookup"><span data-stu-id="b9210-150">When this time is exceeded, the wave is not processed and an error message is displayed.</span></span>  
21. <span data-ttu-id="b9210-151">Spustelėkite Įrašyti.</span><span class="sxs-lookup"><span data-stu-id="b9210-151">Click Save.</span></span>
22. <span data-ttu-id="b9210-152">Uždarykite puslapį.</span><span class="sxs-lookup"><span data-stu-id="b9210-152">Close the page.</span></span>
23. <span data-ttu-id="b9210-153">Pasirinkite Gamybos kontrolė > Nustatymas > Gamybos kontrolės parametrai.</span><span class="sxs-lookup"><span data-stu-id="b9210-153">Go to Production control > Setup > Production control parameters.</span></span>
24. <span data-ttu-id="b9210-154">Lauke Išleisti į sandėlį pasirinkite parinktį.</span><span class="sxs-lookup"><span data-stu-id="b9210-154">In the Release to warehouse field, select an option.</span></span>
    * <span data-ttu-id="b9210-155">Pardavimo užsakymų ir „kanban“ užsakymų atsargos turi būti rezervuotos prie išleidžiant užsakymą į sandėlį.</span><span class="sxs-lookup"><span data-stu-id="b9210-155">For sales orders and kanban orders, inventory must be reserved before the order is released to the warehouse.</span></span> <span data-ttu-id="b9210-156">Kitu atveju prekės arba paskirstymo eilutės negalės būti apdorojamos bangoje.</span><span class="sxs-lookup"><span data-stu-id="b9210-156">Otherwise, the items or allocation lines cannot be processed in a wave.</span></span> <span data-ttu-id="b9210-157">Gamybos užsakymuose taip pat galite pasirinkti Leisti dalinį rezervavimą.</span><span class="sxs-lookup"><span data-stu-id="b9210-157">For production orders, you also have the option of choosing Allow partial reservation.</span></span> <span data-ttu-id="b9210-158">Pvz., tai naudinga, jei turite medžiagų, kurių reikia, kad galėtumėte pradėti gamybą, ir galite palaukti, kol taps pasiekiamos papildomos medžiagos procesui baigti.</span><span class="sxs-lookup"><span data-stu-id="b9210-158">For example, this is useful if you have the materials that you need to start production, and can then wait until the additional materials become available to finish the process.</span></span> <span data-ttu-id="b9210-159">Jei pasirinksite šią parinktį, turėsite rankiniu būdu kartoti išleidimo į sandėlį procesą, kai taps pasiekiama papildomų medžiagų.</span><span class="sxs-lookup"><span data-stu-id="b9210-159">If you select this option, you must manually repeat the release to warehouse process when the additional materials become available.</span></span>  
25. <span data-ttu-id="b9210-160">Uždarykite puslapį.</span><span class="sxs-lookup"><span data-stu-id="b9210-160">Close the page.</span></span>
