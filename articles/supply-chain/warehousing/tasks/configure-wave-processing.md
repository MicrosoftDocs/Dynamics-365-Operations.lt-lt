[!include [banner](../../includes/banner.md)]

<span data-ttu-id="5cd5b-104">Šiame vadove aprašoma, kaip nustatyti kriterijus, pagal kuriuos nustatoma, koks darbas sugeneruojamas sandėliui apdorojus bangą ir ar bangos apdorojamos rankiniu būdu, ar automatiškai.</span><span class="sxs-lookup"><span data-stu-id="5cd5b-104">This guide describes how to set up the criteria that determine what work is generated for a warehouse when a wave is processed, and whether waves are processed manually or automatically.</span></span> <span data-ttu-id="5cd5b-105">Kriterijus galite nurodyti nustatydami bangos šablonus ir užklausas, kurios sugretina bangą su išleistais pardavimo užsakymais, gamybos užsakymais arba „kanban“ užsakymais.</span><span class="sxs-lookup"><span data-stu-id="5cd5b-105">You specify the criteria by setting up wave templates and queries that match a wave with released lines in sales orders, production orders, or kanban orders.</span></span> <span data-ttu-id="5cd5b-106">Bangos apdorojimas naudojamas tuose sandėliuose, kurie šią funkciją naudoja modulyje Sandėlio valdymas, o tuose, kurie šią funkciją naudoja modulyje Atsargų valdymas, jis nenaudojamas.</span><span class="sxs-lookup"><span data-stu-id="5cd5b-106">Wave processing is used in warehouses that use the functionality in the Warehouse management module, and not those that use the functionality in the Inventory management module.</span></span> <span data-ttu-id="5cd5b-107">Šią procedūrą galite vykdyti demonstracinių duomenų įmonėje USMF.</span><span class="sxs-lookup"><span data-stu-id="5cd5b-107">You can run this procedure in demo data company USMF.</span></span>

2. <span data-ttu-id="5cd5b-109">Spustelėkite **Naujas**.</span><span class="sxs-lookup"><span data-stu-id="5cd5b-109">Click **New**.</span></span>
3. <span data-ttu-id="5cd5b-110">Lauke **Bangos šablono pavadinimas** įveskite reikšmę.</span><span class="sxs-lookup"><span data-stu-id="5cd5b-110">In the **Wave template name** field, type a value.</span></span> <span data-ttu-id="5cd5b-111">Kai nustatote bangos šabloną, nurodote seką, kuria šablonai bus gretinami su išleistomis pardavimo užsakymų, gamybos užsakymų ar „kanban‟ eilutėmis.</span><span class="sxs-lookup"><span data-stu-id="5cd5b-111">When you set up a wave template, you specify the sequence in which the templates will be matched to released lines on sales orders, production orders, or kanbans.</span></span> <span data-ttu-id="5cd5b-112">Kai eilutė išleidžiama į sandėlį ar į gamybą, ji naudoja pirmąjį bangos šabloną, kurio kriterijus atitinka.</span><span class="sxs-lookup"><span data-stu-id="5cd5b-112">When a line is released to the warehouse or to production, it uses the first wave template that it meets the criteria for.</span></span> <span data-ttu-id="5cd5b-113">Rekomenduojame padėti šablonus su specifiškiausiais kriterijais sąrašo viršuje.</span><span class="sxs-lookup"><span data-stu-id="5cd5b-113">We recommend that you put templates with the most specific criteria at the top of the list.</span></span> <span data-ttu-id="5cd5b-114">Kuo platesni kriterijai, tuo didesnė tikimybė, kad eilutė atitiks kriterijus, todėl eilutės gali būti priskirtos netinkamai bangai.</span><span class="sxs-lookup"><span data-stu-id="5cd5b-114">The broader the criteria, the more likely it is for a line to meet the criteria, and this could lead to lines being assigned to the wrong wave.</span></span>  
4. <span data-ttu-id="5cd5b-115">Lauke **Bangos šablono aprašas** įveskite reikšmę.</span><span class="sxs-lookup"><span data-stu-id="5cd5b-115">In the **Wave template description** field, type a value.</span></span>
5. <span data-ttu-id="5cd5b-116">Lauke **Teritorija** įveskite arba pasirinkite reikšmę.</span><span class="sxs-lookup"><span data-stu-id="5cd5b-116">In the **Site** field, enter or select a value.</span></span> <span data-ttu-id="5cd5b-117">Jeigu naudojate USMF, galite pasirinkti svetainę „2“.</span><span class="sxs-lookup"><span data-stu-id="5cd5b-117">If you're using USMF, you can select site 2.</span></span>  
6. <span data-ttu-id="5cd5b-118">Lauke **Sandėlis** įveskite arba pasirinkite reikšmę.</span><span class="sxs-lookup"><span data-stu-id="5cd5b-118">In the **Warehouse** field, enter or select a value.</span></span> <span data-ttu-id="5cd5b-119">Jeigu naudojate USMF, galite pasirinkti sandėlį „24“.</span><span class="sxs-lookup"><span data-stu-id="5cd5b-119">If you're using USMF, you can select warehouse 24.</span></span>  
7. <span data-ttu-id="5cd5b-120">Lauką **Automatiškai kurti bangą** nustatykite į **Taip**.</span><span class="sxs-lookup"><span data-stu-id="5cd5b-120">Set the **Automate wave creation** field to **Yes**.</span></span> <span data-ttu-id="5cd5b-121">Pasirinkite šią parinktį, kad banga būtų automatiškai kuriama į sandėlį išleidus pardavimo užsakymą, gamybos užsakymą arba „kanban“.</span><span class="sxs-lookup"><span data-stu-id="5cd5b-121">Select this option to automatically create a wave when a sales order, production order, or kanban is released to the warehouse.</span></span>

9. <span data-ttu-id="5cd5b-124">Nustatykite **Parinktis automatizuoti bangos išleidimą** į **Taip**.</span><span class="sxs-lookup"><span data-stu-id="5cd5b-124">Set the **Automate wave release option** to **Yes**.</span></span> <span data-ttu-id="5cd5b-125">Pasirinkus šią parinktį bus automatiškai išleidžiama banga.</span><span class="sxs-lookup"><span data-stu-id="5cd5b-125">Select this option to automatically release the wave.</span></span> <span data-ttu-id="5cd5b-126">Paėmimo darbas sukuriamas ir prieinamas mobiliuosiuose įrenginiuose.</span><span class="sxs-lookup"><span data-stu-id="5cd5b-126">The picking work is created and made available on mobile devices.</span></span>  
10. <span data-ttu-id="5cd5b-127">Nustatykite **Parinktis priskirti atidaryti bangas** į **Taip**.</span><span class="sxs-lookup"><span data-stu-id="5cd5b-127">Set the **Assign to open waves option** to **Yes**.</span></span> <span data-ttu-id="5cd5b-128">Eilutės priskiriamos bangoms pagal bangos šablono užklausos filtrą.</span><span class="sxs-lookup"><span data-stu-id="5cd5b-128">Lines are assigned to waves based on the query filter for the wave template.</span></span>  
11. <span data-ttu-id="5cd5b-129">Nustatykite **Parinktis apdoroti bangą automatiškai ties ribine reikšme** į **Taip**.</span><span class="sxs-lookup"><span data-stu-id="5cd5b-129">Set the **Process wave automatically at threshold option** to **Yes**.</span></span> <span data-ttu-id="5cd5b-130">Pasirinkite šią parinktį, kad banga būtų automatiškai apdorojama jos vertėms pasiekus svorio, siuntos ir laukų grupėje Bangos slenksčiai nurodytų eilučių slenksčius.</span><span class="sxs-lookup"><span data-stu-id="5cd5b-130">Select this option to automatically process the wave when its values reach the thresholds for weight, shipment, and lines specified in the Wave thresholds field group.</span></span> <span data-ttu-id="5cd5b-131">Šią parinktį galima naudoti tik lauke Bangos šablono tipas pasirinkus Siunta.</span><span class="sxs-lookup"><span data-stu-id="5cd5b-131">This option is available only if Shipping is selected in the Wave template type field.</span></span>

13. <span data-ttu-id="5cd5b-135">Išplėskite dalį **Metodai**.</span><span class="sxs-lookup"><span data-stu-id="5cd5b-135">Expand the **Methods** section.</span></span>

    - <span data-ttu-id="5cd5b-136">Bangos šablonų būdais galima valdyti veiklų, per kurias praeina kiekviena apdorojama banga, seką.</span><span class="sxs-lookup"><span data-stu-id="5cd5b-136">Wave template methods allow you to control the sequence of activities that each wave is going through when it's processed.</span></span> <span data-ttu-id="5cd5b-137">Pavyzdžiui, galite turėti bangų papildymo būdą.</span><span class="sxs-lookup"><span data-stu-id="5cd5b-137">For example, you might have a method for wave replenishment.</span></span> <span data-ttu-id="5cd5b-138">Kai pridedate metodą, jis automatiškai pateikiamas atitinkamoje vietoje pagal sekos veiksmus.</span><span class="sxs-lookup"><span data-stu-id="5cd5b-138">When you add a method, it's automatically listed in the appropriate location in the sequence of steps.</span></span> <span data-ttu-id="5cd5b-139">Jei parinktį Automatiškai leisti papildymo darbą nustatėte į Taip, čia turite įtraukti papildymo būdą.</span><span class="sxs-lookup"><span data-stu-id="5cd5b-139">If you've set the Automate replenishment work release option to Yes, you need to add the replenish method here.</span></span>

17. <span data-ttu-id="5cd5b-145">Išplėskite dalį **Bangos apdorojimas**.</span><span class="sxs-lookup"><span data-stu-id="5cd5b-145">Expand the **Wave processing** section.</span></span>
