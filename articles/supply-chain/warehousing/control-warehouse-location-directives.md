[!include [banner](../includes/banner.md)]

<span data-ttu-id="e2c85-104">Šioje temoje aprašoma, kaip naudoti darbo šablonus ir vietos nurodymus, siekiant nustatyti, kaip ir kur sandėlyje atliekamas darbas.</span><span class="sxs-lookup"><span data-stu-id="e2c85-104">This topic describes how to use work templates and location directives to determine how and where work is carried out in the warehouse.</span></span>

## <a name="work-templates"></a><span data-ttu-id="e2c85-108">Darbo šablonai</span><span class="sxs-lookup"><span data-stu-id="e2c85-108">Work templates</span></span>

<span data-ttu-id="e2c85-109">Puslapyje **Darbo šablonai** galima apibrėžti darbo operacijas, kurį reikia atlikti sandėlyje.</span><span class="sxs-lookup"><span data-stu-id="e2c85-109">The **Work templates** page lets you define the work operations that must be performed in the warehouse.</span></span> <span data-ttu-id="e2c85-110">Paprastai sandėlio darbo operacijas sudaro du veiksmai: sandėlio darbuotojas paima turimas atsargas vienoje vietoje ir tada padeda paimtas atsargas kitoje vietoje.</span><span class="sxs-lookup"><span data-stu-id="e2c85-110">Typically, warehouse work operations consist of a pair of actions: a warehouse worker picks up on-hand inventory in one location and then puts the picked inventory down in another location.</span></span> 

<span data-ttu-id="e2c85-111">Darbo šablonus sudaro antraštė ir susietos eilutės.</span><span class="sxs-lookup"><span data-stu-id="e2c85-111">Work templates consists of a header and associated lines.</span></span> <span data-ttu-id="e2c85-112">Kiekvienas darbo šablonas yra skirtas konkrečiam *darbo užsakymo tipui*.</span><span class="sxs-lookup"><span data-stu-id="e2c85-112">Each work template is for a specific *work order type*.</span></span> <span data-ttu-id="e2c85-113">Daugelis darbo užsakymų tipų yra susieti su šaltinio dokumentais, pvz., pirkimo arba pardavimo užsakymais.</span><span class="sxs-lookup"><span data-stu-id="e2c85-113">Many work order types are associated with source documents, such as purchase or sales orders.</span></span> <span data-ttu-id="e2c85-114">Tačiau kiti darbo užsakymų tipai nurodo atskirus sandėlio procesus, pavyzdžiui, ciklo skaičiavimą.</span><span class="sxs-lookup"><span data-stu-id="e2c85-114">However, other work order types represent separate warehouse processes, such as cycle counting.</span></span> <span data-ttu-id="e2c85-115">*Darbo telkinio ID* leidžia skirstyti darbą į grupes.</span><span class="sxs-lookup"><span data-stu-id="e2c85-115">The *work pool ID* lets you organize work into groups.</span></span> 

<span data-ttu-id="e2c85-116">Darbo antraštės aprašo parametrus galima naudoti norint nustatyti, kada turi būti sukurtas naujas darbas.</span><span class="sxs-lookup"><span data-stu-id="e2c85-116">The settings in the work header definition can be used to determine when a new piece of work should be created.</span></span> <span data-ttu-id="e2c85-117">Pavyzdžiui, galite nustatyti didžiausią paėmimo eilučių skaičių ir didžiausią numatomą paėmimo laiką.</span><span class="sxs-lookup"><span data-stu-id="e2c85-117">For example, you can set a maximum number of pick lines and a maximum expected pick time.</span></span> <span data-ttu-id="e2c85-118">Tada, jei pardavimo užsakymo išrinkimo proceso darbas viršija vieną iš šių reikšmių, tas darbas yra padalijamas į du darbus.</span><span class="sxs-lookup"><span data-stu-id="e2c85-118">Then, if the work for a sales order picking process exceeds either of those values, that work is split into two pieces of work.</span></span> 

<span data-ttu-id="e2c85-119">Darbo eilutės nurodo faktines užduotis, kurių reikia norint vykdyti darbą.</span><span class="sxs-lookup"><span data-stu-id="e2c85-119">The work lines represent the physical tasks that are required in order to process the work.</span></span> <span data-ttu-id="e2c85-120">Pvz., viena siuntimo sandėlio proceso darbo eilutė gali būti skirta prekes paimti sandėlyje, o kita eilutė – tas prekes padėti į išdėstymo sritį.</span><span class="sxs-lookup"><span data-stu-id="e2c85-120">For example, for an outbound warehouse process, there might be a work line for picking up the items within the warehouse and another line for putting those items into a staging area.</span></span> <span data-ttu-id="e2c85-121">Taip pat gali būti papildoma eilutė prekėms iš išdėstymo srities paimti ir dar viena eilutė prekėms į sunkvežimį pakrauti (tai yra krovimo proceso dalis).</span><span class="sxs-lookup"><span data-stu-id="e2c85-121">There can then be an additional line for picking the items from staging, and another line for putting the items into a truck as part of the loading process.</span></span> <span data-ttu-id="e2c85-122">*Krypties kodas* gali būti nustatytas darbo šablono eilutėse.</span><span class="sxs-lookup"><span data-stu-id="e2c85-122">You can set a *directive code* on work template lines.</span></span> <span data-ttu-id="e2c85-123">Krypties kodas yra susietas su vietos nurodymu ir todėl padeda užtikrinti, kad sandėlio darbas yra vykdomas tinkamoje sandėlio vietoje.</span><span class="sxs-lookup"><span data-stu-id="e2c85-123">A directive code is linked to a location directive and therefore helps guarantee that the warehouse work is processed in the correct location in the warehouse.</span></span> 

<span data-ttu-id="e2c85-124">Galite nustatyti užklausą, norėdami kontroliuoti, kada konkretus darbo šablonas naudojamas.</span><span class="sxs-lookup"><span data-stu-id="e2c85-124">You can set up a query to control when a particular work template is used.</span></span> <span data-ttu-id="e2c85-125">Pavyzdžiui, galite nustatyti apribojimą, kad konkretus šablonas būtų naudojamas tik atliekant darbą konkrečiame sandėlyje.</span><span class="sxs-lookup"><span data-stu-id="e2c85-125">For example, you can set a limitation so that a particular template can be used for work only in a specific warehouse.</span></span> <span data-ttu-id="e2c85-126">Taip pat galite nustatyti kelis šablonus, kurie būtų naudojami siuntimo pardavimo užsakymo apdorojimo darbui kurti, atsižvelgiant į pardavimo šaltinį.</span><span class="sxs-lookup"><span data-stu-id="e2c85-126">Alternatively, you might have several templates that are used to create work for outbound sales order processing, depending on the sales origin.</span></span> <span data-ttu-id="e2c85-127">Sistema naudoja lauką **Sekos numeris**, kad nustatytų galimų darbo šablonų vertinimo tvarką.</span><span class="sxs-lookup"><span data-stu-id="e2c85-127">The system uses the **Sequence number** field to determine the order that the available work templates are assessed in.</span></span> <span data-ttu-id="e2c85-128">Todėl, jei turite labai konkrečią užklausą dėl konkretaus darbo šablono, turėtumėte jai priskirti žemą sekos numerį.</span><span class="sxs-lookup"><span data-stu-id="e2c85-128">Therefore, if you have a very specific query for a particular work template, you should give it a low sequence number.</span></span> <span data-ttu-id="e2c85-129">Tada ta užklausa bus vertinama anksčiau nei kitos, bendresnės užklausos.</span><span class="sxs-lookup"><span data-stu-id="e2c85-129">That query will then be evaluated before other, more general queries.</span></span> 

<span data-ttu-id="e2c85-130">Norėdami darbo procesą sustabdyti arba pristabdyti, galite naudoti darbo eilutės parametrą **Stabdyti darbą**.</span><span class="sxs-lookup"><span data-stu-id="e2c85-130">To stop or pause a work process, you can use the **Stop work** setting on the work line.</span></span> <span data-ttu-id="e2c85-131">Tokiu atveju darbą atliekančio darbuotojo nebus prašoma atlikti kitą darbo eilutės veiksmą.</span><span class="sxs-lookup"><span data-stu-id="e2c85-131">In that case, the worker who is performing the work won't be asked to perform the next work line step.</span></span> <span data-ttu-id="e2c85-132">Norėdamas pereiti prie kito veiksmo, tas arba kitas darbuotojas turi pasirinkti darbą dar kartą.</span><span class="sxs-lookup"><span data-stu-id="e2c85-132">To move on to the next step, that worker or another worker must select the work again.</span></span> <span data-ttu-id="e2c85-133">Taip pat galite atskirti darbo užduotis, darbo šablono eilutėse naudodami kitą *darbo klasės ID*.</span><span class="sxs-lookup"><span data-stu-id="e2c85-133">You can also separate the tasks within a piece of work by using a different *work class ID* on the work template lines.</span></span>

## <a name="location-directives"></a><span data-ttu-id="e2c85-134">Vietos nurodymai</span><span class="sxs-lookup"><span data-stu-id="e2c85-134">Location directives</span></span>

<span data-ttu-id="e2c85-135">Vietos nurodymai yra taisyklės, kurios padeda identifikuoti atsargų judėjimo paėmimo ir padėjimo vietas.</span><span class="sxs-lookup"><span data-stu-id="e2c85-135">Location directives are rules that help identify pick and put locations for inventory movement.</span></span> <span data-ttu-id="e2c85-136">Pvz., pardavimo užsakymo operacijoje vietos nurodymas nustato, kur prekės bus paimtos ir kur bus padėtos.</span><span class="sxs-lookup"><span data-stu-id="e2c85-136">For example, in a sales order transaction, a location directive determines where the items will be picked, and where the picked items will be put.</span></span> <span data-ttu-id="e2c85-137">Vietos nurodymus sudaro antraštė ir susietos eilutės, o jie kuriami puslapyje **Vietos nurodymai**.</span><span class="sxs-lookup"><span data-stu-id="e2c85-137">Location directives consist of a header and associated lines, and you create them on the **Location directives** page.</span></span> 

<span data-ttu-id="e2c85-138">Kiekvienas antraštės vietos nurodymas turi būti susietas su *darbo užsakymo tipu*, nurodančiu, kuriam atsargų operacijos tipui nurodymas bus skirtas, pvz., pardavimo užsakymas, papildymas arba žaliavų paėmimas.</span><span class="sxs-lookup"><span data-stu-id="e2c85-138">On the header, each location directive must be associated with a *work order type* that specifies the type of inventory transaction that the directive will be used for, such as sales orders, replenishment, or raw material picking.</span></span> <span data-ttu-id="e2c85-139">*Darbo tipas* nurodo, ar vietos nurodymas bus skirtas išrinkimo darbui, padėjimo darbui, ar kažkuriam kitam sandėlio procesui, pvz., skaičiavimui arba atsargų būsenos keitimams.</span><span class="sxs-lookup"><span data-stu-id="e2c85-139">The *work type* specifies whether the location directive will be used for picking or putting work, or for some other warehouse process, such as counting or inventory status changes.</span></span> <span data-ttu-id="e2c85-140">Taip pat turi būti nurodyta *teritorija* ir *sandėlis*.</span><span class="sxs-lookup"><span data-stu-id="e2c85-140">You must also specify a *site* and a *warehouse*.</span></span> <span data-ttu-id="e2c85-141">Antraštėje nurodytas *krypties kodas* gali būti naudojamas vietos nurodymui su vienu ar daugiau darbo šablonų susieti.</span><span class="sxs-lookup"><span data-stu-id="e2c85-141">A *directive code* that you specify on the header can be used to link the location directive to one or more work templates.</span></span> 

<span data-ttu-id="e2c85-142">Kalbant apie darbo šablonus, nustatydami užklausą galite nurodyti, kada naudojamas konkretus vietos nurodymas.</span><span class="sxs-lookup"><span data-stu-id="e2c85-142">As for work templates, you can set up a query to determine when a particular location directive is used.</span></span> <span data-ttu-id="e2c85-143">Pavyzdžiui, galite nurodyti, kad kai pardavimo užsakymo šaltinis yra e. prekyba, atsargas reikia paimti iš paskirtos sandėlio vietos.</span><span class="sxs-lookup"><span data-stu-id="e2c85-143">For example, you can specify that when e-commerce is the origin of a sales order, the inventory must be picked up from a dedicated area in the warehouse.</span></span> <span data-ttu-id="e2c85-144">Sistema naudoja lauką **Sekos numeris**, kad nustatytų galimų vietos nurodymų vertinimo tvarką.</span><span class="sxs-lookup"><span data-stu-id="e2c85-144">The system uses the **Sequence number** field to determine the order that the available location directives are assessed in.</span></span>

<span data-ttu-id="e2c85-145">Vietos nurodymo eilutės nustato papildomus vietos ieškos taisyklių taikymo apribojimus.</span><span class="sxs-lookup"><span data-stu-id="e2c85-145">The location directive lines set additional restrictions on the application of the location finding rules.</span></span> <span data-ttu-id="e2c85-146">Galite nurodyti mažiausią ir didžiausią kiekius, kuriems nurodymas būtų taikomas, taip pat galite nurodyti, kad nurodymas yra skirtas konkrečiam atsargų vienetui.</span><span class="sxs-lookup"><span data-stu-id="e2c85-146">You can specify a minimum quantity and a maximum quantity that the directive should apply to, and you can specify that the directive should be for a specific inventory unit.</span></span> <span data-ttu-id="e2c85-147">Pavyzdžiui, jei matavimo vienetas yra padėklai, padėklų prekes galima padėti konkrečioje vietoje.</span><span class="sxs-lookup"><span data-stu-id="e2c85-147">For example, if the unit of measure is pallets, the items in pallets can be put in a specific location.</span></span> <span data-ttu-id="e2c85-148">Taip pat galite nurodyti, ar kiekis gali būti suskaidytas keliose vietose.</span><span class="sxs-lookup"><span data-stu-id="e2c85-148">You can also specify whether the quantity can be split across multiple locations.</span></span> <span data-ttu-id="e2c85-149">Kaip ir vietos nurodymo antraštė, kiekviena vietos nurodymo eilutė turi sekos numerį, pagal kurį nustatoma eilučių vertinimo tvarka.</span><span class="sxs-lookup"><span data-stu-id="e2c85-149">Like the location directive header, each location directive line has a sequence number that determines the order that the lines are assessed in.</span></span>
