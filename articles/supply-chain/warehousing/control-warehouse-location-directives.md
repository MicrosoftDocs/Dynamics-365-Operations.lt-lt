<span data-ttu-id="34b3a-124">Galite nustatyti užklausą, norėdami kontroliuoti, kada konkretus darbo šablonas naudojamas.</span><span class="sxs-lookup"><span data-stu-id="34b3a-124">You can set up a query to control when a particular work template is used.</span></span> <span data-ttu-id="34b3a-125">Pavyzdžiui, galite nustatyti apribojimą, kad konkretus šablonas būtų naudojamas tik atliekant darbą konkrečiame sandėlyje.</span><span class="sxs-lookup"><span data-stu-id="34b3a-125">For example, you can set a limitation so that a particular template can be used for work only in a specific warehouse.</span></span> <span data-ttu-id="34b3a-126">Taip pat galite nustatyti kelis šablonus, kurie būtų naudojami siuntimo pardavimo užsakymo apdorojimo darbui kurti, atsižvelgiant į pardavimo šaltinį.</span><span class="sxs-lookup"><span data-stu-id="34b3a-126">Alternatively, you might have several templates that are used to create work for outbound sales order processing, depending on the sales origin.</span></span> <span data-ttu-id="34b3a-127">Sistema naudoja lauką **Sekos numeris**, kad nustatytų galimų darbo šablonų vertinimo tvarką.</span><span class="sxs-lookup"><span data-stu-id="34b3a-127">The system uses the **Sequence number** field to determine the order that the available work templates are assessed in.</span></span> <span data-ttu-id="34b3a-128">Todėl, jei turite labai konkrečią užklausą dėl konkretaus darbo šablono, turėtumėte jai priskirti žemą sekos numerį.</span><span class="sxs-lookup"><span data-stu-id="34b3a-128">Therefore, if you have a very specific query for a particular work template, you should give it a low sequence number.</span></span> <span data-ttu-id="34b3a-129">Tada ta užklausa bus vertinama anksčiau nei kitos, bendresnės užklausos.</span><span class="sxs-lookup"><span data-stu-id="34b3a-129">That query will then be evaluated before other, more general queries.</span></span> 

<span data-ttu-id="34b3a-130">Norėdami darbo procesą sustabdyti arba pristabdyti, galite naudoti darbo eilutės parametrą **Stabdyti darbą**.</span><span class="sxs-lookup"><span data-stu-id="34b3a-130">To stop or pause a work process, you can use the **Stop work** setting on the work line.</span></span> <span data-ttu-id="34b3a-131">Tokiu atveju darbą atliekančio darbuotojo nebus prašoma atlikti kitą darbo eilutės veiksmą.</span><span class="sxs-lookup"><span data-stu-id="34b3a-131">In that case, the worker who is performing the work won't be asked to perform the next work line step.</span></span> <span data-ttu-id="34b3a-132">Norėdamas pereiti prie kito veiksmo, tas arba kitas darbuotojas turi pasirinkti darbą dar kartą.</span><span class="sxs-lookup"><span data-stu-id="34b3a-132">To move on to the next step, that worker or another worker must select the work again.</span></span> <span data-ttu-id="34b3a-133">Taip pat galite atskirti darbo užduotis, darbo šablono eilutėse naudodami kitą *darbo klasės ID*.</span><span class="sxs-lookup"><span data-stu-id="34b3a-133">You can also separate the tasks within a piece of work by using a different *work class ID* on the work template lines.</span></span>

## <a name="location-directives"></a><span data-ttu-id="34b3a-134">Vietos nurodymai</span><span class="sxs-lookup"><span data-stu-id="34b3a-134">Location directives</span></span>

<span data-ttu-id="34b3a-138">Kiekvienas antraštės vietos nurodymas turi būti susietas su *darbo užsakymo tipu*, nurodančiu, kuriam atsargų operacijos tipui nurodymas bus skirtas, pvz., pardavimo užsakymas, papildymas arba žaliavų paėmimas.</span><span class="sxs-lookup"><span data-stu-id="34b3a-138">On the header, each location directive must be associated with a *work order type* that specifies the type of inventory transaction that the directive will be used for, such as sales orders, replenishment, or raw material picking.</span></span> <span data-ttu-id="34b3a-139">*Darbo tipas* nurodo, ar vietos nurodymas bus skirtas išrinkimo darbui, padėjimo darbui, ar kažkuriam kitam sandėlio procesui, pvz., skaičiavimui arba atsargų būsenos keitimams.</span><span class="sxs-lookup"><span data-stu-id="34b3a-139">The *work type* specifies whether the location directive will be used for picking or putting work, or for some other warehouse process, such as counting or inventory status changes.</span></span> <span data-ttu-id="34b3a-140">Taip pat turi būti nurodyta *teritorija* ir *sandėlis*.</span><span class="sxs-lookup"><span data-stu-id="34b3a-140">You must also specify a *site* and a *warehouse*.</span></span> <span data-ttu-id="34b3a-141">Antraštėje nurodytas *krypties kodas* gali būti naudojamas vietos nurodymui su vienu ar daugiau darbo šablonų susieti.</span><span class="sxs-lookup"><span data-stu-id="34b3a-141">A *directive code* that you specify on the header can be used to link the location directive to one or more work templates.</span></span>

<span data-ttu-id="34b3a-145">Vietos nurodymo eilutės nustato papildomus vietos ieškos taisyklių taikymo apribojimus.</span><span class="sxs-lookup"><span data-stu-id="34b3a-145">The location directive lines set additional restrictions on the application of the location finding rules.</span></span> <span data-ttu-id="34b3a-146">Galite nurodyti mažiausią ir didžiausią kiekius, kuriems nurodymas būtų taikomas, taip pat galite nurodyti, kad nurodymas yra skirtas konkrečiam atsargų vienetui.</span><span class="sxs-lookup"><span data-stu-id="34b3a-146">You can specify a minimum quantity and a maximum quantity that the directive should apply to, and you can specify that the directive should be for a specific inventory unit.</span></span> <span data-ttu-id="34b3a-147">Pavyzdžiui, jei matavimo vienetas yra padėklai, padėklų prekes galima padėti konkrečioje vietoje.</span><span class="sxs-lookup"><span data-stu-id="34b3a-147">For example, if the unit of measure is pallets, the items in pallets can be put in a specific location.</span></span> <span data-ttu-id="34b3a-148">Taip pat galite nurodyti, ar kiekis gali būti suskaidytas keliose vietose.</span><span class="sxs-lookup"><span data-stu-id="34b3a-148">You can also specify whether the quantity can be split across multiple locations.</span></span> <span data-ttu-id="34b3a-149">Kaip ir vietos nurodymo antraštė, kiekviena vietos nurodymo eilutė turi sekos numerį, pagal kurį nustatoma eilučių vertinimo tvarka.</span><span class="sxs-lookup"><span data-stu-id="34b3a-149">Like the location directive header, each location directive line has a sequence number that determines the order that the lines are assessed in.</span></span> 

<span data-ttu-id="34b3a-150">Vietos nurodymams priskiriamas vienas papildomas informacijos lygis: *vietos nurodymo veiksmai*.</span><span class="sxs-lookup"><span data-stu-id="34b3a-150">Location directives have one additional level of detail: *location directive actions*.</span></span> <span data-ttu-id="34b3a-151">Galite nurodyti kelis kiekvienos eilutės vietos nurodymo veiksmus.</span><span class="sxs-lookup"><span data-stu-id="34b3a-151">You can define multiple location directive actions for each line.</span></span> <span data-ttu-id="34b3a-152">Primename, kad sekos numeris yra naudojamas veiksmų vertinimo tvarkai nustatyti.</span><span class="sxs-lookup"><span data-stu-id="34b3a-152">Once again, a sequence number is used to determine the order that the actions are assessed in.</span></span> <span data-ttu-id="34b3a-153">Šiame lygyje galite nustatyti užklausą, norėdami nurodyti, kaip rasti geriausią vietą sandėlyje.</span><span class="sxs-lookup"><span data-stu-id="34b3a-153">On this level, you can set up a query to define how to find the best location in the warehouse.</span></span> <span data-ttu-id="34b3a-154">Taip pat galite naudoti iš anksto nustatytus **strategijos** parametrus optimaliausiai vietai rasti.</span><span class="sxs-lookup"><span data-stu-id="34b3a-154">You can also use predefined **Strategy** settings to find an optimal location.</span></span>

### <a name="example-of-the-use-of-location-directives"></a><span data-ttu-id="34b3a-155">Vietos nurodymų naudojimo pavyzdys</span><span class="sxs-lookup"><span data-stu-id="34b3a-155">Example of the use of location directives</span></span>

<span data-ttu-id="34b3a-156">Šiame pavyzdyje apsvarstysime pirkimo užsakymo procesą, kai vietos nurodymas turi sandėlyje rasti laisvos vietos atsargų prekėms, kurios buvo ką tik užregistruotos gavimo rampoje.</span><span class="sxs-lookup"><span data-stu-id="34b3a-156">For this example, we will consider a purchase order process where the location directive must find free capacity within a warehouse for inventory items that have just been registered at the receiving dock.</span></span> <span data-ttu-id="34b3a-157">Pirmiausia pabandysime sandėlyje rasti laisvos vietos, konsoliduodami esamas atsargas.</span><span class="sxs-lookup"><span data-stu-id="34b3a-157">First, we want to try to find free capacity within the warehouse by consolidating with existing on-hand inventory.</span></span> <span data-ttu-id="34b3a-158">Jei konsoliduoti negalima, tada turime rasti tuščią vietą.</span><span class="sxs-lookup"><span data-stu-id="34b3a-158">If consolidation isn't possible, we then want to find an empty location.</span></span>
