<span data-ttu-id="e0f2b-104">Šioje temoje aprašomos papildymo strategijos, skirtos sandėliams, kuriuose naudojamos modulio Sandėlio valdymas funkcijos.</span><span class="sxs-lookup"><span data-stu-id="e0f2b-104">This topic describes the replenishment strategies that are available for warehouses that use the functionality that is available in Warehouse management.</span></span> <span data-ttu-id="e0f2b-105">Šioje temoje pateikiama informacija netaikoma sandėliavimo sprendimui, kuris prieinamas modulyje Atsargų valdymas.</span><span class="sxs-lookup"><span data-stu-id="e0f2b-105">The information in this topic doesn't apply to the warehousing solution that is available in Inventory management.</span></span>

<span data-ttu-id="e0f2b-106">Galima naudoti tolesnes papildymo strategijas.</span><span class="sxs-lookup"><span data-stu-id="e0f2b-106">The following replenishment strategies are available:</span></span>

- <span data-ttu-id="e0f2b-107">**Papildymas pagal bangos poreikį** – naudojantis šia strategija sukuriamas išsiunčiamų užsakymų arba krovinių papildymo darbas, jei bangai kuriant darbą nėra atsargų.</span><span class="sxs-lookup"><span data-stu-id="e0f2b-107">**Wave demand replenishment** – This strategy creates replenishment work for outbound orders or loads if inventory isn't available when the wave creates work.</span></span> <span data-ttu-id="e0f2b-108">Pavyzdžiui, jei apdorojant bangą nėra pardavimo užsakymui reikalingo kiekio, gali būti sukurtas papildymo darbas.</span><span class="sxs-lookup"><span data-stu-id="e0f2b-108">For example, replenishment work can be created if the quantity that is required for a sales order isn't available when a wave is processed.</span></span>
- <span data-ttu-id="e0f2b-109">**Min. / maks. papildymas** – pagal šią strategiją, norint nustatyti, kada reikia papildyti vietas, naudojami sandėliavimo minimumo ir maksimumo limitai.</span><span class="sxs-lookup"><span data-stu-id="e0f2b-109">**Min/Max replenishment** – This strategy uses minimum and maximum stocking limits to determine when locations should be replenished.</span></span> <span data-ttu-id="e0f2b-110">Pagal prekės ir vietos kriterijus nustatomos papildymui reikalingos atsargos.</span><span class="sxs-lookup"><span data-stu-id="e0f2b-110">The item and location criteria define the inventory that is evaluated for replenishment.</span></span> <span data-ttu-id="e0f2b-111">Min. / maks. papildymo šablonai yra pagrindinis mechanizmas, skirtas palaikyti optimalų prekių kiekį išrinkimo vietose.</span><span class="sxs-lookup"><span data-stu-id="e0f2b-111">Min/Max replenishment templates are the primary mechanism for maintaining optimal levels in picking locations.</span></span> <span data-ttu-id="e0f2b-112">Siekiant užtikrinti, kad yra pakankamai atsargų, kurias galima paimti bangos poreikiui patenkinti, kaip priedą tarp min. / maks. papildymo ciklų galima naudoti poreikio papildymą.</span><span class="sxs-lookup"><span data-stu-id="e0f2b-112">To help guarantee that enough pick face inventory is available to meet wave demand, you can use demand replenishment as a supplement between Min/Max replenishment cycles.</span></span>
- <span data-ttu-id="e0f2b-113">**Krovinio poreikio papildymas** – šia strategija susumuojamas kelių krovinių poreikis ir sukuriamas papildymo darbas, kurio reikia siekiant pripildyti atitinkamas paėmimo vietas atsargų.</span><span class="sxs-lookup"><span data-stu-id="e0f2b-113">**Load demand replenishment** – This strategy sums the demand for several loads and creates the replenishment work that is required in order to stock the relevant picking locations.</span></span> <span data-ttu-id="e0f2b-114">Ši strategija padeda užtikrinti, kad sukuriamus krovinius galima galima paimti sandėlyje, kai jie išleidžiami.</span><span class="sxs-lookup"><span data-stu-id="e0f2b-114">This strategy helps guarantee that the loads that are created can be picked in the warehouse after they are released.</span></span>
- <span data-ttu-id="e0f2b-115">**Skubus papildymas** – jei nepavyksta paskirstyti vietos nurodymo eilutės su papildymo šablonu, šia strategija atsargos papildomos prieš vykdant bangą.</span><span class="sxs-lookup"><span data-stu-id="e0f2b-115">**Immediate replenishment** – This strategy replenishes inventory before a wave is run if allocation fails for a location directive line that has a replenishment template.</span></span> 

<span data-ttu-id="e0f2b-116">Naudojant visas keturias strategijas papildymo darbas kuriamas pagal papildymo šabloną.</span><span class="sxs-lookup"><span data-stu-id="e0f2b-116">All four strategies create replenishment work, based on a replenishment template.</span></span>

## <a name="minmax-replenishment"></a><span data-ttu-id="e0f2b-137">Min. / maks. papildymas</span><span class="sxs-lookup"><span data-stu-id="e0f2b-137">Min/Max replenishment</span></span>
