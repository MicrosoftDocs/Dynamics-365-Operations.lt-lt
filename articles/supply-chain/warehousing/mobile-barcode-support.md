[!include [banner](../includes/banner.md)]

<span data-ttu-id="6d1f1-104">„Android“ yra atvirojo kodo projektas, todėl sukurti įrenginį, kuriame veiktų „Android“ operacinė sistema, gali bet kuris sandėlio brūkšninio kodo skaitytuvų aparatūros gamintojas.</span><span class="sxs-lookup"><span data-stu-id="6d1f1-104">Because Android is an open source project, any manufacturer of hardware for warehouse bar code scanners can build a device to run the Android operating system.</span></span> <span data-ttu-id="6d1f1-105">Įrenginys su „Android“ suderinamas tik tuo atveju, jei jame gali veikti programos, sukurtos „Android“ vykdomajai aplinkai.</span><span class="sxs-lookup"><span data-stu-id="6d1f1-105">A device is only Android-compatible if it can run apps that are written for the Android execution environment.</span></span>
<span data-ttu-id="6d1f1-106">Tačiau aparatūros tiekėjas gali modifikuoti arba sukurti konkrečios versijos „Android“ sistemai skirtus perdengimus, kurie veikia tik tiekėjo aparatūroje.</span><span class="sxs-lookup"><span data-stu-id="6d1f1-106">However, a hardware vendor can modify and create overlays for the Android version that runs on their hardware.</span></span> <span data-ttu-id="6d1f1-107">„Microsoft“ negali prisiimti atsakomybės ir užtikrinti, kad „Android“ skirta mobiliųjų brūkšninių kodų nuskaitymo programa bus suderinama su gamintojo brūkšninio kodo aparatūra ir joje veikiančia „Android“ versija.</span><span class="sxs-lookup"><span data-stu-id="6d1f1-107">Microsoft cannot take any responsibility to ensure that a mobile bar code scanning app for Android is compatible with a manufacturer’s bar code scanning hardware and the Android version that runs on it.</span></span> 

<span data-ttu-id="6d1f1-108">„Dynamics 365 Supply Chain Management - Warehousing“ programa buvo išbandyta pasirinktuose brūkšninių kodų nuskaitymo įrenginiuose, kuriuose veikia „Android“ operacinė sistema.</span><span class="sxs-lookup"><span data-stu-id="6d1f1-108">The Dynamics 365 Supply Chain Management - Warehousing app has been tested with a selection of Android powered devices for bar code scanning.</span></span> <span data-ttu-id="6d1f1-109">Atliekant šiuos bandymus patikrinti tik pavyzdiniai rinkoje esantys gaminiai.</span><span class="sxs-lookup"><span data-stu-id="6d1f1-109">These tests only cover a sample of the devices that are available on the market.</span></span>

<span data-ttu-id="6d1f1-110">Prieš nusprendžiant, kurią aparatūrą įsigyti, savo klientams rekomenduojame sandėlio mobiliąją nuskaitymo programą išbandyti pasirinktose aparatūrose.</span><span class="sxs-lookup"><span data-stu-id="6d1f1-110">As a customer, we recommend that you test the Warehouse mobile scanning app on selected hardware before you decide on the hardware that you want to buy.</span></span>
