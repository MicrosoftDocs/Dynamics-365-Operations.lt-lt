# <a name="create-a-formula-by-copying-from-an-existing-formula"></a>Kurti formulę kopijuojant iš esamos formulės

[!include [banner](../../includes/banner.md)]

Ši procedūra nurodo, kaip greitai sukurti naują formulę, ją nukopijavus iš esamos formulės. Norėdami sukurti šią procedūrą galite naudoti demonstracinių duomenų įmonės USMF, USP2 arba USPI.


## <a name="create-a-new-formula"></a>Kurti naują formulę
1. Eikite į Formulės.
2. Spustelėkite Naujas.
3. Lauke „Formulė“ įveskite reikšmę.
4. Lauke Pavadinimas surinkite reikšmę.
5. Lauke Teritorija įveskite arba pasirinkite reikšmę.
6. Lauke Prekių grupė įveskite arba pasirinkite vertę.
7. Spustelėkite Įrašyti.

## <a name="copy-from-an-existing-formula"></a>Kopijuoti iš esamos formulės
1. Spustelėkite Kopijuoti.
2. Lauke Prekės numeris įveskite arba pasirinkite reikšmę.
3. Lauke Formulės versija įveskite arba pasirinkite vertę.
4. Spustelėkite GERAI.

## <a name="revise-the-copied-formula"></a>Peržiūrėti nukopijuotą formulę
1. Sąraše raskite ir pasirinkite norimą įrašą.
2. Spustelėkite Naikinti.
3. Spustelėkite Taip.

## <a name="approve-formula"></a>Tvirtinti formulę
1. Spustelėkite „Tvirtinti formulę“.
2. Lauke Patvirtino įveskite arba pasirinkite vertę.
3. Spustelėkite GERAI.
4. Uždarykite puslapį.
