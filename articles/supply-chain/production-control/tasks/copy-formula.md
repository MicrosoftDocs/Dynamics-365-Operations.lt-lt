# <a name="copy-a-formula"></a>Nukopijuokite formulę

[!include [task guide banner](../../includes/task-guide-banner.md)]

Šios procedūros tikslas yra sukurti formulę, kuri apimtų tuos pačius ingredientus, kaip ir esama formulė, tačiau su nedideliais skirtumais. Norėdami sukurti formulės eilutes, galite kopijavimo funkcijos pagalba nukopijuoti esamą formulę, kuri turi daugumą jums reikalingų ingredientų. Tada galite atlikti būtinus atskirų eilučių pakeitimus naujojoje versijoje. Naudojant kopijavimo funkciją nereikės sukurti keleto beveik identiškų formulių. Kuriant šią užduotį naudojama demonstracinių duomenų įmonė yra USP2.


## <a name="create-a-formula"></a>Kurti formulę
1. Pasirinkite Produkto informacijos valdymas > Komplektavimo specifikacijos ir formulės > Formulės.
2. Spustelėkite Naujas.
3. Lauke „Formulė“ įveskite reikšmę.
4. Lauke Pavadinimas surinkite reikšmę.
    * Įveskite prasmingą formulės pavadinimą.  
5. Lauke Teritorija spustelėkite išplečiamąjį mygtuką, kad atidarytumėte peržvalgą.
6. Sąraše spustelėkite saitą pasirinktoje eilutėje.
7. Lauke Prekių grupė spustelėkite išplečiamąjį mygtuką, kad atidarytumėte peržvalgą.
8. Sąraše raskite ir pasirinkite norimą įrašą.
9. Sąraše spustelėkite saitą pasirinktoje eilutėje.
10. Spustelėkite Įrašyti.

## <a name="copy-formula-lines"></a>Kopijuoti formulės eilutes
1. Veiksmų srityje spustelėkite „Formulė“.
2. Spustelėkite Kopijuoti.
3. Lauke Prekės numeris spustelėkite išplečiamąjį mygtuką, kad atidarytumėte peržvalgą.
4. Sąraše spustelėkite saitą pasirinktoje eilutėje.
5. Lauke „Formulės versija“ spustelėkite išplečiamąjį mygtuką, kad atidarytumėte peržvalgą.
6. Sąraše spustelėkite saitą pasirinktoje eilutėje.
7. Spustelėkite GERAI.

## <a name="adjust-copied-formula-lines"></a>Koreguoti nukopijuotas formulės eilutes
1. Sąraše pažymėkite pasirinktą eilutę.
2. Spustelėkite Naikinti.
3. Spustelėkite Taip.

## <a name="approve-formula"></a>Tvirtinti formulę
1. Veiksmų srityje spustelėkite „Formulė“.
2. Spustelėkite „Tvirtinti formulę“.
3. Lauke Patvirtino spustelėkite išplečiamąjį mygtuką, kad atidarytumėte peržvalgą.
4. Sąraše spustelėkite saitą pasirinktoje eilutėje.
5. Spustelėkite Pažymėti.
6. Spustelėkite GERAI.
