# <a name="copy-co-products-from-an-existing-formula-version"></a>Kopijuoti sudėtinius produktus iš esamos formulės versijos

[!include [task guide banner](../../includes/task-guide-banner.md)]

Ši procedūra nurodo, kaip kopijuoti sudėtinius produktus iš esamos formulės versijos į kitokią patvirtinto produkto formulę. Būtina sąlyga, kad bent viena formulės versija būtų susieta su sudėtiniais produktais. Kuriant šią procedūrą naudojama demonstracinių duomenų įmonės USP2.


## <a name="find-a-released-product"></a>Rasti patvirtinta produktą
1. Eikite į Išleisti produktai.
2. Spustelėkite Rodyti filtrus.
    * Ketinate į filtro dialogo langą įtraukti lauką Gamybos tipas.  
3. Norėdami įtraukti lauką Gamybos tipas, spustelėkite Įtraukti filtro lauką.
    * Atlikdami kitą veiksmą, prieš pasirinkdami Taikyti, lauke Gamybos tipas turite įvesti formulę. Taip nustatomas išleistų produktų sąrašo filtras.  
4. Lauke Gamybos tipas rankiniu būdu įveskite formulę.
5. Spustelėkite Taikyti.

## <a name="select-a-released-product"></a>Pasirinkti patvirtintą produktą
1. Sąraše raskite ir pasirinkite norimą įrašą.
2. Spustelėkite Formulės versijos.
    * Inžinerijos veiksmų srityje spustelėkite Formulės versijas.  

## <a name="copy-co-products"></a>Kopijuoti sudėtinius produktus
1. Veiksmų srityje spustelėkite Formulės versija.
2. Spustelėkite Sudėtiniai produktai.
3. Spustelėkite Kopijuoti.
4. Lauke Prekės numeris įveskite arba pasirinkite reikšmę.
5. Lauke Formulės versija įveskite arba pasirinkite vertę.
6. Spustelėkite GERAI.
7. Uždarykite puslapį.
