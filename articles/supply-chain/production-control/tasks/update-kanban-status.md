# <a name="update-kanban-status"></a>„Kanban“ būsenos naujinimas

[!include [task guide banner](../../includes/task-guide-banner.md)]

Kai per klaidą ištuštinamas „kanban‟ arba reikia ištuštinti gautą „kanban“, turite atnaujinti „kanban“ būseną. Kuriant šią procedūrą naudojama demonstracinių duomenų įmonė yra USMF. Ši procedūra yra skirta parduotuvės prižiūrėtojui.


## <a name="find-the-kanban"></a>Suraskite „kanban“.
1. Pasirinkite Gamybos kontrolė > Kanban > Kanban.
2. Atidarykite stulpelio filtrą Sandėliavimo vieneto būsena.
3. Spustelėkite Valyti.
    * Taip grąžinami filtrai.  
4. Norėdami rasti įrašus, naudokite spartųjį filtrą. Pvz., filtruokite lauką Kortelės numeris reikšme „000149“.

## <a name="change-emptied-status-to-received-status"></a>Būsenos „ištuštintas‟ keitimas į „gautas‟
1. Spustelėkite Atšaukti tuščią sandėliavimo vienetą.
2. Spustelėkite GERAI.
    * Atkreipkite dėmesį, kad sandėliavimo vieneto būsena yra Gautas.  

## <a name="change-received-status-to-emptied-status"></a>Būsenos „gautas‟ keitimas į „ištuštintas‟
1. Spustelėkite Ištuštinti „kanban“.
2. Sąraše pažymėkite pasirinktą eilutę.
    * Atkreipkite dėmesį, kad sandėliavimo vieneto būsena yra Ištuštintas.
