# <a name="cant-end-a-production-order-because-of-a-bom-consumptioncost-value-error"></a>Negalima užbaigti gamybos užsakymo dėl KS consumptionCost vertės klaidos

## <a name="symptoms"></a>Požymiai

Kai bandote patvirtinti galutinį gamybos užsakymą, gaunate tokį klaidos pranešimą:

> KS suvartojimo sąnaudų vertės skaičiavimas turi būti neigiamas išduodant iš atsargų.

## <a name="resolution"></a>Sprendimas

Ši problema buvo ištaisyta 10.0.15 leidime.
