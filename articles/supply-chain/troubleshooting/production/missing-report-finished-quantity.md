# <a name="report-as-finished-update-fails-with-a-missing-quantity-error"></a>Pranešti, kad baigta, nepavyko, nes trūksta kiekio klaidos

## <a name="symptoms"></a>Požymiai

Bandote pranešti apie gamybos užsakymą kaip baigtą pranešdamas klaidos kiekį, bet nepranešdamas kiekio laiką ar medžiagas. Baigtų pranešimo naujinimas galiausiai nepavyks jums bandant užbaigti gamybos užsakymą ir jūs gausite tolesnį klaidos pranešimą:

> Nėra ataskaitos apie kiekį, paskelbtą užbaigtu.

## <a name="resolution"></a>Sprendimas

Jei negalite pranešti klaidos kiekio gamybos užsakyme, nebent turite taip pat pranešti apie gerą kiekį.
