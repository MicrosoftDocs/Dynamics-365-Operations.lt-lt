# <a name="late-selection-isnt-respected-when-production-orders-are-reset-via-a-batch-job"></a>Atsižvelgiama į vėlai pasirinktą užduotį, kai gamybos užsakymai iš naujo nustatomi naudojant paketinę užduotį

KB numeris: 4614634

## <a name="symptoms"></a>Požymiai

Kai naudojate pasikartojančią paketinę užduotį gamybos užsakymo būsenai nustatyti iš naujo, atsižvelgiama į pavėluotus pasirinkimus.

## <a name="resolution"></a>Skiriamoji geba

Dabartinis dizainas nepalaiko pavėluotų pasirinkimų naudojimo būsenos *nustatymo iš naujo* procese.
