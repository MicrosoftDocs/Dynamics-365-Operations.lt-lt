# <a name="resource-name-isnt-updated-after-changing-a-worker-name-in-the-address-book"></a>Ištekliaus pavadinimas nėra naujintas po to, kai keičiu darbuotojo pavadinimą bendroje adresų knygoje

## <a name="symptoms"></a>Požymiai

Jei keičiate darbuotojo vardą bendrų adresų knygyne, atitinkamas resursų pavadinimas nėra naujinamas pagrindinėje resursų grupėje.

## <a name="resolution"></a>Sprendimas

Atitinkantis išteklių pavadinimas šiuo metu nėra atnaujinamas automatiškai. Norėdami ištaisyti problemą, turite rankiniu būdu naujinti ištekliaus pavadinimą.
