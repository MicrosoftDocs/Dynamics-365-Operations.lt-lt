# <a name="users-can-unpick-material-lines-for-products-that-have-been-reported-as-finished"></a>Vartotojai gali atsisakyti produktų, kurie buvo pranešti kaip baigti, medžiagų eilučių paėmimo

KB numeris: 4614721

## <a name="symptoms"></a>Požymiai

Sistema leidžia vartotojams atsisakyti žaliavų medžiagų eilučių produktams, kurie buvo pranešti kaip baigti.

## <a name="resolution"></a>Skiriamoji geba

Sistema veikia kaip sukurta. Žaliavų suvartojimą galima koreguoti, kol gamybos užsakymas pasiekia būseną *Baigta*.
