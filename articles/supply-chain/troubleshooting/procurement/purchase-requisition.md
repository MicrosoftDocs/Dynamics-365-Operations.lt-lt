# <a name="you-cant-add-a-line-to-a-purchase-requisition-after-you-request-a-change"></a>Kai reikalaujate pakeisti, eilutės į pirkimo paraišką įtraukti negalite

KB numeris: 4611211

## <a name="symptoms"></a>Požymiai

Sistema jums neleidžia įtraukti eilutės į pirkimo paraišką, kai reikalaujate pokyčio.

## <a name="resolution"></a>Skiriamoji geba

Turite atšaukti darbo eigą. Kai pirkimo paraiškos būsena yra *Juodraštis*, galite toliau ją redaguoti arba prie jos pridėti eilutę.
