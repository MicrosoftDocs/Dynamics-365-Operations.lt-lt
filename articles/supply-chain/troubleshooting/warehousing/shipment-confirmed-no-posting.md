# <a name="shipment-for-load-has-been-confirmed-but-no-lines-are-posted"></a>Krovinio siuntimas patvirtintas, tačiau eilutės neregistrinės

## <a name="symptoms"></a>Požymiai

Dirbdami su siuntimo sandėlio operacijomis, galite gauti tokį klaidos pranešimą:

> Krovinio 1 % siuntimas buvo patvirtintas.

Tačiau daugiau registruoti nerasta.

## <a name="resolution"></a>Sprendimas

Toks veikimo būdas yra numatytas. Siuntimo patvirtinimas neveikia publikavimo. Jis tik naujina siuntimą ir krovinio būseną. Publikavimas turi vykti kaip atskiras procesas.
