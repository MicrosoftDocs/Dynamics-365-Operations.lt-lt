# <a name="only-one-label-is-printed-for-multiple-work-headers-on-a-single-receipt"></a>Viename kvite spausdinama tik viena kelių darbo antraščių žymė

KB numeris: 4614667

## <a name="symptoms"></a>Požymiai

Tai pačiai paskirties numerio daliai, kaip vienos sandėlio programos gavimo įvykio dalis, sukuriamos kelios darbo antraštės. Tačiau, gautame produkte spausdinama tik viena numerio lentelės žymė.

## <a name="resolution"></a>Skiriamoji geba

Sistema veikia kaip sukurta.

Dabartiniame dizaine visada generuojama viena numerio lentelės žymė, neatsižvelgiant į esamą darbo antraštės skaičių ir darbo eilučių derinius. Sugeneruotoje etiketėje yra informacija tik vienam deriniui.

Norėdami išspręsti šią problemą, įsitikinkite, kad darbo antraštės kūrimas visada susietas su viena paskirties numerio lentelėmis.
