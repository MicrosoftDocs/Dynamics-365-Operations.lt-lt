# <a name="packing-station-doesnt-show-product-notes"></a>Pakavimo stotis nerodo produkto pastabų

KB numeris: 4614615

## <a name="symptoms"></a>Požymiai

Pakavimo pastabos pakavimo formoje nerodomas, kai pakavimo instrukcijos pridėtos prie bendrojo produkto arba produkto varianto.

## <a name="resolution"></a>Skiriamoji geba

Sistema veikia kaip sukurta.

Pagal dabartinę pakavimo formos pakavimo pastabų pakavimo logiką reikia, kad pastabos būtų susietos su siuntomis.
