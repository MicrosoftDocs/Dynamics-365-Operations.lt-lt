# <a name="invalid-license-plate-error-when-scanning-license-plate-id-in-the-mobile-app"></a>Netinkama numerio lentelė arba vietos klaida nuskaitant licencijos numerio ID „mobile app“

## <a name="symptoms"></a>Požymiai

Nuskaitydami licencijos numerio ID „Warehouse Management“ „mobile app“ galite gauti tokį klaidos pranešimą:

> Netinkama numerio lentelė.

## <a name="resolution"></a>Sprendimas

aĮsitikinkite, kad licencijos lentelės ID yra licencijos lentelės lentelėje ir kad prekės ir kiekiai licencijos lentelėje yra prieinami (kitaip tariant, jie nėra blokuojami).
