# <a name="can-only-use-administrator-role-to-access-the-mobile-device-app-emulator"></a>Galite tik naudoti administratoriaus vaidmenį, kad prieitumėte prie mobilaus įrenginio programos emuliatoriaus

## <a name="symptoms"></a>Požymiai

Nustatydami sandėlius „Supply Chain Management“ srityje, negalite naudoti jokio vaidmens, išskyrus administratoriaus vaidmenį, kad būtų galima pasiekti mobiliojo įrenginio emiatorių.

## <a name="resolution"></a>Sprendimas

Mobilaus įrenginio programos emuliatorius yra nustatytas taip, kad dirbtų tik su administratoriaus paskyra. Rekomenduojame jums naudoti sandėlio valdymo mobiliųjų įrenginių programėlę visiems testavimo ir gyvojo proceso tikslams.
