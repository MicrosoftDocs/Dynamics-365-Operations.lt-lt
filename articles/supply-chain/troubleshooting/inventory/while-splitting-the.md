# <a name="when-a-catch-weight-quantity-is-split-minimum-quantity-is-used-instead-of-nominal-quantity"></a>Kai esamo svorio kiekis padalintas, mažiausias kiekis naudojamas vietoje nominalaus kiekio

KB numeris: 4612073

## <a name="symptoms"></a>Požymiai

Kai esamo svorio kiekis padalintas į paketus, lauke **Paimtas kiekis** naudojamas minimalus nustatytas prekės kiekis, o ne nominalusis kiekis.

## <a name="resolution"></a>Skiriamoji geba

Sistema veikia kaip sukurta. Sistema paėmimui naudoja kiekvienos prekės minimalaus kiekio nustatymą.
