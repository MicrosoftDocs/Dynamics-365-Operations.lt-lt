# <a name="picking-list-report-doesnt-show-new-notes-that-are-attached-to-a-journal"></a>Išrinkimo dokumento ataskaitoje nerodo naujų prie žurnalo pridėtų pastabų

KB numeris: 4611660

## <a name="symptoms"></a>Požymiai

Paėmimo sąrašo ataskaitoje nerodo naujų prie pridėtų pastabų į pakavimo sąrašo žurnalą.

## <a name="resolution"></a>Skiriamoji geba

Išrinkimo dokumento ataskaitoje rodomos tik pastabos, kurios pridėtos prie pardavimo užsakymo eilutės ir pardavimo užsakymo antraštės. Jame nerodomos jokios naujos pastabos, pridėtos prie SF žurnalo.
