# <a name="vendor-isnt-specified-when-planned-orders-are-firmed"></a>Tiekėjas nenurodytas, kai tvirtinami suplanuoti užsakymai

Klaidos kodas: SYS23532

## <a name="symptoms"></a>Požymiai

Kai bandote patvirtinti suplanuotus užsakymus, gaunate tokį klaidos pranešimą:

> Tiekėjas nenurodytas

## <a name="resolution"></a>Sprendimas

Norėdami nurodyti tiekėją, atlikite šiuos veiksmus.

1. Atidarykite suplanuotą užsakymą, kuriam trūksta tiekėjo.
1. „FastTab“ skirtuko **Suplanuotas tiekimas** lauke **Tiekėjas** pasirinkite tiekėją.
