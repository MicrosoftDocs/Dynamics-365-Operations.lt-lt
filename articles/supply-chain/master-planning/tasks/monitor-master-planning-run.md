# <a name="monitor-a-master-planning-run"></a>Bendrojo planavimo vykdymo stebėjimas

[!include [task guide banner](../../includes/task-guide-banner.md)]

Gamybos planuotojas nori pamatyti, ar vykdomas bendrasis planavimas. Atlikti šiai procedūrai naudokite demonstracinių duomenų įmonę USMF.


## <a name="run-master-planning"></a>Vykdyti bendrąjį planavimą
1. Spustelėkite Bendrasis planavimas.
    * Tai rasite numatytoje ataskaitų srityje.  
2. Lauke Planas įveskite arba pasirinkite vertę.
    * Pavyzdys: statinis planas  
3. Spustelėkite Vykdyti.
4. Lauke Sekti apdorojimo laiką pasirinkite Taip.
    * Jei laukas jau pasirinktas, šį veiksmą praleiskite.  
5. Lauke Gijų skaičius įveskite skaičių.
6. Išplėskite dalį Įtrauktini įrašai.
7. Spustelėkite Filtras.
8. Sąraše pažymėkite pasirinktą eilutę.
    * Pažymėkite eilutę, kurioje Laukas = Prekės numeris.  
9. Lauke Kriterijai įveskite arba pasirinkite reikšmę.
    * Pavyzdys: T0001  
10. Spustelėkite GERAI.
11. Spustelėkite GERAI.

## <a name="monitor-the-master-planning-run"></a>Bendrojo planavimo vykdymo stebėjimas
1. Spustelėkite Istorija.
2. Spustelėkite Užklausos.
3. Spustelėkite Proceso užduoties trukmė.
4. Sąraše raskite ir pasirinkite norimą įrašą.
    * Galite apžvelgti, kiek laiko truko atlikti kiekvieną kiekvienos prekės planavimo veiksmą.
