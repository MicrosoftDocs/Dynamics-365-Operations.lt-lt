<span data-ttu-id="b408e-104">Galite sugeneruoti darbo užsakymo ataskaitą, kurioje pateikta išsami informacija apie vieną ar daugiau pasirinktų darbo užsakymų.</span><span class="sxs-lookup"><span data-stu-id="b408e-104">You can generate a work order report that shows detailed information about one or more work orders that you select.</span></span>
