<span data-ttu-id="3cbba-104">Šioje temoje aprašoma, kaip nustatyti funkcinės vietos ciklo būsenas ir ciklo modelius modulyje Turto valdymas.</span><span class="sxs-lookup"><span data-stu-id="3cbba-104">This topic describes how to set up functional location lifecycle states and lifecycle models in Asset Management.</span></span> <span data-ttu-id="3cbba-105">Funkcinės vietos ciklo būsenos reiškia galimas funkcinės vietos būsenas, pavyzdžiui, sukurta, aktyvioji ir pasibaigusi.</span><span class="sxs-lookup"><span data-stu-id="3cbba-105">Functional location lifecycle states define the states that a functional location can go through, for example, created, active, and ended.</span></span> <span data-ttu-id="3cbba-106">Visas funkcines vietas, neatsižvelgiant į jų ciklo būseną, galite peržiūrėti sąrašo puslapyje **All functional locations**.</span><span class="sxs-lookup"><span data-stu-id="3cbba-106">You are able to view all functional locations, regardless of their lifecycle state, in the **All functional locations** list page.</span></span> <span data-ttu-id="3cbba-107">Galite pakeisti funkcinės vietos būseną, pasirinkdami ją sąrašo puslapyje **Visos funkcinės vietos** ir pasirinkdami **Naujinti funkcinės vietos būseną**.</span><span class="sxs-lookup"><span data-stu-id="3cbba-107">You can change the state of a functional location by selecting it in the **All functional locations** list page and selecting **Update functional location state**.</span></span>

## <a name="set-up-functional-location-lifecycle-states"></a><span data-ttu-id="3cbba-108">Funkcinės vietos ciklo būsenų konfigūracija</span><span class="sxs-lookup"><span data-stu-id="3cbba-108">Set up functional location lifecycle states</span></span>

1. <span data-ttu-id="3cbba-109">Pasirinkite **Turto valdymas** > **Konfigūracija** > **Funkcinės vietos** > **Ciklo būsenos**.</span><span class="sxs-lookup"><span data-stu-id="3cbba-109">Select **Asset management** > **Setup** > **Functional locations** > **Lifecycle states**.</span></span>
2. <span data-ttu-id="3cbba-110">Pasirinkite **New**, kad sukurtumėte naują funkcinės vietos būseną.</span><span class="sxs-lookup"><span data-stu-id="3cbba-110">Select **New** to create a new functional location state.</span></span>
3. <span data-ttu-id="3cbba-111">Lauke **Ciklo būsena** įterpkite būsenos ID, o lauke **Pavadinimas** įterpkite funkcinės vietos būsenos pavadinimą.</span><span class="sxs-lookup"><span data-stu-id="3cbba-111">Insert the state ID in the **Lifecycle state** field and a name for the functional location state in the **Name** field.</span></span> <span data-ttu-id="3cbba-112">Lauke **Ciklo modeliai** matote funkcinės vietos ciklo modelių, kuriuose naudojama funkcinės vietos būsena, skaičių.</span><span class="sxs-lookup"><span data-stu-id="3cbba-112">In the **Lifecycle models** field, you can see the number of functional location lifecycle models that uses the functional location state.</span></span>
4. <span data-ttu-id="3cbba-113">FastTab **Bendra** perjungiklyje **Aktyvioji** pasirinkite „Taip“, jei funkcinė vieta turi būti aktyvi esant šiai būsenai.</span><span class="sxs-lookup"><span data-stu-id="3cbba-113">On the **General** FastTab, select "Yes" on the **Active** toggle button if the functional location should be active at this state.</span></span>
5. <span data-ttu-id="3cbba-114">Perjungiklyje **Kurti turtą** pasirinkite „Taip“, kad būtų galima automatiškai sukurti turtą, kurio pavadinimas sutaptų su funkcinės vietos pavadinimu, ir įdiegti jį funkcinėje vietoje esant šiai būsenai.</span><span class="sxs-lookup"><span data-stu-id="3cbba-114">Select "Yes" on the **Create assets** toggle button if it should be possible to automatically create an asset with the same name as the functional location and install it on the functional location at this state.</span></span>  
>[!NOTE]
><span data-ttu-id="3cbba-115">Šis perjungiklis yra susijęs su lauku **Asset type**, esančiu **General** FastTab **Functional location types** formoje (**Asset management** > **Setup** > **Functional locations** > **Functional location types**).</span><span class="sxs-lookup"><span data-stu-id="3cbba-115">This toggle button relates to the **Asset type** field on the **General** FastTab in the **Functional location types** form (**Asset management** > **Setup** > **Functional locations** > **Functional location types**).</span></span>
6. <span data-ttu-id="3cbba-116">Perjungiklyje **Pervardyti vietą** pasirinkite „Taip“, kad būtų galima pakeisti funkcinės vietos pavadinimą esant šiai būsenai.</span><span class="sxs-lookup"><span data-stu-id="3cbba-116">Select "Yes" on the **Rename location** toggle button if it should be possible to change the name of the functional location at this state.</span></span>
7. <span data-ttu-id="3cbba-117">Perjungiklyje **Naujos antrinės vietos** pasirinkite „Taip“, kad būtų galima įtraukti naujas funkcinės vietos antrines vietas esant šiai būsenai.</span><span class="sxs-lookup"><span data-stu-id="3cbba-117">Select "Yes" on the **New sub locations** toggle button if it should be possible to add new sub locations to the functional location at this state.</span></span>
8. <span data-ttu-id="3cbba-118">Perjungiklyje **Diegti turtą** pasirinkite „Taip“, kad esant šiai būsenai funkcinėje vietoje būtų galima diegti turtą.</span><span class="sxs-lookup"><span data-stu-id="3cbba-118">Select "Yes" on the **Install assets** toggle button if it should be possible to install assets on the functional location at this state.</span></span>
9. <span data-ttu-id="3cbba-119">Perjungiklyje **Naikinti funkcinę vietą** pasirinkite „Taip“, kad esant šiai būsenai būtų galima panaikinti funkcinę vietą.</span><span class="sxs-lookup"><span data-stu-id="3cbba-119">Select "Yes" on the **Delete functional location** toggle button if it should be possible to delete the functional location at this state.</span></span>
10. <span data-ttu-id="3cbba-120">Lauke **Ciklo būsena** pasirinkite turto būseną, jei norite, kad viso funkcinėje vietoje įdiegto turto ciklo būsena būtų automatiškai atnaujinama esant šiai būsenai.</span><span class="sxs-lookup"><span data-stu-id="3cbba-120">Select an asset state in the **Lifecycle state** field if you want the asset lifecycle state for all assets installed on the functional location to be automatically updated at this state.</span></span> <span data-ttu-id="3cbba-121">Pavyzdžiui: Pavyzdys: uždarius funkcinę vietą ir nustačius funkcinės vietos ciklo būseną „Pasibaigusi“, rekomenduojama automatiškai pakeisti toje funkcinėje vietoje įdiegto turto ciklo būseną į „Nenaudojamas“.</span><span class="sxs-lookup"><span data-stu-id="3cbba-121">Example: If you close down a functional location, and set the functional location lifecycle state to "Ended", you may want to automatically change the lifecycle state of the assets installed on that functional location to "Not in use".</span></span>


>[!NOTE]
><span data-ttu-id="3cbba-122">Funkcinės vietos ciklo būsenos, ciklo modeliai ir tipai yra susiję ir naudojami taip pat, kaip ir darbo užsakymo ciklo būsenos, darbo užsakymo ciklo modeliai ir darbo užsakymų tipai.</span><span class="sxs-lookup"><span data-stu-id="3cbba-122">Functional location lifecycle states, lifecycle models, and types are related and used in the same way as work order lifecycle states, work order lifecycle models, and work order types.</span></span> 

## <a name="set-up-functional-location-lifecycle-models"></a><span data-ttu-id="3cbba-123">Funkcinės vietos ciklo modelių konfigūracija</span><span class="sxs-lookup"><span data-stu-id="3cbba-123">Set up functional location lifecycle models</span></span>

3. <span data-ttu-id="3cbba-129">Lauke **Ciklo modelis** įterpkite ciklo modelio ID, o lauke **Pavadinimas** įterpkite ciklo modelio pavadinimą.</span><span class="sxs-lookup"><span data-stu-id="3cbba-129">Insert the lifecycle model ID in the **Lifecycle model** field and a name for the lifecycle model in the **Name** field.</span></span> <span data-ttu-id="3cbba-130">Laukuose **Funkcinių vietų tipai** ir **Ciklo būsenos** matote funkcinių vietų tipų, naudojančių ciklo modelį, skaičių ir būsenų, pasirinktų ciklo modelyje, skaičių.</span><span class="sxs-lookup"><span data-stu-id="3cbba-130">In the **Functional location types** and **Lifecycle states** fields, you can see the number of functional location types that uses the lifecycle model and the number of states selected in the lifecycle model.</span></span>

5. <span data-ttu-id="3cbba-133">Jei norite pasirinkti visas galimas modelio būsenas, spustelėkite mygtuką ![pasirinkti visus galimus etapus](media/03-setup-for-functional-locations.png).</span><span class="sxs-lookup"><span data-stu-id="3cbba-133">If you want to select all the available states for a model, click the ![select all available stages](media/03-setup-for-functional-locations.png) button.</span></span> <span data-ttu-id="3cbba-134">Visos būsenos perkeliamos į sekciją **Lifecycle states selected**.</span><span class="sxs-lookup"><span data-stu-id="3cbba-134">All states are transferred to the **Lifecycle states selected** section.</span></span>

7. <span data-ttu-id="3cbba-136">Pasirinkę **Lifecycle state updates**, nustatysite, kurios ciklo būsenos gali būti susijusios su pasirinkta būsena.</span><span class="sxs-lookup"><span data-stu-id="3cbba-136">Select **Lifecycle state updates** to define which lifecycle states can follow a selected state.</span></span>
