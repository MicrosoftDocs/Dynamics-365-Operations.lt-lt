# <a name="parameter-configurations-for-retail-statements"></a> Mažmeninės prekybos išrašų parametrų konfigūracijos

[!include[task guide banner](../includes/task-guide-banner.md)]

Ši procedūra nurodo mažmeninės prekybos parametrų konfigūracijas, kurios turi įtakos, kaip kuriami ir registruojami mažmeninės prekybos išrašai. Šioje procedūroje naudojama demonstracinė įmonė USRT.

1. Pasirinkite Mažmeninė prekyba ir prekyba > „Headquarters“ sąranka > Parametrai > Mažmeninės prekybos parametrai.
2. Spustelėkite skirtuką Registravimas.
    * Pasirinkite „Taip“, jei norite konkrečiai registruoti laikotarpio nuolaidos sumas.  
    * Pasirinkite „Standartinė“, jei norite naudoti numatytąsias sąskaitas, arba pasirinkite „Laikotarpio“, jei norite nurodyti, kurią kiekvienos laikotarpio nuolaidos sąskaitą naudoti.  
    * Pasirinkite „Suvestinė“, jei norite sujungti atsargų eilutes, kai įmanoma.  
    * Pasirinkite „Taip“, jei SF ir mokėjimai turėtų būti automatiškai sudengti kaip išrašų registravimo proceso dalis.  
    * Pasirinkite „Taip“, jei norite sujungti pinigų įnešimo į įmonės kasą operacijas.  
    * Pasirinkite „Taip“, jei norite sujungti inkasavimo operacijas.  
    * Pasirinkite „Taip“, norėdami įjungti išrašų registravimo telkimą.  
    * Pasirinkite „Taip“, norėdami kurti ir apdoroti užsakymus, kai registruojate išrašus.  
    * Įveskite maksimalų užsakymų, kurie bus apdoroti atliekant kiekvieną paketinę užduotį, skaičių.  
3. Spustelėkite Įrašyti.
