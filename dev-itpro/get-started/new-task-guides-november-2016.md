# <a name="new-or-updated-task-guides-november-2016"></a>Naujų ar atnaujintų užduočių vadovai (2016 m lapkričio mėn.)

Šioje temoje išvardytos naujų ar atnaujintų užduočių vadovai, buvo išleistas "Microsoft" (lapkritis 2016 - visomis kalbomis) APQC vieningas biblioteka, skirta Microsoft Dynamics 365 operacijoms. Išleista jokia nauja darbo pradžios biblioteka.

(Lapkritis 2016--visomis kalbomis) APQC vieningas bibliotekoje yra užduočių vadovai, kurie skirti naudoti su Microsoft Dynamics 365 operacijoms, Microsoft Dynamics 365 operacijų platformos naujinimas 3, Microsoft Dynamics AX platformos naujinimas 2, Dynamics AX žinyno 2016 m. gegužės naujinimas ir Microsoft Dynamics AX 2016 m. vasario išleidimo. Daugiau informacijos apie šias versijas, rasite [kas yra naujų ar pakeistų](whats-new-changed.md). Šioje bibliotekoje yra lokalizuota užduočių vadovai. Daugiau informacijos apie tai, kaip prie užduočių vadovas biblioteka žinyno sistemoje, rasite [pradėti su pagalba](help-overview.md). **Pastaba:** žvaigždutė (**\***) šalia pavadinimo verslo proceso modeler (BPM) linija rodo, kad užduoties vadovas, kad verslo procesas buvo sukurta dinamika 365 operacijoms 2016 m. rugpjūčio atnaujinti ir nėra palaikoma ankstesnėse versijose Microsoft Dynamics AX.
